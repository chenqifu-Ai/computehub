# 政策申报时间线

> 园区每建成一个模块，就自然满足了某条政策的申报条件。五大板块建设完成之日，就是政策红利自动兑现之时。

## 总体路线

```
Year 1 (建设期)     Year 2 (运营期)     Year 3 (扩张期)      Year 4 (成熟期)       Year 5 (标杆期)
┌────────────┐   ┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐
│ 市级示范     │   │ 省级示范     │    │ 国家级试点   │    │ 多个国家级   │    │ 行业标准    │
│ 源网荷储     │ → │ 储能试点     │ → │ 碳达峰试点   │ → │ 近零碳示范  │ → │ 全国复制    │
│ 绿电交易     │   │ 大宗固废     │    │ 小巨人      │    │ 先进制造集群│    │ 碳金融创新  │
│ 氢能专项     │   │ CCUS示范    │    │ CCER备案    │    │ 生态工业园  │    │ IPO         │
└────────────┘   └────────────┘    └────────────┘    └────────────┘    └────────────┘
```
