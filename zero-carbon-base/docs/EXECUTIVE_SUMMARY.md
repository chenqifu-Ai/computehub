# 执行摘要（一页纸）

## 项目定位

**零碳产业基地**——一个 "清洁能源发电 + 低碳制造 + 碳移除 + 循环经济 + 碳服务" 五环闭环的产业生态。核心使命是让企业 **减碳的同时降低成本**。

## 竞争壁垒

1. **绿电定价权**：度电成本¥0.18-0.25，对外供给¥0.35-0.42（比市电低40-50%）
2. **碳资产开发权**：年碳资产5-25万吨，碳价上涨趋势确定
3. **政策关系**：国家级零碳园区→绿电交易通道+税收优惠+补贴
4. **产业闭环**：能源→制造→固废→再生→能源，内部循环效率>85%

## 一句话

> **用绿电把企业电费降40%，用碳资产把利润再加一层——零碳不是成本，是竞争力。**
