# 产业闭环 · 三流循环设计

> 发电→制造→固废→再生→发电——尽量内部循环。每一度电、每一滴水、每一吨固废都变成两次收入。

## 一、能量流闭环

```
☀️ 光伏 → 🔋 储能 → 绿电直供制造
🌬️ 风电 ↗             ↘ 余电 → 电解制氢 → 氢能重卡/叉车
                                          ↘ 氢燃料电池 → 调峰备用
                                          ↘ 燃料电池余热 → 温室/藻类

效率提升：
  传统工业园：发电35-45% → 输配损耗5-8% → 实际利用30-40%
  闭环园区：发电100% → 储能+制氢吸收 → 弃电率<3%
            热能梯级利用：余热→温室干燥→热水
```

## 二、物质流闭环

```
农林废弃物 → 生物炭 → 土壤改良/建材
                  ↘ 热解气 → 发电/供热

工业固废(粉煤灰/矿渣) → 低碳水泥 → 园区基建
                         ↘ 碳化建材 → 对外销售

工业废水 → 中水回用 → 园区工业/绿化用水
           ↘ 结晶盐 → 工业盐销售

废金属/塑料/纸 → 再生资源分拣 → 对外销售回料

CO₂(自身+企业) → 藻类培养 → 饲料/肥料
                ↘ 矿化固碳 → 碳酸盐建材
```

## 三、碳流闭环

```
CO₂排放源:
  生物炭热解烟气 → CO₂浓度15-20% → 管道直送藻类池
  低碳水泥窑尾气 → CO₂浓度20-25% → 管道直送矿化窑
  园区少量排放 → 胺基吸收装置集中捕获 → 温室/农业

产出:
  生物炭：固碳千年 → CCER碳汇
  矿化建材：CO₂变成石头 → 碳移除信用
  藻粉：光合作用固定CO₂ → 饲料/化妆品
  绿电替代：每万kWh减碳8吨 → 绿证+CCER

最终：园区整体净零甚至净负排放
```

## 四、闭环的经济价值

| 循环环节 | 无闭环(外付) | 有闭环(内部) | 差值(年收益) |
|---------|------------|------------|------------|
| 电费 | ¥0.70/kWh | ¥0.20/kWh | 省¥500万(按1000万kWh) |
| 固废处置 | ¥100-200/吨 | ¥30+产建材 | 省¥200万+赚¥300万 |
| 废水处理 | ¥5-8/吨 | 中水¥1.5/吨 | 省¥200万 |
| 工业用水 | ¥4-6/吨 | 中水¥2/吨 | 省¥150万 |
| 碳配额购买 | ¥60-100/吨CO₂ | 自产CCER卖¥60-100 | 赚¥300-500万 |
| 余热利用 | 排放 | 温室+干燥 | 省¥80万 |
| **合计** | **外流¥1200万+** | **内部循环** | **增收省支~¥1500万/年** |

## 五、闭环设计三原则

1. **就近匹配**：热解车间挨着藻类池（CO₂管道直送），水泥车间靠近固废堆场，不跨区运输（管道比卡车便宜10倍）

2. **余热梯级利用**：高温(500°C+)→发电/热解，中温(100-300°C)→温室/烘干，低温(50-100°C)→供暖/藻类

3. **冗余度设计**：关键环节有备用路径——光伏不出力→风电补，风光都不出→储能放，储能放完→氢燃料电池启动，氢也没有→市电备用电
