# 愿景与战略

## 一句话愿景

> 成为中国领先的零碳产业生态运营商——拥有能源定价权、碳资产开发权和产业招商话语权的平台型公司。

## 核心使命

让企业 **减碳的同时降低成本**——不是帮企业花钱做碳中和，而是在帮企业省钱的过程中顺便碳达标。

## 战略五环

```
              ┌──────────────┐
              │  碳中和服务  │  ← 高毛利、强引流、轻资产
              │  (软产业)    │
              └──────┬───────┘
                     │
    ┌─────────┐  ┌──┴──┐  ┌──────────┐
    │ 清洁能源 │←│ 闭环 │→│ 循环经济  │  ← 能源底座 + 资源闭环
    │ (底座)   │  │ 生态 │  │ (变废为宝)│
    └────┬────┘  └──┬──┘  └─────┬────┘
         │          │           │
         └──────────┴───────────┘
              ┌──────────┐
              │ 低碳制造 │  ← 核心产值、以绿制绿
              │ (主业)   │
              └────┬─────┘
                   │
              ┌────┴─────┐
              │ 碳移除    │  ← 负碳技术、赚碳汇
              │ (壁垒)    │
              └──────────┘
```

## 发展路径

```
Phase 1 (0-2年)：500亩样板 → 验证模式
Phase 2 (2-4年)：2000亩复制 → 规模盈利
Phase 3 (4-6年)：5000亩标杆 → 产业生态
Phase 4 (6-10年)：全国复制10+基地 → 平台型公司
```

## 估值预期

| 阶段 | 业务形态 | 估值逻辑 | 预期估值 |
|------|---------|---------|---------|
| 样板期 | 园区运营商 + 能源资产 | 10-15倍PE | 5-10亿 |
| 复制期 | 能源装备制造商 + 系统方案商 | 20-25倍PE | 30-60亿 |
| 生态期 | 碳资产平台 + 产业生态 | 25-30倍PE | 100-300亿 |
