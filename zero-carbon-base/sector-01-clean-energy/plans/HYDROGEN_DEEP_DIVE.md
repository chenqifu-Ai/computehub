# 氢能 · 深度设计

> 清洁能源板块的核心子赛道。不是单独盈利中心，而是全能源系统的调节器、全产业链的政策杠杆、全园区的招商磁铁。

## 一、在清洁能源板块中的定位

```
风光余电 → 电解制氢 → 储氢 → 加氢站 → 氢能交通
                                  ↘ 氢燃料电池发电 → 调峰备用
                                  ↘ 工业用氢（给园区制造企业）

电池存电最多4-8小时，氢可以存几个月——唯一跨季节储能手段。
```

## 二、系统配置

| 环节 | 500亩配置 | 2000亩配置 | 5000亩配置 |
|------|----------|-----------|-----------|
| 制氢(碱性电解槽) | 1-2MW | 5MW | 10MW |
| 日产氢量 | 500-1000kg | 2.5吨 | 5吨 |
| 储氢 | 高压罐300-500kg | 高压罐+金属氢化物 | 高压罐+盐穴(如条件) |
| 加氢站 | 1座(三级) | 1座(二级) | 2座(一级+二级) |
| 下游 | 园区叉车+内部车辆 | 重卡20-30台+叉车50台 | 重卡100台+公交+物流 |

## 三、制氢成本拆解

```
1kg 绿氢成本：
  电费：52kWh × ¥0.25(谷电) = ¥13.0
  水费：9L × ¥0.005 = ¥0.05
  折旧：¥200万设备 ÷ 10年 ÷ 180吨 = ¥1.1
  运维：¥80万/年 ÷ 180吨 = ¥0.44
  ─────────────────────────────────
  合计：¥14.6/kg H₂

市场售价参考：
  灰氢(煤制氢含碳成本)：¥8-12/kg
  蓝氢(天然气+CCUS)：¥15-20/kg
  加氢站零售：¥30-45/kg(补贴后¥25-35)
  
定价策略：¥28/kg → 比市面便宜20-30%，毛利率40-50%
```

## 五、战略价值（超越氢能本身）

1. **消化余电**：风光弃电率从15%降到3%，提高全能源资产IRR 1-2个百分点
2. **政府关系**：氢能是三级叠加补贴赛道，一年跟市/省领导至少两次深度汇报
3. **招商磁铁**：承诺氢能物流成本低于柴油30%，一公里省¥1.5
4. **产业协同**：电解槽自产→自己验证→对外销售，从"用户"变"供应商"
