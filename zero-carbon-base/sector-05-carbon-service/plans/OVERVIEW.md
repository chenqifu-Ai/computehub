# 碳中和服务 · 软产业、高毛利、强引流

## 定位

利润倍增器、客户粘性增强器、品牌溢价放大器。

## 四大业务线

### 1. 碳管理
- 碳核算/核查（ISO 14064 / GHG Protocol）
- 碳足迹认证（产品碳标签）
- MRV系统开发（数字化能碳管理平台）
- 企业ESG报告编制

### 2. 碳金融
- CCER开发（年10-25万吨资产池）
- 碳配额质押融资
- 碳基金/碳信托
- 绿色保险

### 3. 研发/中试
- 低碳技术研究院（联合高校）
- 零碳实验室
- 中试基地（CCUS/绿氢/新材料）

### 4. 绿色咨询/培训
- ESG战略咨询
- 零碳工厂改造（全流程设计）
- 双碳人才培训（人社部认证）

## 商业模式特点

| 特性 | 价值 |
|------|------|
| 轻资产 | 不需要大额设备投资 |
| 高毛利 | 50-70%毛利率 |
| 强引流 | 咨询客户→能源/制造客户转化率>30% |
| 粘性 | 持续碳管理服务→锁定客户5-10年 |
