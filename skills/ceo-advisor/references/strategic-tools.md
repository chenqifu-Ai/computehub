# 战略规划工具箱

## SWOT分析

### 定义
评估企业内部优势(S)、劣势(W)和外部分机会(O)、威胁(T)

### 矩阵分析
| | 正面 | 负面 |
|---|---|---|
| 内部 | 优势(S) | 劣势(W) |
| 外部 | 机会(O) | 威胁(T) |

### 应用步骤
1. 列出内部优势和劣势（资源、能力、流程）
2. 识别外部机会和威胁（市场、政策、技术）
3. 制定SO（发挥优势抓住机会）战略
4. 制定WO（克服劣势抓住机会）战略
5. 制定ST（发挥优势应对威胁）战略
6. 制定WT（克服劣势规避威胁）战略

## 波特五力分析

### 五大竞争力量
| 力量 | 分析要点 |
|------|----------|
| 供应商议价能力 | 供应商集中度、替代品、转换成本 |
| 买方议价能力 | 客户集中度、替代品、价格敏感度 |
| 新进入者威胁 | 进入壁垒、规模经济、品牌认知 |
| 替代品威胁 | 替代品性价比、转换成本、替代意愿 |
| 行业竞争程度 | 竞争者数量、行业增长率、固定成本 |

### 应用
- 进入新行业前的风险评估
- 制定竞争策略
- 识别盈利能力影响因素

## BCG矩阵

### 四象限
| 市场增长率 | 相对市场份额 | 类型 | 策略 |
|------------|--------------|------|------|
| 高 | 高 | 明星 | 投资增长 |
| 高 | 低 | 问题 | 选择性投资 |
| 低 | 高 | 现金牛 | 维持/收割 |
| 低 | 低 | 瘦狗 | 退出/剥离 |

### 应用
- 产品组合分析
- 资源分配决策
- 业务优先级排序

## 平衡计分卡(BSC)

### 四维度
| 维度 | 关注点 | 典型指标 |
|------|--------|----------|
| 财务 | 盈利能力 | 收入增长、利润率 |
| 客户 | 客户价值 | 满意度、留存率 |
| 流程 | 运营效率 | 周期时间、质量 |
| 学习成长 | 发展能力 | 员工技能、创新 |

### 应用
- 战略分解为可执行目标
- 绩效考核体系设计
- 战略执行监控
