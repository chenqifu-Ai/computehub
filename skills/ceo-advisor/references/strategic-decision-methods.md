# CEO战略决策方法

## 概述

战略决策是企业最高层次的决策，直接影响企业的发展方向和竞争优势。CEO需要掌握科学的决策方法，在复杂多变的环境中做出正确决策。本文系统介绍战略决策的方法、工具和最佳实践。

---

## 一、战略决策概述

### 1. 战略决策定义

**战略决策**：对企业长期发展方向、资源配置、竞争优势等重大问题做出的决策，具有全局性、长期性、不可逆性等特点。

### 2. 战略决策特点

| 特点 | 说明 |
|-----|------|
| 全局性 | 影响企业整体 |
| 长期性 | 影响周期长 |
| 不可逆性 | 逆转成本高 |
| 风险性 | 结果不确定 |
| 复杂性 | 涉及因素多 |

### 3. 决策层次

```
┌─────────────────────────────────────────────────────────────┐
│                    企业决策层次                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  战略决策（CEO层）                                           │
│    ├── 企业使命、愿景、价值观                               │
│    ├── 业务组合、发展方向                                   │
│    ├── 重大投资、并购决策                                   │
│    └── 组织架构、人才战略                                   │
│                                                             │
│  战术决策（高管层）                                          │
│    ├── 业务战略、竞争策略                                   │
│    ├── 资源分配、预算决策                                   │
│    └── 部门目标、绩效指标                                   │
│                                                             │

### 2. 决策时机选择

| 时机类型 | 说明 | 建议 |
|---------|------|------|
| 危机决策 | 时间紧迫、信息不全 | 快速决策，边做边调 |
| 战略决策 | 影响重大、时间充裕 | 充分分析，谨慎决策 |
| 机会窗口 | 时机有限、稍纵即逝 | 快速判断，果断行动 |
| 例行决策 | 常规事项、风险可控 | 流程化、标准化 |

| 方案 | 评分标准 | 权重 | 方案A | 方案B | 方案C |
|-----|---------|------|-------|-------|-------|
| 市场规模 | 1-10分 | 30% | 8 | 6 | 7 |
| 竞争强度 | 1-10分 | 20% | 5 | 7 | 6 |
| 资源匹配 | 1-10分 | 25% | 7 | 8 | 5 |
| 风险程度 | 1-10分 | 25% | 6 | 7 | 8 |
| **加权总分** | | | 6.55 | 6.95 | 6.55 |

```

### 3. 群体决策方法

**头脑风暴**：
- 原则：不批评、自由发挥、追求数量
- 步骤：准备→发散→收敛→筛选

**德尔菲法**：
```
┌─────────────────────────────────────────────────────────────┐
│                    德尔菲法流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │

│     ↓                                                       │
│  汇总结果，反馈给专家                                        │
│     ↓                                                       │
│  第二轮：专家根据反馈修正                                    │
│     ↓                                                       │
│  再次汇总，再次反馈                                          │
│     ↓                                                       │
│  第三轮：专家再次修正                                        │
│     ↓                                                       │
│  最终共识                                                    │
│                                                             │

```

**投票决策**：
| 方法 | 说明 | 适用场景 |
|-----|------|---------|
| 简单多数 | 票数最多者胜出 | 方案差异不大 |
| 加权投票 | 按权重投票 | 决策者影响力不同 |
| 多轮投票 | 多轮淘汰 | 方案较多 |
| 共识决策 | 全体同意 | 重大决策、团队协作 |

### 4. 决策陷阱与规避

| 陷阱 | 表现 | 规避方法 |
|-----|------|---------|
| 锚定效应 | 过度依赖第一信息 | 多角度分析、独立判断 |
| 确认偏见 | 只关注支持观点的信息 | 寻找反面证据 |
| 沉没成本 | 因已投入而继续坚持 | 忽略沉没成本、只看未来 |
| 过度自信 | 高估自己的判断能力 | 寻求反对意见、量化不确定性 |
| 群体思维 | 追求一致而压制异议 | 指定反对者、匿名表达 |
| 近期偏见 | 过度重视最近发生的事 | 长期视角、数据分析 |

### 2. 资源配置决策

**投资决策方法**：

| 方法 | 说明 | 适用场景 |
|-----|------|---------|
| 净现值（NPV） | 未来现金流的现值 | 长期投资 |
| 内部收益率（IRR） | 投资回报率 | 项目比较 |
| 回收期 | 收回投资时间 | 流动性考虑 |
| 实物期权 | 灵活性价值 | 不确定性高 |

**NPV计算示例**：
```
NPV = Σ [CFt / (1+r)^t] - 初始投资

其中：
CFt = 第t期现金流
r = 折现率
t = 时间周期

NPV > 0：项目可行
NPV < 0：项目不可行
```
