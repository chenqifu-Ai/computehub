# 战略决策方法

## 战略决策概述

### 定义
战略决策是企业对重大、长期问题的决策，影响企业发展方向和资源配置。

### 特点
| 特点 | 说明 |
|------|------|
| 全局性 | 影响企业整体 |
| 长期性 | 影响周期长 |
| 风险性 | 不确定性高 |
| 不可逆性 | 调整成本高 |

## 战略决策流程

### 决策步骤
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  问题识别 → 信息收集 → 方案制定 → 方案评估 → 决策执行 │
│                                                     │
│  1. 问题识别：明确决策问题                          │
│  2. 信息收集：收集相关信息                          │
│  3. 方案制定：制定多个方案                          │
│  4. 方案评估：评估各方案优劣                        │
│  5. 决策执行：实施决策并跟踪                        │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## 决策方法

### SWOT分析
| 维度 | 说明 | 策略 |
|------|------|------|
| S-优势 | 内部优势 | 利用优势 |
| W-劣势 | 内部劣势 | 克服劣势 |
| O-机会 | 外部机会 | 抓住机会 |
| T-威胁 | 外部威胁 | 规避威胁 |

**战略组合**：
| 组合 | 策略 |
|------|------|
| SO | 发挥优势抓住机会 |
| WO | 克服劣势抓住机会 |
| ST | 发挥优势规避威胁 |
| WT | 克服劣势规避威胁 |

## 决策树分析

### 决策树方法
```
决策节点（□）→ 方案选择
机会节点（○）→ 不确定事件
结果节点（△）→ 最终结果

计算：
期望值 = Σ（结果 × 概率）
```

### 示例
```
是否进入新市场？
├── 进入（投资100万）
│   ├── 成功（60%）→ 收益300万
│   └── 失败（40%）→ 亏损50万
│   期望收益 = 0.6×300 + 0.4×(-50) - 100 = 160 - 20 - 100 = 40万
│
└── 不进入
    └── 收益 = 0
```

## 情景分析

### 情景规划
| 情景 | 描述 | 应对策略 |
|------|------|----------|
| 乐观情景 | 市场增长超预期 | 扩大投资 |
| 基准情景 | 市场正常增长 | 按计划执行 |
| 悲观情景 | 市场下滑 | 收缩成本 |

### 情景分析步骤
1. 确定关键因素
2. 构建情景
3. 分析影响
4. 制定应对

## 德尔菲法

### 方法介绍
| 步骤 | 说明 |
|------|------|
| 专家选择 | 选择相关领域专家 |
| 匿名征询 | 发送问卷 |
| 汇总反馈 | 汇总意见反馈 |
| 多轮迭代 | 2-3轮 |
| 形成结论 | 达成共识 |

### 优点
- 避免群体压力
- 专家独立判断
- 多轮反馈修正

## 决策常见误区

### 误区类型
| 误区 | 说明 | 应对 |
|------|------|------|
| 过度自信 | 高估自己判断 | 收集更多信息 |
| 锚定效应 | 受第一印象影响 | 多角度分析 |
| 群体思维 | 追求一致 | 鼓励不同意见 |
| 沉没成本 | 不愿放弃已投入 | 看未来不看过去 |

### 决策原则
1. 数据驱动决策
2. 多方案比较
3. 考虑风险
4. 保持灵活性
5. 及时复盘
