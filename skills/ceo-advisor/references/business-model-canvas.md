# 商业模式画布 - 一页纸商业模式设计工具

## 概述

商业模式画布（Business Model Canvas，BMC）是由Alexander Osterwalder提出的商业模式分析工具，用一页纸描述、分析和设计商业模式，是创业者和企业战略制定的核心工具。

---

## 二、九大模块详解

### 1. 客户细分

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          客户细分（Customer Segments）                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  定义：企业为谁创造价值？谁是最重要的客户？                                    │
│                                                                             │
│  客户细分类型：                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  1. 大众市场（Mass Market）                                         │   │
│  │     ├── 特征：广泛的客户群体                                         │   │
│  │     ├── 需求：相似的需求和问题                                       │   │
│  │     └── 示例：消费品、快消品                                         │   │
│  │                                                                     │   │
│  │  2. 利基市场（Niche Market）                                        │   │
│  │     ├── 特征：专门化的客户群体                                       │   │
│  │     ├── 需求：特定需求                                               │   │
│  │     └── 示例：高端定制、专业服务                                     │   │
│  │                                                                     │   │
│  │  3. 区隔市场（Segmented）                                           │   │
│  │     ├── 特征：多个细分市场                                          │   │
│  │     ├── 需求：略有不同的需求                                         │   │
│  │     └── 示例：银行、航空公司                                         │   │
│  │                                                                     │   │
│  │  4. 多边市场（Diversified）                                          │   │
│  │     ├── 特征：多个独立的客户群体                                     │   │
│  │     ├── 需求：完全不同的需求                                         │   │
│  │     └── 示例：平台型企业                                             │   │
│  │                                                                     │   │
│  │  5. 多边平台（Multi-sided Platform）                                │   │
│  │     ├── 特征：连接多个客户群体                                       │   │
│  │     ├── 需求：相互依赖的需求                                         │   │
│  │     └── 示例：Uber、Airbnb                                          │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  分析问题：                                                                  │
│  ├── 我们为谁创造价值？                                                     │
│  ├── 谁是我们最重要的客户？                                                  │
│  ├── 客户群体可细分吗？                                                     │
│  └── 不同客户群体的需求差异是什么？                                          │
│                                                                             │
│  客户细分工具：                                                              │
│  ├── 人口统计：年龄、性别、收入、教育                                        │
│  ├── 地理位置：地区、城市、国家                                             │
│  ├── 心理特征：生活方式、价值观、兴趣                                       │
│  ├── 行为特征：购买习惯、使用频率、忠诚度                                    │
│  └── 需求特征：痛点、需求、期望                                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2. 价值主张

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          价值主张（Value Propositions）                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  定义：我们为客户提供什么价值？解决什么问题？                                  │
│                                                                             │
│  价值主张类型：                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  1. 创新（Newness）                                                 │   │
│  │     └── 提供前所未有的产品或服务                                     │   │
│  │                                                                     │   │
│  │  2. 性能（Performance）                                             │   │
│  │     └── 优于竞争对手的产品性能                                       │   │
│  │                                                                     │   │
│  │  3. 定制（Customization）                                           │   │
│  │     └── 满足个别客户需求                                             │   │
│  │                                                                     │   │
│  │  4. 设计（Design）                                                  │   │
│  │     └── 卓越的设计和美感                                            │   │
│  │                                                                     │   │
│  │  5. 品牌/地位（Brand/Status）                                       │   │
│  │     └── 品牌价值和身份象征                                           │   │
│  │                                                                     │   │
│  │  6. 价格（Price）                                                   │   │
│  │     └── 更低的价格                                                  │   │
│  │                                                                     │   │
│  │  7. 成本缩减（Cost Reduction）                                      │   │
│  │     └── 帮助客户降低成本                                            │   │
│  │                                                                     │   │
│  │  8. 风险缩减（Risk Reduction）                                      │   │
│  │     └── 降低客户风险                                                │   │
│  │                                                                     │   │
│  │  9. 可达性（Accessibility）                                         │   │
│  │     └── 使产品更容易获得                                            │   │
│  │                                                                     │   │
│  │  10. 便利性/可用性（Convenience/Usability）                          │   │
│  │      └── 使产品更易用                                               │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  分析问题：                                                                  │
│  ├── 我们为客户解决什么问题？                                                │
│  ├── 我们满足客户什么需求？                                                  │
│  ├── 我们提供什么价值？                                                     │
│  ├── 为什么客户选择我们？                                                   │
│  └── 我们与竞争对手有什么不同？                                               │
│                                                                             │
│  价值主张设计：                                                              │
│  ├── 客户痛点：客户面临的问题和困难                                           │
│  ├── 客户收益：客户期望的结果和利益                                           │
│  ├── 产品/服务：我们提供什么                                                 │
│  ├── 痛点缓解：如何解决客户问题                                              │
│  └── 收益创造：如何创造客户价值                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│  │                                                                     │   │
```

```
│  │     ├── 基础设施：网络、服务器                                      │   │
│  │     ├── 分销网络                                                    │   │
│  │     └── 示例：亚马逊仓储、沃尔玛门店                                 │   │

│  │     ├── 版权                                                        │   │

│  │     └── 示例：可口可乐配方、谷歌算法                                 │   │

│  │     ├── 关键人才                                                   │   │
│  │     ├── 专业技能                                                   │   │
```

## 五、总结

### 商业模式画布核心要点

```
1. 九大模块：客户、价值、渠道、关系、收入、资源、业务、合作、成本
2. 前台：客户细分、价值主张、渠道通路、客户关系、收入来源
3. 后台：核心资源、关键业务、重要合作、成本结构
4. 价值主张是核心，连接前后台
5. 迭代优化，持续验证
```

```
九大模块画布展，前五后四价值连。
客户细分定对象，价值主张解痛点。

收入来源看变现，核心资源是关键。
关键业务日日忙，重要合作资源连。
成本结构算一算，商业模式画布全。
```
