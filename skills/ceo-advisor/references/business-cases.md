# 商业案例分析

## 案例分析方法

### 案例分析框架
| 步骤 | 内容 | 说明 |
|------|------|------|
| 背景 | 公司背景、行业背景 | 了解环境 |
| 问题 | 核心问题是什么 | 明确问题 |
| 分析 | 分析原因、影响 | 深入分析 |
| 方案 | 解决方案是什么 | 提出方案 |
| 结果 | 实施结果如何 | 评估效果 |
| 启示 | 有什么启示 | 总结经验 |

| 创始人 | 王兴 |

| 因素 | 说明 |

### 核心战略
```
美团战略：

1. 高频打低频
   ├── 外卖（高频）
   ├── 到店（中频）
   └── 酒旅（低频）

2. 网络效应
   ├── 商家越多 → 用户越多
   └── 用户越多 → 商家越多

3. 运营效率
   ├── 配送网络优化
   ├── 商家管理优化
   └── 成本控制

4. 多元化
   ├── 外卖
   ├── 到店
   ├── 酒旅
   └── 新业务（买菜、共享单车）
```

| 启示 | 应用 |
|------|------|
| 高频打低频 | 找到高频入口 |
| 网络效应 | 构建双边网络 |
| 运营效率 | 成本控制是核心 |
| 多元化 | 业务协同 |
