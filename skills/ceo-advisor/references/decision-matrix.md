# 决策矩阵与多准则决策分析 - 系统化决策工具

## 概述

决策矩阵（Decision Matrix）是一种系统化的决策工具，用于在多个方案之间进行选择。当面临多个选择标准时，决策矩阵帮助决策者客观地评估和比较各个方案。

---

## 一、决策矩阵基本概念

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        决策矩阵定义                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  决策矩阵是一种定量决策工具，通过以下步骤进行：                               │
│  1. 确定决策标准                                                            │
│  2. 为标准分配权重                                                          │
│  3. 评估各方案在标准上的得分                                                │
│  4. 计算加权总分                                                            │
│  5. 选择得分最高的方案                                                      │
│                                                                             │
│  决策矩阵结构：                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │              │   标准1   │   标准2   │   标准3   │   总分   │      │   │
│  │              │  权重w1   │  权重w2   │  权重w3   │         │      │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │   方案A     │   得分    │   得分    │   得分    │   加权   │      │   │
│  │             │   S1A     │   S2A     │   S3A     │   总分   │      │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │   方案B     │   得分    │   得分    │   得分    │   加权   │      │   │
│  │             │   S1B     │   S2B     │   S3B     │   总分   │      │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │   方案C     │   得分    │   得分    │   得分    │   加权   │      │   │
│  │             │   S1C     │   S2C     │   S3C     │   总分   │      │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  加权总分计算公式：                                                          │
│  总分 = Σ(标准权重 × 方案得分)                                              │
│  总分 = w1×S1 + w2×S2 + w3×S3 + ... + wn×Sn                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 二、决策矩阵步骤

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        决策矩阵步骤                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  步骤一：明确决策问题                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  要解决的问题：                                                      │   │
│  │  ├── 决策目标是什么？                                                │   │
│  │  ├── 有哪些约束条件？                                                │   │
│  │  ├── 决策时间框架？                                                  │   │
│  │  └── 谁是决策者？                                                    │   │
│  │                                                                     │   │
│  │  示例：                                                              │   │
│  │  决策目标：选择供应商                                                │   │
│  │  约束条件：预算100万以内                                            │   │
│  │  时间框架：本季度完成                                                │   │
│  │  决策者：采购部门                                                    │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤二：列出备选方案                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  头脑风暴：                                                          │   │
│  │  ├── 列出所有可能的方案                                              │   │
│  │  ├── 不评估可行性，先收集                                            │   │
│  │  └── 鼓励团队成员参与                                                │   │
│  │                                                                     │   │
│  │  筛选可行方案：                                                      │   │
│  │  ├── 排除明显不可行的方案                                            │   │
│  │  ├── 确保方案之间的可比性                                            │   │
│  │  └── 控制方案数量（通常3-7个）                                       │   │
│  │                                                                     │   │
│  │  示例：                                                              │   │
│  │  方案A：供应商甲                                                      │   │
│  │  方案B：供应商乙                                                      │   │
│  │  方案C：供应商丙                                                      │   │
│  │  方案D：供应商丁                                                      │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤三：确定评估标准                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  标准要求：                                                          │   │
│  │  ├── 与决策目标相关                                                  │   │
│  │  ├── 可衡量或可评估                                                  │   │
│  │  ├── 独立性（标准之间不重叠）                                        │   │
│  │  └── 完整性（覆盖重要方面）                                          │   │
│  │                                                                     │   │
│  │  标准类型：                                                          │   │
│  │  ├── 定量标准：成本、时间、数量                                      │   │
│  │  ├── 定性标准：质量、服务、风险                                      │   │
│  │  └── 混合标准：综合评估                                              │   │
│  │                                                                     │   │
│  │  示例（供应商选择）：                                                │   │
│  │  标准1：价格                                                          │   │
│  │  标准2：质量                                                          │   │
│  │  标准3：交期                                                          │   │
│  │  标准4：服务                                                          │   │
│  │  标准5：资质                                                          │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤四：分配标准权重                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  权重方法：                                                          │   │
│  │  ├── 直接分配法：直接分配权重                                        │   │
│  │  ├── 成对比较法：两两比较重要性                                      │   │
│  │  └── 层次分析法（AHP）：系统化权重确定                                │   │
│  │                                                                     │   │
│  │  直接分配法：                                                        │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │  标准  │  重要性  │  权重  │                                │   │   │
│  │  │  ────────────────────────────────────────────────────────  │   │   │
│  │  │  价格  │   很重要 │  0.3   │                                │   │   │
│  │  │  质量  │   最重要 │  0.35  │                                │   │   │
│  │  │  交期  │   重要   │  0.15  │                                │   │   │
│  │  │  服务  │   一般   │  0.1   │                                │   │   │
│  │  │  资质  │   重要   │  0.1   │                                │   │   │
│  │  │  合计  │         │  1.0   │                                │   │   │
│  │  └─────────────────────────────────────────────────────────────┘   │   │
│  │                                                                     │   │
│  │  成对比较法：                                                        │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │  比较        │  更重要  │  得分                              │   │   │
│  │  │  ────────────────────────────────────────────────────────  │   │   │
│  │  │  价格vs质量  │  质量   │  质量+1                            │   │   │
│  │  │  价格vs交期  │  价格   │  价格+1                            │   │   │
│  │  │  质量vs交期  │  质量   │  质量+1                            │   │   │
│  │  │  ...         │  ...    │  ...                               │   │   │
│  │  └─────────────────────────────────────────────────────────────┘   │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤五：评分各方案                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  评分方法：                                                          │   │
│  │  ├── 定量标准：根据数值直接评分                                      │   │
│  │  ├── 定性标准：根据评估标准评分                                      │   │
│  │  └── 评分量表：1-5分或1-10分                                          │   │
│  │                                                                     │   │
│  │  示例（1-5分量表）：                                                  │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │  分数  │  描述                                                │   │   │
│  │  │  ────────────────────────────────────────────────────────  │   │   │
│  │  │   5    │  优秀：远超预期                                      │   │   │
│  │  │   4    │  良好：超过预期                                      │   │   │
│  │  │   3    │  一般：符合预期                                      │   │   │
│  │  │   2    │  较差：低于预期                                      │   │   │
│  │  │   1    │  很差：远低于预期                                    │   │   │
│  │  └─────────────────────────────────────────────────────────────┘   │   │
│  │                                                                     │   │
│  │  评分示例：                                                          │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │          │ 价格  │ 质量  │ 交期  │ 服务  │ 资质  │        │   │   │
│  │  │  权重    │  0.3  │  0.35 │  0.15 │  0.1  │  0.1  │        │   │   │
│  │  │  ────────────────────────────────────────────────────────  │   │   │
│  │  │  供应商甲 │   4   │   5   │   4   │   3   │   4   │        │   │   │
│  │  │  供应商乙 │   5   │   3   │   5   │   4   │   3   │        │   │   │
│  │  │  供应商丙 │   3   │   4   │   3   │   5   │   5   │        │   │   │
│  │  │  供应商丁 │   4   │   4   │   4   │   4   │   4   │        │   │   │
│  │  └─────────────────────────────────────────────────────────────┘   │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤六：计算加权总分                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  计算公式：                                                          │   │
│  │  加权总分 = Σ(标准权重 × 方案得分)                                  │   │
│  │                                                                     │   │
│  │  计算示例：                                                          │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │          │ 价格  │ 质量  │ 交期  │ 服务  │ 资质  │ 总分   │   │   │
│  │  │  权重    │  0.3  │  0.35 │  0.15 │  0.1  │  0.1  │       │   │   │
│  │  │  ────────────────────────────────────────────────────────  │   │   │
│  │  │  供应商甲 │   4   │   5   │   4   │   3   │   4   │  4.25 │   │   │
│  │  │  供应商乙 │   5   │   3   │   5   │   4   │   3   │  3.95 │   │   │
│  │  │  供应商丙 │   3   │   4   │   3   │   5   │   5   │  3.75 │   │   │
│  │  │  供应商丁 │   4   │   4   │   4   │   4   │   4   │  4.00 │   │   │
│  │  └─────────────────────────────────────────────────────────────┘   │   │
│  │                                                                     │   │
│  │  计算：                                                              │   │

│  │  供应商乙 = 0.3×5 + 0.35×3 + 0.15×5 + 0.1×4 + 0.1×3 = 3.95        │   │

│  │  供应商丁 = 0.3×4 + 0.35×4 + 0.15×4 + 0.1×4 + 0.1×4 = 4.00        │   │
│  │                                                                     │   │
│  │  排名：                                                              │   │
│  │  1. 供应商甲：4.25                                                   │   │
│  │  2. 供应商丁：4.00                                                   │   │
│  │  3. 供应商乙：3.95                                                   │   │
│  │  4. 供应商丙：3.75                                                   │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  步骤七：分析与决策                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  分析结果：                                                          │   │
│  │  ├── 查看总分排名                                                    │   │
│  │  ├── 分析各标准表现                                                  │   │
│  │  ├── 识别优势和劣势                                                  │   │
│  │  └── 考虑敏感性分析                                                  │   │
│  │                                                                     │   │
│  │  敏感性分析：                                                        │   │
│  │  ├── 改变权重，看排名是否变化                                        │   │
│  │  ├── 改变得分，看排名是否变化                                        │   │
│  │  └── 评估决策稳定性                                                  │   │
│  │                                                                     │   │
│  │  决策建议：                                                          │   │
│  │  ├── 选择总分最高的方案                                              │   │
│  │  ├── 考虑其他因素（风险、时间等）                                    │   │
│  │  ├── 征求相关方意见                                                  │   │
│  │  └── 记录决策过程                                                    │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 三、层次分析法（AHP）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        层次分析法（AHP）                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  层次分析法（Analytic Hierarchy Process）由Saaty提出，                        │
│  用于处理复杂决策问题的多准则决策方法。                                       │
│                                                                             │
│  AHP步骤：                                                                  │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  步骤1：建立层次结构                                                 │   │
│  │         目标层                                                      │   │
│  │           │                                                         │   │
│  │         准则层                                                      │   │
│  │         /│\                                                        │   │
│  │        / │ \                                                       │   │
│  │       /  │  \                                                      │   │
│  │      方案层                                                        │   │
│  │                                                                     │   │
│  │  步骤2：构造判断矩阵                                                 │   │
│  │         两两比较，确定相对重要性                                     │   │
│  │                                                                     │   │
│  │  步骤3：计算权重向量                                                 │   │
│  │         计算特征向量，归一化                                         │   │
│  │                                                                     │   │
│  │  步骤4：一致性检验                                                   │   │
│  │         检查判断矩阵的一致性                                         │   │
│  │                                                                     │   │
│  │  步骤5：综合评价                                                     │   │
│  │         计算各方案综合得分                                           │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  成对比较量表（1-9量表）：                                                    │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  标度  │  含义                                                    │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │   1    │  同等重要                                                │   │
│  │   3    │  稍微重要                                                │   │
│  │   5    │  明显重要                                                │   │
│  │   7    │  强烈重要                                                │   │
│  │   9    │  极端重要                                                │   │
│  │  2,4,6,8│  中间值                                                 │   │
│  │  倒数  │  反向比较                                                │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  判断矩阵示例：                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │            │  价格  │  质量  │  交期  │  服务  │                    │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │   价格    │   1    │  1/3   │   3    │   5    │                    │   │
│  │   质量    │   3    │   1    │   5    │   7    │                    │   │
│  │   交期    │  1/3   │  1/5   │   1    │   3    │                    │   │
│  │   服务    │  1/5   │  1/7   │  1/3   │   1    │                    │   │
│  │                                                                     │   │
│  │  解读：                                                              │   │
│  │  - 质量 vs 价格 = 3：质量稍微比价格重要                               │   │
│  │  - 质量 vs 交期 = 5：质量明显比交期重要                               │   │
│  │  - 价格 vs 服务 = 5：价格明显比服务重要                               │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  一致性检验：                                                                │
│  ├── 计算一致性指标CI = (λmax - n) / (n - 1)                                │
│  ├── 计算一致性比率CR = CI / RI                                             │
│  ├── 若CR < 0.1，则一致性可接受                                              │
│  └── 若CR ≥ 0.1，需调整判断矩阵                                              │
│                                                                             │
│  随机一致性指标RI：                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  n   │  1    │  2    │  3    │  4    │  5    │  6    │  7   │    │   │
│  │  ─────────────────────────────────────────────────────────────    │   │
│  │  RI  │  0    │  0    │ 0.58  │ 0.90  │ 1.12  │ 1.24  │ 1.32 │    │   │
│  └─────────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 四、决策矩阵应用场景

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        决策矩阵应用场景                                      │
```

## 五、决策矩阵优缺点

## 六、总结

### 决策矩阵核心要点

```
1. 确定标准：明确评估维度
2. 分配权重：反映标准重要性
3. 评分评估：量化各方案表现
4. 加权计算：得出综合得分
5. 分析决策：考虑敏感性
6. 文档记录：保留决策过程
```

### 记忆口诀

```
决策矩阵六步走，标准权重方案分。
明确标准定维度，权重分配显重要。
评分评估量化看，加权计算得总分。
分析决策要敏感，文档记录好查询。
AHP法更系统，成对比较一致性。
多准则决策不迷，客观决策有依据。
```
