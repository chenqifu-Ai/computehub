# 蓝海战略 - 创造无竞争市场空间

## 概述

蓝海战略（Blue Ocean Strategy）是由W. Chan Kim和Renée Mauborgne提出的战略管理理论，主张企业应该寻找或创造"蓝海"——无竞争的市场空间，而不是在"红海"中与竞争对手激烈厮杀。

---

## 二、价值创新

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          价值创新                                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  价值创新是蓝海战略的核心：                                                   │
│                                                                             │
│  传统战略：差异化 OR 低成本                                                   │
│  ├── 差异化战略：提高价值，成本上升                                          │
│  └── 成本领先战略：降低成本，价值下降                                        │
│                                                                             │
│  蓝海战略：差异化 AND 低成本                                                  │
│  └── 价值创新：同时实现差异化和低成本                                        │
│                                                                             │
│  价值创新公式：                                                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │     价值创新 = 价值创造 + 成本降低                                   │   │
│  │                                                                     │   │
│  │              ↑                 ↑                                    │   │
│  │           买方价值          成本结构优化                              │   │
│  │                                                                     │   │
│  │  ┌───────────────────────────────────────────────────────────────┐ │   │
│  │  │                     │                                        │ │   │
│  │  │        差异化      │           蓝海战略                      │ │   │
│  │  │        ↑          │         价值创新                        │ │   │
│  │  │        │          │            区                            │ │   │
│  │  │        │          │           /│\                           │ │   │
│  │  │        │          │          / │ \                          │ │   │
│  │  │        │          │         /  │  \                         │ │   │
│  │  │        │          │        /   │   \                        │ │   │
│  │  │        │          │       /    │    \                       │ │   │
│  │  │   ─────┼──────────┼──────/─────┼─────\────────────────────  │ │   │
│  │  │        │          │     /      │      \                     │ │   │
│  │  │        │          │    /       │       \                    │ │   │
│  │  │        │          │   /        │        \                   │ │   │
│  │  │        │          │  /         │         \                  │ │   │
│  │  │        ↓          │ /          │          \                 │ │   │
│  │  │      成本         │/           │           \                │ │   │
│  │  │                   │红海区      │红海区      │                │ │   │
│  │  │                   │低成本      │差异化      │                │ │   │
│  │  └───────────────────────────────────────────────────────────────┘ │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  价值创新实现路径：                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  1. 剔除（Eliminate）                                               │   │
│  │     └── 剔除行业中不再创造价值的要素                                 │   │
│  │                                                                     │   │
│  │  2. 减少（Reduce）                                                  │   │
│  │     └── 减少低于行业标准的要素                                       │   │
│  │                                                                     │   │
│  │  3. 增加（Raise）                                                   │   │
│  │     └── 增加高于行业标准的要素                                       │   │
│  │                                                                     │   │
│  │  4. 创造（Create）                                                  │   │
│  │     └── 创造行业中从未有过的要素                                     │   │
│  │                                                                     │   │
│  │  效果：                                                              │   │
│  │  ├── 剔除和减少 → 降低成本                                          │   │
│  │  ├── 增加和创造 → 提升买方价值                                      │   │
│  │  └── 四个动作同时进行 → 价值创新                                    │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```
│  │                                                                     │   │
```

## 五、蓝海战略制定方法

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                      蓝海战略制定方法                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │

│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                                                                     │   │
│  │  六个路径：                                                          │   │
│  │                                                                     │   │
│  │  1. 跨越替代行业                                                    │   │
│  │     ├── 分析顾客在选择替代品时的考量                                │   │
│  │     └── 示例：咖啡店 vs 书店 vs 图书馆                              │   │
│  │                                                                     │   │
│  │  2. 跨越战略集团                                                    │   │
│  │     ├── 分析不同战略集团的顾客选择                                  │   │
│  │     └── 示例：经济型酒店 vs 豪华酒店                                │   │
│  │                                                                     │   │
│  │  3. 跨越买方链                                                      │   │
│  │     ├── 重新定义买方：购买者、使用者、影响者                          │   │
│  │     └── 示例：药品关注使用者而非购买者                              │   │
│  │                                                                     │   │
│  │  4. 跨越互补产品和服务                                              │   │
│  │     ├── 分析顾客在产品前后的需求                                    │   │
│  │     └── 示例：电影院提供保姆服务                                    │   │
│  │                                                                     │   │
│  │  5. 跨越功能-情感导向                                                │   │
│  │     ├── 功能导向转向情感或反之                                      │   │
│  │     └── 示例：星巴克的情感体验                                      │   │
│  │                                                                     │   │
│  │  6. 跨越时间                                                        │   │
│  │     ├── 预判行业趋势                                                │   │
│  │     └── 示例：苹果预见智能手机趋势                                  │   │
│  │                                                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  方法二：关注大局，而非数字                                                  │
│  ├── 绘制战略布局图                                                        │
│  ├── 直观展示竞争格局                                                      │
│  ├── 识别创新机会                                                          │
│  └── 让团队达成共识                                                        │
│                                                                             │
│  方法三：超越现有需求                                                        │
│  ├── 关注非客户：三层非客户                                                 │
│  │   ├── 第一层：即将购买的非客户                                         │   │
│  │   ├── 第二层：拒绝的非客户                                            │   │
│  │   └── 第三层：未探索市场的非客户                                        │   │
│  ├── 挖掘新需求                                                            │
│  └── 扩大市场边界                                                          │
│                                                                             │
│  方法四：遵循战略顺序                                                        │
│  ├── 买方效用：产品是否创造买方效用？                                        │
│  ├── 价格：价格是否被大众接受？                                              │
│  ├── 成本：能否以目标成本生产？                                              │
│  └── 采用：是否解决采用障碍？                                                │
```

## 九、总结

### 蓝海战略核心要点

```
1. 蓝海：创造无竞争的市场空间
2. 价值创新：同时实现差异化和低成本
3. 四步动作：剔除、减少、增加、创造
4. 六个路径：跨越替代、战略集团、买方、互补、功能情感、时间
5. 战略布局图：可视化竞争格局
6. 执行障碍：认知、资源、动力、政治
```

### 记忆口诀

```
红海竞争血淋淋，蓝海创新利润增。
剔除减少降成本，增加创造提价值。
四步动作同时做，六个路径找空间。
战略布局图分析，价值曲线看差异。
```
