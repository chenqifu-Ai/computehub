# PESTEL分析 - 宏观环境分析框架

## 概述

PESTEL分析（又称PESTLE分析）是用于分析企业宏观环境的战略工具，通过六个维度系统地评估外部环境对组织的影响。由PEST分析发展而来，增加了伦理和法律维度。

---

## 二、各维度详解

### 1. Political（政治因素）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          政治因素分析                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  关键要素：                                                                 │
│  ├── 政府稳定性：政权更迭风险                                               │
│  ├── 政策方向：产业政策、财政政策                                           │
│  ├── 贸易政策：关税、进出口限制                                              │
│  ├── 税收政策：税率、税收优惠                                               │
│  ├── 政府干预：补贴、监管                                                   │
│  └── 国际关系：地缘政治、贸易战                                              │
│                                                                             │
│  影响类型：                                                                 │
│  ├── 机会：政策支持、补贴、减税                                              │
│  └── 威胁：政策限制、加税、监管收紧                                          │
│                                                                             │
│  分析问题：                                                                 │
│  ├── 当前政府的政策方向是什么？                                              │
│  ├── 政治环境稳定吗？                                                       │
│  ├── 政府对行业有什么影响？                                                 │
│  ├── 贸易政策如何影响业务？                                                 │
│  └── 有哪些政治风险需要关注？                                               │
│                                                                             │
│  示例 - 新能源汽车行业：                                                     │
│  ├── 机会：新能源汽车补贴政策                                               │
│  ├── 机会：碳中和目标支持                                                   │
│  ├── 威胁：补贴退坡                                                        │
│  └── 威胁：国际贸易壁垒                                                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 2. Economic（经济因素）

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          经济因素分析                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  关键要素：                                                                 │
│  ├── 经济增长：GDP增长率                                                    │
│  ├── 利率：贷款利率、储蓄利率                                               │
│  ├── 通胀率：物价水平                                                       │
│  ├── 汇率：货币价值                                                         │
│  ├── 就业：失业率、工资水平                                                 │
│  ├── 消费：消费支出、消费信心                                               │
│  └── 投资：投资水平、投资回报                                               │
│                                                                             │
│  影响分析：                                                                 │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                     经济因素对业务的影响                              │   │
│  │                                                                     │   │
│  │   高经济增长    → 市场需求增加 → 扩张机会                            │   │
│  │   低经济增长    → 消费减少     → 收入下降风险                        │   │
│  │   高利率       → 融资成本增加 → 投资减少                            │   │
│  │   低利率       → 融资成本降低 → 投资机会                            │   │
│  │   高通胀       → 成本上升     → 利润压力                            │   │
│  │   汇率波动     → 进出口成本变化 → 利润影响                           │   │
│  │   高失业率     → 消费下降     → 需求减少                            │   │
│  │   低失业率     → 消费旺盛     → 市场机会                            │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  分析问题：                                                                 │
│  ├── 经济处于什么周期？                                                     │
│  ├── 利率变化对融资有何影响？                                               │
│  ├── 通胀对成本和定价有何影响？                                             │
│  ├── 汇率波动如何影响业务？                                                 │
│  └── 消费者信心如何？                                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 三、PESTEL分析流程

```
PESTEL分析五步法：

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  Step 1: 确定分析范围                                                       │
│  ├── 分析对象：企业、产品、市场、项目                                        │
│  ├── 时间范围：短期、中期、长期                                             │
│  └── 地理范围：全球、国家、区域                                              │
│                                                                             │
│  Step 2: 收集环境信息                                                       │
│  ├── 政治信息：政府报告、政策文件                                            │
│  ├── 经济数据：统计局、央行、国际组织                                        │
│  ├── 社会趋势：研究报告、社会调查                                            │
│  ├── 技术动态：科技媒体、专利数据                                            │
│  ├── 环境法规：环保部门、国际公约                                           │
│  └── 法律法规：法律数据库、政府公告                                          │
│                                                                             │
│  Step 3: 识别关键因素                                                       │
│  ├── 列出各维度所有因素                                                     │
│  ├── 评估因素影响程度（高/中/低）                                            │
│  ├── 评估发生概率（高/中/低）                                                │
│  └── 筛选出关键因素                                                         │
│                                                                             │
│  Step 4: 分析影响                                                          │
│  ├── 区分机会和威胁                                                         │
│  ├── 评估影响的严重程度                                                     │
│  ├── 评估时间紧迫性                                                         │
│  └── 确定优先级                                                             │
│                                                                             │
│  Step 5: 制定应对策略                                                       │
│  ├── 机会：如何利用？                                                       │
│  ├── 威胁：如何规避？                                                       │
│  ├── 短期应对措施                                                           │
│  └── 长期战略调整                                                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### PESTEL核心价值

```
1. 全面性：覆盖宏观环境所有重要维度

3. 前瞻性：识别未来趋势和变化
4. 风险管理：识别外部威胁
5. 机会发现：发现市场机会
```
