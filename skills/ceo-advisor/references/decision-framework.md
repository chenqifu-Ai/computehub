# 决策框架与方法

## 决策类型

### 按重要性分类
| 类型 | 特点 | 示例 |
|------|------|------|
| 战略决策 | 长期、全局 | 进入新市场 |
| 战术决策 | 中期、部门 | 产品定价 |
| 业务决策 | 短期、日常 | 采购数量 |

### 按确定性分类
| 类型 | 不确定性 | 方法 |
|------|----------|------|
| 确定型决策 | 结果确定 | 线性规划 |
| 风险型决策 | 已知概率 | 决策树 |
| 不确定型决策 | 未知概率 | 最大最小准则 |

## 决策模型

### RAPID模型
| 角色 | 职责 | 说明 |
|------|------|------|
| R-Recommend | 建议 | 提出方案 |
| A-Agree | 同意 | 审批签字 |
| P-Perform | 执行 | 落实决策 |
| I-Input | 输入 | 提供信息 |
| D-Decide | 决定 | 最终拍板 |

### RACI矩阵
| 角色 | 含义 |
|------|------|
| R-Responsible | 负责执行 |
| A-Accountable | 对结果负责 |
| C-Consulted | 提供意见 |
| I-Informed | 知情 |

| A | 8 | 7 | 6 | 7.0 |

### 决策树分析
```
          ┌─ 好市场(0.6) → 利润100万
          │
进入市场 ─┤
          │
          └─ 差市场(0.4) → 亏损30万
          
期望值 = 0.6×100 + 0.4×(-30) = 48万
```

### SWOT决策
| 战略 | 组合 | 含义 |
|------|------|------|
| SO | 优势+机会 | 发挥优势抓机会 |

## 决策常见偏差

### 认知偏差
| 偏差 | 表现 | 应对 |
|------|------|------|
| 锚定效应 | 过度依赖首次信息 | 多方验证 |
| 确认偏差 | 偏好支持己见的信息 | 反向思考 |
| 沉没成本 | 难以放弃已投入 | 只看未来 |
| 过度自信 | 高估能力 | 数据说话 |

### 群体偏差
| 现象 | 表现 | 应对 |
|------|------|------|
| 群体思维 | 避免冲突、从众 | 鼓励异议 |
| 权威影响 | 附和领导 | 匿名决策 |
| 信息瀑布 | 后面人跟前面 | 独立思考 |

## 决策流程优化

### 六步法
1. **明确问题**：定义决策目标
2. **收集信息**：获取相关数据
3. **生成方案**：头脑风暴
4. **评估方案**：优缺点分析
5. **做出决策**：选择最优方案
6. **执行复盘**：跟踪结果
