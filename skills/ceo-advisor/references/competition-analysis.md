# 竞争分析框架

## 波特五力模型

### 五大竞争力量
```
┌─────────────────────────────────────────┐
│           潜在进入者威胁                   │
│         （进入壁垒分析）                   │
├─────────────────────────────────────────┤
│  供应商    │    行业竞争    │    买方      │
│  议价能力   │    （现有竞争）  │   议价能力   │
├─────────────────────────────────────────┤
│           替代品威胁                      │
│         （替代品分析）                     │
└─────────────────────────────────────────┘
```

### 各力量分析要点

| 力量 | 分析维度 | 小微企业关注点 |
|------|----------|---------------|
| 现有竞争 | 竞争对手数量、行业增长率、退出壁垒 | 本地竞品、价格战风险 |
| 潜在进入 | 资本需求、政策壁垒、渠道壁垒 | 低门槛意味着高威胁 |
| 替代品 | 替代品性价比、转换成本 | 线上替代线下、自动化替代人工 |
| 供应商议价 | 供应商集中度、替代供应商 | 单一供应商风险 |
| 买方议价 | 客户集中度、价格敏感度 | 大客户依赖风险 |

## 竞争对手分析

### 分析维度
1. **身份识别**：竞争对手是谁？
2. **战略分析**：他们的战略是什么？
3. **能力评估**：优势劣势是什么？
4. **反应预测**：他们会如何反应？

### 竞争情报收集
- 公开信息：官网、年报、新闻
- 用户反馈：评价、投诉、口碑
- 市场调研：价格、服务、渠道
- 离职员工：战略、运营细节

## 差异化策略

### 小微企业差异化路径
| 路径 | 说明 | 案例 |
|------|------|------|
| 服务差异化 | 更好、更快、更贴心 | 24小时响应、上门服务 |
| 区域差异化 | 本地化优势 | 本地人脉、快速响应 |
| 细分专业化 | 聚焦细分领域 | 只做餐饮企业服务 |
| 价格差异化 | 成本优势 | 低运营成本、高性价比 |

## 竞争定位

### 定位三角
```
         差异化
           △
          ╱  ╲
         ╱    ╲
        ╱      ╲
       ╱        ╲
      ╱          ╲
     ▼────────────▼
  成本领先      聚焦
```

### 小微企业定位建议
- **避免正面竞争**：不与大企业硬刚
- **聚焦细分市场**：做深做透一个领域
- **建立护城河**：客户关系、本地资源、专业能力
- **快速迭代**：灵活调整策略
