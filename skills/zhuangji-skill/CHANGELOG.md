# 📋 装机技能更新日志

## v1.2.0 - 2026-04-08
### 新增功能
- **交互式版本选择**: 在OpenClaw安装阶段添加用户交互界面
- **版本选择菜单**: 提供4个版本选项供用户选择
- **默认值支持**: 回车默认选择稳定版，保持自动化流程
- **实时反馈**: 显示用户选择的具体版本信息

### 技术改进
- 使用 `read -p` 实现用户交互输入
- 设置默认值 `${VERSION_CHOICE:-1}` 确保流程连续性
- 完善版本选择逻辑和错误处理
- 更新详细文档说明交互特性

### 文件更新
