### 协同学习场景

**场景1：新客户咨询**
- CEO顾问：分析客户需求、制定服务方案
- 财务专家：财税可行性分析
- HR专家：人力资源配套建议
- 营销专家：市场推广建议
- 法律顾问：合规风险评估

**场景2：复杂问题解决**
- 主导专家提出问题
- 其他专家从各自领域补充
- 综合各方意见形成方案

**场景3：知识交叉学习**
- 每周一次跨领域学习
- 例如：财务专家学习基础合同法
- 例如：HR专家学习基础财税知识

## 知识共享

### 共享知识库
位置：`/root/.openclaw/workspace/skills/shared-knowledge/`

目录结构：
```
shared-knowledge/
├── cases/           # 服务案例库
├── templates/       # 常用模板
├── policies/        # 政策法规库
└── insights/        # 行业洞察
```

### 知识同步机制
1. 各专家学习成果存入个人知识库
2. 高价值知识提取到共享知识库
3. 其他专家可引用共享知识
