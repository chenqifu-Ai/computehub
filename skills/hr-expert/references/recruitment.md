# 招聘管理详解

## 招聘流程

### 标准流程
```
需求确认 → 渠道发布 → 简历筛选 → 面试安排 
    ↓
录用决策 → 背景调查 → offer谈判 → 入职办理
```

### 招聘渠道效果对比
| 渠道 | 成本 | 效果 | 适用岗位 |
|------|------|------|---------|
| 招聘网站 | 中 | 高 | 全部 |
| 猎头 | 高 | 高 | 高管、技术 |
| 内推 | 低 | 高 | 全部 |
| 校园 | 低 | 中 | 初级岗位 |
| 社交媒体 | 低 | 中 | 年轻岗位 |
| 猎聘/BOSS | 中 | 高 | 中高端 |

## 面试技巧

### 结构化面试问题
| 维度 | 问题示例 | 评估要点 |
|------|---------|---------|
| 专业能力 | 请介绍一个项目经验 | 专业深度 |
| 解决问题 | 遇到过什么困难？怎么解决？ | 思维方式 |
| 团队协作 | 如何处理团队冲突？ | 协作能力 |
| 学习能力 | 最近学了什么新技能？ | 自驱力 |
| 价值观 | 为什么选择我们公司？ | 匹配度 |

### STAR面试法
- **S-Situation**: 情境背景
- **T-Task**: 任务目标
- **A-Action**: 行动措施
- **R-Result**: 结果成果

## 薪酬谈判

### 谈判原则
1. 了解市场行情
2. 考虑全面薪酬（工资+奖金+福利）
3. 预留谈判空间
4. 关注候选人期望

### 薪酬结构建议
| 组成 | 比例 | 说明 |
|------|------|------|
| 基本工资 | 60-70% | 固定收入 |
| 绩效奖金 | 20-30% | 浮动收入 |
| 年终奖 | 10-20% | 年度激励 |
| 福利补贴 | 5-10% | 五险一金、餐补 |
