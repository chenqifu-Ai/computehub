# 劳动法规：劳动合同法实务

## 一、劳动合同法概述

### 1. 立法目的
```markdown
劳动合同法立法目的：
- 完善劳动合同制度
- 明确劳动合同双方权利义务
- 保护劳动者合法权益
- 构建和谐稳定劳动关系
```

### 2. 适用范围
```markdown
适用主体：
- 中华人民共和国境内的企业
- 个体经济组织
- 民办非企业单位
- 国家机关、事业单位、社会团体

适用人员：
- 与用人单位建立劳动关系的劳动者
- 包括全职、兼职、临时工
- 不包括公务员、事业单位编制人员
```

### 3. 基本原则
```markdown
合法原则：
- 劳动合同内容必须合法
- 劳动合同形式必须合法
- 劳动合同程序必须合法

公平原则：
- 双方权利义务对等
- 不得显失公平

平等自愿原则：
- 双方地位平等
- 意思表示真实

协商一致原则：
- 劳动合同内容协商确定
- 变更劳动合同需协商一致

诚实信用原则：
- 如实告知义务
- 遵守承诺
```

## 二、劳动合同的订立

### 1. 劳动合同形式
```markdown
书面形式要求：
- 建立劳动关系应当订立书面劳动合同
- 口头合同的法律效力

未签订书面合同的后果：
- 一个月内：用人单位无过错
- 一个月至一年：支付双倍工资
- 超过一年：视为无固定期限劳动合同

书面劳动合同内容：
- 必备条款
- 约定条款
```

### 2. 劳动合同期限
```markdown
固定期限劳动合同：
- 约定终止时间
- 可以续签
- 期满终止

无固定期限劳动合同：
- 没有约定终止时间
- 除非法定情形，不得终止
- 符合条件应当签订

无固定期限劳动合同订立条件：
- 连续工作满十年
- 连续订立二次固定期限劳动合同
- 用人单位自用工之日起满一年不与劳动者订立书面劳动合同

以完成一定工作任务为期限的劳动合同：
- 以任务完成为终止条件
- 任务完成即终止
```

### 3. 试用期
```markdown
试用期期限：

劳动合同期限三个月以上不满一年：
- 试用期不得超过一个月

劳动合同期限一年以上不满三年：
- 试用期不得超过二个月

三年以上固定期限和无固定期限劳动合同：
- 试用期不得超过六个月

禁止规定：
- 同一用人单位与同一劳动者只能约定一次试用期
- 以完成一定工作任务为期限的劳动合同不得约定试用期
- 劳动合同期限不满三个月的不得约定试用期

试用期工资：
- 不得低于本单位相同岗位最低档工资的80%
- 不得低于劳动合同约定工资的80%
- 不得低于用人单位所在地的最低工资标准

试用期解除：
- 在试用期间被证明不符合录用条件的
- 用人单位可以解除劳动合同
- 应当说明理由
```

### 4. 劳动合同必备条款
```markdown
法定必备条款：
- 用人单位名称、住所、法定代表人
- 劳动者姓名、住址、身份证号码
- 劳动合同期限
- 工作内容和工作地点
- 工作时间和休息休假
- 劳动报酬
- 社会保险
- 劳动保护、劳动条件和职业危害防护
- 法律法规规定的其他事项

约定条款：
- 试用期
- 培训
- 保守秘密
- 补充保险
- 福利待遇
```

### 5. 劳动合同效力
```markdown
生效时间：
- 双方签字或盖章生效
- 约定生效条件的，条件成就时生效

无效劳动合同：

无效情形：
- 以欺诈、胁迫手段订立
- 用人单位免除法定责任
- 排除劳动者权利
- 违反法律法规强制性规定

无效后果：
- 自始无效
- 部分无效不影响其他部分
- 劳动者已付出劳动的，用人单位应当支付劳动报酬
```

## 三、劳动合同的履行与变更

### 1. 劳动合同履行
```markdown
履行原则：
- 全面履行
- 诚实信用履行

用人单位义务：
- 按照合同约定提供劳动条件
- 及时足额支付劳动报酬
- 缴纳社会保险
- 提供劳动保护

劳动者义务：
- 完成劳动任务
- 提高职业技能
- 执行劳动安全卫生规程
- 遵守劳动纪律和职业道德

工资支付：
- 按月支付
- 不得克扣
- 不得拖欠
- 不得低于最低工资标准
```

### 2. 劳动合同变更
```markdown
变更原则：
- 协商一致
- 书面形式
- 变更后的劳动合同文本由双方各执一份

变更情形：
- 双方协商变更
- 客观情况发生重大变化
- 企业转制、改制

变更程序：
- 提出变更要求
- 协商变更内容
- 达成书面协议
- 变更后双方签字盖章

调岗调薪：
- 双方协商一致可以调岗调薪
- 劳动者不能胜任工作，可以调岗
- 调岗合理性判断
```

## 四、劳动合同的解除与终止

### 1. 协商解除
```markdown
双方协商一致：
- 可以解除劳动合同
- 用人单位提出协商解除的，应当支付经济补偿
- 劳动者提出协商解除的，用人单位不支付经济补偿
```

### 2. 劳动者单方解除
```markdown
提前通知解除：

试用期：
- 提前三日通知用人单位

试用期后：
- 提前三十日以书面形式通知用人单位

随时通知解除（劳动者权利）：

用人单位过错情形：
- 未按照劳动合同约定提供劳动保护
- 未及时足额支付劳动报酬
- 未依法缴纳社会保险
- 规章制度违反法律法规
- 以欺诈、胁迫手段订立劳动合同

立即解除（无需通知）：

用人单位严重过错：
- 暴力、威胁强迫劳动
- 违章指挥、强令冒险作业
- 危及人身安全
```

### 3. 用人单位单方解除
```markdown
过失性辞退（无需支付经济补偿）：

劳动者过错情形：
- 在试用期间被证明不符合录用条件
- 严重违反用人单位规章制度
- 严重失职，营私舞弊，给用人单位造成重大损害
- 同时与其他用人单位建立劳动关系，影响完成本单位工作
- 以欺诈手段使对方违背真实意思订立劳动合同
- 被依法追究刑事责任

非过失性辞退（需提前三十日通知或支付一个月工资）：

劳动者无过错情形：
- 劳动者患病或非因工负伤，医疗期满不能从事原工作
- 劳动者不能胜任工作，培训或调岗后仍不能胜任
- 客观情况发生重大变化，劳动合同无法履行，协商不成

经济性裁员：
- 依照企业破产法规定进行重整
- 生产经营发生严重困难
- 企业转产、重大技术革新，经变更劳动合同后仍需裁员
- 其他客观经济情况发生重大变化

裁员限制：
- 一次性裁员二十人以上或裁减不足二十人但占职工总数百分之十以上
- 提前三十日向工会或全体职工说明情况
- 听取工会或职工意见
- 向劳动行政部门报告

优先留用：
- 与本单位订立较长期限的固定期限劳动合同
- 与本单位订立无固定期限劳动合同
- 家庭无其他就业人员，有需要扶养的老人或未成年人
```

### 4. 不得解除情形
```markdown
禁止解除情形：

职业病相关：
- 从事接触职业病危害作业未进行离岗前职业健康检查
- 疑似职业病病人在诊断或医学观察期间

职业病与工伤：
- 在本单位患职业病或因工负伤并被确认丧失或部分丧失劳动能力

医疗期：
- 患病或非因工负伤，在规定的医疗期内

女职工保护：
- 女职工在孕期、产期、哺乳期

其他：
- 在本单位连续工作满十五年，且距法定退休年龄不足五年
- 法律法规规定的其他情形
```

### 5. 劳动合同终止
```markdown
终止情形：

期满终止：
- 固定期限劳动合同期满
- 以完成一定工作任务为期限的劳动合同，任务完成

主体消失：
- 劳动者开始依法享受基本养老保险待遇
- 劳动者死亡，或被人民法院宣告死亡或宣告失踪
- 用人单位被依法宣告破产
- 用人单位被吊销营业执照、责令关闭、撤销
- 用人单位决定提前解散

终止续延：
- 有禁止解除情形的，劳动合同应当续延至相应情形消失时终止
- 丧失或部分丧失劳动能力的，按国家工伤保险规定执行
```

## 五、经济补偿

### 1. 经济补偿情形
```markdown
用人单位支付经济补偿：

劳动者解除劳动合同：
- 用人单位过错，劳动者随时解除

用人单位解除劳动合同：
- 协商解除（用人单位提出）
- 非过失性辞退
- 经济性裁员

劳动合同终止：
- 固定期限劳动合同期满终止（用人单位不维持或提高条件续订）
- 用人单位破产、吊销、关闭、解散

其他：
- 以完成一定工作任务为期限的劳动合同因任务完成终止
```

### 2. 经济补偿标准
```markdown
计算方法：
- 按劳动者在本单位工作年限
- 每满一年支付一个月工资
- 六个月以上不满一年的按一年计算
- 不满六个月的支付半个月工资

工资标准：
- 劳动合同解除或终止前十二个月的平均工资
- 包括计时工资、计件工资、奖金、津贴、补贴等

封顶规定：
- 月工资高于本地区上年度职工月平均工资三倍的
- 按职工月平均工资三倍支付
- 支付年限最高不超过十二年

特殊情形：
- 违法解除或终止劳动合同
- 应当支付赔偿金
- 赔偿金为经济补偿标准的二倍
```

### 3. 经济补偿计算
```markdown
计算公式：
经济补偿 = 工作年限 × 月工资

示例计算：

工作5年，月平均工资8000元：
- 经济补偿 = 5 × 8000 = 40000元

工作3年8个月，月平均工资6000元：
- 经济补偿 = 4 × 6000 = 24000元（按4年计算）

工作6个月，月平均工资5000元：
- 经济补偿 = 0.5 × 5000 = 2500元

高工资封顶计算：
- 月工资30000元，当地平均工资8000元
- 封顶工资 = 8000 × 3 = 24000元
- 工作10年
- 经济补偿 = 10 × 24000 = 240000元（封顶12年为120000元）
```

## 六、劳务派遣

### 1. 劳务派遣概念
```markdown
定义：
- 劳务派遣单位与劳动者订立劳动合同
- 将劳动者派遣到用工单位工作
- 用工单位对劳动者进行管理使用

三方关系：
- 劳务派遣单位：用人单位
- 用工单位：实际使用劳动者
- 劳动者：被派遣劳动者

适用岗位：
- 临时性岗位：存续期不超过六个月
- 辅助性岗位：非主营业务岗位
- 替代性岗位：替代脱产学习、休假劳动者
```

### 2. 劳务派遣规定
```markdown
派遣单位要求：
- 注册资本不得少于二百万元
- 有与开展业务相适应的固定经营场所和设施
- 有符合法律规定的劳务派遣管理制度
- 取得劳务派遣经营许可证

劳动合同：
- 应当与被派遣劳动者订立二年以上固定期限劳动合同
- 按月支付劳动报酬
- 在无工作期间，按照当地最低工资标准支付报酬

用工单位义务：
- 执行国家劳动标准
- 提供劳动条件和劳动保护
- 告知工作要求和劳动报酬
- 支付加班费、绩效奖金
- 提供培训
- 实行同工同酬

禁止规定：
- 不得设立劳务派遣单位向本单位派遣劳动者
- 不得将非临时性、辅助性、替代性岗位实施劳务派遣
- 劳务派遣用工不得超过其用工总量的10%
```

## 七、违反劳动合同法的法律责任

### 1. 用人单位违法责任
```markdown
规章制度违法：
- 给劳动者造成损害的，应当承担赔偿责任

劳动合同违法：
- 未签订书面劳动合同：支付双倍工资
- 违法约定试用期：支付赔偿金
- 违法解除或终止：支付赔偿金

工资违法：
- 克扣或拖欠工资：支付赔偿金
- 支付工资低于最低工资标准：补足差额

社会保险违法：
- 未缴纳社会保险：补缴
- 造成损失：赔偿

其他违法：
- 招用未解除劳动合同的劳动者：连带赔偿责任
- 劳务派遣违法：行政处罚
```

### 2. 劳动者违法责任
```markdown
违反劳动合同：
- 给用人单位造成损失的，应当承担赔偿责任

违反保密义务：
- 支付违约金
- 赔偿损失

违反竞业限制：
- 支付违约金
- 赔偿损失

违反培训协议：
- 按约定支付违约金
```

### 3. 赔偿责任
```markdown
用人单位赔偿情形：
- 解除或终止劳动合同未支付经济补偿
- 违法解除或终止劳动合同
- 规章制度违法造成损害
- 劳动合同无效造成损害
- 劳务派遣违法造成损害

劳动者赔偿情形：
- 违反劳动合同约定
- 违反保密义务
- 违反竞业限制约定
- 违反服务期约定
- 造成用人单位损失
```

## 八、劳动争议处理

### 1. 劳动争议范围
```markdown
劳动争议类型：
- 确认劳动关系争议
- 订立、履行、变更、解除、终止劳动合同争议
- 劳动报酬、工伤医疗费、经济补偿争议
-工作时间、休息休假、社会保险争议
- 其他劳动争议

不属于劳动争议：
- 劳动者请求社会保险经办机构发放社会保险金
- 劳动者与用人单位因住房制度改革发生的纠纷
- 劳动者对劳动能力鉴定委员会的鉴定结论不服
- 家庭与家政服务人员与家庭发生的纠纷
- 个体工匠与帮工、学徒之间的纠纷
```

### 2. 劳动争议处理程序
```markdown
协商：
- 双方自行协商
- 协商不成可以申请调解

调解：
- 企业劳动争议调解委员会
- 依法设立的基层人民调解组织
- 乡镇、街道劳动争议调解组织

仲裁：
- 劳动争议仲裁委员会
- 劳动争议仲裁前置
- 申请仲裁时效为一年

诉讼：
- 对仲裁裁决不服
- 收到仲裁裁决书之日起十五日内向人民法院提起诉讼
```

### 3. 举证责任
```markdown
举证责任分配：

劳动者举证：
- 劳动关系存在
- 劳动合同内容
- 工资数额
- 工作年限

用人单位举证：
- 工资支付凭证
- 考勤记录
- 规章制度
- 解除劳动合同的理由
- 劳动者过失的事实
- 其他用人单位掌握管理的证据

特殊情形：
- 用人单位作出开除、除名、辞退、解除劳动合同决定
- 用人单位负举证责任
```
