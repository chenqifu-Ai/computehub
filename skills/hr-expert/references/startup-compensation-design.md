# 创业公司薪酬设计指南

## 薪酬设计原则

### 基本原则
| 原则 | 说明 |
|------|------|
| 市场竞争力 | 薪酬水平与市场相当 |
| 内部公平 | 同类岗位薪酬相近 |
| 激励导向 | 绩效与薪酬挂钩 |
| 成本可控 | 符合公司财务状况 |
| 灵活性 | 适应不同阶段需求 |

## 薪酬结构

### 典型薪酬组成
```
总薪酬 = 基本工资 + 绩效奖金 + 股权激励 + 福利补贴

├── 基本工资：固定收入
├── 绩效奖金：浮动收入
├── 股权激励：长期激励
└── 福利补贴：社保、公积金、餐补等
```

### 各部分比例
| 岗位类型 | 基本工资 | 绩效奖金 | 股权激励 |
|----------|----------|----------|----------|
| 技术岗 | 70-80% | 10-20% | 10-20% |
| 销售岗 | 40-60% | 30-50% | 5-10% |
| 管理岗 | 60-70% | 20-30% | 10-20% |
| 行政岗 | 80-90% | 5-10% | 5-10% |

### 岗位等级体系
| 等级 | 典型岗位 | 薪酬范围 |
|------|----------|----------|
| P1 | 实习生/助理 | 3-6K |
| P2 | 初级专员 | 6-10K |
| P3 | 中级专员 | 10-15K |
| P4 | 高级专员 | 15-25K |

| P6 | 经理 | 30-50K |
| P7 | 总监 | 40-70K |
| P8 | VP | 60-100K |

### 薪酬宽带
| 宽带设置 | 说明 |
|----------|------|
| 宽带幅度 | 同级别薪酬差异可达30-50% |
| 重叠度 | 相邻级别薪酬有20-30%重叠 |
| 调整空间 | 晋升不调薪vs调薪不晋升 |

### 定薪流程
```
1. 岗位评估 → 确定岗位等级
2. 市场调研 → 了解市场薪酬水平
3. 候选人评估 → 能力、经验匹配度
4. 确定薪酬区间 → 根据预算
5. 薪酬谈判 → 双方协商
6. 正式录用 → 发Offer
```

## 绩效奖金设计

### 绩效考核体系
| 等级 | 描述 | 比例 | 奖金系数 |
|------|------|------|----------|
| S | 超出预期 | 10% | 1.3-1.5 |
| A | 满足预期 | 20% | 1.1-1.2 |
| B | 基本满足 | 50% | 1.0 |
| C | 需改进 | 15% | 0.7-0.9 |
| D | 不合格 | 5% | 0 |

### 奖金计算公式
```
绩效奖金 = 目标奖金 × 绩效系数 × 公司系数

其中：
- 目标奖金：基本工资 × 奖金比例
- 绩效系数：个人绩效考核结果
- 公司系数：公司整体业绩达成率
```

### 绩效考核周期
| 周期 | 适用岗位 | 说明 |
|------|----------|------|
| 月度 | 销售、运营 | 短期业绩导向 |
| 季度 | 技术、产品 | 项目交付周期 |
| 半年 | 行政、财务 | 稳定岗位 |
| 年度 | 全员 | 年终奖 |

## 股权激励设计

### 股权激励类型
| 类型 | 说明 | 适用阶段 |
|------|------|----------|
| 期权(Option) | 未来购买权 | 早期创业公司 |
| 限制性股票(RSU) | 免费授予但受限 | 成熟期公司 |
| 虚拟股票 | 享受分红无所有权 | 不想稀释股权 |
| 股票增值权 | 分享股价增值收益 | 上市公司 |

### 期权分配
| 岗位 | 典型期权池占比 | 行权期限 |
|------|----------------|----------|
| 联合创始人 | 10-30% | 无限制 |
| CXO | 1-3% | 4年归属 |
| 总监 | 0.5-1% | 4年归属 |
| 经理 | 0.1-0.5% | 4年归属 |
| 核心员工 | 0.01-0.1% | 4年归属 |

### 期权条款
| 条款 | 说明 |
|------|------|
| 行权价 | 通常为当前估值的一定折扣 |
| 归属期 | 通常4年，1年cliff |
| 行权窗口 | 离职后90天内行权 |
| 回购权 | 公司可优先回购 |

### 期权授予流程
```
1. 董事会批准期权池
2. 确定授予对象和数量
3. 签署期权协议
4. 办理工商变更（如需）
5. 员工签署授予协议
6. 定期归属确认
```

## 福利补贴设计

### 法定福利
| 福利 | 公司缴纳 | 个人缴纳 |
|------|----------|----------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 9.5% | 2% |
| 失业保险 | 0.5% | 0.5% |
| 工伤保险 | 0.2-1.9% | - |
| 生育保险 | 0.8% | - |
| 公积金 | 5-12% | 5-12% |

### 公司福利
| 福利类型 | 内容 | 成本估算 |
|----------|------|----------|
| 餐饮补贴 | 午餐补贴 | 20-30元/天 |
| 交通补贴 | 打车报销 | 200-500元/月 |
| 通讯补贴 | 话费报销 | 100-300元/月 |
| 健康体检 | 年度体检 | 500-2000元/年 |
| 团建活动 | 团建预算 | 200-500元/人/季 |
| 节日福利 | 礼品/礼金 | 500-2000元/年 |

### 福利成本
```
员工成本 = 基本工资 × (1 + 社保公积金比例 + 福利比例)

示例：
基本工资10000元
社保公积金比例：公司35%左右
福利比例：5%左右
总成本 = 10000 × (1 + 0.35 + 0.05) = 14000元

即：员工成本约为基本工资的1.4倍
```

## 常见问题

### Q1: 如何确定创业公司薪酬？
```
步骤：
1. 了解市场薪酬（招聘网站、人脉）
2. 评估公司支付能力
3. 设计薪酬结构（现金+期权）
4. 预留调整空间
5. 建立调薪机制
```

### Q2: 期权和现金怎么平衡？
```
经验公式：
现金薪酬缺口 = 市场薪酬 - 实际薪酬
期权价值 = 缺口 × 风险系数 × 年限

示例：
市场薪酬30K，实际给15K
缺口15K/月，风险系数2，年限4年
期权价值 = 15K × 12 × 2 × 4 = 144万
相当于公司估值0.1%，期权1-2%
```

### Q3: 如何防止薪酬倒挂？
```
方法：
1. 定期市场调研
2. 建立调薪机制
3. 薪酬宽带设计
4. 核心人才专项调整
```

### Q4: 销售人员薪酬怎么设计？
```
典型结构：
底薪：30-50%（保障基本生活）
提成：50-70%（业绩挂钩）

提成设计：
- 阶梯式：销售额越高，提成比例越高
- 分段式：基本目标+超额目标
- 团队制：个人+团队双提成
```

## 总结

### 薪酬设计要点
1. **市场导向**：参考市场薪酬水平
2. **结构合理**：基本+绩效+股权+福利
3. **激励有效**：绩效与奖金挂钩
4. **成本可控**：符合公司财务状况
5. **公平透明**：制度公开，个人保密
