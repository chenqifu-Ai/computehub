# 员工入职流程

## 入职准备

### 入职前准备
| 步骤 | 内容 | 责任人 |
|------|------|--------|
| 发送offer | 薪资、入职时间、地点 | HR |
| 准备材料 | 入职清单、表格 | HR |
| 安排工位 | 电脑、账号、工位 | 行政 |
| 通知部门 | 入职时间、培训安排 | HR |

### 入职材料清单
| 材料 | 说明 |
|------|------|
| 身份证复印件 | 正反面 |
| 学历证书 | 原件+复印件 |
| 离职证明 | 前单位离职证明 |
| 体检报告 | 近期体检报告 |
| 银行卡 | 工资卡 |
| 照片 | 一寸照片 |

## 入职当天

### 流程安排
| 时间 | 事项 | 负责人 |
|------|------|--------|
| 9:00 | 签到、材料审核 | HR |
| 9:30 | 填写入职表 | HR |
| 10:00 | 领取工牌、办公用品 | 行政 |
| 10:30 | 部门介绍 | 部门负责人 |
| 11:00 | 参观公司 | HR |
| 12:00 | 欢迎午餐 | 部门同事 |
| 14:00 | 系统培训 | IT |
| 15:00 | 岗位培训 | 部门负责人 |
| 17:00 | 入职确认 | HR |

### 入职表格
- 入职登记表
- 劳动合同
- 保密协议
- 员工手册签收
- 社保公积金确认

## 入职培训

### 培训内容
| 模块 | 内容 | 时长 |
|------|------|------|
| 公司介绍 | 历史、文化、业务 | 1小时 |
| 制度培训 | 考勤、报销、请假 | 1小时 |
| 系统培训 | OA、邮件、协作工具 | 1小时 |
| 安全培训 | 信息安全、保密 | 30分钟 |
| 岗位培训 | 部门流程、工作内容 | 2小时 |

### 培训考核
- 新员工培训考试
- 培训效果评估
- 培训记录归档

## 试用期管理

### 试用期规定
| 合同期限 | 试用期 |
|----------|--------|
| 3个月-1年 | 最多1个月 |
| 1年-3年 | 最多2个月 |
| 3年以上 | 最多6个月 |

### 考核要点
| 维度 | 权重 | 内容 |
|------|------|------|
| 工作态度 | 30% | 积极性、责任心 |
| 专业能力 | 40% | 技能、效率 |
| 团队协作 | 20% | 沟通、配合 |
| 学习能力 | 10% | 进步速度 |

### 转正流程
1. 提前15天提交转正申请
2. 部门负责人评估
3. HR审核
4. 领导审批
5. 转正谈话
6. 签订正式合同

## 入职常见问题

| 问题 | 解决方案 |
|------|----------|
| 材料不齐 | 先入职后补齐 |
| 无法按时入职 | 提前沟通改期 |
| 试用期不合格 | 延长或辞退 |
| 适应困难 | 安排导师带教 |
