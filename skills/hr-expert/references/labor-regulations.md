# 劳动法规要点

## 劳动法基本原则

### 核心原则
1. 劳动者权益优先保护
2. 劳动关系契约化
3. 劳动关系协调机制
4. 劳动标准法定化

## 劳动合同法规

### 合同签订
| 情形 | 要求 |
|------|------|
| 用工之日起 | 一个月内签订书面合同 |
| 未签订合同 | 支付双倍工资（超一个月） |
| 无固定期限合同 | 连续工作满10年或连续签两次固定期 |

### 试用期规定
| 合同期限 | 试用期 |
|----------|--------|
| 不足3个月 | 不得约定 |
| 3个月-1年 | 不得超过1个月 |
| 1年-3年 | 不得超过2个月 |
| 3年以上或无固定 | 不得超过6个月 |

### 违法解除后果
- 继续履行合同
- 或支付赔偿金（双倍经济补偿）

## 工资支付法规

### 工资构成
| 项目 | 说明 |
|------|------|
| 基本工资 | 固定报酬 |
| 绩效工资 | 浮动报酬 |
| 津贴补贴 | 特殊岗位补偿 |
| 加班工资 | 法定计算 |

### 加班工资标准
| 时间 | 计算基数 |
|------|----------|
| 工作日延时 | 150% |
| 休息日 | 200%（或调休） |
| 法定节假日 | 300% |

### 最低工资
- 各地区标准不同
- 加班费、津贴不计入
- 试用期不得低于80%

## 工时与休假

### 标准工时
- 每日8小时
- 每周40小时
- 每周至少休息1天

### 特殊工时
| 类型 | 适用 | 审批 |
|------|------|------|
| 综合计算工时 | 季节性行业 | 劳动部门审批 |
| 不定时工作制 | 高管、销售 | 劳动部门审批 |

### 法定假期
| 类型 | 天数 |
|------|------|
| 年休假 | 5-15天（按工龄） |
| 婚假 | 3天（各地有差异） |
| 丧假 | 1-3天 |
| 产假 | 98天+各地奖励假 |
| 陪产假 | 7-30天（各地不同） |

## 社会保险法规

### 五险一金
| 险种 | 单位比例 | 个人比例 |
|------|----------|----------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 9%+ | 2% |
| 失业保险 | 0.5-2% | 0.5-1% |
| 工伤保险 | 0.2-1.9% | 0 |
| 生育保险 | 0.5-1% | 0 |
| 住房公积金 | 5-12% | 5-12% |

### 缴费基数
- 下限：社平工资60%
- 上限：社平工资300%
- 新员工：首月工资
