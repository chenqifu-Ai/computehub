# 劳动法规合规要点

## 一、劳动法律法规体系

### 1. 法律层级
| 层级 | 名称 | 效力 |
|------|------|------|
| 法律 | 《劳动法》《劳动合同法》《社会保险法》 | 最高 |
| 行政法规 | 《劳动合同法实施条例》等 | 次高 |
| 部门规章 | 劳动部规章、规定 | 中等 |
| 地方性法规 | 各省劳动合同条例 | 区域性 |

### 2. 核心法律
- **《劳动法》**：劳动领域基本法
- **《劳动合同法》**：劳动合同专门法
- **《劳动争议调解仲裁法》**：争议处理程序法
- **《社会保险法》**：社保权益保障法

## 二、劳动合同合规

### 1. 合同签订

#### 签订时限
- **新员工**：入职一个月内签订
- **未签合同**：超过一个月不满一年，支付双倍工资
- **满一年未签**：视为无固定期限合同

#### 合同必备条款
```markdown
1. 用人单位信息（名称、住所、法定代表人）
2. 劳动者信息（姓名、身份证号）
3. 合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护、条件和职业危害防护
```

#### 试用期规定
| 合同期限 | 试用期上限 | 备注 |
|----------|------------|------|
| 不满3个月 | 不得约定 | - |
| 3个月~1年 | 1个月 | - |
| 1年~3年 | 2个月 | - |
| 3年以上/无固定 | 6个月 | - |

**注意**：试用期工资不得低于转正工资的80%

### 2. 合同变更
- **原则**：协商一致，书面形式
- **禁止**：单方强制变更
- **例外**：客观情况发生重大变化，可协商变更

### 3. 合同解除

#### 用人单位解除（合法情形）

**即时解除（无需补偿）：**
- 试用期不符合录用条件
- 严重违反规章制度
- 严重失职、营私舞弊，造成重大损害
- 同时与其他单位建立劳动关系，拒不改正
- 以欺诈手段订立合同
- 被追究刑事责任

**预告解除（需支付补偿）：**
- 劳动者患病或非因工负伤，医疗期满不能从事原工作
- 劳动者不能胜任工作，经培训或调岗仍不能胜任
- 客观情况发生重大变化，合同无法履行

**经济性裁员：**
- 需裁减20人以上或占职工总数10%以上
- 提前30日向工会或全体职工说明情况
- 需向劳动行政部门报告

#### 劳动者解除
- **提前30日书面通知**：正式员工
- **提前3日通知**：试用期员工
- **即时解除**：用人单位违法、未及时足额支付报酬等

### 4. 经济补偿金

#### 计算标准
```
经济补偿金 = 工作年限 × 月工资

- 工作年限：每满一年支付一个月工资
- 不满半年：按半年计算
- 半年以上不满一年：按一年计算
- 月工资：解除合同前12个月平均工资
```

#### 限制上限
- 月工资 > 当地上年度职工月平均工资3倍
- 年限最高不超过12年

## 三、工资支付合规

### 1. 工资构成
```markdown
工资总额 = 基本工资 + 绩效工资 + 津贴补贴 + 加班费 + 奖金

不计入工资：
- 社保公积金单位缴纳部分
- 福利费
- 劳动保护费
```

### 2. 支付要求
- **周期**：至少每月支付一次
- **形式**：货币形式，不得以实物代替
- **记录**：书面记录支付项目、数额、时间
- **凭证**：向劳动者提供个人工资清单

### 3. 最低工资
- 不得低于当地最低工资标准
- 加班费、夜班津贴等不计入最低工资

### 4. 加班工资
| 时间 | 标准 |
|------|------|
| 工作日延长 | 150% |
| 休息日安排工作 | 200%（或安排补休） |
| 法定节假日 | 300% |

## 四、社会保险合规

### 1. 五险一金
| 险种 | 单位比例 | 个人比例 | 备注 |
|------|----------|----------|------|
| 养老保险 | 16% | 8% | - |
| 医疗保险 | 9.5% | 2% | 含生育 |
| 失业保险 | 0.5% | 0.5% | - |
| 工伤保险 | 0.2-1.9% | - | 行业差异 |
| 住房公积金 | 5-12% | 5-12% | - |

### 2. 缴费基数
- **下限**：当地社平工资60%
- **上限**：当地社平工资300%
- **新员工**：首月工资为基数

### 3. 未缴/少缴风险
- 劳动者可随时解除合同并要求补偿
- 社保部门责令补缴+滞纳金（每日万分之五）
- 严重者列入失信名单

## 五、工作时间与休息休假

### 1. 标准工时
- **每日**：8小时
- **每周**：40小时
- **延长限制**：每日不超过1小时，特殊情况不超过3小时，每月不超过36小时

### 2. 休息休假
| 类型 | 天数 | 备注 |
|------|------|------|
| 休息日 | 每周2天 | 可调休 |
| 法定节假日 | 11天 | 不可调休 |
| 年休假 | 5-15天 | 工作年限决定 |
| 病假 | - | 医疗期内 |
| 婚假 | 3天+各地增补 | - |
| 产假 | 98天+各地增补 | - |
| 陪产假 | 15天+各地增补 | - |

### 3. 年休假规定
| 工作年限 | 年休假天数 |
|----------|------------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

**未休年休假补偿**：300%日工资

## 六、劳动争议处理

### 1. 处理渠道
```markdown
协商 → 调解 → 仲裁 → 诉讼

- 协商：双方自行协商
- 调解：企业调解委员会、基层调解组织
- 仲裁：劳动争议仲裁委员会（前置程序）
- 诉讼：人民法院
```

### 2. 仲裁时效
- **一般时效**：知道权利被侵害之日起1年
- **劳动报酬争议**：劳动关系存续期间不受时效限制
- **劳动关系终止**：终止之日起1年

### 3. 举证责任
- **用人单位**：掌握管理的证据由用人单位举证
- **劳动者**：其他事实由劳动者举证

## 七、常见违规风险与防范

### 1. 高频违规点
| 违规行为 | 法律后果 | 防范措施 |
|----------|----------|----------|
| 未签劳动合同 | 双倍工资 | 入职即签，30日内完成 |
| 试用期违法 | 支付赔偿金 | 严格遵守期限和工资标准 |
| 拖欠工资 | 补发+补偿金+滞纳金 | 按月足额发放 |
| 未缴社保 | 补缴+滞纳金+行政处罚 | 入职即缴，足额缴纳 |
| 违法解雇 | 支付双倍补偿金 | 依法解除，保留证据 |
| 超时加班 | 行政处罚+加班费 | 控制加班时长，发放加班费 |

### 2. 用工合规检查清单

```markdown
□ 劳动合同签订情况
□ 试用期约定是否合法
□ 工资是否按时足额发放
□ 社保公积金是否缴纳
□ 加班是否按规定支付
□ 年休假是否安排
□ 规章制度是否民主程序制定
□ 解除合同是否符合法定情形
□ 是否支付经济补偿金
□ 劳动合同是否依法解除
```

## 八、规章制度合规

### 1. 制定程序
```markdown
1. 起草制度草案
2. 职工代表大会或全体职工讨论
3. 与工会或职工代表平等协商
4. 公示告知劳动者
```

### 2. 公示方式
- 员工手册签收
- 内部公告栏张贴
- 公司内部网站发布
- 集中培训并签到

### 3. 无效情形
- 内容违法
- 程序不合法
- 未公示告知
