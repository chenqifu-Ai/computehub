# 面试技巧与方法

## 面试类型

### 按面试形式
| 类型 | 特点 | 适用 |
|------|------|------|
| 结构化面试 | 固定题目、标准评分 | 批量招聘 |
| 半结构化面试 | 有核心问题+追问 | 专业岗位 |
| 非结构化面试 | 灵活自由 | 高管岗位 |
| 情景面试 | 模拟工作场景 | 管理岗位 |
| 无领导小组讨论 | 团队协作观察 | 管理培训生 |

### 按面试轮次
| 轮次 | 目的 | 面试官 |
|------|------|--------|
| 初试 | 基本匹配度 | HR |
| 复试 | 专业能力 | 部门负责人 |
| 终试 | 综合素质 | 公司高管 |

## 面试准备

### 岗位分析
- 岗位职责
- 任职要求
- 薪资范围
- 发展空间

### 简历筛选
| 筛选要点 | 关注内容 |
|----------|----------|
| 基本信息 | 年龄、学历、工作年限 |
| 工作经历 | 公司、职位、业绩 |
| 跳槽频率 | 稳定性判断 |
| 期望薪资 | 匹配度 |

### 面试提纲
**结构化面试提纲模板：**
1. 自我介绍（2分钟）
2. 工作经历深挖（10分钟）
3. 专业能力考察（15分钟）
4. 综合素质评估（10分钟）
5. 候选人提问（5分钟）

## 面试技巧

### 开场技巧
| 技巧 | 说明 |
|------|------|
| 破冰 | 寒暄、自我介绍 |
| 说明流程 | 让候选人放松 |
| 明确时间 | 控制面试节奏 |

### 提问技巧
**STAR提问法**
| 维度 | 问题示例 |
|------|----------|
| S-Situation | 当时的背景是什么？ |
| T-Task | 你的任务是什么？ |
| A-Action | 你做了什么？ |
| R-Result | 结果如何？ |

**追问技巧**
- "能具体说说吗？"
- "还有呢？"
- "结果怎么样？"
- "如果重来会怎么做？"

### 观察要点
| 维度 | 观察内容 |
|------|----------|
| 表达能力 | 逻辑清晰、表达流畅 |
| 思维能力 | 分析问题、解决问题 |
| 情绪控制 | 压力下表现 |
| 价值观 | 与企业文化匹配 |

### 常见面试问题
| 类别 | 问题 |
|------|------|
| 自我认知 | 请做个自我介绍 |
| 工作经历 | 离职原因是什么 |
| 专业能力 | 如何处理XX问题 |
| 职业规划 | 未来3年规划 |
| 薪资期望 | 期望薪资是多少 |
| 反向提问 | 有什么想问我们的 |

## 面试评估

### 评估维度
| 维度 | 权重 | 评分 |
|------|------|------|
| 专业能力 | 40% | 1-5分 |
| 综合素质 | 30% | 1-5分 |
| 沟通表达 | 15% | 1-5分 |
| 价值观匹配 | 15% | 1-5分 |

### 评估记录
- 记录关键回答
- 记录观察感受
- 给出建议结论

### 决策建议
| 评估结果 | 建议 |
|----------|------|
| 优秀(4.5+) | 推荐进入下一轮 |
| 良好(3.5-4.5) | 可考虑，需对比 |
| 一般(2.5-3.5) | 谨慎考虑 |
| 较差(<2.5) | 不推荐 |

## 面试注意事项

### 面试官注意事项
| 禁止 | 说明 |
|------|------|
| 歧视性提问 | 性别、年龄、婚育等 |
| 引导性提问 | "你不觉得..." |
| 主观臆断 | 不要第一印象定论 |
| 泄露薪资 | 不要透露他人薪资 |

### 面试礼仪
- 准时开始
- 认真倾听
- 不打断候选人
- 尊重候选人隐私
- 及时反馈结果

## 面试后跟进

### 面试总结
- 填写面试评估表
- 汇总各轮评估结果
- 形成录用建议

### 结果反馈
| 结果 | 反馈方式 |
|------|----------|
| 通过 | 电话通知+发offer |
| 待定 | 电话沟通+等待 |
| 不通过 | 邮件感谢 |
