# 招聘渠道与选择

## 招聘渠道分类

### 内部招聘
| 方式 | 优点 | 缺点 |
|------|------|------|
| 内部竞聘 | 成本低、熟悉企业 | 选择范围有限 |
| 员工推荐 | 信任度高、快速 | 可能形成小团体 |
| 人才储备 | 快速响应 | 需长期培养 |

### 外部招聘
| 渠道 | 适用岗位 | 特点 |
|------|----------|------|
| 招聘网站 | 各类岗位 | 流量大、成本中 |
| 猎头 | 高管、稀缺人才 | 成本高、精准 |
| 校园招聘 | 应届生 | 人才储备 |
| 社交媒体 | 年轻群体 | 成本低、互动强 |
| 招聘会 | 批量岗位 | 直接接触 |

## 渠道选择策略

### 按岗位层级
| 层级 | 推荐渠道 |
|------|----------|
| 高管 | 猎头、圈内推荐 |
| 中层 | 招聘网站、猎头 |
| 基层 | 招聘网站、招聘会 |
| 应届生 | 校园招聘、实习生转正 |

### 按行业特点
| 行业 | 特殊渠道 |
|------|----------|
| 互联网 | 技术社区、开源项目 |
| 金融 | 行业协会、认证机构 |
| 制造 | 职业院校、技能大赛 |

## 招聘网站选择

### 主流平台
| 平台 | 定位 | 适用企业 |
|------|------|----------|
| BOSS直聘 | 直聊、效率高 | 中小企业 |
| 前程无忧 | 综合性强 | 各类企业 |
| 智联招聘 | 传统企业 | 大中型企业 |
| 猎聘 | 中高端 | 有预算企业 |
| 拉勾 | 互联网 | 科技公司 |

### 投放技巧
1. **标题优化**：突出岗位亮点
2. **JD清晰**：职责、要求明确
3. **薪资透明**：写明范围更有效
4. **及时回复**：24小时内响应
