# 劳动法规：劳动合同管理实务

## 一、劳动合同概述

### 1. 劳动合同定义
劳动合同是劳动者与用人单位确立劳动关系、明确双方权利和义务的协议。

### 2. 劳动合同特征
```markdown
主体特定：
- 一方是劳动者
- 一方是用人单位

内容法定：
- 必备条款由法律规定
- 部分内容可协商约定

关系从属：
- 劳动者从属于用人单位
- 用人单位对劳动者有管理权

继续性：
- 合同关系持续存在
- 不是一次性给付
```

### 3. 劳动合同类型

#### 固定期限劳动合同
```markdown
定义：
- 约定合同终止时间的劳动合同

特点：
- 期限明确
- 到期自然终止
- 可续签

适用情形：
- 一般用工情形
- 临时性、辅助性岗位
```

#### 无固定期限劳动合同
```markdown
定义：
- 约定无确定终止时间的劳动合同

应当订立情形：
- 劳动者在该用人单位连续工作满10年
- 用人单位初次实行劳动合同制度时，劳动者在该用人单位连续工作满10年且距法定退休年龄不足10年
- 连续订立二次固定期限劳动合同，劳动者提出续订的

视为订立情形：
- 用人单位自用工之日起满一年不与劳动者订立书面劳动合同

权利义务：
- 劳动者可随时提出解除
- 用人单位需符合法定条件才能解除
```

#### 以完成一定工作任务为期限的劳动合同
```markdown
定义：
- 约定以某项工作的完成为合同期限的劳动合同

特点：
- 期限不确定
- 以工作完成为终止条件

适用情形：
- 项目制工作
- 季节性工作
- 临时性任务
```

## 二、劳动合同订立

### 1. 订立原则
```markdown
合法原则：
- 主体合法
- 内容合法
- 程序合法

公平原则：
- 权利义务对等
- 条款公平合理

平等自愿原则：
- 双方地位平等
- 意思表示真实

协商一致原则：
- 充分协商
- 达成一致

诚实信用原则：
- 如实告知
- 履行承诺
```

### 2. 必备条款
```markdown
法定必备条款：
1. 用人单位名称、住所、法定代表人
2. 劳动者姓名、住址、身份证号
3. 劳动合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护、劳动条件和职业危害防护
9. 法律法规规定的其他事项

缺少必备条款后果：
- 由劳动行政部门责令改正
- 给劳动者造成损害的应当赔偿
```

### 3. 约定条款

#### 试用期
```markdown
期限限制：
- 合同期限3个月以上不满1年：试用期不得超过1个月
- 合同期限1年以上不满3年：试用期不得超过2个月
- 合同期限3年以上或无固定期限：试用期不得超过6个月

禁止情形：
- 同一用人单位与同一劳动者只能约定一次试用期
- 以完成一定工作任务为期限的劳动合同不得约定试用期
- 合同期限不满3个月的不得约定试用期
- 非全日制用工不得约定试用期

工资标准：
- 试用期工资不得低于本单位同岗位最低档工资或劳动合同约定工资的80%
- 并不得低于用人单位所在地的最低工资标准

违法后果：
- 违法约定的试用期无效
- 用人单位需支付赔偿金
```

#### 保密条款
```markdown
保密内容：
- 商业秘密
- 技术秘密
- 经营信息

保密期限：
- 可约定在职期间
- 可约定离职后一定期限

违约责任：
- 约定违约金
- 赔偿损失

注意：
- 保密义务是法定义务
- 即使未约定保密条款，劳动者也有保密义务
```

#### 竞业限制
```markdown
适用人员：
- 高级管理人员
- 高级技术人员
- 其他负有保密义务的人员

限制内容：
- 不得到竞争单位工作
- 不得自办竞争企业

限制期限：
- 不得超过2年

经济补偿：
- 用人单位需在竞业限制期限内给予经济补偿
- 月补偿标准不低于劳动者离职前12个月平均工资的30%
- 且不得低于当地最低工资标准

违约责任：
- 劳动者违反竞业限制约定，应支付违约金
- 用人单位可要求继续履行竞业限制

解除竞业限制：
- 用人单位可随时解除竞业限制协议
- 劳动者解除需满足特定条件
```

#### 服务期
```markdown
约定条件：
- 用人单位为劳动者提供专项培训费用
- 对其进行专业技术培训

服务期限：
- 双方协商约定
- 应当合理

违约金：
- 可约定违约金
- 违约金数额不得超过用人单位提供的培训费用
- 用人单位要求劳动者支付的违约金不得超过服务期尚未履行部分所应分摊的培训费用

注意：
- 普通培训（如入职培训、安全培训）不能约定服务期
- 只有专业技术培训才能约定服务期
```

### 4. 订立程序
```markdown
第一步：告知义务
- 用人单位告知工作内容、条件、地点、职业危害等
- 劳动者告知学历、经历、健康状况等

第二步：协商条款
- 就合同条款进行协商
- 达成一致意见

第三步：签订合同
- 双方签字或盖章
- 文本由用人单位和劳动者各执一份

第四步：建立劳动关系
- 用工之日起劳动关系建立
- 应当自用工之日起一个月内订立书面劳动合同

未签订合同的法律后果：
- 用工超过一个月不满一年未签订：支付双倍工资
- 用工超过一年未签订：视为订立无固定期限劳动合同
```

## 三、劳动合同履行与变更

### 1. 劳动合同履行
```markdown
履行原则：
- 全面履行：按照约定全面履行义务
- 亲自履行：劳动者应当亲自履行

用人单位义务：
- 按照约定提供劳动条件
- 按时足额支付劳动报酬
- 缴纳社会保险
- 提供劳动保护

劳动者义务：
- 遵守规章制度
- 完成工作任务
- 遵守劳动纪律
- 保守商业秘密
```

### 2. 劳动合同变更
```markdown
变更原则：
- 协商一致
- 书面形式
- 合法合规

变更情形：
- 协商变更：双方协商一致变更
- 法定变更：根据法律规定变更

常见变更：
- 工作岗位变更
- 工作地点变更
- 劳动报酬变更
- 合同期限变更

变更程序：
1. 一方提出变更建议
2. 双方协商变更内容
3. 达成一致后签订变更协议
4. 变更协议作为合同附件

注意：
- 变更应当采用书面形式
- 变更后的合同文本双方各执一份
- 未经协商一致，不得单方变更
```

### 3. 薪酬调整
```markdown
用人单位权利：
- 根据经营状况调整薪酬
- 根据员工绩效调整薪酬
- 调整需符合法律和合同约定

调整方式：
- 升薪：绩效优秀、能力提升
- 降薪：绩效不达标、岗位调整
- 调整需有合理依据

注意事项：
- 薪酬调整应符合合同约定
- 降薪需有合理依据
- 降薪幅度应合理
- 建议协商一致后调整
```

## 四、劳动合同解除

### 1. 协商解除
```markdown
定义：
- 用人单位与劳动者协商一致解除劳动合同

程序：
- 一方提出解除建议
- 双方协商一致
- 签订解除协议
- 办理离职手续

经济补偿：
- 用人单位提出解除：支付经济补偿
- 劳动者提出解除：不支付经济补偿
```

### 2. 劳动者单方解除

#### 预告解除
```markdown
条件：
- 劳动者提前30日以书面形式通知用人单位
- 试用期内提前3日通知用人单位

程序：
- 提交书面辞职申请
- 工作交接
- 办理离职手续

注意：
- 无需用人单位批准
- 期满即可解除劳动关系
```

#### 即时解除（用人单位有过错）
```markdown
情形：
- 未按照劳动合同约定提供劳动保护或劳动条件
- 未及时足额支付劳动报酬
- 未依法为劳动者缴纳社会保险费
- 用人单位的规章制度违反法律法规
- 劳动合同无效
- 法律法规规定的其他情形
- 用人单位以暴力、威胁或非法限制人身自由强迫劳动
- 用人单位违章指挥、强令冒险作业危及人身安全

程序：
- 劳动者可以即时解除
- 无需提前通知
- 用人单位应支付经济补偿
```

### 3. 用人单位单方解除

#### 过失性解除（无需支付经济补偿）
```markdown
情形：
- 在试用期间被证明不符合录用条件
- 严重违反用人单位规章制度
- 严重失职、营私舞弊，给用人单位造成重大损害
- 同时与其他用人单位建立劳动关系，对完成工作任务造成严重影响，或经提出拒不改正
- 因欺诈、胁迫等使劳动合同无效
- 被依法追究刑事责任

程序：
- 用人单位需举证证明
- 通知工会（有工会的）
- 书面通知劳动者

注意事项：
- "严重违反"需要规章制度明确规定
- "重大损害"需要量化标准
- 需要保留相关证据
```

#### 非过失性解除（需支付经济补偿）
```markdown
情形：
- 劳动者患病或非因工负伤，医疗期满后不能从事原工作，也不能从事另行安排的工作
- 劳动者不能胜任工作，经过培训或调整岗位仍不能胜任工作
- 客观情况发生重大变化，致使劳动合同无法履行，经协商未能就变更合同达成协议

程序：
- 提前30日书面通知劳动者，或额外支付一个月工资
- 通知工会（有工会的）
- 支付经济补偿

注意事项：
- 医疗期需满，且需证明不能胜任
- 不能胜任需经培训或调岗
- 客观情况重大变化需充分举证
```

#### 经济性裁员
```markdown
条件：
- 依照企业破产法规定进行重整
- 生产经营发生严重困难
- 企业转产、重大技术革新或经营方式调整，经变更劳动合同后仍需裁减人员
- 其他因劳动合同订立时所依据的客观情况发生重大变化，致使劳动合同无法履行

裁减人员数量：
- 裁减人员20人以上
- 或裁减不足20人但占企业职工总数10%以上

程序：
- 提前30日向工会或全体职工说明情况
- 听取工会或职工意见
- 裁减人员方案向劳动行政部门报告
- 支付经济补偿

优先留用人员：
- 与本单位订立较长期限固定期限劳动合同
- 与本单位订立无固定期限劳动合同
- 家庭无其他就业人员，有需要扶养的老人或未成年人

不得裁减人员：
- 患职业病或因工负伤并被确认丧失或部分丧失劳动能力
- 患病或非因工负伤在规定医疗期内
- 女职工在孕期、产期、哺乳期
- 法律法规规定的其他情形
```

### 4. 不得解除的情形
```markdown
法定不得解除情形：
- 从事接触职业病危害作业未进行离岗前职业健康检查，或疑似职业病在诊断或医学观察期间
- 在本单位患职业病或因工负伤并被确认丧失或部分丧失劳动能力
- 患病或非因工负伤在规定医疗期内
- 女职工在孕期、产期、哺乳期
- 在本单位连续工作满15年且距法定退休年龄不足5年
- 法律法规规定的其他情形

例外：
- 劳动者有过失性解除情形的，用人单位仍可解除
```

### 5. 经济补偿计算
```markdown
计算标准：
- 按劳动者在本单位工作年限计算
- 每满一年支付一个月工资
- 六个月以上不满一年按一年计算
- 不满六个月支付半个月工资

月工资标准：
- 劳动者在劳动合同解除或终止前12个月的平均工资
- 月工资高于当地职工月平均工资三倍的，按三倍支付
- 年限最高不超过12年

工作年限：
- 本单位工作年限
- 非因劳动者原因从原单位安排到新单位的，工作年限合并计算

赔偿金（违法解除）：
- 用人单位违法解除或终止劳动合同
- 应当支付赔偿金
- 赔偿金为经济补偿的二倍
```

## 五、劳动合同终止

### 1. 终止情形
```markdown
法定终止：
- 劳动合同期满
- 劳动者开始依法享受基本养老保险待遇
- 劳动者死亡或被人民法院宣告死亡或失踪
- 用人单位被依法宣告破产
- 用人单位被吊销营业执照、责令关闭、撤销
- 用人单位决定提前解散
- 法律法规规定的其他情形

注意：
- 终止是法定情形，不是单方行为
- 符合法定条件时自动终止
```

### 2. 终止续延
```markdown
应当续延情形：
- 劳动合同期满，有不得解除情形的
- 续延至相应情形消失时终止

女职工特别保护：
- 孕期、产期、哺乳期劳动合同期满
- 应当续延至相应情形消失

职业病特别保护：
- 疑似职业病在诊断或医学观察期间
- 续延至诊断或观察结束
```

### 3. 终止经济补偿
```markdown
用人单位应当支付经济补偿：
- 劳动合同期满，用人单位不续签
- 劳动合同期满，用人单位降低条件续签，劳动者不同意
- 用人单位被宣告破产
- 用人单位被吊销营业执照、责令关闭、撤销
- 用人单位决定提前解散

用人单位不需支付经济补偿：
- 劳动合同期满，用人单位维持或提高条件续签，劳动者不同意
- 劳动者开始享受基本养老保险待遇
- 劳动者死亡或被宣告死亡、失踪
```

## 六、劳动合同管理实务要点

### 1. 合同签订管理
```markdown
签订时限：
- 用工前签订：最佳实践
- 用工之日起一个月内签订：法定时限
- 超期签订：承担双倍工资风险

签订流程：
- 入职前发送合同文本供审阅
- 入职当日签订劳动合同
- 双方各执一份
- 建立劳动合同台账

注意事项：
- 合同条款必须完备
- 避免约定违法条款
- 注意合同文本管理
- 及时送达劳动合同
```

### 2. 合同履行管理
```markdown
薪酬管理：
- 按时足额支付
- 保留支付凭证
- 工资条签收

考勤管理：
- 建立考勤制度
- 记录出勤情况
- 保存考勤记录

规章制度：
- 内容合法
- 程序民主
- 公示告知

档案管理：
- 建立员工档案
- 保存相关材料
- 档案保管期限
```

### 3. 合同变更管理
```markdown
变更原则：
- 协商一致
- 书面形式
- 合理合法

变更流程：
- 发出变更通知
- 协商变更内容
- 签订变更协议
- 保存变更记录

注意事项：
- 单方变更风险
- 降薪需要依据
- 调岗需要合理
```

### 4. 合同解除管理
```markdown
解除程序：
- 审查解除依据
- 收集相关证据
- 通知工会
- 书面通知员工
- 办理离职手续
- 支付经济补偿

证据保存：
- 违纪证据
- 考核证据
- 培训记录
- 调岗记录
- 通知送达证明

风险防范：
- 程序合法
- 依据充分
- 证据完整
- 通知到位
```

### 5. 劳动争议处理
```markdown
争议类型：
- 确认劳动关系争议
- 劳动报酬争议
- 经济补偿争议
- 赔偿金争议
- 社会保险争议

处理途径：
- 协商：双方协商解决
- 调解：企业调解、调解组织调解
- 仲裁：劳动争议仲裁
- 诉讼：人民法院诉讼

仲裁时效：
- 一般时效：一年
- 劳动报酬争议：劳动关系存续期间不受时效限制

举证责任：
- 用人单位作出的开除、除名、辞退、解除劳动合同、减少劳动报酬、计算劳动者工作年限等决定发生争议，用人单位负举证责任
```

## 七、常见法律风险及防范

### 1. 未签订书面劳动合同
```markdown
风险：
- 支付双倍工资
- 视为订立无固定期限劳动合同

防范：
- 用工前或用工当日签订
- 建立签订台账
- 拒签员工书面催告
```

### 2. 试用期违法
```markdown
风险：
- 违法约定试用期无效
- 支付赔偿金

防范：
- 严格按法定期限约定
- 试用期工资符合标准
- 试用期考核留痕
```

### 3. 规章制度违法
```markdown
风险：
- 规章制度无效
- 劳动争议败诉
```
