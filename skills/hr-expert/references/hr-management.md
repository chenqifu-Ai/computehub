# 小微企业人力管理

## 招聘流程

### 标准流程
1. **需求确认**：岗位职责、任职要求、薪资范围
2. **渠道发布**：招聘网站、朋友圈、内推
3. **简历筛选**：关键词匹配、背景核实
4. **面试安排**：初试、复试、终试
5. **背景调查**：学历、工作经历、诚信
6. **录用决策**：offer谈判、入职时间
7. **入职办理**：合同、档案、培训

## 薪酬设计

### 薪酬结构
| 组成部分 | 占比 | 说明 |
|---------|-----|------|
| 基本工资 | 60-70% | 固定收入 |
| 绩效奖金 | 20-30% | 浮动收入 |
| 年终奖 | 10-20% | 年度激励 |
| 福利补贴 | 5-10% | 五险一金、餐补等 |

### 小微企业薪酬策略
- **核心岗位**：略高于市场水平
- **支持岗位**：市场水平
- **股权激励**：核心员工可选

## 劳动法规要点

### 劳动合同
- 必须书面签订（入职1个月内）
- 试用期不得超过法定期限
- 违法解除需支付2N赔偿

### 社保公积金
| 社保类型 | 单位比例 | 个人比例 |
|---------|---------|---------|
| 养老 | 16% | 8% |
| 医疗 | 10% | 2% |
| 失业 | 0.5% | 0.5% |
| 工伤 | 0.2-1.9% | 0 |
| 生育 | 0.8% | 0 |
| 公积金 | 5-12% | 5-12% |

### 离职处理
- 试用期：提前3天通知
- 正式期：提前30天书面通知
- 经济补偿：N（工作年限）×月工资
