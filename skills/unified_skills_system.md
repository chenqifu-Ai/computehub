## 📊 统一展示标准

### 仪表盘显示规则
1. **分类显示**：按系统管理、专业领域、技术能力三类显示
2. **状态标识统一**：✅ 活跃 / ✅ 就绪 / ⚠️ 禁用 / 🔄 运行中
3. **功能描述标准化**：统一格式的功能描述
4. **调用方式统一**：统一的技能调用接口

### 技能调用规则
1. **系统管理技能**：自动运行，无需手动调用
2. **专业领域技能**：按需调用，用户触发
3. **技术能力技能**：工具类，辅助其他技能
