# 知识产权：知识产权法律实务

## 一、知识产权概述

### 1. 知识产权定义
知识产权是指人们就其智力劳动成果所依法享有的专有权利，通常是国家赋予创造者对其智力成果在一定时期内享有的专有权利。

### 2. 知识产权类型
```markdown
著作权：
- 文学、艺术和科学作品
- 著作权人身权
- 著作权财产权

专利权：
- 发明专利
- 实用新型专利
- 外观设计专利

商标权：
- 商品商标
- 服务商标
- 集体商标
- 证明商标

其他知识产权：
- 地理标志
- 商业秘密
- 集成电路布图设计
- 植物新品种
```

### 3. 知识产权特征
```markdown
无形性：
- 知识产权客体是无形的
- 不占有空间
- 可以同时被多人使用

专有性：
- 权利人独占使用
- 排除他人使用
- 许可他人使用

地域性：
- 在授权国范围内有效
- 不同国家需要分别申请
- 国际保护需要国际条约

时间性：
- 有保护期限
- 期满后进入公有领域
- 不同权利期限不同
```

### 2. 著作权内容
```markdown
著作权人身权：

发表权：
- 决定作品是否公之于众
- 只能行使一次

署名权：
- 在作品上署名
- 不署名或署假名
- 永久保护

修改权：
- 修改或授权他人修改作品
- 永久保护

保护作品完整权：
- 保护作品不受歪曲、篡改
- 永久保护

著作权财产权：

复制权：
- 以印刷、复印、拓印、录音、录像、
  翻录、翻拍等方式制作作品复制件

发行权：
- 以出售或者赠与方式向公众提供作品

出租权：
- 有偿许可他人临时使用电影作品、
  计算机软件

展览权：
- 公开陈列美术作品、摄影作品

表演权：
- 公开表演作品
- 公开播送表演

放映权：
- 通过放映机、幻灯机等技术设备
  公开再现美术、摄影、电影等作品

广播权：
- 以有线或者无线方式公开传播

信息网络传播权：
- 以有线或者无线方式向公众提供作品
- 使公众可以在其个人选定的时间和
  地点获得作品

摄制权：
- 以摄制电影或者以类似摄制电影的
  方法将作品固定在载体上

改编权：
- 改变作品，创作出具有独创性的新作品

翻译权：
- 将作品从一种语言文字转换成
  另一种语言文字

汇编权：
- 将作品或者作品的片段通过选择或者
  编排，汇集成新作品
```

### 3. 著作权保护期限
```markdown
作者为公民：
- 发表权、财产权：作者终生及其死亡后50年
- 截止于作者死亡后第50年的12月31日

作者为法人或其他组织：
- 发表权、财产权：50年
- 截止于作品首次发表后第50年的12月31日

合作作品：
- 截止于最后死亡的作者死亡后第50年的12月31日

著作权人身权：
- 署名权、修改权、保护作品完整权永久保护
- 发表权有保护期限
```

### 4. 著作权侵权
```markdown
侵权行为：

直接侵权：
- 未经许可使用他人作品
- 未经许可复制、发行他人作品
- 未经许可信息网络传播他人作品

间接侵权：
- 提供侵权便利
- 教唆、帮助他人侵权

侵权责任：

民事责任：
- 停止侵害
- 消除影响
- 赔礼道歉
- 赔偿损失

行政责任：
- 责令停止侵权
- 没收违法所得
- 罚款

刑事责任：
- 侵犯著作权罪
- 销售侵权复制品罪
- 刑期、罚金

赔偿计算：
- 实际损失
- 侵权人违法所得
- 法定赔偿（500元-500万元）
```

### 5. 合理使用
```markdown
合理使用情形：

个人学习：
- 为个人学习、研究或者欣赏
- 使用他人已经发表的作品

适当引用：
- 为介绍、评论某一作品
- 适当引用他人已经发表的作品

新闻报道：
- 为报道新闻
- 在媒体中不可避免地再现或引用

课堂教学：
- 为学校课堂教学或者科学研究
- 翻译或者少量复制

免费表演：
- 免费表演已经发表的作品
- 未向公众收费，未向表演者支付报酬

公共图书馆：
- 图书馆为陈列或者保存版本的需要
- 复制本馆收藏的作品

合理使用条件：
- 已发表作品
- 注明作者姓名、作品名称
- 不影响作品正常使用
- 不合理损害权利人利益
```

## 三、专利权

### 1. 专利类型
```markdown
发明专利：
- 技术革新
- 产品、方法或其改进
- 新的技术方案
- 保护期20年

实用新型专利：
- 产品形状、构造
- 实用技术方案
- 小发明
- 保护期10年

外观设计专利：
- 产品形状、图案、色彩
- 富有美感
- 适于工业应用
- 保护期15年
```

### 2. 专利授予条件
```markdown
发明专利和实用新型专利：

新颖性：
- 不属于现有技术
- 未在国内外出版物公开发表
- 未公开使用

创造性：
- 与现有技术相比
- 具有突出的实质性特点和显著的进步
- 发明专利要求较高
- 实用新型专利要求较低

实用性：
- 能够制造或使用
- 能够产生积极效果

外观设计专利：

新颖性：
- 不属于现有设计
- 与现有设计不相同也不近似

区别性：
- 与现有设计相比
- 具有明显区别

不冲突：
- 不与他人在先权利相冲突
```

### 3. 专利申请
```markdown
申请文件：

发明专利：
- 请求书
- 说明书
- 权利要求书
- 说明书摘要

实用新型专利：
- 请求书
- 说明书
- 权利要求书
- 说明书摘要

外观设计专利：
- 请求书
- 外观设计图片或照片
- 简要说明

申请流程：
- 提交申请
- 初步审查
- 公布（发明专利）
- 实质审查（发明专利）
- 授权登记

申请原则：
- 先申请原则
- 单一性原则
- 书面原则
```

### 4. 专利权内容
```markdown
专利权人的权利：

独占实施权：
- 制造
- 使用
- 许诺销售
- 销售
- 进口

许可权：
- 许可他人实施专利
- 独占许可
- 排他许可
- 普通许可

转让权：
- 转让专利权
- 书面合同
- 登记生效

标记权：
- 在专利产品上标明专利标记
- 标明专利号

专利权人的义务：
- 缴纳年费
- 不得滥用专利权
```

### 5. 专利侵权
```markdown
侵权判定：

全面覆盖原则：
- 被控侵权产品包含权利要求全部技术特征
- 字面侵权

等同原则：
- 以等同特征替代
- 实现相同功能
- 达到相同效果

侵权行为：
- 制造专利产品
- 使用专利产品
- 许诺销售专利产品
- 销售专利产品
- 进口专利产品

不视为侵权的情形：
- 权利用尽
- 先用权
- 临时过境
- 科研实验
- 行政审批

侵权责任：

民事责任：
- 停止侵权
- 赔偿损失
- 消除影响

赔偿计算：
- 实际损失
- 侵权人获利
- 许可费倍数
- 法定赔偿（3万元-500万元）
```

## 四、商标权

### 1. 商标类型
```markdown
按构成要素分：

文字商标：
- 汉字
- 字母
- 数字

图形商标：
- 图形
- 图案

组合商标：
- 文字+图形
- 多要素组合

三维商标：
- 产品形状
- 包装形状

声音商标：
- 音乐
- 声音

颜色商标：
- 颜色组合
- 单一颜色

按用途分：

商品商标：
- 用于商品
- 区别商品来源

服务商标：
- 用于服务
- 区别服务来源

集体商标：
- 团体、协会注册
- 成员共同使用

证明商标：
- 证明商品或服务的
  特定品质
- 由具有监督能力的组织注册
```

### 2. 商标注册条件
```markdown
积极条件：
- 有显著特征
- 便于识别
- 不与他人在先权利冲突

消极条件：

不得作为商标使用：
- 同国家名称、国旗、国徽相同或近似
- 同外国国家名称、国旗相同或近似
- 同政府间国际组织名称相同或近似
- 同官方标志相同或近似
- 带有民族歧视
- 带有欺骗性
- 有害于社会主义道德风尚

不得作为商标注册：
- 仅有本商品的通用名称、图形、型号
- 仅直接表示商品的质量、原料、功能、用途等
- 缺乏显著特征
```

### 3. 商标注册程序
```markdown
申请：
- 提交申请书
- 商标图样
- 缴纳费用

审查：
- 形式审查
- 实质审查

公告：
- 初步审定公告
- 公告期3个月

异议：
- 公告期内可以提出异议
- 异议理由
- 异议审查

核准注册：
- 公告期满无异议
- 核准注册
- 发给商标注册证

注册有效期：
- 注册有效期10年
- 期满前12个月可以续展
- 续展有效期10年
```

### 4. 商标权内容
```markdown
商标权人的权利：

专用权：
- 在核定商品上使用注册商标
- 禁止他人使用相同或近似商标

许可权：
- 许可他人使用注册商标
- 签订商标使用许可合同
- 许可合同备案

转让权：
- 转让注册商标
- 书面合同
- 共同向商标局申请

商标权人的义务：
- 规范使用注册商标
- 保证商品质量
- 不得自行改变注册事项
```

### 5. 商标侵权
```markdown
侵权行为：
- 未经许可在同一种商品上使用
  与注册商标相同的商标
- 未经许可在同一种商品上使用
  与注册商标近似的商标
- 销售侵犯注册商标专用权的商品
- 伪造、擅自制造注册商标标识
- 更换注册商标并将更换商标的商品
  投入市场

侵权认定：
- 商标相同或近似
- 商品相同或类似
- 容易导致混淆

侵权责任：

民事责任：
- 停止侵权
- 赔偿损失
- 消除影响

行政责任：
- 责令立即停止侵权
- 没收、销毁侵权商品
- 罚款

刑事责任：
- 假冒注册商标罪
- 销售假冒注册商标商品罪
- 非法制造注册商标标识罪

赔偿计算：
- 实际损失
- 侵权人获利
- 许可费倍数
- 法定赔偿（500万元以下）
```

## 五、商业秘密

### 1. 商业秘密定义
```markdown
商业秘密：
- 不为公众所知悉
- 具有商业价值
- 经权利人采取保密措施
- 技术信息、经营信息等

商业秘密类型：

技术信息：
- 设计图纸
- 配方
- 工艺流程
- 技术数据

经营信息：
- 客户名单
- 营销策略
- 财务数据
- 经营计划
```

### 2. 商业秘密保护
```markdown
保密措施：
- 签订保密协议
- 制定保密制度
- 限制知悉范围
- 标注保密标识

员工保密：
- 入职签订保密协议
- 离职保密义务
- 竞业限制

商业秘密侵权：

侵权行为：
- 以不正当手段获取
- 披露、使用或允许他人使用
- 违反约定披露、使用

侵权责任：
- 停止侵权
- 赔偿损失
- 消除影响
```
