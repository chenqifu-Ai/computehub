# 公司法律实务

## 公司类型

### 公司类型对比
| 类型 | 特点 | 适用 | 责任 |
|------|------|------|------|
| 有限责任公司 | 股东人数1-50人 | 中小企业 | 以出资额为限 |
| 股份有限公司 | 发起人2-200人 | 大企业 | 以股份为限 |
| 个人独资企业 | 1人 | 小微企业 | 无限责任 |
| 合伙企业 | 2人以上 | 专业机构 | 无限/有限责任 |

## 公司设立

### 设立条件
| 条件 | 有限责任公司 | 股份有限公司 |
|------|--------------|--------------|
| 股东人数 | 1-50人 | 2-200人 |
| 注册资本 | 认缴制 | 认缴制 |
| 公司章程 | 必须制定 | 必须制定 |
| 公司名称 | 符合规定 | 符合规定 |
| 公司住所 | 必须有 | 必须有 |

### 设立流程
| 步骤 | 内容 | 时间 |
|------|------|------|
| 核名 | 名称核准 | 1-3天 |
| 提交材料 | 工商材料 | 3-5天 |
| 领取执照 | 营业执照 | 1天 |
| 刻章 | 公章、财务章 | 1天 |
| 银行开户 | 基本户 | 1-2天 |
| 税务登记 | 税务报到 | 1天 |

| 监事会 | 监督机构 | 监督董事、高管 |

### 股东权利
| 权利 | 说明 |
|------|------|
| 表决权 | 参与重大决策 |
| 分红权 | 按出资比例分红 |
| 知情权 | 查阅公司资料 |
| 转让权 | 转让股权 |
| 优先购买权 | 其他股东转让时优先购买 |

## 股权设计

### 股权结构
| 类型 | 结构 | 控制权 |
|------|------|--------|
| 绝对控制 | 67%以上 | 完全控制 |
| 相对控制 | 51%以上 | 一般控制 |
| 否决权 | 34%以上 | 重大事项否决 |
| 参与权 | 10%以上 | 参与决策 |

### 股权分配原则
1. **明确控制权**：创始人保持控制
2. **预留期权池**：10-15%给团队
3. **避免50:50**：防止决策僵局
4. **动态调整**：根据贡献调整

## 公司章程

### 必备条款
| 条款 | 内容 |
|------|------|
| 名称住所 | 公司名称、地址 |
| 经营范围 | 主营业务 |
| 注册资本 | 认缴金额 |
| 股东信息 | 股东姓名、出资 |
| 组织机构 | 股东会、董事会等 |
| 财务制度 | 财务会计规定 |

### 自主约定条款
- 股权转让限制
- 表决权安排
- 分红比例
- 继承问题

## 公司变更

### 常见变更
| 变更类型 | 材料 | 时间 |
|----------|------|------|
| 变更名称 | 股东会决议、章程修正案 | 3-5天 |
| 变更地址 | 新地址证明 | 3-5天 |
| 变更经营范围 | 股东会决议 | 3-5天 |
| 变更法定代表人 | 股东会决议、新法人身份 | 3-5天 |
| 变更注册资本 | 股东会决议、验资报告 | 5-7天 |
| 股权转让 | 股权转让协议 | 3-5天 |

## 公司解散

### 解散原因
| 原因 | 说明 |
|------|------|
| 章程规定 | 营业期限届满 |
| 股东决议 | 股东会决议解散 |
| 合并分立 | 公司合并分立 |
| 吊销 | 被吊销营业执照 |
| 法院判决 | 法院判决解散 |

### 清算流程
1. 成立清算组
2. 通知债权人
3. 清理债权债务
4. 制定清算方案
5. 分配剩余财产
6. 注销登记
