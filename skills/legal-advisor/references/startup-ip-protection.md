# 初创企业知识产权保护指南

## 为什么知识产权重要

### 知识产权价值
| 类型 | 价值体现 | 保护期限 |
|------|----------|----------|
| 商标 | 品牌识别、市场竞争 | 10年（可续展） |
| 专利 | 技术壁垒、融资估值 | 发明20年/实用新型10年 |
| 著作权 | 代码保护、内容资产 | 作者终身+50年 |
| 商业秘密 | 核心竞争力 | 永久（需保密） |

### 知识产权风险
| 风险类型 | 后果 | 案例 |
|----------|------|------|
| 被侵权 | 市场份额被抢占 | 技术被抄袭 |
| 侵权他人 | 赔偿+停止侵权 | 产品下架 |
| 品牌被抢注 | 被迫改名 | 商标被抢注 |
| 代码泄露 | 核心资产流失 | 员工带走代码 |

## 商标保护

### 商标注册流程
| 步骤 | 时间 | 费用 | 说明 |
|------|------|------|------|
| 商标查询 | 1-3天 | 免费 | 避免近似商标 |
| 提交申请 | 1天 | 300元/类 | 网上申请 |
| 形式审查 | 1-2月 | - | 材料审核 |
| 实质审查 | 4-6月 | - | 是否可注册 |
| 初审公告 | 3月 | - | 异议期 |
| 注册公告 | 1月 | - | 获得证书 |
| **总计** | **9-12月** | **300元/类** | - |

## 著作权保护

## 知识产权风险防范

### 侵权风险排查
| 排查项 | 方法 | 工具 |
|--------|------|------|
| 商标侵权 | 商标局查询 | 中国商标网 |

### 开源软件合规
| 许可证类型 | 特点 | 注意事项 |
|------------|------|----------|
| MIT | 最宽松 | 保留版权声明 |
| Apache 2.0 | 宽松+专利授权 | 保留声明、变更说明 |
| GPL | 传染性强 | 衍生作品必须开源 |
| LGPL | 较弱传染性 | 动态链接可不开源 |

### 应对措施
| 风险 | 应对 |
|------|------|
| 被侵权 | 发律师函→行政投诉→诉讼 |
| 侵权他人 | 立即停止→协商和解→赔偿 |
| 商标被抢注 | 异议→无效宣告→购买 |

## 知识产权避坑

### 常见错误
| 错误 | 后果 | 避免方法 |
|------|------|----------|
| 产品上线后才想起商标 | 被抢注风险 | 产品未动，商标先行 |
| 忽视开源协议 | 侵权风险 | 代码合规检查 |
| 员工没有保密协议 | 商业秘密泄露 | 入职必签NDA |
| 专利公开过早 | 丧失新颖性 | 先申请后公开 |
| 不监控侵权 | 市场份额被抢 | 定期市场监控 |

### 关键提醒
| 提醒 | 说明 |
|------|------|
| 早申请 | 知识产权具有先申请原则 |
| 全覆盖 | 核心资产全面保护 |
| 常监控 | 定期监控市场和竞争对手 |
| 重保密 | 商业秘密严格保密 |
| 留证据 | 创作过程留痕 |
