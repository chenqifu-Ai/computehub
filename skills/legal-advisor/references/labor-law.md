# 劳动法实务要点

## 劳动合同

### 合同类型
| 类型 | 适用情形 | 期限 |
|------|----------|------|
| 固定期限 | 一般用工 | 约定年限 |
| 无固定期限 | 连续签2次固定期或工龄满10年 | 无终止日期 |
| 以完成一定工作为期限 | 项目制用工 | 工作完成时终止 |

### 合同必备条款
1. 用人单位信息
2. 劳动者信息
3. 合同期限
4. 工作内容和地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护条件

### 试用期规定
| 合同期限 | 试用期上限 |
|----------|------------|
| < 3个月 | 不得约定 |
| 3个月-1年 | 1个月 |
| 1-3年 | 2个月 |
| > 3年或无固定期限 | 6个月 |

## 工资支付

### 最低工资标准
- 各地区不同，需参照当地规定
- 加班工资不计入最低工资
- 试用期工资不低于最低工资

### 加班工资计算
| 时间段 | 计算基数 |
|--------|----------|
| 工作日延时 | 150% |
| 休息日加班 | 200%（可调休） |
| 法定节假日 | 300% |

## 社会保险

### 五险缴纳比例（示例）
| 险种 | 单位比例 | 个人比例 |
|------|----------|----------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 9% | 2% |
| 失业保险 | 0.5% | 0.5% |
| 工伤保险 | 0.2-1.9% | 0 |
| 生育保险 | 0.8% | 0 |

### 缴纳基数
- 下限：当地社平工资60%
- 上限：当地社平工资300%
- 新入职员工：首月工资

## 解除劳动合同

### 用人单位解除
| 类型 | 条件 | 经济补偿 |
|------|------|----------|
| 协商解除 | 双方同意 | N |
| 过失性辞退 | 严重违纪、不胜任 | 无 |
| 无过失性辞退 | 试用期不符、医疗期后 | N+1 |
| 经济性裁员 | 经营困难 | N |

### 劳动者解除
- 提前30天书面通知
- 试用期提前3天
- 用人单位违法可即时解除

### 经济补偿计算
- N = 工作年限 × 月工资
- 不满半年按半年计算
- 半年以上不满一年按一年计算
- 月工资 = 离职前12个月平均工资
