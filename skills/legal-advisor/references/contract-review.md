# 合同审核实务指南

## 合同审核流程

### 审核步骤
| 步骤 | 内容 | 责任人 |
|------|------|--------|
| 形式审查 | 格式、签章、附件 | 法务初审 |
| 实质审查 | 条款、权利义务 | 法务复核 |
| 风险评估 | 法律风险、商业风险 | 法务+业务 |
| 审批签署 | 领导审批 | 授权人 |

### 审核要点
- 合同主体资格
- 合同内容合法性
- 条款完整性
- 表述准确性
- 风险可控性

## 合同主体审核

### 主体资格
| 审核项 | 方法 |
|--------|------|
| 营业执照 | 查询国家企业信用信息公示系统 |
| 经营范围 | 核对是否超范围经营 |
| 法定代表人 | 核对身份证明 |
| 授权委托 | 核对授权书 |

### 主体审查
- 自然人：身份证、民事行为能力
- 法人：营业执照、法人代表身份
- 非法人组织：主体资格证明

## 核心条款审核

### 标的条款
| 要点 | 说明 |
|------|------|
| 标的明确 | 名称、规格、型号清晰 |
| 数量准确 | 单位、数量明确 |
| 质量标准 | 国家标准/行业标准/约定标准 |
| 验收方式 | 验收标准、期限、方式 |

### 价格条款
| 要点 | 说明 |
|------|------|
| 价格明确 | 单价、总价、币种 |
| 结算方式 | 银行转账、票据、现金 |
| 支付条件 | 预付、分期、验收后 |
| 税费约定 | 含税/不含税、发票类型 |

### 交付条款
| 要点 | 说明 |
|------|------|
| 交付时间 | 具体日期或期限 |
| 交付地点 | 明确地址 |
| 交付方式 | 自提/送货/邮寄 |
| 风险转移 | 交付时风险转移 |

### 违约责任
| 要点 | 说明 |
|------|------|
| 违约情形 | 明确列举违约行为 |
| 违约金 | 具体金额或计算方式 |
| 损失赔偿 | 赔偿范围、计算方式 |
| 免责条款 | 不可抗力等免责情形 |

### 争议解决
| 方式 | 特点 | 适用 |
|------|------|------|
| 协商 | 成本低、效率高 | 争议不大 |
| 调解 | 第三方主持 | 有调解意愿 |
| 仲裁 | 一裁终局 | 合同约定仲裁条款 |
| 诉讼 | 两审终审 | 未约定仲裁 |

## 常见合同风险

### 条款风险
| 风险类型 | 表现 | 防范 |
|----------|------|------|
| 条款缺失 | 关键条款未约定 | 使用标准合同模板 |
| 条款模糊 | 表述不清晰 | 明确具体表述 |
| 条款矛盾 | 前后不一致 | 统一条款表述 |
| 格式条款 | 不公平格式条款 | 重点审查免责条款 |

### 履约风险
| 风险类型 | 防范措施 |
|----------|----------|
| 对方违约 | 违约金、担保 |
| 情势变更 | 变更解除条款 |
| 合同诈骗 | 尽职调查 |

## 合同管理建议

### 合同归档
| 内容 | 要求 |
|------|------|
| 合同原件 | 妥善保管 |
| 履约记录 | 完整记录 |
| 往来函件 | 归档保存 |
| 变更补充 | 及时归档 |

### 合同台账
- 合同编号
- 合同名称
- 签订日期
- 合同金额
- 履约状态
- 结算情况

### 定期清理
| 频率 | 内容 |
|------|------|
| 月度 | 新签合同登记 |
| 季度 | 履约情况检查 |
| 年度 | 合同归档清理 |
