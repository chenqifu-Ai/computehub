# 劳动法：劳动合同法律实务

## 一、劳动合同概述

### 1. 劳动合同定义
劳动合同是劳动者与用人单位确立劳动关系、明确双方权利和义务的协议。建立劳动关系应当订立书面劳动合同。

### 2. 劳动合同特征
```markdown
主体特定：
- 一方是劳动者
- 一方是用人单位
- 主体资格有特殊要求

内容法定：
- 法律规定必备条款
- 当事人可约定条款
- 内容不得违法

标的特定：
- 劳动者的劳动行为
- 用人单位支付报酬

人身属性：
- 劳动者必须亲自履行
- 不得转让
- 具有人身依附性
```

### 3. 劳动合同原则
```markdown
合法原则：
- 订立程序合法
- 合同内容合法
- 履行方式合法

公平原则：
- 权利义务对等
- 内容公平合理
- 不得显失公平

平等自愿原则：
- 双方地位平等
- 意思表示真实
- 无欺诈胁迫

协商一致原则：
- 双方协商达成一致
- 充分表达意愿
- 达成合意

诚实信用原则：
- 如实告知
- 诚实守信
- 履行承诺
```

## 二、劳动合同订立

### 1. 订立形式
```markdown
书面形式：
- 应当订立书面劳动合同
- 自用工之日起一个月内订立
- 未订立的承担法律责任

口头形式：
- 非全日制用工可以口头订立
- 临时性、辅助性工作

事实劳动关系：
- 用工事实存在
- 未订立书面合同
- 依法承担法律责任

法律后果：
- 超过一个月不满一年未订立：
  - 支付双倍工资
- 超过一年未订立：
  - 支付11个月双倍工资
  - 视为无固定期限劳动合同
```

### 2. 劳动合同期限
```markdown
固定期限劳动合同：
- 约定合同终止时间
- 可续签
- 到期终止或续签

无固定期限劳动合同：
- 没有约定终止时间
- 符合法定条件应当订立
- 只要不出现法定情形就一直有效

应当订立无固定期限合同的情形：
- 劳动者在该用人单位连续工作满十年的
- 用人单位初次实行劳动合同制度时，
  劳动者在该用人单位连续工作满十年且距法定退休年龄不足十年的
- 连续订立二次固定期限劳动合同续订的
- 用人单位自用工之日起满一年不与劳动者订立书面劳动合同的

以完成一定工作任务为期限的劳动合同：
- 以工作任务完成时间为合同期限
- 任务完成合同终止
- 一般不续签
```

### 3. 试用期规定
```markdown
试用期期限：
- 合同期限三个月以上不满一年：
  - 试用期不得超过一个月
- 合同期限一年以上不满三年：
  - 试用期不得超过二个月
- 三年以上固定期限和无固定期限：
  - 试用期不得超过六个月

试用期规定：
- 同一用人单位与同一劳动者只能约定一次试用期
- 以完成一定工作任务为期限的不得约定试用期
- 试用期包含在劳动合同期限内
- 试用期工资不得低于本单位相同岗位最低档工资的80%
- 试用期工资不得低于劳动合同约定工资的80%
- 试用期工资不得低于用人单位所在地的最低工资标准

违法约定试用期：
- 违法约定的试用期已经履行的
- 由用人单位以试用期满月工资为标准
- 按已经履行的超过法定试用期的期间
- 向劳动者支付赔偿金
```

### 4. 劳动合同必备条款
```markdown
法定必备条款：
- 用人单位的名称、住所和法定代表人
- 劳动者的姓名、住址和身份证号码
- 劳动合同期限
- 工作内容和工作地点
- 工作时间和休息休假
- 劳动报酬
- 社会保险
- 劳动保护、劳动条件和职业危害防护
- 法律、法规规定应当纳入劳动合同的其他事项

约定条款：
- 试用期
- 培训
- 保守秘密
- 补充保险
- 福利待遇
- 其他事项
```

### 5. 劳动合同无效
```markdown
无效情形：

以欺诈、胁迫手段订立：
- 提供虚假学历证书
- 提供虚假工作经历
- 威胁劳动者订立合同

用人单位免除责任：
- 免除法定责任条款
- 排除劳动者权利条款
- 如：工伤概不负责

违反法律强制性规定：
- 内容违法
- 程序违法
- 如：约定工资低于最低工资标准

无效劳动合同后果：
- 无效劳动合同自始无效
- 不发生法律效力
- 劳动者已付出劳动的，用人单位应当支付劳动报酬
- 给对方造成损害的，有过错的一方承担赔偿责任
```

## 三、劳动合同履行

### 1. 履行原则
```markdown
全面履行原则：
- 按照合同约定的条款履行
- 按照法律规定履行
- 不得擅自变更

亲自履行原则：
- 劳动者必须亲自履行
- 不得委托他人代为履行
- 用人单位不得强迫劳动者代为履行

协作履行原则：
- 双方互相配合
- 互相协作
- 共同完成工作任务
```

### 2. 工资支付
```markdown
工资支付规定：

支付形式：
- 货币形式支付
- 不得以实物替代

支付周期：
- 按月支付
- 不得克扣或者无故拖欠

支付标准：
- 不得低于当地最低工资标准
- 不得低于劳动合同约定标准
- 同工同酬

特殊情况支付：
- 加班工资：按规定支付加班费
- 法定节假日：支付工资
- 年休假：未休年休假支付300%工资

工资支付保护：
- 不得克扣工资
- 不得无故拖欠工资
- 不得以实物替代工资
```

### 3. 工作时间
```markdown
标准工作时间：
- 每日工作时间不超过8小时
- 每周工作时间不超过40小时
- 每周至少休息一日

缩短工作时间：
- 从事有害身体健康工作的
- 在特殊条件下工作的
- 如：高温、井下作业

延长工作时间限制：
- 一般每日不超过1小时
- 特殊原因每日不超过3小时
- 每月不超过36小时

加班工资标准：
- 工作日延长：150%工资
- 休息日工作：200%工资
- 法定节假日工作：300%工资
```

### 4. 休息休假
```markdown
法定节假日：
- 元旦：1天
- 春节：3天
- 清明节：1天
- 劳动节：1天
- 端午节：1天
- 中秋节：1天
- 国庆节：3天

带薪年休假：
- 累计工作满1年不满10年：5天
- 累计工作满10年不满20年：10天
- 累计工作满20年：15天
- 未休年休假支付300%工资

其他假期：
- 婚假：3天
- 丧假：1-3天
- 产假：98天+各地延长
- 病假：按规定支付病假工资

探亲假：
- 探望配偶：每年30天
- 未婚探望父母：每年20天
- 已婚探望父母：每4年20天
```

## 四、劳动合同变更

### 1. 变更原则
```markdown
协商一致原则：
- 双方协商一致
- 达成变更合意
- 不得单方变更

书面形式原则：
- 变更应当采用书面形式
- 口头变更无效
- 变更后的合同文本各执一份

合法原则：
- 变更内容合法
- 变更程序合法
- 不得违反法律强制性规定
```

### 2. 变更情形
```markdown
协商变更：
- 双方协商一致
- 可以变更劳动合同内容
- 应当采用书面形式

用人单位单方变更：
- 劳动者不能胜任工作
- 可以调整工作岗位
- 应当合理调整

客观情况重大变化：
- 劳动合同订立时所依据的客观情况发生重大变化
- 致使劳动合同无法履行
- 可以协商变更

企业转产、重大技术革新：
- 可以变更劳动合同
- 可以裁减人员
- 应当优先留用人员
```

### 3. 调岗调薪
```markdown
合理调岗：

用人单位可以调岗的情形：
- 劳动者不能胜任工作
- 劳动者患病或非因工负伤
- 医疗期满后不能从事原工作
- 企业转产、重大技术革新
- 客观情况发生重大变化

调岗的合理性：
- 调整后的岗位与原岗位相关
- 薪资待遇相当
- 不具有侮辱性
- 不具有惩罚性

调薪：

合理调薪的情形：
- 劳动者不能胜任工作
- 绩效考核不合格
- 岗位调整
- 企业经营困难

调薪的限制：
- 不得低于最低工资标准
- 不得低于劳动合同约定
- 应当合理
- 应当协商
```

## 五、劳动合同解除

### 1. 协商解除
```markdown
协商一致解除：
- 双方协商一致
- 可以解除劳动合同
- 用人单位提出解除的，应当支付经济补偿

协商解除程序：
- 一方提出解除提议
- 另一方同意
- 办理工作交接
- 办理解除手续
```

### 2. 劳动者单方解除
```markdown
预告解除：
- 提前30日以书面形式通知用人单位
- 试用期提前3日通知
- 不需要用人单位同意

即时解除（无需预告）：

用人单位过错情形：
- 未按照劳动合同约定提供劳动保护或者劳动条件的
- 未及时足额支付劳动报酬的
- 未依法为劳动者缴纳社会保险费的
- 用人单位的规章制度违反法律、法规的规定，损害劳动者权益的
- 以欺诈、胁迫手段订立劳动合同
- 用人单位以暴力、威胁或者非法限制人身自由的手段强迫劳动者劳动的
- 用人单位违章指挥、强令冒险作业危及劳动者人身安全的

后果：
- 劳动者可以立即解除劳动合同
- 不需要事先告知用人单位
- 用人单位应当支付经济补偿
```

### 3. 用人单位单方解除
```markdown
过失性辞退（无需支付经济补偿）：

劳动者有过错情形：
- 在试用期间被证明不符合录用条件的
- 严重违反用人单位规章制度的
- 严重失职，营私舞弊，给用人单位造成重大损害的
- 劳动者同时与其他用人单位建立劳动关系，
  对完成本单位工作任务造成严重影响，或者经用人单位提出，拒不改正的
- 以欺诈手段使用人单位在违背真实意思的情况下订立劳动合同的
- 被依法追究刑事责任的

非过失性辞退（需支付经济补偿）：

劳动者无过错情形：
- 劳动者患病或非因工负伤，
  医疗期满后不能从事原工作，也不能从事另行安排的工作
- 劳动者不能胜任工作，
  经过培训或者调整工作岗位，仍不能胜任工作
- 劳动合同订立时所依据的客观情况发生重大变化，
  致使劳动合同无法履行，经协商未能就变更劳动合同内容达成协议

程序要求：
- 提前30日以书面形式通知劳动者本人
- 或者额外支付劳动者一个月工资
```

### 4. 经济性裁员
```markdown
裁员条件：
- 依照企业破产法规定进行重整
- 生产经营发生严重困难
- 企业转产、重大技术革新，
  经变更劳动合同后，仍需裁减人员
- 其他因劳动合同订立时所依据的客观情况发生重大变化，
  致使劳动合同无法履行

裁员人数：
- 裁减人员二十人以上
- 或者裁减不足二十人但占企业职工总数百分之十以上

裁员程序：
- 提前三十日向工会或者全体职工说明情况
- 听取工会或者职工的意见
- 裁减人员方案向劳动行政部门报告

优先留用人员：
- 与本单位订立较长期限的固定期限劳动合同的
- 与本单位订立无固定期限劳动合同的
- 家庭无其他就业人员，有需要扶养的老人或者未成年人的

重新录用：
- 裁减人员后六个月内录用人员的，
  应当优先录用被裁减人员
```

### 5. 不得解除的情形
```markdown
劳动者有下列情形之一的，
用人单位不得解除劳动合同：

职业病或工伤：
- 从事接触职业病危害作业的劳动者
  未进行离岗前职业健康检查，
  或者疑似职业病病人在诊断或者医学观察期间的
- 在本单位患职业病或者因工负伤并被确认丧失或者部分丧失劳动能力的

医疗期：
- 患病或者非因工负伤，在规定的医疗期内的

女职工保护：
- 女职工在孕期、产期、哺乳期的

其他情形：
- 在本单位连续工作满十五年，且距法定退休年龄不足五年的
- 法律、行政法规规定的其他情形

例外：
- 以上情形不适用于过失性辞退
```

## 六、劳动合同终止

### 1. 终止情形
```markdown
劳动合同期满：
- 固定期限劳动合同期满
- 以完成一定工作任务为期限的劳动合同任务完成

劳动者主体资格消灭：
- 劳动者开始依法享受基本养老保险待遇
- 劳动者死亡
- 劳动者被人民法院宣告死亡或者宣告失踪

用人单位主体资格消灭：
- 用人单位被依法宣告破产
- 用人单位被吊销营业执照、责令关闭、撤销
- 用人单位决定提前解散

其他情形：
- 法律、行政法规规定的其他情形
```

### 2. 终止的经济补偿
```markdown
应当支付经济补偿的情形：
- 劳动合同期满终止（用人单位不愿续签或降低条件续签）
- 用人单位被依法宣告破产
- 用人单位被吊销营业执照、责令关闭、撤销
- 用人单位决定提前解散

不需要支付经济补偿的情形：
- 劳动合同期满，用人单位维持或提高条件续签，劳动者不愿续签
- 劳动者开始享受基本养老保险待遇
- 劳动者死亡
- 劳动者被宣告死亡或失踪
```

## 七、经济补偿

### 1. 经济补偿标准
```markdown
计算标准：
- 按劳动者在本单位工作的年限
- 每满一年支付一个月工资
- 六个月以上不满一年的，按一年计算
- 不满六个月的，向劳动者支付半个月工资

月工资标准：
- 劳动者在劳动合同解除或者终止前十二个月的平均工资
- 包括计时工资、计件工资、奖金、津贴和补贴等

高收入封顶：
- 劳动者月工资高于用人单位所在地区上年度职工月平均工资三倍的
- 支付经济补偿的标准按职工月平均工资三倍的数额支付
- 支付经济补偿的年限最高不超过十二年

计算公式：
经济补偿 = 工作年限 × 月工资
```

### 2. 经济补偿情形
```markdown
劳动者解除，用人单位需支付经济补偿：
- 用人单位未提供劳动条件
- 用人单位未支付劳动报酬
- 用人单位未缴纳社会保险
- 用人单位规章制度违法
- 用人单位以欺诈、胁迫手段订立合同
- 用人单位强迫劳动

用人单位解除，需支付经济补偿：
- 协商解除（用人单位提出）
- 非过失性辞退
- 经济性裁员

劳动合同终止，需支付经济补偿：
- 合同期满（用人单位不愿续签）
- 用人单位破产
- 用人单位被吊销执照
- 用人单位提前解散
```

### 3. 赔偿金
```markdown
违法解除或终止劳动合同赔偿金：

用人单位违法解除或终止劳动合同：
- 应当支付赔偿金
- 赔偿金标准为经济补偿标准的二倍

计算公式：
赔偿金 = 工作年限 × 月工资 × 2

劳动者要求继续履行劳动合同：
- 用人单位应当继续履行
- 不支付赔偿金

劳动者不要求继续履行或者已经不能继续履行：
- 用人单位应当支付赔偿金
```

## 八、劳动争议处理

### 1. 劳动争议范围
```markdown
劳动争议类型：
- 确认劳动关系争议
- 订立、履行、变更、解除和终止劳动合同争议
- 除名、辞退和辞职、离职争议
- 工作时间、休息休假、社会保险争议
- 劳动报酬、工伤医疗费、经济补偿争议
- 法律、法规规定的其他劳动争议
```

### 2. 劳动争议处理程序
```markdown
协商：
- 劳动者与用人单位协商
- 自愿原则
- 协商不成可申请调解

调解：
- 向劳动争议调解组织申请调解
- 调解组织：企业劳动争议调解委员会、
  依法设立的基层人民调解组织、乡镇街道劳动争议调解组织
- 调解期限：15日
- 调解不成可申请仲裁

仲裁：
- 向劳动争议仲裁委员会申请仲裁
- 劳动争议仲裁委员会由劳动行政部门代表、
  工会代表和企业代表组成
- 仲裁时效：一年
- 仲裁期限：45日，可延长15日

诉讼：
- 对仲裁裁决不服的，可向人民法院起诉
- 起诉期限：收到仲裁裁决书之日起15日内
- 一审、二审程序

一裁终局：
- 追索劳动报酬、工伤医疗费、经济补偿不超过当地月最低工资标准12个月金额的争议
- 因执行国家的劳动标准在工作时间、休息休假、社会保险等方面发生的争议
- 仲裁裁决为终局裁决
- 劳动者不服的，仍可向法院起诉
```

### 3. 举证责任

劳动争议举证规则：

用人单位举证责任：

- 减少劳动报酬
- 计算劳动者工作年限
- 用人单位应当提供证据

劳动者举证责任：
- 劳动者主张权利的存在
- 劳动者应当提供证据

举证责任倒置：
- 用人单位掌握管理的证据
- 用人单位不提供的
- 用人单位承担不利后果
