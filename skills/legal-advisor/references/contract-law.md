# 合同法要点

## 合同基础知识

### 合同类型
| 类型 | 特点 | 示例 |
|------|------|------|
| 买卖合同 | 货币换货物 | 采购合同 |
| 服务合同 | 货币换服务 | 服务协议 |
| 租赁合同 | 使用权转移 | 房屋租赁 |
| 借款合同 | 货币借贷 | 贷款合同 |
| 劳动合同 | 劳动关系 | 雇佣合同 |

### 合同成立要件
1. 主体合格：具有民事行为能力
2. 意思真实：真实意愿表达
3. 内容合法：不违反法律强制性规定
4. 形式合规：法律规定需要书面形式的必须书面

## 合同条款

### 必备条款
| 条款 | 内容要求 | 常见问题 |
|------|---------|---------|
| 当事人 | 名称、地址、联系方式 | 主体不准确 |
| 标的 | 交易对象明确 | 描述模糊 |
| 数量 | 计量单位和数量 | 单位不统一 |
| 质量 | 质量标准或要求 | 标准不明 |
| 价款 | 金额、支付方式、时间 | 付款条件不清 |
| 履行 | 履行期限、地点、方式 | 时间不明 |
| 违约责任 | 违约情形和责任 | 责任不对等 |
| 争议解决 | 仲裁或诉讼 | 约定不明 |

### 重要条款

**保密条款**
```
双方对在合同履行过程中知悉的对方商业秘密
负有保密义务，未经对方书面同意，
不得向第三方披露或使用。
保密期限为合同终止后X年。
```

**不可抗力条款**
```
因自然灾害、战争、政府行为等不可抗力
导致合同无法履行的，双方互不承担违约责任，
但应及时通知对方并提供证明。
```

**解约条款**
```
一方严重违约，守约方有权解除合同，
并要求违约方赔偿损失。
```

## 合同审核要点

### 风险审查清单
- [ ] 主体资格审查
- [ ] 经营范围核实
- [ ] 签约权限确认
- [ ] 标的物描述
- [ ] 价格条款核对
- [ ] 付款条件明确
- [ ] 违约责任对等
- [ ] 争议解决约定
- [ ] 签章真实性
- [ ] 日期完整性

### 常见陷阱
| 陷阱 | 表现 | 应对 |
|------|------|------|
| 霸王条款 | 单方免责、加重对方责任 | 修改或拒绝 |
| 模糊条款 | 合理期限、适当方式 | 明确具体标准 |
| 跳坑条款 | "最终解释权归甲方" | 删除或修改 |
| 担保陷阱 | 连带责任担保 | 慎重评估风险 |

## 常用合同模板要点

### 服务合同
1. 服务内容明确具体
2. 服务标准可量化
3. 服务期限和交付时间
4. 验收标准和程序
5. 费用及支付方式
6. 保密义务
7. 知识产权归属

### 合作协议
1. 合作内容与分工
2. 利润分配方式
3. 费用承担
4. 合作期限
5. 退出机制
6. 违约责任

### 保密协议
1. 保密信息范围
2. 保密义务
3. 保密期限
4. 违约责任
5. 例外情形
