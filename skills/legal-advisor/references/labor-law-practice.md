# 劳动法实务：初创企业用工指南

## 劳动法基础

### 适用范围
| 类型 | 适用法律 | 说明 |
|------|----------|------|
| 劳动关系 | 劳动合同法 | 标准用工 |
| 劳务关系 | 民法典 | 灵活用工 |
| 劳务派遣 | 劳动合同法 | 第三方派遣 |

### 劳动关系认定
| 因素 | 劳动关系特征 | 劳务关系特征 |
|------|--------------|--------------|
| 主体 | 用人单位+劳动者 | 平等主体 |
| 从属性 | 人格+经济从属 | 独立主体 |
| 管理 | 受用人单位管理 | 自主完成 |
| 报酬 | 定期工资 | 项目/约定报酬 |
| 社保 | 强制缴纳 | 无强制 |

## 劳动合同

### 合同类型
| 类型 | 期限 | 适用情况 | 注意事项 |
|------|------|----------|----------|
| 固定期限 | 约定时间 | 一般用工 | 最常用 |
| 无固定期限 | 无终止 | 满十年或连续两次固定期限 | 较难解除 |
| 完成一定任务 | 任务完成 | 项目制用工 | 明确任务 |

### 合同必备条款
```
劳动合同必备条款：
1. 用人单位信息（名称、住所、法定代表人）
2. 劳动者信息（姓名、身份证、住址）
3. 合同期限（起止时间）
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护和劳动条件
9. 法律法规规定的其他事项
```

### 试用期规定
| 合同期限 | 试用期上限 | 说明 |
|----------|------------|------|
| < 3个月 | 不得约定试用期 | 短期合同 |
| 3个月-1年 | ≤ 1个月 | 较短期 |
| 1年-3年 | ≤ 2个月 | 中期 |
| ≥ 3年或无固定期限 | ≤ 6个月 | 长期 |

**注意**：
- 试用期工资不低于正式工资的80%
- 同一用人单位只能约定一次试用期
- 试用期包含在合同期限内

### 合同签订时间
| 时间 | 要求 | 后果 |
|------|------|------|
| 用工前 | 最佳 | 合规 |
| 用工起1个月内 | 应签订 | 支付双倍工资风险 |
| 用工起1个月-1年未签 | 违法 | 支付11个月双倍工资 |
| 用工满1年未签 | 视为无固定期限 | 签订无固定期限合同 |

## 薪酬福利

### 工资支付
| 要求 | 说明 |
|------|------|
| 支付时间 | 按月支付，最迟次月15日 |
| 支付方式 | 货币支付，禁止实物 |
| 工资条 | 提供工资清单 |
| 不得克扣 | 法定情形外不得克扣 |

### 最低工资
| 项目 | 说明 |
|------|------|
| 标准 | 各地区不同 |
| 内容 | 不含加班、津贴、福利 |
| 试用期待遇 | 不低于最低工资 |

### 加班工资
| 时间 | 计算标准 |
|------|----------|
| 工作日延长 | 150%工资 |
| 休息日工作 | 200%工资（或调休） |
| 法定节假日 | 300%工资 |

### 社会保险
| 险种 | 缴费比例（单位+个人） | 说明 |
|------|------------------------|------|
| 养老保险 | 16% + 8% | 累计缴费满15年退休领取 |
| 医疗保险 | 9.5% + 2% | 看病报销 |
| 失业保险 | 0.5% + 0.5% | 失业领取 |
| 工伤保险 | 0.2-1.9% | 单位缴纳，工作伤害赔付 |
| 生育保险 | 0.8% | 单位缴纳，生育报销 |

**注意**：2024年起，部分地区五险合并申报，具体以当地政策为准。

### 公积金
| 项目 | 说明 |
|------|------|
| 缴费比例 | 5%-12%（单位+个人） |
| 缴费基数 | 上年度月平均工资 |
| 用途 | 购房、租房、装修等 |

## 工作时间与休假

### 标准工时
| 类型 | 时长 | 说明 |
|------|------|------|
| 标准工时 | 每日8小时，每周40小时 | 最常用 |
| 综合计算工时 | 按周期计算 | 季度、年度 |
| 不定时工时 | 不计算加班 | 需审批 |

### 休息休假
| 假期 | 天数 | 条件 |
|------|------|------|
| 法定节假日 | 11天 | 全员 |
| 年休假 | 5-15天 | 工作满1年 |
| 婚假 | 3天+各地规定 | 结婚 |
| 丧假 | 1-3天 | 直系亲属去世 |
| 产假 | 98天+各地规定 | 女职工生育 |
| 陪产假 | 7-30天（各地不同） | 男职工 |
| 病假 | 根据工龄 | 生病 |

### 年休假标准
| 工龄 | 年休假天数 |
|------|------------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

## 劳动关系解除

### 解除方式
| 方式 | 通知期 | 经济补偿 | 说明 |
|------|--------|----------|------|
| 协商解除 | 无 | 约定 | 双方同意 |
| 员工辞职 | 提前30日 | 无 | 员工自愿 |
| 员工即时辞职 | 无 | 有 | 单位过错 |
| 单位辞退（过失） | 无 | 无 | 员工过错 |
| 单位辞退（无过失） | 提前30日或支付一月工资 | 有 | 单位原因 |
| 经济性裁员 | 提前30日通知工会 | 有 | 经营困难 |

### 单位可解除情形（过失性）
| 情形 | 说明 |
|------|------|
| 试用期不符合录用条件 | 需证明 |
| 严重违反规章制度 | 制度需合法且公示 |
| 严重失职，营私舞弊，造成重大损害 | 需证明损害 |
| 同时与其他单位建立劳动关系 | 经提出拒不改正 |
| 被追究刑事责任 | 刑事处罚 |

### 单位可解除情形（无过失性）
| 情形 | 说明 |
|------|------|
| 患病或非因工负伤，不能从事原工作 | 需医疗期满 |
| 不能胜任工作 | 需培训或调岗 |
| 客观情况发生重大变化 | 合同无法履行 |

### 经济补偿标准
```
经济补偿 = 工作年限 × 月工资

规则：
- 每满1年支付1个月工资
- 6个月以上不满1年按1年计算
- 不满6个月支付半个月工资
- 月工资指离职前12个月平均工资
- 月工资高于当地平均工资3倍的，按3倍支付，最高12年
```

### 单位不得解除情形
| 情形 | 说明 |
|------|------|
| 患职业病或因工负伤 | 被确认丧失劳动能力 |
| 患病或非因工负伤 | 在规定的医疗期内 |
| 女职工孕期、产期、哺乳期 | 三期内 |
| 法律规定的其他情形 | |

## 工伤认定

### 工伤范围
| 情形 | 说明 |
|------|------|
| 工作时间和工作场所内，因工作原因受到事故伤害 | 典型工伤 |
| 工作时间前后，从事与工作有关的预备性或收尾性工作受到事故伤害 | 工作延伸 |
| 工作时间和工作场所内，因履行工作职责受到暴力等意外伤害 | 工作原因 |
| 患职业病 | 职业病目录 |
| 因工外出期间，由于工作原因受到伤害或发生事故下落不明 | 外出工伤 |
| 上下班途中，受到非本人主要责任的交通事故伤害 | 通勤工伤 |

### 工伤认定流程
```
1. 发生工伤 → 及时送医 → 保留证据
2. 30日内 → 单位申请工伤认定
3. 单位不申请 → 员工/家属可在1年内申请
4. 社保部门 → 调查核实 → 作出认定
5. 认定工伤 → 劳动能力鉴定 → 工伤待遇
```

### 工伤待遇
| 项目 | 支付方 | 标准 |
|------|--------|------|
| 医疗费 | 工伤保险 | 符合目录范围 |
| 停工留薪期工资 | 用人单位 | 原工资福利 |
| 伤残津贴 | 工伤保险 | 按伤残等级 |
| 一次性伤残补助金 | 工伤保险 | 按伤残等级 |
| 一次性工伤医疗补助金 | 工伤保险 | 按当地标准 |
| 一次性伤残就业补助金 | 用人单位 | 按当地标准 |

## 劳动争议处理

### 处理流程
```
协商 → 调解 → 仲裁 → 诉讼

时效：
- 协商、调解：无时效限制
- 劳动仲裁：知道权利被侵害之日起1年内
- 民事诉讼：收到仲裁裁决书之日起15日内
```

### 证据收集
| 类型 | 示例 |
|------|------|
| 劳动关系证据 | 劳动合同、工资条、社保记录、工牌 |
| 工资证据 | 银行流水、工资条、转账记录 |
| 考勤证据 | 考勤表、打卡记录、工作群记录 |
| 解除证据 | 解除通知书、录音、邮件 |

### 常见争议类型
| 争议类型 | 处理要点 |
|----------|----------|
| 确认劳动关系 | 证明从属性 |
| 劳动报酬争议 | 提供工资约定证据 |
| 经济补偿争议 | 计算工作年限、工资标准 |
| 工伤待遇争议 | 工伤认定、伤残等级 |
| 违法解除争议 | 解除原因、程序是否合法 |

## 初创企业用工风险

### 常见风险
| 风险 | 后果 | 防范 |
|------|------|------|
| 未签劳动合同 | 双倍工资（最多11个月） | 入职30日内签订 |
| 未缴社保 | 补缴+滞纳金+罚款 | 按时缴纳 |
| 违法解除 | 继续履行或双倍赔偿 | 依法解除 |
| 加班争议 | 加班费+赔偿 | 规范加班管理 |
| 工伤争议 | 工伤待遇+罚款 | 缴纳工伤保险 |

### 用工成本估算
| 项目 | 占比（约） | 说明 |
|------|------------|------|
| 工资 | 100% | 基础 |
| 社保单位部分 | 约30% | 养老16%+医疗9.5%+失业0.5%+工伤1%+生育0.8%+公积金5-12% |
| 公积金单位部分 | 5-12% | 各地标准不同 |
| 其他福利 | 5-10% | 餐补、交通补等 |
| **总成本** | **140-155%** | - |

## 灵活用工方式

### 劳务外包
| 项目 | 说明 |
|------|------|
| 适用 | 非核心业务 |
| 关系 | 与外包公司签约 |
| 风险 | 较低 |
| 成本 | 按项目/时长计费 |

### 劳务派遣
| 项目 | 说明 |
|------|------|
| 适用 | 临时性、辅助性、替代性岗位 |
| 限制 | 派遣工不超过用工总量10% |
| 关系 | 与派遣公司签约 |
| 责任 | 用工单位连带责任 |

### 实习生
| 项目 | 说明 |
|------|------|
| 适用 | 在校学生 |
| 关系 | 实习协议（非劳动合同） |
| 待遇 | 实习补贴（非工资） |
| 社保 | 无需缴纳社保 |

### 兼职
| 项目 | 说明 |
|------|------|
| 适用 | 灵活用工 |
| 关系 | 劳务关系或非全日制用工 |
| 时长 | 每周不超过24小时 |
| 社保 | 仅工伤保险 |

## 常用文书模板

### 劳动合同要点
```markdown
## 劳动合同（要点）

甲方：VideoInsight科技有限公司
乙方：[员工姓名]

一、合同期限
自 [日期] 至 [日期]，试用期 [天数]。

二、工作内容
岗位：[岗位名称]
工作地点：[地址]
工作职责：[职责描述]

三、工作时间
标准工时制，每日8小时，每周40小时。

四、劳动报酬
试用期工资：[金额] 元/月
转正工资：[金额] 元/月
发薪日：每月15日

五、社会保险
依法缴纳社会保险和住房公积金。

六、劳动保护
提供必要的劳动保护和劳动条件。

七、合同解除
按《劳动合同法》规定执行。

八、违约责任
按法律规定执行。

九、争议解决
协商不成，可申请劳动仲裁。

甲方盖章：              乙方签字：
日期：                  日期：
```


## 总结

### 初创企业用工要点
| 要点 | 说明 |
|------|------|
| 签合同 | 入职30日内必须签订书面劳动合同 |
| 缴社保 | 依法缴纳，不可约定不缴 |
| 定制度 | 员工手册、考勤制度需民主程序制定并公示 |
| 留证据 | 招聘、考核、解除等环节保留证据 |
| 按程序 | 解除劳动合同需依法定程序 |
