# 公司法：公司治理结构

## 一、公司治理概述

### 1. 公司治理定义
公司治理是指通过一套制度安排，明确公司各利益相关者的权利与义务，规范公司决策、执行、监督机制，确保公司高效运行并保护各方合法权益的制度体系。

### 2. 公司治理原则
```markdown
原则一：股东主权原则
- 股东是公司的所有者
- 股东享有最终控制权
- 保护股东合法权益

原则二：权责清晰原则
- 明确各机构职责边界
- 权利与责任对等
- 避免职责交叉或真空

原则三：制衡有效原则
- 决策、执行、监督分离
- 形成有效制约机制
- 防止权力滥用

原则四：信息公开原则
- 确保信息真实准确
- 及时披露重大事项
- 保护知情权
```

### 3. 公司治理结构
```
                    ┌─────────────┐
                    │   股东会    │ ← 权力机构
                    │  (股东大   │
                    └──────┬──────┘
                           │ 选举/罢免
                    ┌──────▼──────┐
                    │   董事会    │ ← 决策机构
                    └──────┬──────┘
                           │ 聘任/解聘
              ┌────────────┼────────────┐
              │            │            │
       ┌──────▼──────┐ ┌────▼────┐ ┌─────▼─────┐
       │   经理层    │ │ 监事会  │ │ 各专门委员会 │
       │  (管理层)   │ │ (监督机构)│ │           │
       └─────────────┘ └─────────┘ └───────────┘
```

## 二、股东会（股东大会）

### 1. 性质与职权
股东会是公司的权力机构，由全体股东组成。

#### 法定职权（《公司法》第37条）
```markdown
1. 决定公司的经营方针和投资计划
2. 选举和更换非由职工代表担任的董事、监事
3. 审议批准董事会、监事会报告
4. 审议批准公司年度财务预决算方案
5. 审议批准公司利润分配方案和弥补亏损方案
6. 对公司增加或减少注册资本作出决议
7. 对发行公司债券作出决议
8. 对公司合并、分立、解散、清算作出决议
9. 修改公司章程
10. 公司章程规定的其他职权
```

### 2. 会议类型
| 类型 | 召开时间 | 召开条件 | 审议事项 |
|------|----------|----------|----------|
| 年度股东会 | 每年一次 | 应当召开 | 年度报告、利润分配等 |
| 临时股东会 | 适时 | 符合条件时 | 特定重大事项 |

#### 临时股东会召开条件
```markdown
1. 董事人数不足法定人数或章程规定人数的2/3
2. 公司未弥补的亏损达实收股本总额1/3
3. 单独或合计持有公司10%以上股份的股东请求
4. 董事会认为必要
5. 监事会提议召开
6. 公司章程规定的其他情形
```

### 3. 召开程序
```markdown
会议通知：
- 有限责任公司：召开15日前通知全体股东（章程另有规定除外）
- 股份有限公司：召开20日前通知各股东（临时会议15日前）

通知内容：
- 会议时间、地点
- 审议事项
- 表决权行使方式

表决规则：
- 有限责任公司：按出资比例行使表决权（章程另有规定除外）
- 股份有限公司：按持股比例行使表决权
- 特别决议：须经出席会议股东所持表决权2/3以上通过
```

### 4. 表决机制
| 决议类型 | 通过标准 | 适用事项 |
|----------|----------|----------|
| 普通决议 | 出席股东表决权过半数 | 一般事项 |
| 特别决议 | 出席股东表决权2/3以上 | 重大事项 |

#### 特别决议事项
```markdown
- 增加或减少注册资本
- 公司合并、分立、解散
- 变更公司形式
- 修改公司章程
- 公司章程规定的其他事项
```

## 三、董事会

### 1. 性质与组成
董事会是公司的决策机构，对股东会负责。

#### 董事会组成
```markdown
人数要求：
- 有限责任公司：3-13人
- 股份有限公司：5-19人
- 规模较小的有限公司：可设1名执行董事

董事任期：
- 每届任期不超过3年
- 连选可以连任
- 无正当理由不得辞职

职工董事：
- 国有独资公司：应当有职工董事
- 其他公司：可以设立
```

### 2. 董事会职权
```markdown
1. 召集股东会会议，执行股东会决议
2. 决定公司的经营计划和投资方案
3. 制订公司年度财务预决算方案
4. 制订公司利润分配方案和弥补亏损方案
5. 制订公司增减注册资本、发行债券方案
6. 制订公司合并、分立、解散方案
7. 决定公司内部管理机构的设置
8. 决定聘任或解聘公司经理及其报酬
9. 制定公司的基本管理制度
10. 公司章程规定的其他职权
```

### 3. 会议制度
```markdown
会议召开：
- 定期会议：每年至少2次（股份有限公司）
- 临时会议：代表1/10以上表决权的股东、1/3以上董事或监事会提议

表决规则：
- 一人一票（不是按持股比例）
- 须经全体董事过半数通过

会议记录：
- 董事应当对决议承担责任
- 决议违反法律法规或章程的，参与决议的董事承担责任
- 表决时表明异议的董事可免除责任
```

### 4. 专门委员会
上市公司应设立以下专门委员会：
| 委员会 | 职责 | 独立董事占比 |
|--------|------|--------------|
| 战略委员会 | 发展战略、重大投资决策 | 过半数 |
| 审计委员会 | 财务审计、内控监督 | 过半数 |
| 提名委员会 | 董事、高管人选 | 过半数 |
| 薪酬与考核委员会 | 薪酬方案、绩效考核 | 过半数 |

## 四、经理层

### 1. 经理的地位
经理由董事会聘任或解聘，对董事会负责，是公司的执行机构。

### 2. 经理职权
```markdown
1. 主持公司的生产经营管理工作
2. 组织实施董事会决议
3. 组织实施公司年度经营计划和投资方案
4. 拟订公司内部管理机构设置方案
5. 拟订公司的基本管理制度
6. 制定公司的具体规章
7. 提请聘任或解聘公司副经理、财务负责人
8. 决定聘任或解聘除应由董事会决定的人员
9. 董事会授予的其他职权
```

### 3. 高级管理人员
```markdown
高级管理人员范围：
- 总经理/CEO
- 副总经理
- 财务负责人（CFO）
- 董事会秘书（上市公司）
- 公司章程规定的其他人员

任职资格禁止：
- 无民事行为能力或限制民事行为能力
- 因贪污、贿赂、侵占财产等被判刑，执行期满未逾5年
- 担任破产企业董事、厂长、经理，对企业破产负有个人责任，自破产清算完结未逾3年
- 担任违法被吊销营业执照企业的法定代表人，负有个人责任，自被吊销未逾3年
- 个人所负数额较大的债务到期未清偿
```

## 五、监事会

### 1. 性质与组成
监事会是公司的监督机构，对股东会负责。

#### 监事会组成
```markdown
人数要求：
- 有限责任公司：不少于3人（规模小的可设1-2名监事）
- 股份有限公司：不少于3人

组成结构：
- 股东代表监事：由股东会选举产生
- 职工代表监事：由职工代表大会选举产生
- 职工代表比例：不得低于1/3

任期：
- 每届任期3年
- 连选可以连任
```

### 2. 监事会职权
```markdown
1. 检查公司财务
2. 对董事、高级管理人员执行职务的行为进行监督
3. 对违反法律、法规、章程的董事、高管提出罢免建议
4. 当董事、高管损害公司利益时要求其纠正
5. 提议召开临时股东会会议
6. 向股东会会议提出提案
7. 对董事、高管提起诉讼
8. 发现公司经营情况异常时进行调查
```

### 3. 监事会会议
```markdown
召开频率：
- 有限责任公司：每年至少1次
- 股份有限公司：每6个月至少1次

表决规则：
- 决议应当经全体监事过半数通过
- 一人一票

会议记录：
- 应当对所议事项的决定作成会议记录
- 出席会议的监事应当在会议记录上签名
```

## 六、股东权利保护

### 1. 知情权
```markdown
股东有权查阅、复制：
- 公司章程
- 股东会会议记录
- 董事会会议决议
- 监事会会议决议
- 财务会计报告

有限公司股东还可查阅：
- 会计账簿（需书面说明目的）
```

### 2. 表决权
```markdown
- 按出资比例或持股比例行使
- 可委托他人代为行使
- 公司不得接受本公司股票作为质押标的
```

### 3. 利润分配请求权
```markdown
分配条件：
- 弥补亏损
- 提取法定公积金
- 提取任意公积金（由股东会决定）

分配方式：
- 有限责任公司：按实缴出资比例分配（章程另有规定除外）
- 股份有限公司：按股东持有的股份比例分配（章程另有规定除外）
```

### 4. 股东诉讼权利
```markdown
股东代表诉讼：
- 条件：连续180日单独或合计持有1%以上股份
- 程序：先书面请求监事会/董事会起诉，被拒绝或在30日内未起诉
- 结果：股东以自己名义起诉，利益归于公司

股东直接诉讼：
- 董事、高管违反法律法规或章程规定，损害股东利益
- 股东可向人民法院提起诉讼
```

## 七、公司治理常见问题

### 1. 一股独大
```markdown
表现：
- 大股东控制股东会、董事会
- 中小股东权益得不到保护
- 公司决策缺乏制衡

对策：
- 完善累积投票制度
- 引入独立董事
- 加强中小股东保护
- 信息披露制度
```

### 2. 内部人控制
```markdown
表现：
- 管理层实际控制公司
- 股东会形同虚设
- 损害股东利益

对策：
- 加强监事会监督职能
- 完善董事会结构
- 引入外部董事
- 建立有效的激励机制
```

### 3. 董事会失灵
```markdown
表现：
- 董事会决策流于形式
- 独立董事不独立
- 专门委员会虚设

对策：
- 优化董事会结构
- 完善董事选任机制
- 强化董事责任
- 建立董事评价机制
```

## 八、公司治理最佳实践

### 1. 股东会运作规范
```markdown
- 严格履行会议通知程序
- 规范会议记录和签到
- 充分保障股东知情权
- 合理安排会议议程
- 规范表决程序
```

### 2. 董事会建设要点
```markdown
- 优化董事会人员结构
- 引入独立董事和外部董事
- 设立专门委员会并有效运作
- 完善董事会议事规则
- 建立董事考核评价机制
```

### 3. 监事会履职保障
```markdown
- 保证监事会独立性
- 提供必要的工作条件
- 监事有权查阅相关资料
- 建立监事履职保障机制
```

### 4. 信息披露管理
```markdown
- 建立信息披露管理制度
- 确保信息真实、准确、完整
- 及时披露重大事项
- 保护投资者知情权
```
