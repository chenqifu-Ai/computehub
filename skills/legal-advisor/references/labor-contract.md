# 劳动合同法律实务

## 劳动合同签订

### 必备条款
| 条款 | 内容 |
|------|------|
| 用人单位信息 | 名称、住所、法定代表人 |
| 劳动者信息 | 姓名、住址、身份证号 |
| 合同期限 | 固定期限/无固定期限 |
| 工作内容 | 岗位、职责 |
| 工作地点 | 工作场所 |
| 工作时间 | 标准工时/综合工时 |
| 休息休假 | 法定休假 |
| 劳动报酬 | 工资数额、支付方式 |
| 社会保险 | 社保缴纳 |
| 劳动保护 | 安全卫生条件 |

### 约定条款
| 条款 | 说明 |
|------|------|
| 试用期 | 约定试用期 |
| 培训 | 培训服务期 |
| 保密 | 商业保密 |
| 竞业限制 | 竞业限制 |
| 补充福利 | 补充保险、福利 |

## 试用期规定

### 试用期期限
| 合同期限 | 试用期上限 |
|----------|------------|
| 3个月-1年 | 1个月 |
| 1年-3年 | 2个月 |
| 3年以上 | 6个月 |

### 试用期工资
| 规定 | 说明 |
|------|------|
| 最低标准 | 不得低于本单位同岗位最低档工资80% |
| 最低工资 | 不得低于用人单位所在地区最低工资标准 |

### 试用期解除
| 情形 | 说明 |
|------|------|
| 不胜任 | 提供证据，可解除 |
| 违纪 | 证明违纪事实 |
| 协商 | 协商一致解除 |

## 劳动合同变更

### 变更原则
- 协商一致
- 书面形式
- 不违反法律

### 变更情形
| 情形 | 说明 |
|------|------|
| 协商变更 | 双方协商一致 |
| 客观情况变更 | 重大客观情况变化 |
| 医疗期满 | 医疗期满不能从事原工作 |
| 不胜任 | 不能胜任工作 |

## 劳动合同解除

### 员工解除
| 类型 | 条件 | 程序 |
|------|------|------|
| 提前解除 | 提前30天书面通知 | 书面通知 |
| 即时解除 | 企业违法 | 无需提前通知 |

### 企业解除
| 类型 | 条件 | 补偿 |
|------|------|------|
| 过失性解除 | 严重违纪、不胜任等 | 无补偿 |
| 非过失性解除 | 医疗期满、不能胜任等 | N或N+1 |
| 经济性裁员 | 经营困难、转产等 | N+1 |

### 不能解除情形
| 情形 | 说明 |
|------|------|
| 职业病 | 患职业病或工伤 |
| 医疗期 | 在规定的医疗期内 |
| 孕期 | 女职工孕期、产期、哺乳期 |
| 临近退休 | 在本单位连续工作满15年 |

## 经济补偿

### 补偿标准
```
经济补偿 = 工作年限 × 离职前12个月平均工资

工作年限计算：
- 满一年：1个月工资
- 6个月-1年：1个月工资
- 不满6个月：0.5个月工资

高收入封顶：
- 月工资高于当地平均工资3倍
- 补偿年限最高12年
```

### 补偿情形
| 情形 | 补偿标准 |
|------|----------|
| 协商解除 | N |
| 企业提出解除 | N或N+1 |
| 经济性裁员 | N+1 |
| 合同到期不续签 | N |

## 劳动争议处理

### 争议类型
| 类型 | 案例 |
|------|------|
| 工资争议 | 拖欠工资、加班费 |
| 解除争议 | 违法解除、补偿金 |
| 工伤争议 | 工伤认定、赔偿 |
| 合同争议 | 合同签订、变更 |

### 处理流程
```
协商 → 调解 → 仲裁 → 诉讼

协商：双方协商解决
调解：企业调解委员会
仲裁：劳动争议仲裁委员会（60日内）
诉讼：人民法院（15日内）
```

### 举证责任
| 情形 | 举证责任 |
|------|----------|
| 工资争议 | 用人单位举证 |
| 解除争议 | 用人单位举证 |
| 工伤争议 | 用人单位举证 |

## 劳动合同风险防范

### 签订风险
| 风险 | 防范措施 |
|------|----------|
| 未签合同 | 入职30日内签订 |
| 条款缺失 | 使用标准合同 |
| 约定违法 | 合法约定 |

### 履行风险
| 风险 | 防范措施 |
|------|----------|
| 变更未书面 | 及时签订变更协议 |
| 加班争议 | 规范加班审批 |
| 工资争议 | 工资条签字确认 |

### 解除风险
| 风险 | 防范措施 |
|------|----------|
| 违法解除 | 依法解除 |
| 证据不足 | 保留证据 |
| 程序违规 | 规范程序 |
