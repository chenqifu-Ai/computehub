# 劳动法：劳动争议处理

## 一、劳动争议概述

### 1. 劳动争议定义
劳动争议是指劳动者与用人单位之间因劳动权利和义务发生分歧而引起的争议。

### 2. 劳动争议范围
```markdown
因确认劳动关系发生的争议
因订立、履行、变更、解除和终止劳动合同发生的争议
因除名、辞退和辞职、离职发生的争议
因工作时间、休息休假、社会保险、福利、培训以及劳动保护发生的争议
因劳动报酬、工伤医疗费、经济补偿或者赔偿金等发生的争议
法律法规规定的其他劳动争议
```

### 3. 劳动争议特点
| 特点 | 说明 |
|------|------|
| 主体特定 | 一方是劳动者，一方是用人单位 |
| 内容特定 | 围绕劳动权利义务 |
| 处理程序特定 | 仲裁前置 |
| 时效特定 | 仲裁时效为一年 |

## 二、劳动争议处理原则

### 1. 基本原则
```markdown
合法原则：
- 依据法律法规处理
- 处理程序合法
- 处理结果合法

公正原则：
- 平等对待双方当事人
- 客观认定事实
- 公正适用法律

及时处理原则：
- 遵守法定时限
- 提高处理效率
- 防止争议扩大

着重调解原则：
- 调解贯穿处理全过程
- 鼓励协商解决
- 调解不成及时裁决
```

### 2. 处理方式
```markdown
协商：双方自行协商解决
调解：通过调解组织调解
仲裁：通过劳动争议仲裁委员会仲裁
诉讼：通过人民法院诉讼
```

## 三、劳动争议处理程序

### 1. 处理流程
```
┌────────┐    ┌────────┐    ┌────────┐    ┌────────┐
│  协商  │ →  │  调解  │ →  │  仲裁  │ →  │  诉讼  │
└────────┘    └────────┘    └────────┘    └────────┘
   自愿           自愿         必经程序        终局
```

### 2. 仲裁前置原则
```markdown
法律规定：
- 劳动争议必须先经过仲裁
- 对仲裁裁决不服才能向法院起诉
- 仲裁是诉讼的前置程序

例外情况：
- 追索劳动报酬、工伤医疗费、经济补偿金，不超过当地月最低工资标准12个月金额的争议
- 因执行国家的劳动标准在工作时间、休息休假、社会保险等方面发生的争议
- 以上情况，仲裁裁决为终局裁决
```

## 四、协商与调解

### 1. 协商
```markdown
协商特点：
- 双方自愿
- 自主进行
- 没有固定程序
- 协商结果不具有强制力

协商优势：
- 成本最低
- 时间最快
- 保护双方关系
- 自主灵活

协商建议：
- 明确争议焦点
- 准备相关证据
- 了解法律规定
- 保持理性沟通
```

### 2. 调解

#### 调解组织
| 类型 | 说明 |
|------|------|
| 企业劳动争议调解委员会 | 用人单位内部设立 |
| 人民调解委员会 | 基层群众自治组织 |
| 乡镇街道劳动争议调解中心 | 政府设立的调解组织 |
| 行业性调解组织 | 行业协会设立的调解组织 |

#### 调解程序
```markdown
申请调解：
- 当事人提出口头或书面申请
- 调解组织收到申请后登记受理

调解过程：
- 调解员听取双方意见
- 调查核实相关事实
- 进行调解沟通
- 促使双方达成协议

调解结果：
- 调解成功：制作调解协议书，双方签字
- 调解不成：告知当事人可以申请仲裁
- 调解期限：一般15日内完成

调解协议效力：
- 调解协议书经双方签字后生效
- 具有法律约束力，当事人应当履行
- 一方不履行，另一方可以申请仲裁
- 经司法确认后可申请强制执行
```

## 五、劳动仲裁

### 1. 仲裁委员会

#### 管辖规定
```markdown
一般管辖：
- 劳动争议由劳动合同履行地劳动争议仲裁委员会管辖
- 劳动争议由用人单位所在地的劳动争议仲裁委员会管辖

双方当事人分别申请：
- 由劳动合同履行地的劳动争议仲裁委员会管辖

选择管辖：
- 当事人可以书面协议选择管辖
- 但不得违反级别管辖和专属管辖
```

#### 仲裁时效
```markdown
一般时效：
- 仲裁时效期间为一年
- 从当事人知道或者应当知道权利被侵害之日起计算

特殊规定：
- 劳动关系存续期间因拖欠劳动报酬发生争议的，不受一年仲裁时效限制
- 劳动关系终止的，应当自劳动关系终止之日起一年内提出

时效中止：
- 因不可抗力或有其他正当理由，仲裁时效中止
- 中止原因消除后，仲裁时效期间继续计算

时效中断：
- 一方当事人通过协商、申请调解等方式向对方主张权利
- 一方当事人向有关部门投诉、举报
- 对方当事人同意履行义务
- 从中断时起，仲裁时效重新计算
```

### 2. 仲裁程序

#### 申请与受理
```markdown
申请条件：
- 有明确的被申请人
- 有具体的仲裁请求和事实理由
- 属于劳动争议仲裁委员会受理范围
- 属于该仲裁委员会管辖

申请材料：
- 劳动争议仲裁申请书
- 申请人身份证明
- 被申请人信息
- 证据材料
- 委托代理人的授权委托书

受理期限：
- 收到申请后5日内决定是否受理
- 受理后5日内将申请书副本送达被申请人
- 被申请人收到后10日内提交答辩书
```

#### 开庭审理
```markdown
开庭准备：
- 仲裁庭组成
- 开庭通知（开庭5日前）
- 当事人阅卷

庭审程序：
1. 开庭：核对当事人，宣布案由
2. 调查：申请人陈述请求、被申请人答辩、举证质证
3. 辩论：双方就争议焦点进行辩论
4. 调解：仲裁庭应当先行调解
5. 裁决：调解不成的，作出裁决

审理期限：
- 仲裁庭应当自仲裁委员会受理仲裁申请之日起45日内结案
- 案情复杂需要延期的，经批准可以延期15日
```

#### 仲裁裁决
```markdown
裁决类型：

终局裁决：
适用范围：
- 追索劳动报酬、工伤医疗费、经济补偿金，不超过当地月最低工资标准12个月金额的争议
- 因执行国家劳动标准在工作时间、休息休假、社会保险等方面发生的争议

效力：
- 裁决书自作出之日起发生法律效力
- 用人单位不得向法院起诉
- 劳动者不服可以自收到裁决书之日起15日内向法院起诉

非终局裁决：
- 除终局裁决外的其他裁决
- 当事人不服可以自收到裁决书之日起15日内向法院起诉
- 期满不起诉的，裁决书发生法律效力
```

### 3. 仲裁费用
```markdown
费用规定：
- 劳动争议仲裁不收费
- 仲裁委员会的经费由财政予以保障
```

## 六、劳动诉讼

### 1. 诉讼管辖
```markdown
级别管辖：
- 基层人民法院管辖第一审劳动争议案件
- 中级人民法院管辖重大复杂案件

地域管辖：
- 用人单位所在地或劳动合同履行地基层人民法院
- 劳动合同履行地不明确的，由用人单位所在地基层人民法院管辖
```

### 2. 起诉条件
```markdown
起诉条件：
- 有明确的被告
- 有具体的诉讼请求和事实理由
- 属于人民法院受理民事诉讼的范围和受诉人民法院管辖
- 已经过劳动争议仲裁委员会仲裁（仲裁前置）

起诉期限：
- 自收到仲裁裁决书之日起15日内
- 超过期限的，人民法院不予受理
```

### 3. 诉讼程序

#### 第一审程序
```markdown
起诉：
- 提交起诉状
- 人民法院审查立案

审理前准备：
- 送达起诉状副本
- 送达答辩状副本
- 组织证据交换

开庭审理：
- 开庭3日前通知当事人
- 法庭调查
- 法庭辩论
- 最后陈述
- 调解或判决

审理期限：
- 适用普通程序的，应当在立案之日起6个月内审结
- 适用简易程序的，应当在立案之日起3个月内审结
```

#### 第二审程序
```markdown
上诉条件：
- 当事人不服一审判决
- 在判决书送达之日起15日内提起上诉

上诉审理：
- 第二审人民法院对上诉案件进行审理
- 第二审判决是终审判决

审理期限：
- 第二审案件应当在第二审立案之日起3个月内审结
```

### 4. 执行程序
```markdown
执行依据：
- 生效的仲裁裁决书
- 生效的民事判决书、调解书

执行申请：
- 向人民法院申请强制执行
- 申请执行的期限为二年

执行措施：
- 查询、冻结、划拨存款
- 扣留、提取收入
- 查封、扣押、拍卖财产
- 限制高消费
- 列入失信被执行人名单
```

## 七、常见劳动争议类型

### 1. 劳动报酬争议
```markdown
常见争议：
- 拖欠工资
- 加班费计算
- 奖金发放
- 工资标准争议

举证责任：
- 用人单位掌握工资支付凭证
- 用人单位应当提供工资支付凭证
- 用人单位不提供的，承担不利后果

计算标准：
- 加班工资：工作日150%、休息日200%、法定节假日300%
- 工资标准：按合同约定，约定不明的按集体合同或同工同酬
```

### 2. 解除劳动合同争议
```markdown
常见争议：
- 违法解除
- 经济补偿金计算
- 代通知金
- 未提前通知

用人单位违法解除：
- 继续履行劳动合同
- 或支付赔偿金（经济补偿金的二倍）

经济补偿金计算：
- 按劳动者在本单位工作年限
- 每满一年支付一个月工资
- 六个月以上不满一年的按一年计算
- 不满六个月的支付半个月工资
- 月工资指解除前12个月平均工资
```

### 3. 工伤争议
```markdown
常见争议：
- 工伤认定
- 工伤待遇
- 工伤赔偿

工伤认定：
- 职工发生事故伤害或被诊断鉴定为职业病
- 所在单位应当自事故伤害发生之日或被诊断鉴定为职业病之日起30日内申请工伤认定
- 用人单位未申请的，工伤职工可以在1年内申请

工伤待遇：
- 医疗费
- 停工留薪期工资
- 护理费
- 一次性伤残补助金
- 一次性工伤医疗补助金
- 一次性伤残就业补助金
```

### 4. 社会保险争议
```markdown
常见争议：
- 未缴纳社会保险
- 少缴社会保险
- 社会保险待遇

处理方式：
- 缴费争议：由社会保险费征收机构处理
- 待遇争议：可以申请劳动仲裁

注意：
- 社会保险补缴一般不属于劳动争议受案范围
- 应当由社会保险行政部门处理
- 但因社会保险待遇造成的损失争议可以申请仲裁
```

## 八、劳动争议处理技巧

### 1. 劳动者角度
```markdown
证据收集：
- 保留劳动合同
- 收集工资条、银行流水
- 保存考勤记录
- 收集工作证、工作服等
- 保留微信聊天记录、邮件
- 录音录像（注意合法性）

仲裁申请：
- 明确仲裁请求
- 列明事实理由
- 提供证据材料
- 注意仲裁时效

诉讼准备：
- 准备起诉状
- 整理证据目录
- 了解诉讼程序
- 考虑是否委托代理人
```

### 2. 用人单位角度
```markdown
制度建设：
- 完善规章制度
- 签订劳动合同
- 规范用工管理
- 保留管理证据

争议应对：
- 积极参与调解
- 依法应诉答辩
- 提供相关证据
- 合法合规处理

风险防范：
- 依法签订劳动合同
- 按时足额支付工资
- 依法缴纳社会保险
- 规范解除劳动合同
```

## 九、劳动争议处理期限汇总

| 程序 | 时限 | 起点 |
|------|------|------|
| 仲裁时效 | 1年 | 知道权利被侵害之日起 |
| 仲裁受理 | 5日 | 收到仲裁申请 |
| 答辩期 | 10日 | 收到申请书副本 |
| 仲裁审理 | 45日 | 受理之日（可延15日） |
| 起诉期限 | 15日 | 收到裁决书 |
| 一审审理 | 6个月 | 立案之日（普通程序） |
| 一审审理 | 3个月 | 立案之日（简易程序） |
| 二审审理 | 3个月 | 第二审立案 |
| 申请执行 | 2年 | 法律文书生效 |
