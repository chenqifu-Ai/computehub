# 公司法务实务

## 公司设立

### 设立流程
| 步骤 | 内容 | 时间 | 费用 |
|------|------|------|------|
| 核名 | 公司名称预先核准 | 1-3天 | 免费 |
| 提交材料 | 提交设立申请 | 3-5天 | 免费 |
| 领取执照 | 领取营业执照 | 1天 | 免费 |
| 刻章 | 刻制公章、财务章等 | 1-2天 | 200-500元 |
| 银行开户 | 开设基本账户 | 3-7天 | 免费-500元 |
| 税务登记 | 税务登记、领发票 | 1-3天 | 免费 |
| 社保开户 | 社保、公积金开户 | 1-3天 | 免费 |

### 设立材料
| 材料 | 说明 |
|------|------|
| 公司章程 | 公司基本制度 |
| 股东会决议 | 股东决议 |
| 股东身份证明 | 身份证复印件 |
| 注册地址证明 | 租赁合同或房产证 |
| 法定代表人身份证明 | 身份证复印件 |

### 公司类型选择
| 类型 | 说明 | 适用 |
|------|------|------|
| 有限责任公司 | 股东有限责任 | 一般公司 |
| 股份有限公司 | 股份可以转让 | 大型企业 |
| 个人独资企业 | 个人无限责任 | 个体户 |
| 合伙企业 | 合伙人共同经营 | 合伙经营 |

## 公司变更

### 变更类型
| 变更 | 说明 | 流程 |
|------|------|------|
| 名称变更 | 公司名称变更 | 核名→变更登记→换照 |
| 地址变更 | 注册地址变更 | 变更登记→换照 |
| 股东变更 | 股东股权转让 | 股东会决议→变更登记 |
| 注册资本变更 | 增资或减资 | 股东会决议→变更登记 |
| 经营范围变更 | 经营范围调整 | 变更登记 |
| 法定代表人变更 | 法人变更 | 变更登记 |

### 变更材料
| 变更 | 材料 |
|------|------|
| 名称变更 | 新名称核准、变更申请、股东会决议 |
| 地址变更 | 新地址证明、变更申请、股东会决议 |
| 股东变更 | 股权转让协议、变更申请、股东会决议 |
| 注册资本变更 | 验资报告、变更申请、股东会决议 |

## 公司解散

### 解散原因
| 原因 | 说明 |
|------|------|
| 公司章程规定 | 章程规定期限届满 |
| 股东会决议 | 股东会决议解散 |
| 合并分立 | 公司合并或分立 |
| 吊销执照 | 被吊销营业执照 |
| 破产 | 资不抵债 |

### 解散流程
| 步骤 | 内容 | 时间 |
|------|------|------|
| 成立清算组 | 清算公司资产 | 15天内 |
| 公告 | 公告债权人 | 60天内 |
| 清算 | 清算资产、债务 | 视情况 |
| 注销登记 | 工商注销 | 清算结束后 |
| 税务注销 | 税务注销 | 工商注销前 |
| 银行账户注销 | 银行账户注销 | 最后 |

### 清算
| 顺序 | 内容 |
|------|------|
| 清算费用 | 支付清算费用 |
| 职工工资 | 支付职工工资、社保 |
| 税款 | 缴纳税款 |
| 债务 | 清偿债务 |
| 分配 | 剩余财产分配 |

## 公司税务

### 主要税种
| 税种 | 税率 | 说明 |
|------|------|------|
| 增值税 | 6%、13% | 销售收入 |
| 企业所得税 | 25%（小微企业20%） | 利润 |
| 个人所得税 | 3-45% | 工资、分红 |
| 印花税 | 0.03%-0.1% | 合同、账簿 |

### 小微企业优惠
| 优惠 | 条件 | 内容 |
|------|------|------|
| 所得税减免 | 年应纳税所得额<300万 | 减按25%计入，税率20% |
| 增值税减免 | 月销售额<10万 | 免征增值税 |
| 六税两费减免 | 小微企业 | 减半征收 |

### 税务合规
| 要求 | 说明 |
|------|------|
| 按时申报 | 按月/季度申报 |
| 如实申报 | 真实申报收入 |
| 保存凭证 | 保存发票、凭证 |
| 接受检查 | 配合税务检查 |
