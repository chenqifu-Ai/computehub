# 合同法：合同审查实务指南

## 一、合同审查概述

### 1. 合同审查定义
合同审查是指对合同条款进行全面、系统的检查和分析，确保合同内容合法、条款完备、权利义务明确、风险可控的过程。

### 2. 合同审查目的
```markdown
防范法律风险：
- 发现合同中的法律风险
- 避免或减少合同纠纷
- 保护当事人合法权益

保障商业利益：
- 明确双方权利义务
- 保护商业秘密
- 确保合同可执行

提高合同质量：
- 完善合同条款
- 规范合同表述
- 避免歧义争议
```

### 3. 合同审查原则
```markdown
合法性原则：
- 合同内容符合法律规定
- 合同形式符合法定要求
- 不违反法律强制性规定

公平性原则：
- 权利义务对等
- 风险分配合理
- 不存在显失公平条款

完备性原则：
- 条款完整
- 内容全面
- 没有遗漏

明确性原则：
- 条款表述清晰
- 没有歧义
- 易于理解和执行
```

## 二、合同审查流程

### 1. 审查准备
```markdown
收集材料：
- 合同文本
- 背景材料
- 相关文件
- 参考资料

了解背景：
- 交易背景
- 商业目的
- 谈判情况
- 对方情况

确定审查重点：
- 风险点识别
- 重点条款
- 关注事项
```

### 2. 形式审查
```markdown
合同主体审查：
- 主体资格是否合法
- 是否具有签约能力
- 是否具有履约能力
- 是否需要授权

合同形式审查：
- 形式是否符合法律要求
- 是否需要书面形式
- 是否需要公证或备案
- 是否需要审批

合同完整性审查：
- 条款是否完整
- 结构是否合理
- 附件是否齐全
- 签章是否规范
```

### 3. 实质审查
```markdown
合法性审查：
- 合同内容是否合法
- 是否违反强制性规定
- 是否损害国家、集体或第三人利益

条款审查：
- 条款是否完备
- 条款是否明确
- 权利义务是否对等
- 风险分配是否合理

重点条款审查：
- 标的条款
- 价款条款
- 履行条款
- 违约责任条款
- 争议解决条款
```

### 4. 审查意见
```markdown
审查报告：
- 审查发现的问题
- 风险分析
- 修改建议

风险提示：
- 高风险条款
- 中风险条款
- 低风险条款

修改建议：
- 条款修改建议
- 条款补充建议
- 条款删除建议
```

## 三、合同主体审查

### 1. 主体资格审查
```markdown
自然人：
- 是否具有完全民事行为能力
- 身份证真伪核实
- 是否具备签约能力

法人：
- 营业执照是否有效
- 是否具有法人资格
- 经营范围是否涵盖合同内容
- 注册资本是否到位
- 是否年检合格

其他组织：
- 是否具有合法主体资格
- 是否具有经营资格
- 是否具有履约能力

特殊主体：
- 分公司：需总公司授权
- 事业单位：需审查其职能范围
- 社会团体：需审查其业务范围
```

### 2. 履约能力审查
```markdown
财务能力：
- 注册资本
- 实缴资本
- 财务状况
- 偿债能力

经营能力：
- 经营年限
- 经营规模
- 行业地位
- 过往业绩

信用状况：
- 信用记录
- 央行征信报告
- 司法涉诉情况
- 行政处罚记录

资质能力：
- 行业资质证书
- 专业许可证
- 安全生产许可
- 环保审批
```

### 3. 代理权限审查
```markdown
法定代表人：
- 核对法定代表人身份
- 确认法定代表人签字权限
- 无需授权委托书

授权代理人：
- 核对代理人身份
- 审查授权委托书
- 确认授权范围和期限
- 核对签字样式

授权委托书审查：
- 委托人签章是否真实
- 代理人信息是否准确
- 授权事项是否明确
- 授权期限是否有效
- 是否加盖公章
```

## 四、合同条款审查

### 1. 合同首部条款
```markdown
合同名称：
- 名称应与合同内容一致
- 反映合同性质

合同编号：
- 便于合同管理
- 按公司编码规则

签订时间地点：
- 明确签订日期
- 明确签订地点
- 影响法律适用和管辖

当事人信息：
- 名称准确完整
- 地址、联系方式准确
- 法定代表人/代理人信息
```

### 2. 标的条款审查
```markdown
标的名称：
- 名称准确规范
- 使用标准名称
- 避免使用简称

标的数量：
- 数量明确具体
- 计量单位规范
- 允许误差范围

标的质量：
- 质量标准明确
- 适用国家标准/行业标准
- 约定检验标准和方式

标的特征：
- 规格、型号、款式
- 性能、功能描述
- 配置清单
```

### 3. 价款条款审查
```markdown
价款金额：
- 金额明确具体
- 大小写一致
- 是否含税

价款构成：
- 单价、总价
- 是否包含其他费用
- 价格调整机制

支付方式：
- 支付时间
- 支付方式（转账、现金、票据）
- 支付条件
- 分期付款安排

发票条款：
- 发票类型（增值税专用/普通发票）
- 发票开具时间
- 发票金额
- 发票信息

税费承担：
- 明确税费承担方
- 税率变化处理
```

### 4. 履行条款审查
```markdown
履行期限：
- 起止时间明确
- 分阶段履行的时间节点
- 延期履行的处理

履行地点：
- 交付地点明确
- 风险转移地点
- 运输方式和费用

履行方式：
- 具体履行方式
- 履行标准
- 验收标准

履行条件：
- 履行前提条件
- 先决条件
- 后续条件

验收条款：
- 验收标准
- 验收方式
- 验收期限
- 验收不合格的处理
```

### 5. 权利义务条款审查
```markdown
权利条款：
- 权利内容明确
- 权利行使条件
- 权利行使方式

义务条款：
- 义务内容明确
- 义务履行标准
- 义务履行期限

配合义务：
- 需要对方配合的事项
- 配合的方式和时间
- 不配合的处理

保密义务：
- 保密范围明确
- 保密期限
- 保密例外情形
- 违反保密义务的责任
```

### 6. 违约责任条款审查
```markdown
违约情形：
- 具体列明违约情形
- 违约行为的界定
- 重大违约和一般违约

违约责任：
- 违约金数额或计算方式
- 赔偿损失的范围和计算方式
- 继续履行的条件
- 解除合同的条件

违约金条款：
- 违约金数额合理
- 不超过实际损失的30%
- 约定过高或过低的调整

免责条款：
- 免责事由明确
- 不可抗力的定义和处理
- 免责的范围和条件
```

### 7. 争议解决条款审查
```markdown
争议解决方式：
- 协商、调解、仲裁、诉讼
- 选择仲裁需明确仲裁机构
- 选择诉讼可约定管辖法院

仲裁条款：
- 仲裁机构名称准确
- 仲裁事项明确
- 仲裁地点约定
- 仲裁裁决的效力

诉讼管辖：
- 约定管辖法院
- 管辖法院选择合法有效
- 不违反级别管辖和专属管辖

适用法律：
- 适用中国法律
- 涉外合同可约定适用法律
```

### 8. 合同变更解除条款审查
```markdown
变更条款：
- 变更的条件和程序
- 变更的形式要求（书面）
- 变更的效力

解除条款：
- 解除的条件
- 解除的程序
- 解除的后果
- 解除通知方式

终止条款：
- 终止的条件
- 终止的程序
- 终止后的处理
```

### 9. 其他条款审查
```markdown
保密条款：
- 保密信息范围
- 保密期限
- 保密义务
- 违反保密的责任

知识产权条款：
- 知识产权归属
- 使用许可范围
- 侵权责任

不可抗力条款：
- 不可抗力定义
- 不可抗力范围
- 不可抗力的处理
- 不可抗力的证明

通知条款：
- 通知方式
- 通知地址
- 通知效力

合同效力条款：
- 生效条件
- 生效时间
- 有效期限

附件条款：
- 附件清单
- 附件效力
- 附件与正文关系
```

## 六、合同审查常见问题

### 1. 条款不完备
```markdown
常见问题：
- 缺少必要条款
- 条款约定不明确
- 条款存在空白

解决方案：
- 补充完整条款
- 明确模糊条款
- 删除无效空白
```

### 2. 表述不准确
```markdown
常见问题：
- 用词不规范
- 表述有歧义
- 概念不清晰

解决方案：
- 使用规范法律用语
- 明确界定概念
- 消除歧义表述
```

### 3. 权利义务不对等
```markdown
常见问题：
- 一方权利多义务少
- 一方义务多权利少
- 风险分配不均

解决方案：
- 平衡双方权利义务
- 合理分配风险
- 调整不对等条款
```

### 4. 违约责任不明确
```markdown
常见问题：
- 违约情形不具体
- 违约责任不明确
- 赔偿范围不清楚

解决方案：
- 明确违约情形
- 明确违约责任
- 明确赔偿范围和计算方式
```

### 5. 管辖约定不规范
```markdown
常见问题：
- 管辖法院约定不明确
- 管辖约定违反法律规定
- 仲裁条款不规范

解决方案：
- 明确管辖法院或仲裁机构
- 确保管辖约定合法有效
- 规范仲裁条款
```

## 七、合同审查工具与技巧

### 1. 审查清单
```markdown
合同基本要素：
- 合同名称
- 合同编号
- 签订日期
- 签订地点
- 当事人信息

主要条款：
- 标的条款
- 价款条款
- 履行条款
- 权利义务条款
- 违约责任条款
- 争议解决条款

重要条款：
- 生效条款
- 解除条款
- 保密条款
- 不可抗力条款

附件审查：
- 附件完整
- 附件与正文一致
- 附件签章
```

### 2. 审查技巧
```markdown
整体把握：
- 先通读全文
- 了解合同性质和目的
- 把握整体框架

分项审查：
- 按条款逐项审查
- 对照法律规定
- 查找问题和风险

重点突出：
- 关注核心条款
- 关注风险条款
- 关注责任条款

前后对照：
- 条款之间是否矛盾
- 前后表述是否一致
- 相关条款是否衔接
```

### 3. 审查记录
```markdown
问题记录：
- 问题描述
- 条款位置
- 风险等级

建议记录：
- 修改建议
- 补充建议
- 删除建议

跟踪记录：
- 问题处理结果
- 修改落实情况
- 最终版本确认
```
