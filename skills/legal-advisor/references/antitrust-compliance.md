# 企业反垄断合规实务指南

## 一、核心概念

### 什么是反垄断法？

反垄断法是规范市场竞争秩序、预防和制止垄断行为、保护市场公平竞争的法律规范。中国《反垄断法》于2008年实施，2022年完成首次重大修订。

### 垄断行为的四大类型

| 类型 | 具体表现 | 典型案例 |
|------|----------|----------|
| 垄断协议 | 固定价格、分割市场、联合抵制 | 汽车零部件企业价格串通 |
| 滥用市场支配地位 | 不公平高价、拒绝交易、搭售 | 某电商平台"二选一"案 |
| 经营者集中 | 并购未申报、收购导致垄断 | 多起互联网并购案 |
| 行政性垄断 | 滥用行政权力排除限制竞争 | 地方保护主义案例 |

## 二、高频风险点与防范

### 1. 横向垄断协议（竞争对手之间）

**高风险行为：**
- 价格串通：联合涨价、固定折扣
- 市场分割：约定销售区域或客户群
- 联合抵制：共同拒绝与某供应商/客户交易
- 限制产量：联合减产保价

**实际案例：**

> **某建材行业协会案**
> 
> 某省建材行业协会组织20余家水泥企业签订《自律公约》，约定：
> - 统一上调价格50元/吨
> - 划分销售区域，不得跨区销售
> - 违约企业罚款10万元
>
> **处罚结果：** 协会被罚款50万元，参与企业合计罚款超2000万元。
>
> **合规要点：** 行业协会不得组织企业串通价格，企业应主动抵制并举报此类行为。

### 2. 纵向垄断协议（上下游之间）

**高风险行为：**
- 转售价格维持（RPM）：固定或限定转售价格
- 限定销售区域或客户
- 独家交易条款

**实际案例：**

> **某品牌汽车转售价格限制案**
> 
> 某汽车品牌要求经销商：
> - 销售价格不得低于指导价的95%
> - 违规经销商扣除返利
> - 限制跨区域销售
>
> **处罚结果：** 罚款1.2亿元。
>
> **合规要点：** 
> - 建议价格（非强制）通常合规
> - 可设定最高限价保护消费者利益
> - 避免"建议价格+处罚机制"的组合

### 3. 滥用市场支配地位

**认定标准：**
1. 相关市场界定
2. 市场份额超过50%（推定支配地位）
3. 无正当理由实施排除、限制竞争行为

**高风险行为：**

| 行为类型 | 典型表现 | 合规建议 |
|----------|----------|----------|
| 不公平定价 | 明显高于合理成本的价格 | 定价需有成本依据 |
| 掠夺性定价 | 低于成本销售排挤对手 | 避免倾销式定价 |
| 拒绝交易 | 无正当理由拒绝供应/采购 | 合理理由需书面记录 |
| 搭售 | 强制购买非必要产品 | 捆绑需有正当性理由 |
| 差别待遇 | 相同条件下不同价格/条件 | 价格差异需成本依据 |

**实际案例：**

> **某电商平台"二选一"案**
> 
> 某平台要求入驻商家：
> - 不得在其他平台开店或促销
> - 违规商家降权、限流
> - 通过数据监控商家行为
>
> **处罚结果：** 罚款182亿元（按销售额4%计）
>
> **合规要点：**
> - 平台不得强制独家合作
> - 数据使用需获得授权
> - 算法决策需透明可解释

### 4. 经营者集中申报

**申报门槛：**
- 全球营业额合计超100亿元 + 中国境内营业额超8亿元
- 或中国境内营业额合计超20亿元 + 至少两家企业各超4亿元

**常见误区：**

| 误区 | 正确理解 |
|------|----------|
| 小企业并购无需申报 | 标准看集团整体营业额 |
| 股权比例低无需申报 | 控制权变更即需申报 |
| 完成交易后再申报 | 应当申报前申报，获批后方可实施 |
| 子公司并购不申报 | 需穿透计算实际控制人营业额 |

**实际案例：**

> **某互联网企业并购未申报案**
> 
> 某互联网巨头收购多家初创企业，未申报即完成交割。
>
> **处罚结果：** 
> - 未申报行为罚款50万元/案
> - 部分交易被要求剥离
>
> **合规要点：**
> - 投资前评估是否触发申报
> - 设立内部申报审查机制
> - 保留交易决策证据链

## 三、合规管理体系建设

### 组织架构

```
董事会
    │
    └── 反垄断合规委员会
            │
            ├── 合规部（日常管理）
            ├── 法务部（法律审核）
            └── 业务部门（执行落实）
```

### 核心制度清单

1. **反垄断合规手册** - 明确禁止行为
2. **定价审批制度** - 价格调整需审批留痕
3. **合同审查制度** - 重点审查排他性条款
4. **并购申报制度** - 投资前强制合规审查
5. **举报与调查制度** - 内部违规举报通道

### 风险自查清单

**月度自查项：**
- [ ] 本月是否有与竞争对手的价格沟通
- [ ] 是否有限制经销商定价的行为
- [ ] 是否有搭售或强制交易
- [ ] 投资并购是否经过合规审查

**年度审查项：**
- [ ] 市场份额测算（是否超50%）
- [ ] 定价策略合理性评估
- [ ] 合作协议排他性条款审查
- [ ] 合规培训覆盖率

## 四、执法应对指南

### 配合调查义务

| 调查内容 | 企业义务 | 违法后果 |
|----------|----------|----------|
| 提供文件资料 | 按要求提交，不得隐匿 | 罚款最高100万元 |
| 接受询问 | 如实回答，不得作伪证 | 个人罚款/刑事责任 |
| 配合现场检查 | 允许进入场所检查 | 罚款+强制措施 |

### 突发调查应对流程

```
执法机关到达
    │
    ├── 验证执法证件（2人以上+证件）
    ├── 通知法务/合规负责人到场
    ├── 要求说明调查范围
    │
    ├── 配合但不自证其罪
    │   ├── 提供要求文件，不主动提供额外信息
    │   � 如实回答，不作推测性陈述
    │
    └── 记录调查过程（全程录音/录像/笔录）
```

### 宽大制度（Leniency）

**申请条件：**
- 主动报告垄断协议行为
- 提供关键证据
- 全面配合调查
- 在行业中首先申请

**宽大力度：**
- 第一申请者：免除处罚
- 第二申请者：减轻30%-50%
- 第三申请者：减轻20%-30%

## 五、行业特殊合规要点

### 互联网平台

- **算法共谋风险**：定价算法可能被认定为变相价格串通
- **数据使用边界**：不得利用数据优势实施歧视性待遇
- **平台责任**：对平台内经营者垄断行为负有监督义务

### 医药行业

- **原料药垄断**：控制原料药供应可能构成滥用支配地位
- **药品定价**：专利期内定价需考虑反垄断限制
- **带量采购**：联合采购不构成垄断协议

### 汽车行业

- **经销商管理**：转售价格限制是执法重点
- **售后配件**：不得强制使用原厂配件
- **销售区域限制**：需有正当理由

## 六、合规实务建议

### 给中小企业

1. **定价留痕** - 所有定价决策需有成本依据文件
2. **会议记录** - 行业会议避免价格议题讨论
3. **合同审查** - 排他性条款需法律审核
4. **员工培训** - 销售、采购人员必须懂红线

### 给大型企业

1. **市场份额监测** - 定期评估是否触发支配地位认定
2. **合规前置** - 并购决策前完成合规尽调
3. **申诉机制** - 建立内部违规行为举报通道
4. **应急预案** - 制定突发调查应对方案

## 七、法律责任概览

| 行为类型 | 行政责任 | 民事责任 | 刑事责任 |
|----------|----------|----------|----------|
| 垄断协议 | 上一年度销售额1%-10%罚款 | 赔偿损失 | 情节严重可追刑责 |
| 滥用支配地位 | 上一年度销售额1%-10%罚款 | 赔偿损失 | 情节严重可追刑责 |
| 违法集中 | 50万元罚款+责令恢复 | 赔偿损失 | 无 |
| 阻碍调查 | 个人最高20万，单位最高100万 | - | 情节严重可追刑责 |

> **核心合规口诀：**
> 
> 不串通，不固定，竞争对手要记清。
> 不搭售，不歧视，支配地位莫滥用。
> 并购前，先申报，集中审查要走掉。
> 遇调查，要配合，主动宽大可减免。
