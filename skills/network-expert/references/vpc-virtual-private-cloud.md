# 云计算网络 - VPC虚拟私有云

## 一、VPC概述

### 定义
VPC（Virtual Private Cloud）是在公有云中创建的逻辑隔离的虚拟网络，用户可以在其中部署云资源，完全控制网络拓扑和IP地址规划。

### 核心特点
- **隔离性**：与公共网络逻辑隔离
- **可控性**：自主规划IP、子网、路由
- **灵活性**：按需扩展，弹性配置
- **安全性**：安全组、网络ACL保护

---

## 二、VPC核心组件

### 1. 子网（Subnet）
- VPC内的IP地址段划分
- 分为公有子网和私有子网
- 每个子网对应一个可用区

### 2. 路由表（Route Table）
- 控制子网流量走向
- 定义目标地址的下一跳
- 关联到子网

### 3. 网关
- **互联网网关（IGW）**：连接公网
- **NAT网关**：私有子网访问公网
- **VPN网关**：连接本地数据中心
- **对等连接**：连接其他VPC

### 4. 安全组（Security Group）
- 虚拟防火墙
- 实例级别访问控制
- 有状态（自动允许返回流量）

### 5. 网络ACL
- 子网级别访问控制
- 无状态（需配置双向规则）
- 优先级规则匹配

## 三、VPC网络架构

### 典型三层架构
```
┌─────────────────────────────────────┐
│           公有子网                    │
│  ┌─────┐    ┌─────┐    ┌─────┐     │
│  │ LB  │    │ NAT │    │Bastion│   │
│  └─────┘    └─────┘    └─────┘     │
│       │          │          │        │
└───────┼──────────┼──────────┼───────┘
        │          │          │
┌───────┼──────────┼──────────┼───────┐
│       │    私有子网           │       │
│  ┌────┴────┐  ┌────┴────┐  ┌────┴───┐│
│  │Web服务器│  │应用服务器│  │数据库   ││
│  └─────────┘  └─────────┘  └────────┘│
└─────────────────────────────────────┘
```

### 网络流量路径
```
公网 → IGW → 公有子网 → NAT → 私有子网 → 数据库
                      ↓
                  安全组检查
                      ↓
                  网络ACL检查
```

## 四、VPC规划最佳实践

### 1. IP地址规划
| 类型 | CIDR | 主机数 | 用途 |
|------|------|--------|------|
| 大型VPC | 10.0.0.0/16 | 65536 | 企业级 |
| 中型VPC | 10.0.0.0/18 | 16384 | 部门级 |
| 小型VPC | 10.0.0.0/20 | 4096 | 项目级 |

### 2. 子网划分原则
- 预留扩展空间
- 每个可用区至少一个子网
- 公有子网和私有子网分离

### 3. 安全层级
```
边界防护 → 网络ACL → 安全组 → 主机防火墙
```

## 五、VPC连接方案

### 1. VPC对等连接
- 不同VPC之间通信
- 跨账号连接
- 跨区域连接

### 2. VPN连接
- Site-to-Site VPN
- 连接本地数据中心
- 加密传输

### 3. 专线连接
- 专用物理链路
- 高带宽、低延迟
- 金融级安全

### 4. 私有链接
- 私有访问云服务
- 不经过公网
- 安全可靠

## 七、VPC常见场景

### 场景1：Web应用
- 公有子网：负载均衡、Web服务器
- 私有子网：应用服务器、数据库
- NAT网关：私有子网访问公网

### 场景2：微服务架构
- 服务网格
- 服务发现
- 内部负载均衡

### 场景3：混合云
- VPN/专线连接本地DC
- 云上扩展资源
- 统一管理

### 场景4：多环境隔离
- 开发、测试、生产环境分离
- 共享服务VPC
- 跨环境访问控制

## 九、VPC故障排查

### 常见问题

#### 1. 无法访问公网
- 检查路由表是否有IGW路由
- 检查安全组是否放行
- 检查实例是否有公网IP

#### 2. 子网间无法通信
- 检查路由表配置
- 检查安全组规则
- 检查网络ACL规则

#### 3. VPN连接失败
- 检查VPN配置
- 检查IPSec参数
- 检查路由配置
