# SDN软件定义网络 - 网络架构革命

## 概述

SDN（Software-Defined Networking，软件定义网络）是一种新型网络架构，将网络的控制平面与数据平面分离，通过软件编程实现网络控制，使网络更加灵活、可编程、易于管理。

## 八、总结

### SDN核心要点

```
1. 转控分离：控制平面与数据平面分离
2. 集中控制：SDN控制器作为网络大脑
3. 可编程：通过API编程网络
4. OpenFlow：核心南向协议
5. 应用场景：数据中心、广域网、运营商网络
```

### 记忆口诀

```
转控分离是核心，集中控制做大脑。
南向协议OpenFlow，北向API应用来。
数据平面只转发，流表匹配执行快。
数据中心SD-WAN，运营商网都用它。
```
