# 网络架构 - 软件定义网络SDN

## 一、SDN概述

### 定义
软件定义网络（Software-Defined Networking，SDN）是一种新型网络架构，通过将控制平面与数据平面分离，实现网络的可编程控制。

### 核心特点
- **控制与转发分离**：控制平面集中管理，数据平面只负责转发
- **集中式控制**：通过SDN控制器统一管理全网
- **可编程**：通过API编程实现网络自动化
- **开放接口**：标准化的北向和南向接口

## 三、OpenFlow协议

### 核心概念
- **流表（Flow Table）**：定义数据包处理规则
- **流表项**：匹配字段 + 计数器 + 动作
- **多级流表**：流水线处理

### 流表项组成
```
| 匹配字段 | 优先级 | 计数器 | 动作 | 超时 |
|---------|--------|--------|------|------|
| 入端口   |        |        |      |      |
| MAC地址  |        |        |      |      |
| IP地址   |        |        |      |      |
| TCP端口  |        |        |      |      |
```

### 动作类型
- **Output**：转发到指定端口
- **Group**：组播或负载均衡
- **Meter**：带宽限制
- **Drop**：丢弃数据包
