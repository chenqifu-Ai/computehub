# 小学科学：植物的生长

## 植物的基本结构

### 植物的六大器官
| 器官 | 功能 | 特点 |
|------|------|------|
| 根 | 吸收水分和养分 | 生长在地下 |
| 茎 | 支撑和运输 | 连接根和叶 |
| 叶 | 光合作用 | 绿色、扁平 |
| 花 | 繁殖 | 色彩鲜艳 |
| 果实 | 保护种子 | 可食用 |
| 种子 | 繁殖 | 可发芽 |

### 根的作用
| 类型 | 说明 | 例子 |
|------|------|------|
| 直根 | 主根明显，侧根少 | 大豆、胡萝卜 |
| 须根 | 没有明显主根 | 水稻、玉米 |

**根的功能：**
- 固定植物
- 吸收水分和无机盐
- 储存养分（如萝卜）

### 茎的作用
| 类型 | 说明 | 例子 |
|------|------|------|
| 直立茎 | 直立生长 | 大树 |
| 攀援茎 | 攀附他物 | 葡萄 |
| 缠绕茎 | 缠绕他物 | 牵牛花 |
| 匍匐茎 | 贴地生长 | 草莓 |

**茎的功能：**
- 支撑植物
- 运输水分和养分
- 储存养分（如土豆）

### 叶的作用
| 结构 | 说明 |
|------|------|
| 叶片 | 光合作用主要场所 |
| 叶脉 | 运输水分和养分 |
| 叶柄 | 连接茎和叶片 |

**叶的功能：**
- 光合作用
- 蒸腾作用
- 呼吸作用

## 光合作用

### 什么是光合作用
```
二氧化碳 + 水 + 光能 → 有机物 + 氧气

在叶绿体中进行
需要光
产生氧气
```

### 光合作用的意义
| 意义 | 说明 |
|------|------|
| 提供氧气 | 地球上的氧气主要来源 |
| 提供食物 | 产生有机物 |
| 能量转化 | 光能转化为化学能 |

### 影响光合作用的因素
| 因素 | 影响 |
|------|------|
| 光照强度 | 光越强，光合作用越强 |
| 二氧化碳浓度 | 浓度越高，光合作用越强 |
| 温度 | 适温下光合作用最强 |
| 水分 | 水分充足，光合作用强 |

## 植物的生长过程

### 种子萌发
| 条件 | 说明 |
|------|------|
| 内在条件 | 种子完整、有活力 |
| 外在条件 | 水分、空气、适宜温度 |

### 萌发过程
```
种子吸水 → 种皮破裂 → 胚根伸出 → 胚芽伸出 → 长成幼苗
```

### 植物生长阶段
| 阶段 | 特点 |
|------|------|
| 发芽期 | 种子萌发 |
| 幼苗期 | 根茎叶生长 |
| 成熟期 | 开花结果 |
| 衰老期 | 逐渐衰老 |

## 植物的繁殖

### 种子繁殖
| 步骤 | 说明 |
|------|------|
| 传粉 | 花粉传到柱头 |
| 受精 | 精子与卵细胞结合 |
| 结果 | 子房发育成果实 |
| 传播 | 种子传播到各处 |

### 无性繁殖
| 方式 | 说明 | 例子 |
|------|------|------|
| 扦插 | 剪枝插入土中 | 月季 |
| 嫁接 | 接穗接到砧木 | 苹果 |
| 分株 | 分开母株 | 草莓 |
| 压条 | 枝条压入土中 | 石榴 |

## 植物的营养

### 需要的营养物质
| 类型 | 元素 | 作用 |
|------|------|------|
| 大量元素 | N、P、K | 生长必需 |
| 中量元素 | Ca、Mg、S | 重要补充 |
| 微量元素 | Fe、Zn等 | 适量需要 |

### 缺素症状
| 缺乏元素 | 症状 |
|----------|------|
| 氮(N) | 叶片发黄、生长慢 |
| 磷(P) | 根系发育差 |
| 钾(K) | 抗病力弱 |

## 实验：观察植物生长

### 实验一：种子萌发
**材料**：绿豆种子、杯子、水、纸巾

**步骤**：
1. 准备两个杯子
2. 一个放湿润纸巾，一个放干燥纸巾
3. 各放入10粒绿豆种子
4. 观察并记录发芽情况

**结论**：种子萌发需要水分

### 实验二：光合作用
**材料**：盆栽植物、黑纸片、回形针

**步骤**：
1. 选一片叶子
2. 用黑纸片遮住一部分
3. 放在阳光下照射几小时
4. 取下叶子，放入酒精中煮
5. 用碘液染色

**结论**：见光部分变蓝，说明产生了淀粉

### 实验三：茎的运输
**材料**：芹菜、红墨水、杯子

**步骤**：
1. 杯中倒入红墨水
2. 放入芹菜
3. 观察几小时后芹菜的变化

**结论**：茎可以运输水分
