# 小学数学加减法完整教程

## 一、加法基础

### 1. 加法概念

**加法**：把两个或多个数合并成一个数的运算。

**符号**：+（加号）

**读法**：加、加上、和

**例**：3 + 2 = 5
- 读作：三加二等于五
- 意思：把3和2合起来是5

### 2. 加法交换律

**规律**：两个数相加，交换加数的位置，和不变。

**公式**：a + b = b + a

**例**：
- 3 + 5 = 8
- 5 + 3 = 8
- 所以：3 + 5 = 5 + 3

### 3. 加法结合律

**规律**：三个数相加，先把前两个数相加，或者先把后两个数相加，和不变。

**公式**：(a + b) + c = a + (b + c)

**例**：
- (2 + 3) + 4 = 5 + 4 = 9
- 2 + (3 + 4) = 2 + 7 = 9
- 所以：(2 + 3) + 4 = 2 + (3 + 4)

## 三、减法基础

### 1. 减法概念

**减法**：已知两个数的和与其中一个加数，求另一个加数的运算。

**符号**：-（减号）

**读法**：减、减去、差

**例**：5 - 3 = 2
- 读作：五减三等于二
- 意思：5比3多多少

### 2. 减法的性质

**性质**：一个数减去另一个数，再减去第三个数，等于这个数减去后两个数的和。

**公式**：a - b - c = a - (b + c)

**例**：
- 15 - 3 - 2 = 12 - 2 = 10
- 15 - (3 + 2) = 15 - 5 = 10
- 所以：15 - 3 - 2 = 15 - (3 + 2)

## 五、100以内加减法

### 1. 不进位加法

**方法**：个位加个位，十位加十位

**例**：23 + 45 = ?

```
方法：
十位：2 + 4 = 6
个位：3 + 5 = 8
结果：23 + 45 = 68
```

### 2. 进位加法

**方法**：个位相加满十，向十位进一

**例**：36 + 47 = ?

```
方法：
个位：6 + 7 = 13（写3进1）
十位：3 + 4 + 1 = 8
结果：36 + 47 = 83
```

### 3. 不退位减法

**方法**：个位减个位，十位减十位

**例**：78 - 35 = ?

```
方法：
十位：7 - 3 = 4
个位：8 - 5 = 3
结果：78 - 35 = 43
```

### 4. 退位减法

**方法**：个位不够减，从十位退一当十

**例**：52 - 38 = ?

```
方法：
个位：2 - 8 不够减
从十位退1，变成12 - 8 = 4
十位：5 - 1 - 3 = 1
结果：52 - 38 = 14
```

## 六、加减法速算技巧

### 1. 凑整法

**方法**：把数凑成整十、整百

**例**：
- 28 + 43 = (28 + 2) + (43 - 2) = 30 + 41 = 71
- 99 + 56 = (99 + 1) + (56 - 1) = 100 + 55 = 155

### 2. 补数法

**方法**：利用补数（凑成10或100的数）

**例**：
- 8的补数是2（因为8 + 2 = 10）
- 75的补数是25（因为75 + 25 = 100）

### 3. 找规律法

**规律1**：加法中，一个数增加，另一个数减少，和不变

**例**：28 + 42 = (28 + 2) + (42 - 2) = 30 + 40 = 70

**规律2**：减法中，被减数和减数同时增加或减少，差不变

**例**：56 - 38 = (56 + 4) - (38 + 4) = 60 - 42 = 18

## 七、加减法应用题

### 1. 加法应用题

**例**：小明有8个苹果，妈妈又给了他5个，小明现在有多少个苹果？

**解题步骤**：
1. 找关键词："又给了"表示加法
2. 列算式：8 + 5 = ?
3. 计算：8 + 5 = 13
4. 写答案：小明现在有13个苹果。

### 2. 减法应用题

**例**：篮子里有15个橘子，拿走了7个，还剩多少个？

**解题步骤**：
1. 找关键词："拿走了"、"还剩"表示减法
2. 列算式：15 - 7 = ?
3. 计算：15 - 7 = 8
4. 写答案：还剩8个橘子。

### 3. 加减混合应用题

**例**：公交车上有12人，下车5人，又上车8人，现在车上有多少人？

**解题步骤**：
1. 分析过程：下车5人 → 减少 → 12 - 5 = 7
2. 分析过程：上车8人 → 增加 → 7 + 8 = 15
3. 写答案：现在车上有15人。

## 八、常见错误与纠正

### 1. 进位忘记加

**错误**：36 + 47 = 73
**正确**：36 + 47 = 83
**原因**：忘记进位
**纠正**：记住"满十进一"

### 2. 退位忘记减

**错误**：52 - 38 = 24
**正确**：52 - 38 = 14
**原因**：忘记十位退了1
**纠正**：在十位上点个点提醒自己

### 3. 数位对齐错误

**错误**：36 + 5 = 86
**正确**：36 + 5 = 41
**原因**：5对齐了十位
**纠正**：个位对个位，十位对十位
