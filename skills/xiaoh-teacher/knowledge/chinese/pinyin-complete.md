## 一、声母（23个）

- **p** (坡)：p-ā → pā（爬）

#### 翘舌音

#### 特殊声母

## 四、声调（4个+轻声）

### 1. 声调标记

- **第一声**（阴平）：ā ˉ 高平调，55
- **第二声**（阳平）：á ˊ 上升调，35
- **第三声**（上声）：ǎ ˇ 先降后升，214
- **第四声**（去声）：à ˋ 下降调，51
- **轻声**：不标调，如：ma（吗）

### 2. 标调规则

```
有 a 不放过，
无 a 找 o e，
i u 并列标在后，
单个韵母不用说。

i 上标调去掉点，
ü 上标调点保留。
```

### 3. 声调练习

**ā á ǎ à**
- ā：阿姨、安全
- á：啊呀、熬煎
- ǎ：好吧、可以
- à：啊哈、那好

## 五、拼读规则

### 1. 两拼法

**声母 + 韵母 = 音节**

例：
- b + a → ba（爸）
- p + o → po（破）
- m + i → mi（米）
- f + u → fu（父）

### 2. 三拼法

**声母 + 介母 + 韵母 = 音节**

例：
- j + i + a → jia（家）
- q + i + a → qia（恰）
- x + i + a → xia（夏）
- g + u + a → gua（瓜）

### 3. 拼读口诀

```
声母在前韵母后，
两音相连猛一碰，
声轻韵重声母短，
韵母发音要响亮。
```

## 六、特殊规则

### 1. j q x y 与 ü 相拼

**规则**：j q x y 与 ü 相拼时，ü 上两点去掉，但读音不变。

例：
- j + ü → ju（居）
- q + ü → qu（曲）
- x + ü → xu（许）
- y + ü → yu（雨）

**口诀**：
```
小ü见到j q x，
擦掉眼泪笑嘻嘻，
去掉两点还读ü。
```

### 2. i u 并列标在后

**规则**：当 i 和 u 并列时，声调标在后面的韵母上。

例：
- iu → iū（优）
- ui → uī（威）

## 七、常见易错字

### 1. 形近字区分

| 易混淆 | 区分方法 |
|-------|---------|
| b/d | b像收音机，d像马蹄声 |
| p/q | p像小旗，q像蝌蚪 |
| zh/z | zh翘舌，z平舌 |
| ch/c | ch翘舌，c平舌 |
| sh/s | sh翘舌，s平舌 |

### 2. 前后鼻音区分

| 前鼻音 | 后鼻音 | 区分方法 |
|-------|-------|---------|
| an | ang | an收舌尖，ang收喉音 |
| en | eng | en收舌尖，eng收喉音 |
| in | ing | in收舌尖，ing收喉音 |

## 八、练习题

- （ ）ā → bā（八）
- （ ）ī → mī（米）
- （ ）ū → tū（土）
