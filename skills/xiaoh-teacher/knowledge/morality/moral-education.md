# 道德与法治：小学道德与法治基础知识

## 一、个人与家庭

### 1. 认识自己
```markdown
认识自己的特点：
- 外貌特征：长相、身高、体型
- 性格特点：开朗、内向、活泼
- 兴趣爱好：喜欢的活动、运动、学习

认识自己的优点和不足：
- 每个人都有自己的优点
- 也有需要改进的地方
- 正确看待自己

培养良好习惯：
- 生活习惯：早睡早起、讲究卫生
- 学习习惯：认真听讲、按时完成作业
- 行为习惯：礼貌待人、遵守规则
```

```markdown
- 兄弟姐妹：互相关心、互相帮助

家庭美德：
- 尊老爱幼
- 夫妻和睦
- 勤俭持家
- 邻里团结
```

### 3. 家庭生活

## 四、规则与法律

### 1. 认识规则
```markdown
什么是规则：
- 大家共同遵守的规定
- 保证生活有序进行
- 保护每个人的利益

生活中的规则：
- 学校规则：校规、班规
- 家庭规则：家规
- 社会规则：公共秩序

遵守规则的重要性：
- 维护秩序
- 保护安全
- 方便生活
- 保障公平

违反规则的后果：
- 受到批评
- 受到处罚
- 影响他人
- 造成危险
```

### 2. 认识法律
```markdown
什么是法律：
- 国家制定的行为规范
- 人人必须遵守
- 由国家强制力保证实施

法律的作用：
- 保护公民权利
- 维护社会秩序
- 惩罚违法犯罪
- 教育引导

法律与规则的关系：
- 法律是一种特殊的规则
- 法律比一般规则更加严格
- 法律由国家制定
- 法律具有强制力
```

### 3. 与小学生相关的法律
```markdown
《中华人民共和国宪法》：
- 国家的根本大法
- 规定公民的基本权利和义务
- 所有人都必须遵守

《中华人民共和国义务教育法》：
- 保障适龄儿童接受义务教育
- 父母送子女上学是义务
- 学校不得开除学生

《中华人民共和国未成年人保护法》：
- 保护未成年人的合法权益
- 家庭保护、学校保护
- 社会保护、司法保护

《中华人民共和国预防未成年人犯罪法》：
- 预防未成年人犯罪
- 不良行为和严重不良行为
- 预防措施
```

### 4. 拒绝校园欺凌
```markdown
什么是校园欺凌：
- 殴打、辱骂同学
- 勒索财物
- 孤立、排挤同学
- 在网上传播谣言

校园欺凌的危害：
- 身体伤害
- 心理伤害
- 影响学习
- 影响成长

如何应对校园欺凌：
- 不要害怕，勇敢说"不"
- 及时告诉老师、家长
- 不做旁观者
- 帮助被欺凌的同学

预防校园欺凌：
- 团结同学
- 不欺负他人
- 学会保护自己
- 及时报告
```

## 五、权利与义务

### 1. 认识权利
```markdown
公民的基本权利：

人身权利：
- 生命健康权
- 人身自由权
- 人格尊严权
- 隐私权

财产权利：

- 不被随意开除

其他权利：

- 肖像权
- 姓名权
```

### 2. 认识义务
```markdown
公民的基本义务：

遵守宪法和法律：
- 学习法律知识
- 遵守法律规定
- 不做违法的事

维护国家利益：
- 热爱祖国
- 维护国家安全
- 维护国家荣誉

其他义务：
- 接受教育
- 劳动的义务
- 计划生育
```

### 3. 权利与义务的关系
```markdown
权利与义务的关系：
- 相互依存
- 相互促进
- 不可分割

正确行使权利：
- 在法律范围内行使

- 不得损害国家利益

自觉履行义务：
- 法律规定的义务必须履行
```

## 七、安全教育

### 1. 交通安全
```markdown
步行安全：
- 走人行道
- 过马路走斑马线
- 看红绿灯
- 不在马路上玩耍

乘车安全：
- 系好安全带
- 不把头手伸出窗外
- 等车停稳再下车
- 不乘坐超载车辆

骑车安全：
- 未满12周岁不骑自行车上路
- 不逆行、不闯红灯
- 骑车要戴头盔
- 注意避让行人

认识交通标志：
- 禁止标志：红色圆圈
- 警告标志：黄色三角形
- 指示标志：蓝色圆形
```

### 2. 消防安全
```markdown
火灾预防：
- 不玩火
- 不玩弄电器
- 不乱动燃气
- 不燃放烟花爆竹

火灾逃生：
- 发现火灾及时拨打119
- 用湿毛巾捂住口鼻
- 弯腰低姿逃生
- 不乘坐电梯

火灾自救：
- 身上着火就地打滚
- 被困时发出求救信号
- 等待救援
- 不要跳楼

认识消防设施：
- 灭火器
- 消火栓
- 安全出口
- 疏散指示标志
```

### 3. 人身安全
```markdown
防拐骗：
- 不跟陌生人走
- 不吃陌生人的东西
- 不给陌生人开门
- 记住家长电话

防性侵：
- 认识身体隐私部位
- 不让别人触碰隐私部位
- 告诉父母被侵犯
- 学会拒绝

防溺水：
- 不私自下水游泳
- 不在无家长陪同下游泳
- 不到不熟悉的水域游泳
- 发现溺水及时呼救

网络安全：
- 不泄露个人信息
- 不轻信网络陌生人
- 不浏览不良网站
- 合理安排上网时间
```

### 4. 突发事件应对
```markdown
地震：
- 躲在桌子下
- 保护头部
- 远离玻璃窗
- 有序撤离

洪水：
- 往高处转移
- 不涉水行走
- 等待救援
- 注意卫生防疫

雷电：
- 不在树下躲避
- 不在空旷地带
- 关闭电器
- 不接打电话

紧急求助电话：
- 报警：110
- 火警：119
- 急救：120
- 交通事故：122
```
