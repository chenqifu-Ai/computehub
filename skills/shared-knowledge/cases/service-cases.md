# 服务案例库

## 案例001：初创公司一站式服务

### 客户背景
- 行业：科技初创公司
- 阶段：天使轮后
- 需求：工商注册+财税+人事+法律

### 服务方案
| 服务 | 内容 | 周期 | 费用 |
|------|------|------|------|
| 工商注册 | 公司设立 | 5天 | 2000 |
| 代理记账 | 月度账务 | 月度 | 500/月 |
| 人事代理 | 社保公积金 | 月度 | 200/月 |
| 法律顾问 | 年度服务 | 年度 | 12000/年 |

### 客户价值
- 一站式解决，节省时间
- 专业团队，降低风险
- 年度套餐，性价比高

## 案例002：个体户转公司

### 客户背景
- 行业：餐饮连锁
- 原主体：个体户
- 需求：转型为公司

### 服务内容
1. 新公司注册
2. 商标转让
3. 人员转移
4. 税务变更
5. 银行账户变更

### 关键节点
- 时间：30天完成转型
- 风险：税务筹划（个体户税率不同）
- 成本：合理规划转型成本

## 案例003：股权激励设计

### 客户背景
- 行业：互联网
- 阶段：A轮
- 需求：核心员工股权激励

### 方案设计
| 项目 | 内容 |
|------|------|
| 激励对象 | 10人核心团队 |
| 激励比例 | 10%股权池 |
| 行权条件 | 服务满4年 |
| 退出机制 | 回购条款 |

### 法律文件
- 股权激励协议
- 股权代持协议
- 公司章程修改

## 案例004：税务筹划

### 客户背景
- 行业：咨询服务
- 年收入：500万
- 税负高：企业所得税+分红个税

### 筹划方案
1. 小微企业优惠：应税所得100万以内实际税率2.5%
2. 合理费用：工资、社保、办公费用
3. 利润留存：再投资计划

### 效果
- 税负降低30%
- 合规无风险

## 案例005：劳动争议处理

### 案情
- 员工主张违法解除
- 要求赔偿2N
- 公司无证据证明合规

### 处理方案
1. 协商调解
2. 完善证据链
3. 达成和解协议

### 教训
- 保留书面证据
- 规章制度要签收
- 绩效考核要有记录
