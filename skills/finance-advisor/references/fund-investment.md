# 基金理财知识

## 基金基础概念

### 什么是基金
基金是一种集合投资工具，由专业管理人运作，为投资者赚取收益。

### 基金分类
| 类型 | 投资方向 | 风险等级 | 适合人群 |
|------|----------|----------|----------|
| 货币基金 | 短期债券、存款 | 低 | 保守型 |
| 债券基金 | 债券为主 | 中低 | 稳健型 |
| 混合基金 | 股票+债券 | 中 | 平衡型 |
| 股票基金 | 股票为主 | 高 | 进取型 |
| 指数基金 | 跟踪指数 | 中高 | 长期投资 |

## 基金投资策略

### 定投策略
**原理**：定期定额投资，摊低成本，平滑波动。

| 定投方式 | 说明 | 适合 |
|----------|------|------|
| 普通定投 | 固定金额 | 新手 |
| 智能定投 | 低位多投 | 有经验者 |
| 价值平均 | 目标市值 | 专业投资者 |

### 定投技巧
- 周四定投效果较好（历史数据）
- 下跌坚持定投，不轻易停止
- 止盈不止损，目标收益10-30%
- 牛市分批止盈，熊市坚持定投

### 资产配置
| 年龄 | 股基比例 | 债基比例 |
|------|----------|----------|
| 20-30岁 | 70-80% | 20-30% |
| 30-40岁 | 60-70% | 30-40% |
| 40-50岁 | 50-60% | 40-50% |
| 50岁以上 | 30-40% | 60-70% |

## 基金费用

### 费用构成
| 费用 | 费率 | 说明 |
|------|------|------|
| 申购费 | 0.1-1.5% | 买入时收取 |
| 赎回费 | 0-1.5% | 卖出时收取，持有越久越低 |
| 管理费 | 0.5-2% | 年化，每日计提 |
| 托管费 | 0.1-0.25% | 年化，每日计提 |

### 省费技巧
1. 选择C类份额（无申购费）
2. 持有满7天免惩罚性赎回费
3. 选择费率较低的指数基金
4. 大额申购享受折扣

## 基金买卖时机

### 买入时机
| 信号 | 说明 |
|------|------|
| 估值低位 | 市盈率低于历史30%分位 |
| 市场恐慌 | 大跌后逐步买入 |
| 定期定投 | 不择时，长期坚持 |

### 卖出时机
| 信号 | 说明 |
|------|------|
| 达到目标收益 | 按计划止盈 |
| 估值高位 | 市盈率高于历史70%分位 |
| 基金变差 | 经理更换、业绩下滑 |
| 资金需求 | 需要用钱 |

## 基金投资误区

### 常见错误
| 错误 | 说明 | 正确做法 |
|------|------|----------|
| 追涨杀跌 | 涨买跌卖 | 低买高卖 |
| 频繁操作 | 短期买卖 | 长期持有 |
| 只看收益 | 忽略风险 | 综合评估 |
| 集中投资 | 单只基金 | 分散配置 |
| 迷信冠军 | 买去年冠军 | 看长期业绩 |

### 持有周期
- 基金投资建议持有3年以上
- 长期持有胜率更高
- 避免短期频繁交易
