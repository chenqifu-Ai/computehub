# 初创企业保险规划指南

## 保险基础概念

### 什么是保险
保险是一种风险转移机制，通过支付保费，将潜在风险损失转移给保险公司。

### 保险分类
| 类型 | 说明 | 适用场景 |
|------|------|----------|
| 财产保险 | 保障财产损失 | 企业财产 |
| 责任保险 | 保障第三方责任 | 产品责任 |
| 人身保险 | 保障人身风险 | 员工保障 |
| 信用保险 | 保障信用风险 | 应收账款 |

### 初创企业保险需求
| 阶段 | 主要风险 | 保险需求 |
|------|----------|----------|
| 种子期 | 创业失败 | 低 |
| 天使轮 | 团队稳定 | 中 |
| A轮 | 业务扩张 | 高 |
| B轮+ | 全面风险 | 全面保障 |

## 保险管理

### 保险管理制度
| 内容 | 说明 |
|------|------|
| 保险台账 | 记录所有保险 |
| 续保提醒 | 提前30天提醒 |
| 理赔流程 | 明确理赔流程 |
| 定期评估 | 年度保险评估 |

### 理赔流程
```
事故发生 → 及时报案 → 收集材料 → 提交理赔 → 
配合调查 → 获得赔偿
```

### 理赔要点
| 要点 | 说明 |
|------|------|
| 及时报案 | 48小时内报案 |
| 材料完整 | 准备完整材料 |
| 实事求是 | 如实描述事故 |
| 保留证据 | 保留相关证据 |

## 常见问题

### Q1: 必须买哪些保险？
```
法定必须：
- 社保（养老、医疗、失业、工伤、生育）

建议购买：
- 雇主责任险（工伤补充）
- 公众责任险（经营场所）
- 意外险（创始人）
```

### Q2: 保险费用怎么预算？
```
建议比例：年营业收入的0.5-1%
或：年利润的2-5%
```

### Q3: 保险理赔难怎么办？
```
预防措施：
1. 投保前了解条款
2. 选择信誉好的公司
3. 通过经纪人投保
4. 出险及时报案
5. 准备完整材料
```
