# 风险管理实务

## 风险管理概述

### 风险定义
风险是指未来结果的不确定性，可能带来损失或收益。

### 风险分类
| 类型 | 说明 | 案例 |
|------|------|------|
| 市场风险 | 价格波动风险 | 股价、汇率下跌 |
| 信用风险 | 违约风险 | 客户欠款不还 |
| 操作风险 | 人为错误风险 | 系统故障、操作失误 |
| 流动性风险 | 资金周转风险 | 无法变现 |
| 法律风险 | 法律纠纷风险 | 合同纠纷 |
| 声誉风险 | 声誉损失风险 | 负面新闻 |

## 风险管理流程

### 风险管理步骤
| 步骤 | 内容 | 方法 |
|------|------|------|
| 风险识别 | 发现风险 | 头脑风暴、检查表 |
| 风险评估 | 评估影响 | 定性/定量分析 |
| 风险应对 | 制定对策 | 规避、转移、控制 |
| 风险监控 | 持续跟踪 | 预警机制、定期检查 |

### 风险评估方法
| 方法 | 说明 | 适用 |
|------|------|------|
| 风险矩阵 | 可能性×影响程度 | 通用 |
| 敏感性分析 | 变量变化影响 | 财务风险 |
| 情景分析 | 不同情景模拟 | 战略风险 |
| 压力测试 | 极端情况测试 | 金融风险 |

## 风险应对策略

### 四大策略
| 策略 | 说明 | 适用 |
|------|------|------|
| 规避 | 放弃风险活动 | 高风险、低收益 |
| 转移 | 转嫁风险 | 保险、外包 |
| 控制 | 降低风险概率和影响 | 可控风险 |
| 接受 | 承担风险成本 | 低风险、低影响 |

### 风险转移方式
| 方式 | 说明 | 案例 |
|------|------|------|
| 保险 | 购买保险 | 财产险、责任险 |
| 外包 | 外包风险业务 | IT外包 |
| 合同条款 | 合同约定风险 | 违约责任 |
| 衍生品 | 对冲风险 | 期货、期权 |

## 风险预警

### 预警指标
| 类型 | 指标 | 预警值 |
|------|------|--------|
| 财务 | 资产负债率 | >70% |
| 财务 | 流动比率 | <1 |
| 财务 | 现金比率 | <20% |
| 运营 | 库存周转天数 | >90天 |
| 运营 | 应收账款周转天数 | >60天 |

### 预警机制
| 级别 | 标准 | 行动 |
|------|------|------|
| 绿色 | 正常范围 | 正常经营 |
| 黄色 | 接近预警 | 关注观察 |
| 橙色 | 达到预警 | 分析原因 |
| 红色 | 超过预警 | 立即行动 |

## 风险管理案例

### 案例一：应收账款风险
**问题**：某企业应收账款300万，超期6个月
**原因**：客户信用评估缺失、账期管理混乱
**措施**：
1. 建立客户信用评估体系
2. 设定账期上限
3. 加强催收
4. 购买信用保险

### 案例二：供应链风险
**问题**：主要供应商断供
**原因**：过度依赖单一供应商
**措施**：
1. 开发备用供应商
2. 建立安全库存
3. 签订长期合同
4. 垂直整合
