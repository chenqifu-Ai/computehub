# 初创企业银行业务指南

## 企业银行账户

### 账户类型
| 类型 | 用途 | 特点 |
|------|------|------|
| 基本存款账户 | 日常转账结算、现金收付 | 每个企业只能开一个 |
| 一般存款账户 | 借款转存等 | 可开多个，不能取现 |
| 临时存款账户 | 临时经营 | 有期限限制 |
| 专用存款账户 | 专项资金管理 | 专款专用 |

### 开户流程
| 步骤 | 内容 | 时间 | 材料 |
|------|------|------|------|
| 1 | 准备材料 | - | 营业执照、法人身份证、公章等 |
| 2 | 预约银行 | 1-2天 | 电话/网上预约 |
| 3 | 柜面办理 | 1-2小时 | 法人到场，签署协议 |
| 4 | 等待审核 | 1-3天 | 银行审核材料 |
| 5 | 领取账户 | 1天 | 领取开户许可证/账号 |

### 开户材料清单
```
必备材料：
□ 营业执照正本
□ 法定代表人身份证原件
□ 公司公章、财务章、法人章
□ 公司章程（部分银行需要）
□ 公司地址证明（租赁合同或房产证）

如非法人办理：
□ 经办人身份证原件
□ 授权委托书（法人签字盖章）
```

### 银行选择
| 银行类型 | 优势 | 劣势 | 适合 |
|----------|------|------|------|
| 国有大行 | 网点多、稳定 | 手续繁琐、服务一般 | 大企业 |
| 股份制银行 | 服务好、产品多 | 网点相对少 | 中小企业 |
| 城商行 | 政策灵活 | 跨区域受限 | 本地企业 |
| 互联网银行 | 便捷、费用低 | 无网点 | 线上企业 |

## 企业贷款

### 贷款类型
| 类型 | 说明 | 额度 | 期限 |
|------|------|------|------|
| 流动资金贷款 | 日常经营周转 | 较小 | 1年以内 |
| 固定资产贷款 | 购买设备等 | 较大 | 1-5年 |
| 经营性物业贷款 | 物业抵押 | 中等 | 较长 |
| 票据融资 | 票据贴现 | 视票据 | 短期 |
| 信用贷款 | 无抵押 | 较小 | 1年以内 |
| 抵押贷款 | 有抵押 | 较大 | 较长 |

### 初创企业贷款难点
| 难点 | 原因 | 解决方法 |
|------|------|----------|
| 无抵押物 | 资产少 | 信用贷款、担保贷款 |
| 经营时间短 | 无历史数据 | 提供商业计划书 |
| 无流水 | 收入少 | 个人资产抵押 |
| 信用记录少 | 新企业 | 法人个人信用 |

### 贷款申请材料
```
基本材料：
□ 营业执照
□ 组织机构代码证（如有）
□ 税务登记证（如有）
□ 公司章程
□ 财务报表（近2年）
□ 银行流水（近6个月）
□ 经营场所证明

补充材料：
□ 贷款申请书
□ 贷款用途说明
□ 还款计划
□ 抵押物证明（如需）
□ 担保人资料（如需）
```

### 贷款利率
| 类型 | 利率范围 | 说明 |
|------|----------|------|
| LPR基准 | 3.45%-4.2% | 贷款市场报价利率 |
| 信用贷款 | LPR+2%-5% | 无抵押，利率高 |
| 抵押贷款 | LPR+0.5%-2% | 有抵押，利率低 |
| 政策性贷款 | 优惠利率 | 政府支持项目 |

## 支付结算

### 结算方式
| 方式 | 说明 | 特点 | 适用 |
|------|------|------|------|
| 网银转账 | 网上银行转账 | 快捷、方便 | 日常结算 |
| 支票 | 纸质支票 | 传统方式 | 大额支付 |
| 银行汇票 | 银行承兑 | 信用高 | 异地结算 |
| 商业汇票 | 企业承兑 | 期限灵活 | 延期支付 |
| 托收承付 | 委托银行收款 | 银行介入 | 交易结算 |

### 网银功能
| 功能 | 说明 | 限制 |
|------|------|------|
| 查询 | 余额、明细查询 | 无限制 |
| 转账 | 对公/对私转账 | 需设置限额 |
| 代发工资 | 批量发薪 | 需签约 |
| 理财 | 购买银行理财 | 需签约 |
| 票据 | 票据业务 | 需开通 |

### 转账限额
| 类型 | 一般限额 | 说明 |
|------|----------|------|
| 普通网银 | 50-500万/天 | 视银行 |
| U盾/动态密码 | 500万-无限制 | 安全认证 |
| 短信验证 | 5-50万/天 | 较低 |

## 现金管理

### 现金管理原则
| 原则 | 说明 |
|------|------|
| 公私分离 | 公司账户与个人账户分开 |
| 日清月结 | 每日清点，每月核对 |
| 专人管理 | 出纳专人负责 |
| 定期盘点 | 定期核对账实 |
| 留存限额 | 超额及时存银行 |

### 现金使用范围
| 允许 | 不允许 |
|------|--------|
| 员工工资（特殊情况下） | 大额采购 |
| 个人劳务报酬 | 超过结算起点（1000元） |
| 出差人员差旅费 | - |
| 结算起点以下零星支出 | - |

### 银行存款管理
| 项目 | 要求 |
|------|------|
| 对账 | 每月与银行对账 |
| 账实核对 | 账面余额与银行余额一致 |
| 未达账项 | 编制银行存款余额调节表 |
| 支票管理 | 登记领用、作废 |

## 企业网银操作

### 开通流程
| 步骤 | 内容 |
|------|------|
| 1 | 开立对公账户 |
| 2 | 申请开通网银 |
| 3 | 领取U盾/动态密码器 |
| 4 | 设置操作员权限 |
| 5 | 开始使用 |

### 权限设置
| 角色 | 权限 | 说明 |
|------|------|------|
| 主管 | 全部权限 | 最高权限 |
| 制单员 | 填写单据 | 发起转账 |
| 复核员 | 审核单据 | 审核转账 |
| 查询员 | 仅查询 | 只能查看 |

### 安全措施
| 措施 | 说明 |
|------|------|
| U盾/动态密码 | 二次认证 |
| 权限分离 | 制单与复核分开 |
| 限额控制 | 设置转账限额 |
| IP限制 | 限制登录IP |
| 定期更换密码 | 增强安全性 |

## 常见问题

### Q1: 公司刚成立可以贷款吗？
```
答：较难，但可以尝试：
1. 个人经营性贷款（以法人个人名义）
2. 小微企业信用贷款（额度较小）
3. 创业担保贷款（政府支持）
4. 供应链金融（如有上下游）
```

### Q2: 网银转账限额太低怎么办？
```
方法：
1. 申请U盾/高级认证
2. 设置更高的转账限额
3. 分多笔转账
```

### Q3: 银行账户被冻结怎么办？
```
原因：
1. 涉嫌违法
2. 大额可疑交易
3. 长期不动户
4. 司法冻结

解决：
1. 联系银行了解原因
2. 提供相关证明材料
3. 配合调查
```

### Q4: 如何提高贷款成功率？
```
方法：
1. 保持良好的银行流水
2. 按时还款积累信用
3. 提供完整真实的材料
4. 选择适合的贷款产品
5. 与银行建立良好关系
```
