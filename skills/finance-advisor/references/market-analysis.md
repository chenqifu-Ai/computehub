# 金融市场分析

## 宏观经济指标

### GDP指标
- **定义**：国内生产总值，衡量经济总产出
- **影响**：GDP增长利好股市，下跌利空
- **发布频率**：季度

### 通胀指标
- **CPI**：消费者物价指数，反映生活成本
- **PPI**：生产者物价指数，反映生产成本
- **影响**：高通胀→加息→股市下跌

### 利率
- **央行政策利率**：存贷款基准利率
- **市场利率**：银行间拆借利率
- **影响**：利率上升→借贷成本增加→利空股市

### 汇率
- **人民币汇率**：USD/CNY
- **影响**：升值利好进口，贬值利好出口

## 技术分析基础

### K线图
| 形态 | 含义 |
|------|------|
| 阳线 | 收盘>开盘，上涨 |
| 阴线 | 收盘<开盘，下跌 |
| 上影线 | 最高价与实体差距 |
| 下影线 | 最低价与实体差距 |

### 均线系统
- **MA5**：5日均线，短线参考
- **MA10**：10日均线，短线支撑
- **MA20**：20日均线，中期趋势
- **MA60**：60日均线，中期支撑

### 常用指标
| 指标 | 用途 | 信号 |
|------|------|------|
| MACD | 趋势 | 金叉买入，死叉卖出 |
| KDJ | 超买超卖 | K线上穿D线买入 |
| RSI | 强弱 | >70超买，<30超卖 |
| BOLL | 波动 | 上轨压力，下轨支撑 |

## 基本面分析

### 财务指标
| 指标 | 公式 | 含义 |
|------|------|------|
| PE | 股价/每股收益 | 估值水平 |
| PB | 股价/每股净资产 | 资产估值 |
| ROE | 净利润/净资产 | 盈利能力 |
| 毛利率 | (营收-成本)/营收 | 盈利质量 |

### 行业分析
- 行业周期：导入期、成长期、成熟期、衰退期
- 竞争格局：垄断、寡头、完全竞争
- 政策影响：监管政策、产业政策
