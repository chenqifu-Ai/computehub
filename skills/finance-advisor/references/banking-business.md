# 银行业务：商业银行核心业务体系

## 一、商业银行概述

### 1. 商业银行定义
商业银行是以盈利为目的，以吸收公众存款、发放贷款、办理结算为主要业务的金融机构。

### 2. 商业银行功能
```markdown
信用中介功能：
- 吸收公众存款
- 发放贷款
- 资金融通
- 信用创造

支付中介功能：
- 办理结算
- 资金清算
- 支付服务
- 资金划转

信用创造功能：
- 存款货币创造
- 信用扩张
- 货币供给

金融服务功能：
- 代理业务
- 咨询服务
- 投资理财
- 财富管理
```

### 3. 商业银行组织架构
```markdown
公司治理：

股东大会：
- 最高权力机构
- 重大事项决策
- 选举董事监事

董事会：
- 经营决策机构
- 制定发展战略
- 聘任高级管理层

监事会：
- 监督机构
- 监督董事高管
- 检查财务

高级管理层：
- 执行机构
- 经营管理
- 执行董事会决议

组织架构类型：

总分行制：
- 总行-分行-支行
- 统一法人

事业部制：
- 按业务线划分

- 专业化经营

矩阵式：
- 业务线与区域结合
- 双重汇报

```

## 二、负债业务

### 1. 存款业务
```markdown
存款类型：

活期存款：
- 随时存取
- 不计利息或低息
- 流动性强
- 结算账户

定期存款：
- 约定存期
- 固定利率
- 到期支取
- 利率较高

储蓄存款：
- 个人储蓄
- 活期储蓄、定期储蓄
- 定活两便
- 存本取息

大额存单：
- 大额存款凭证
- 固定期限
- 可转让
- 利率市场化

结构性存款：
- 本金保障
- 收益浮动
- 与金融工具挂钩
- 风险较低

存款定价：

影响因素：
- 市场利率水平
- 资金成本
- 竞争状况
- 客户价值

定价方法：
- 成本加成定价
- 市场跟随定价
- 差异化定价
- 关系定价

存款营销：
- 产品创新
- 渠道拓展
- 客户维护
- 品牌建设
```

## 三、资产业务

### 1. 贷款业务
```markdown
贷款分类：

按期限分：
- 短期贷款：一年以内
- 中期贷款：一年至五年
- 长期贷款：五年以上

按对象分：
- 企业贷款
- 个人贷款
- 政府贷款
- 同业贷款

按用途分：
- 流动资金贷款
- 固定资产贷款
- 项目贷款
- 消费贷款

按担保方式分：
- 信用贷款
- 保证贷款
- 抵押贷款
- 质押贷款

贷款流程：

贷前调查：
- 客户资格调查
- 借款用途调查
- 还款能力调查
- 担保调查

贷时审查：
- 贷款条件审查
- 贷款额度审查
- 贷款期限审查
- 贷款利率审查

贷后检查：
- 资金用途检查
- 还款情况检查
- 担保情况检查
- 风险预警

贷款定价：

定价因素：
- 资金成本
- 运营成本
- 风险成本
- 资本成本
- 利润目标

定价方法：
- 成本加成定价
- 基准利率定价
- 风险定价
- 关系定价

贷款风险分类：

正常类：
- 能够履行合同
- 没有足够理由怀疑

关注类：
- 目前有能力偿还
- 存在一些不利因素

次级类：
- 依靠正常经营收入无法足额偿还
- 损失率30%以下

可疑类：
- 无法足额偿还
- 损失率30%-70%

损失类：
- 采取所有措施后仍无法收回
- 损失率70%以上
```

```markdown

- 中国银行保险监督管理委员会
- 中国人民银行

监管目标：
- 保护存款人利益
- 维护银行体系稳定
- 促进银行业健康发展

监管原则：
- 依法监管
- 审慎监管
- 有效监管
- 协同监管
```

### 2. 监管指标
```markdown
资本充足率：
- 核心一级资本充足率 ≥ 5%
- 一级资本充足率 ≥ 6%
- 资本充足率 ≥ 8%

流动性指标：
- 流动性覆盖率 ≥ 100%
- 净稳定资金比例 ≥ 100%
- 流动性比例 ≥ 25%

资产质量指标：
- 不良贷款率
- 拨备覆盖率 ≥ 150%
- 贷款拨备率 ≥ 2.5%

集中度指标：
- 单一客户贷款集中度 ≤ 10%
- 单一集团客户授信集中度 ≤ 15%
- 全部关联度 ≤ 50%

盈利性指标：
- 资产利润率
- 资本利润率
- 成本收入比 ≤ 45%
```
