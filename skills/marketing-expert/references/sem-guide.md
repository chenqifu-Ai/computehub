# SEM搜索引擎营销

## SEM基础概念

### 什么是SEM
SEM（Search Engine Marketing）是通过付费广告在搜索引擎获得流量的营销方式。

### SEM vs SEO
| 维度 | SEM | SEO |
|------|-----|-----|
| 获流方式 | 付费广告 | 自然排名 |
| 见效速度 | 立即见效 | 需要3-6个月 |
| 成本 | 按点击付费 | 人力成本 |
| 稳定性 | 停止即停止 | 长期有效 |

## SEM投放流程

### 账户结构
| 层级 | 说明 | 示例 |
|------|------|------|
| 账户 | 最高层级 | 企业账户 |
| 计划 | 按产品/地区划分 | 品牌计划、产品计划 |
| 单元 | 按关键词主题划分 | 品牌词单元、竞品词单元 |
| 关键词 | 用户搜索词 | "SEM培训" |
| 创意 | 广告文案 | 标题+描述 |

### 关键词策略
| 类型 | 特点 | 策略 |
|------|------|------|
| 品牌词 | 转化高、成本低 | 必买、低价 |
| 产品词 | 转化中等 | 重点竞价 |
| 竞品词 | 流量精准 | 适度投放 |
| 通用词 | 流量大、转化低 | 控制预算 |
| 长尾词 | 流量小、转化高 | 批量覆盖 |

### 创意撰写
**创意公式**：关键词 + 卖点 + 行动号召

| 要素 | 要求 |
|------|------|
| 标题 | 包含关键词、吸引眼球 |
| 描述1 | 核心卖点、产品优势 |
| 描述2 | 促销信息、行动号召 |
| 显示URL | 品牌词、信任感 |

### 创意技巧
- 使用数字：限时优惠、立省XX
- 疑问句式：还在找...？来这里
- 利益点：免费试用、送资料
- 紧迫感：限时、今日截止

## 出价与排名

### 排名公式
**广告排名 = 出价 × 质量度**

### 质量度影响因素
| 因素 | 权重 | 优化方向 |
|------|------|----------|
| 点击率 | 高 | 优化创意、提升相关性 |
| 相关性 | 高 | 关键词与创意匹配 |
| 着陆页体验 | 中 | 加载速度、内容质量 |

### 出价策略
| 策略 | 适用场景 |
|------|----------|
| 手动出价 | 有经验的优化师 |
| 智能出价 | 预算有限、新手 |
| OCPC | 目标成本导向 |
| 目标ROI | 以ROI为目标 |

## 数据分析与优化

### 核心指标
| 指标 | 公式 | 健康标准 |
|------|------|----------|
| 展现量 | 广告展示次数 | 持续增长 |
| 点击率(CTR) | 点击/展现 | >2% |
| 平均点击价格(CPC) | 费用/点击 | 行业对比 |
| 转化率(CVR) | 转化/点击 | >3% |
| 获客成本(CPA) | 费用/转化 | <客单价30% |
| ROI | 收入/费用 | >3 |

### 优化方向
| 问题 | 原因 | 优化 |
|------|------|------|
| 展现少 | 关键词少、出价低 | 加词、提价 |
| 点击率低 | 创意不吸引 | 优化创意 |
| 转化率低 | 着陆页问题 | 优化页面 |
| 成本高 | 竞价激烈 | 长尾词策略 |

### A/B测试
| 测试项 | 方法 |
|--------|------|
| 创意测试 | 多创意轮播 |
| 着陆页测试 | 多页面分流 |
| 出价测试 | 不同出价对比 |

## 负面关键词

### 什么是负面关键词
排除不相关搜索，避免无效点击。

### 常见负面词
| 行业 | 负面关键词 |
|------|------------|
| 培训 | 免费、自学、下载 |
| 产品 | 代理、加盟、二手 |
| 服务 | DIY、教程、方法 |

### 添加策略
- 定期搜索词报告
- 排除无关搜索
- 持续优化账户

## SEM投放策略

### 预算分配
| 阶段 | 预算占比 |
|------|----------|
| 测试期 | 品牌词30% + 长尾词70% |
| 成长期 | 品牌词20% + 产品词50% + 竞品词30% |
| 成熟期 | 按ROI分配 |

### 时段策略
- 分析用户活跃时段
- 高峰时段提高出价
- 低峰时段降低出价

### 地域策略
- 核心地域提高出价
- 非目标地域屏蔽
- 本地化创意文案
