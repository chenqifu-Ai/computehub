# SEO/SEM：搜索引擎优化与营销

## 一、SEO概述

### 1. SEO定义
SEO（Search Engine Optimization）搜索引擎优化，是指通过优化网站结构、内容和外部链接等，提高网站在搜索引擎自然搜索结果中的排名，从而获得更多免费流量的技术和方法。

### 2. SEO的重要性
```markdown
流量价值：
- 搜索引擎是主要流量入口
- 自然搜索流量占比高
- 流量质量好，转化率高

成本优势：
- SEO流量免费
- 相比付费广告成本更低
- 长期效果好

品牌价值：
- 搜索排名靠前增强品牌信任
- 提升品牌知名度
- 增强品牌权威性

用户体验：
- SEO优化提升网站质量
- 改善用户体验
- 提高网站可用性
```

### 3. 搜索引擎工作原理
```markdown
抓取（Crawling）：
- 搜索引擎蜘蛛访问网页
- 发现新页面，跟踪链接
- 读取页面内容

索引（Indexing）：
- 分析抓取的页面
- 存储页面信息
- 建立索引数据库

排名（Ranking）：
- 用户搜索请求
- 搜索引擎匹配索引
- 根据算法排序返回结果

核心算法因素：
- 内容相关性
- 页面质量
- 用户体验
- 网站权威性
- 技术因素
```

## 二、关键词优化

### 1. 关键词研究

#### 关键词分类
```markdown
按搜索意图分类：

导航型关键词：
- 用户寻找特定网站
- 如："淘宝"、"京东登录"
- 优化重点：品牌词、官网

信息型关键词：
- 用户寻找信息
- 如："如何减肥"、"感冒怎么办"
- 优化重点：内容质量、长尾词

交易型关键词：
- 用户准备购买
- 如："iPhone价格"、"运动鞋品牌"
- 优化重点：产品页、转化优化

按竞争程度分类：

核心关键词：
- 搜索量大、竞争激烈
- 如："手机"、"旅游"
- 难度大，需要长期优化

长尾关键词：
- 搜索量小、竞争小
- 如："2024年最新款手机推荐"
- 容易排名，流量精准

按词性分类：

品牌词：
- 包含品牌名称
- 如："华为手机"、"小米官网"

产品词：
- 产品或服务名称
- 如："跑步鞋"、"网页设计"

属性词：
- 描述产品特征
- 如："红色"、"大码"、"便宜"

地域词：
- 包含地理位置
- 如："北京搬家"、"上海租房"
```

#### 关键词研究方法
```markdown
头脑风暴：
- 站在用户角度思考
- 列出可能搜索的词
- 分类整理关键词

竞品分析：
- 分析竞争对手关键词
- 查看竞品排名情况
- 借鉴成功经验

工具研究：
- 百度指数
- 5118关键词工具
- 站长工具
- Google Keyword Planner
- Ahrefs、SEMrush

搜索建议：
- 搜索框下拉建议
- 底部相关搜索
- 了解用户搜索习惯

数据分析：
- 网站流量数据
- 关键词排名数据
- 转化数据分析
```

#### 关键词选择原则
```markdown
相关性：
- 与业务高度相关
- 与页面内容匹配
- 符合用户搜索意图

搜索量：
- 有一定搜索量
- 流量价值可观
- 不选择搜索量过低的词

竞争度：
- 评估排名难度
- 根据资源选择合适难度
- 新站从长尾词开始

商业价值：
- 关键词转化价值
- 是否能带来业务
- 优先高价值关键词
```

### 2. 关键词布局

#### 关键词密度
```markdown
定义：
- 关键词在页面中出现的频率
- 密度 = 关键词出现次数 / 页面总词数

建议密度：
- 一般2%-8%
- 自然出现为主
- 不要刻意堆砌

注意事项：
- 过高会被视为关键词堆砌
- 过低可能不够相关性
- 自然融入内容最重要
```

#### 关键词位置
```markdown
页面标题（Title）：
- 最重要的位置
- 关键词尽量靠前
- 控制在60个字符内

页面描述（Description）：
- 概括页面内容
- 包含关键词
- 控制在160个字符内

H标签：
- H1标签最重要
- 包含核心关键词
- 每个页面只有一个H1

正文内容：
- 第一段出现关键词
- 自然分布全文
- 使用同义词变体

图片Alt属性：
- 描述图片内容
- 包含关键词
- 提升图片搜索排名

URL：
- URL包含关键词
- 使用连字符分隔
- 简短易读

锚文本：
- 内链锚文本包含关键词
- 外链锚文本多样化
- 自然相关最重要
```

## 五、SEM付费推广

### 1. SEM概述

#### SEM定义
SEM（Search Engine Marketing）搜索引擎营销，是指通过付费广告在搜索引擎结果页面获得展示位置的营销方式。

#### SEM与SEO对比
```markdown
SEM特点：
- 见效快：投放即可展示
- 付费：按点击计费
- 精准：可定向投放
- 可控：预算可控

SEO特点：
- 见效慢：需要长期优化
- 免费：自然排名免费
- 长效：排名稳定持久
- 需要持续投入

结合策略：
- SEO打基础，长期获客
- SEM短期见效，快速引流
- SEO和SEM关键词互补
- 数据共享，相互促进
```

### 2. SEM账户结构

#### 账户层级
```markdown
账户：
- 顶层结构
- 管理所有推广计划
- 设置预算

推广计划：
- 按产品/服务分类
- 按地域分类
- 按推广目标分类

推广单元：
- 按关键词主题分组
- 一组相关关键词
- 对应一组创意

关键词：
- 最小单位
- 触发广告展示
- 设置出价和匹配方式
```

#### 账户搭建原则
```markdown
结构清晰：
- 计划划分合理
- 单元主题明确
- 便于管理和优化

覆盖全面：
- 覆盖各类关键词
- 包含不同购买阶段
- 覆盖核心词和长尾词

便于优化：
- 结构便于数据分析
- 便于调整出价
- 便于优化创意
```

### 3. 关键词策略

#### 关键词选择
```markdown
核心词：
- 产品/服务核心关键词
- 搜索量大
- 竞争激烈
- 出价较高

长尾词：
- 搜索量较小
- 竞争较小
- 转化率高
- 出价较低

竞品词：
- 竞争对手品牌词
- 获取竞品流量
- 注意合规风险

品牌词：
- 自己品牌词
- 保护品牌流量
- 成本低、转化高
```

#### 关键词匹配方式
```markdown
精确匹配：
- 搜索词完全相同
- 流量最精准
- 流量最小

短语匹配：
- 包含关键词短语
- 流量适中
- 精准度适中

广泛匹配：
- 相关搜索词都可能触发
- 流量最大
- 精准度最低

否定关键词：
- 排除不相关搜索
- 避免无效点击
- 提高投放效率
```

#### 关键词出价
```markdown
出价原则：
- 核心词高出价
- 长尾词低出价
- 品牌词低价保护

出价策略：
- 根据转化价值出价
```

### 6. SEM数据分析

#### 核心指标
```markdown
展现量：
- 广告展示次数
- 反映覆盖范围

点击量：
- 广告点击次数
- 反映吸引力

点击率（CTR）：
- 点击量/展现量
- 反映广告吸引力
- 一般2%-5%为正常

平均点击价格（CPC）：
- 总花费/点击量
- 反映点击成本

转化量：
- 完成目标行动次数
- 如咨询、注册、购买

转化率（CVR）：
- 转化量/点击量
- 反映落地页效果

获客成本（CPA）：
- 总花费/转化量
- 反映获客效率

投资回报率（ROI）：
- 收益/成本
- 反映投放效果
```

#### 数据分析方法
```markdown
账户分析：
- 计划效果对比
- 单元效果对比
- 预算分配优化

关键词分析：
- 高消费低转化词：优化或暂停
- 低消费高转化词：增加投入
- 高展现低点击词：优化创意
- 低展现高点击词：提高出价

创意分析：
- 点击率对比
- 创意效果测试
- 优化创意版本

时段地域分析：
- 高转化时段：增加预算
- 低转化时段：减少预算
- 地域效果对比
- 调整投放策略
```
