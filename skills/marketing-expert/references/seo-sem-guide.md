# SEO/SEM：搜索引擎优化与营销

## 一、SEO基础概念

### 1. SEO定义
SEO（Search Engine Optimization，搜索引擎优化）是指通过优化网站结构、内容和外部链接等因素，提高网站在搜索引擎自然搜索结果中的排名，从而获得更多免费流量的技术和方法。

### 2. SEO核心目标
```markdown
提高排名：让网站关键词排名更靠前
增加流量：获得更多搜索引擎自然流量
提升体验：改善用户体验和网站质量
建立信任：获得搜索引擎信任和用户信任
```

### 3. 搜索引擎工作原理
```markdown
抓取（Crawling）：
- 搜索引擎蜘蛛爬行网页
- 发现新页面和更新内容
- 通过链接发现页面

索引（Indexing）：
- 解析网页内容
- 存储到数据库
- 建立索引结构

排名（Ranking）：
- 用户输入搜索词
- 从索引中检索相关页面
- 根据算法排序
- 展示搜索结果
```

### 4. 搜索引擎排名因素
| 因素类别 | 具体因素 | 权重趋势 |
|----------|----------|----------|
| 内容因素 | 内容质量、相关性、原创性 | 高 ↑ |
| 技术因素 | 网站速度、移动友好、安全性 | 高 |
| 外链因素 | 链接数量、质量、相关性 | 中 |
| 用户体验 | 点击率、停留时间、跳出率 | 高 ↑ |
| 品牌因素 | 品牌搜索量、品牌提及 | 中 ↑ |

## 二、SEO技术优化

### 1. 网站架构优化

#### URL结构
```markdown
优化原则：
- URL简洁明了
- 包含关键词
- 使用连字符分隔
- 避免动态参数
- 使用HTTPS

示例：
好：https://example.com/seo-guide
差：https://example.com/page?id=123&cat=5
```

#### 网站导航
```markdown
优化要点：
- 导航清晰简洁
- 层级不超过3层
- 使用面包屑导航
- 重要页面离首页不超过3次点击
- 移动端导航友好
```

#### 内部链接
```markdown
内链作用：
- 传递页面权重
- 帮助蜘蛛爬行
- 提升用户体验
- 增加页面相关性

内链优化：
- 相关页面互相链接
- 锚文本自然多样
- 重要页面获得更多内链
- 避免死链和孤立页面
```

### 2. 页面优化

#### 标题标签（Title）
```markdown
优化要点：
- 长度：50-60字符（中文25-30字）
- 包含核心关键词
- 品牌词放后面
- 每个页面标题唯一
- 吸引用户点击

示例：
主关键词 - 品牌名
【2024】SEO优化指南 | 免费学习搜索引擎优化
```

#### 元描述（Meta Description）
```markdown
优化要点：
- 长度：150-160字符（中文75-80字）
- 包含关键词
- 描述页面内容
- 吸引用户点击
- 每个页面唯一

注意：元描述不直接影响排名，但影响点击率
```

#### 标题层级（H1-H6）
```markdown
优化要点：
- 每个页面只有一个H1
- H1包含核心关键词
- 按层级使用H2-H6
- 内容结构清晰
- 不要跳跃使用

示例：
H1：SEO优化完整指南
H2：什么是SEO
H3：SEO的作用
H2：SEO技术优化
H3：网站架构优化
```

#### 图片优化
```markdown
优化要点：
- 文件名描述性
- 添加alt属性
- 压缩图片大小
- 使用响应式图片
- 添加图片结构化数据

示例：
<img src="seo-guide.jpg" alt="SEO优化指南示意图">
```

### 3. 网站速度优化

#### 页面加载速度
```markdown
优化方法：
- 压缩CSS、JS文件
- 图片压缩和懒加载
- 使用CDN加速
- 开启浏览器缓存
- 减少HTTP请求
- 服务器响应优化

速度标准：
- 移动端：< 3秒
- 桌面端：< 2秒
- 首屏渲染：< 1.5秒
```

#### Core Web Vitals
```markdown
核心指标：
- LCP（最大内容渲染）：< 2.5秒
- FID（首次输入延迟）：< 100ms
- CLS（累积布局偏移）：< 0.1

优化方法：
- LCP：优化首屏内容加载
- FID：减少JavaScript阻塞
- CLS：预留图片位置、避免动态内容
```

### 4. 移动端优化

#### 移动友好要求
```markdown
响应式设计：
- 自适应不同屏幕
- 同一URL、同一内容
- 用户无需缩放

移动端体验：
- 字体大小适中（16px+）
- 按钮大小适中（44px+）
- 避免弹窗遮挡内容
- 触摸元素间距合理
```

### 2. 网站地图（Sitemap）

#### XML网站地图
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/page1</loc>
    <lastmod>2024-01-01</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.8</priority>
  </url>
</urlset>
```

#### 网站地图要点
```markdown
包含内容：
- 重要页面URL
- 更新时间
- 更新频率
- 优先级

提交方式：
- 搜索引擎站长平台
- robots.txt文件

更新频率：
- 新页面及时添加
- 定期更新修改
- 删除失效页面
```

### 3. Robots.txt

#### Robots.txt配置
```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /private/
Sitemap: https://example.com/sitemap.xml
```

#### 配置要点
```markdown
允许访问：
- 公开页面
- 静态资源（CSS、JS）

禁止访问：
- 管理后台
- 用户隐私页面
- 临时页面
- 搜索结果页面

注意事项：
- 不要禁止重要页面
- 不要禁止CSS、JS文件
- 定期检查配置
```

#### Canonical标签使用

```html
<link rel="canonical" href="https://example.com/page" />
```

#### 使用场景
```markdown

参数页面：
- 带参数的URL
- 分页URL

跨域重复：
```

## 六、SEM基础

### 1. SEM定义
SEM（Search Engine Marketing，搜索引擎营销）是指通过搜索引擎平台进行付费推广的营销方式，主要包括搜索广告和展示广告。

### 2. SEM与SEO对比
| 维度 | SEO | SEM |
|------|-----|-----|
| 费用 | 免费（优化成本） | 付费点击 |
| 见效时间 | 较慢（3-6个月） | 较快（即时） |
| 排名稳定性 | 相对稳定 | 预算用完即无 |
| 点击成本 | 无 | 每次点击付费 |
| 流量可控性 | 较低 | 较高 |
| 可信度 | 较高（自然结果） | 一般（广告标识） |

### 3. SEM账户结构
```markdown
账户层级：
账户 → 广告系列 → 广告组 → 广告 → 关键词

账户结构设计：
- 按产品/服务分广告系列
- 按主题/关键词分广告组
- 每个广告组关键词相关
- 广告创意与关键词匹配
```

### 4. 关键词策略

#### 关键词类型
```markdown
品牌词：包含品牌名的关键词
- 竞争度：低
- 转化率：高
- 点击成本：低

产品词：具体产品名称关键词
- 竞争度：中
- 转化率：中高
- 点击成本：中

通用词：行业通用词汇
- 竞争度：高
- 转化率：中低
- 点击成本：高

竞品词：竞争对手品牌词
- 竞争度：中
- 转化率：中
- 点击成本：中
```

#### 关键词匹配方式
```markdown
精确匹配：搜索词与关键词完全相同
- 示例："SEO优化" 匹配 "SEO优化"
- 特点：精准、量少

短语匹配：搜索词包含关键词
- 示例："SEO优化" 匹配 "SEO优化服务"
- 特点：相关、量中

广泛匹配：搜索词与关键词相关
- 示例："SEO优化" 匹配 "网站推广"
- 特点：范围广、量多、需否词

否定匹配：排除不相关的搜索词
- 示例：添加 "免费" 为否定词
- 特点：排除无效流量
```

### 5. 广告创意优化

#### 创意要素
```markdown
标题：吸引注意，包含关键词
描述：说明产品优势，引导点击
显示URL：增强可信度
附加信息：突出卖点
```

#### 创意优化技巧
```markdown
标题优化：
- 包含核心关键词
- 突出卖点优势
- 使用数字和符号
- 制造紧迫感

描述优化：
- 说明产品特点
- 突出用户利益
- 包含行动号召
- 区别于竞争对手

附加信息：
- 站点链接
- 评价评分

- 位置信息
```

### 6. 出价与排名

#### 排名因素
```markdown
广告排名 = 出价 × 质量得分

质量得分因素：
- 点击率（CTR）
- 广告相关性
- 落地页体验
- 广告历史表现
```

#### 出价策略
```markdown
手动出价：
- 自主控制出价
- 适合有经验投放者
- 精细化管理

自动出价：

出价调整：
- 设备调整

- 人群调整
```
