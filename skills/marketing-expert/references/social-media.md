## 微信私域运营

### 私域架构
```
公众号（内容）→ 个人号（服务）→ 企业微信（客户管理）
                    ↓
              社群（互动）
```

## 抖音运营

### 账号定位
- **人设**：专业顾问、创业导师
- **内容**：行业知识、创业故事
- **风格**：专业、亲切、接地气

### 内容创作
| 类型 | 形式 | 示例 |
|------|------|------|
| 知识科普 | 口播+字幕 | "注册公司需要多少钱？" |
| 案例分享 | 剧情+讲解 | "帮客户解决XXX问题" |
| 日常vlog | 记录 | "代账公司的一天" |
| 热点解读 | 时事+观点 | "新政解读" |

### 爆款公式
```
爆款 = 选题(50%) + 脚本(30%) + 表现(20%)

选题：用户关心的问题
脚本：黄金开头+干货内容+互动引导
表现：形象、语速、剪辑
```

## 知乎运营

### 内容策略
| 类型 | 特点 | 效果 |
|------|------|------|
| 回答问题 | 专业解答 | 获取流量 |
| 发文章 | 系统内容 | 建立权威 |
| 想法 | 短内容 | 保持活跃 |
| 专栏 | 系列内容 | 粉丝沉淀 |

### 回答技巧
1. **选题**：关注度高、竞争小的问题
2. **开头**：吸引点击（故事、数据、观点）
3. **结构**：总分总，层次分明
4. **配图**：增加可读性
5. **结尾**：引导关注
