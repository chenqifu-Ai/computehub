# SEO搜索引擎优化

## SEO基础概念

### 什么是SEO
SEO（Search Engine Optimization）是通过优化网站，提高在搜索引擎自然排名的技术和方法。

### 核心目标
- 提高关键词排名
- 增加网站流量
- 提升用户体验
- 降低获客成本

## SEO三大要素

### 技术SEO
| 项目 | 要点 |
|------|------|
| 网站速度 | 加载时间<3秒 |
| 移动适配 | 响应式设计 |
| HTTPS | 安全证书 |
| 结构化数据 | Schema标记 |
| 网站地图 | XML sitemap |

### 内容SEO
| 要素 | 说明 |
|------|------|
| 标题标签 | 包含关键词，60字符内 |
| 描述标签 | 吸引点击，160字符内 |
| H标签 | H1-H6层级分明 |
| 关键词密度 | 2-8%自然分布 |
| 内容质量 | 原创、有价值 |

### 外链建设
| 类型 | 价值 |
|------|------|
| 自然外链 | 最高 |
| 友情链接 | 中等 |
| 目录提交 | 较低 |
| 论坛签名 | 很低 |

## 关键词策略

### 关键词分类
| 类型 | 搜索量 | 竞争度 | 转化率 |
|------|--------|--------|--------|
| 核心词 | 高 | 高 | 低 |
| 长尾词 | 低 | 低 | 高 |
| 品牌词 | 中 | 低 | 高 |

### 关键词研究工具
- 百度指数
- 5118
- 站长工具
- Google Keyword Planner

## 关键词布局

### 页面分布
| 页面 | 关键词数量 |
|------|-----------|
| 首页 | 3-5个核心词 |
| 栏目页 | 2-3个相关词 |
| 内容页 | 1个长尾词 |

### 布局位置
- 标题（Title）
- 描述（Description）
- 正文首段
- H1-H2标签
- 图片Alt属性
- URL路径

## SEO效果监测

### 核心指标
| 指标 | 说明 |
|------|------|
| 收录量 | 被搜索引擎收录页面数 |
| 排名 | 关键词搜索排名 |
| 流量 | SEO来源访问量 |
| 跳出率 | 单页访问比例 |
| 转化率 | 完成目标的比例 |
