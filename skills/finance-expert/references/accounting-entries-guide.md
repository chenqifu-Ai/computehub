# 会计实务：常见业务账务处理

## 一、会计分录基础

### 1. 会计分录要素
```markdown
会计分录 = 借方科目 + 借方金额 + 贷方科目 + 贷方金额

基本原则：
- 有借必有贷，借贷必相等
- 资产、成本、费用类：借增贷减
- 负债、权益、收入类：借减贷增
```

### 2. 科目分类
| 类型 | 借方 | 贷方 | 余额 |
|------|------|------|------|
| 资产类 | 增加 | 减少 | 借方 |
| 负债类 | 减少 | 增加 | 贷方 |
| 所有者权益类 | 减少 | 增加 | 贷方 |
| 成本类 | 增加 | 结转 | 借方或无 |
| 损益类（收入） | 减少 | 增加 | 无 |
| 损益类（费用） | 增加 | 减少 | 无 |

## 二、资金筹集业务

### 1. 接受投资
```markdown
借：银行存款
    贷：实收资本（或股本）
        资本公积——资本溢价（如有溢价）

例：企业收到股东投资100万元
借：银行存款 1,000,000
    贷：实收资本 1,000,000
```

### 2. 银行借款
```markdown
借：银行存款
    贷：短期借款（一年内）/ 长期借款（超过一年）

例：从银行取得一年期借款50万元
借：银行存款 500,000
    贷：短期借款 500,000
```

### 3. 借款利息
```markdown
借：财务费用——利息支出
    贷：应付利息 / 银行存款

例：计提本月借款利息5,000元
借：财务费用——利息支出 5,000
    贷：应付利息 5,000
```

## 三、采购业务

### 1. 采购材料（一般纳税人）
```markdown
借：原材料（材料成本）
    应交税费——应交增值税（进项税额）
    贷：银行存款 / 应付账款 / 应付票据

例：采购材料10万元，增值税1.3万元，款已付
借：原材料 100,000
    应交税费——应交增值税（进项税额） 13,000
    贷：银行存款 113,000
```

### 2. 采购材料（小规模纳税人）
```markdown
借：原材料（含税价格）
    贷：银行存款 / 应付账款

例：小规模纳税人采购材料5万元（含税）
借：原材料 50,000
    贷：银行存款 50,000
```

### 3. 采购运费
```markdown
借：原材料 / 销售费用（如为销售）
    应交税费——应交增值税（进项税额）
    贷：银行存款

例：支付材料运费2,000元，增值税180元
借：原材料 2,000
    应交税费——应交增值税（进项税额） 180
    贷：银行存款 2,180
```

## 四、生产业务

### 1. 领用材料
```markdown
借：生产成本（直接材料）
    制造费用（间接材料）
    管理费用（管理部门用）
    贷：原材料

例：生产领用材料8万元，车间一般消耗5,000元
借：生产成本 80,000
    制造费用 5,000
    贷：原材料 85,000
```

### 2. 分配工资
```markdown
借：生产成本（生产工人工资）
    制造费用（车间管理人员工资）
    管理费用（管理人员工资）
    销售费用（销售人员工资）
    贷：应付职工薪酬

例：分配本月工资：生产工人3万，车间管理5千，行政1万，销售8千
借：生产成本 30,000
    制造费用 5,000
    管理费用 10,000
    销售费用 8,000
    贷：应付职工薪酬 53,000
```

### 3. 计提折旧
```markdown
借：制造费用（生产用）
    管理费用（管理用）
    销售费用（销售用）
    贷：累计折旧

例：本月计提折旧：生产设备1万，办公设备3千，销售设备2千
借：制造费用 10,000
    管理费用 3,000
    销售费用 2,000
    贷：累计折旧 15,000
```

### 4. 结转制造费用
```markdown
借：生产成本
    贷：制造费用

例：结转本月制造费用2万元
借：生产成本 20,000
    贷：制造费用 20,000
```

### 5. 完工产品入库
```markdown
借：库存商品
    贷：生产成本

例：完工产品成本15万元入库
借：库存商品 150,000
    贷：生产成本 150,000
```

## 五、销售业务

### 1. 销售商品（一般纳税人）
```markdown
借：银行存款 / 应收账款 / 应收票据
    贷：主营业务收入
        应交税费——应交增值税（销项税额）

例：销售商品20万元，增值税2.6万元，款已收
借：银行存款 226,000
    贷：主营业务收入 200,000
        应交税费——应交增值税（销项税额） 26,000
```

### 2. 结转销售成本
```markdown
借：主营业务成本
    贷：库存商品

例：销售商品成本12万元
借：主营业务成本 120,000
    贷：库存商品 120,000
```

### 3. 销售退回
```markdown
借：主营业务收入
    应交税费——应交增值税（销项税额）
    贷：银行存款 / 应收账款

借：库存商品
    贷：主营业务成本

例：客户退回商品1万元，成本6千元
借：主营业务收入 10,000
    应交税费——应交增值税（销项税额） 1,300
    贷：银行存款 11,300

借：库存商品 6,000
    贷：主营业务成本 6,000
```

### 4. 销售费用
```markdown
借：销售费用
    应交税费——应交增值税（进项税额）
    贷：银行存款

例：支付广告费1万元，增值税600元
借：销售费用 10,000
    应交税费——应交增值税（进项税额） 600
    贷：银行存款 10,600
```

## 六、期间费用

### 1. 管理费用
```markdown
借：管理费用——明细科目
    贷：银行存款 / 库存现金 / 应付账款等

常见明细：
- 办公费
- 差旅费
- 业务招待费
- 咨询费
- 水电费
- 租赁费
```

### 2. 财务费用
```markdown
借：财务费用——利息支出 / 手续费 / 汇兑损益
    贷：银行存款 / 应付利息 / 长期借款

例：支付银行手续费500元
借：财务费用——手续费 500
    贷：银行存款 500
```

## 七、税费业务

### 1. 计提增值税
```markdown
应交增值税 = 销项税额 - 进项税额

如为应交：
借：应交税费——应交增值税（转出未交增值税）
    贷：应交税费——未交增值税

如为留抵：不做账务处理，留待以后抵扣
```

### 2. 计提附加税
```markdown
借：税金及附加
    贷：应交税费——应交城建税
                ——应交教育费附加
                ——应交地方教育附加

例：计提城建税、教育费附加、地方教育附加共3,000元
借：税金及附加 3,000
    贷：应交税费——应交城建税 2,100
                ——应交教育费附加 600
                ——应交地方教育附加 300
```

### 3. 计提企业所得税
```markdown
借：所得税费用
    贷：应交税费——应交企业所得税

例：计提企业所得税5万元
借：所得税费用 50,000
    贷：应交税费——应交企业所得税 50,000
```

### 4. 缴纳税款
```markdown
借：应交税费——明细税种
    贷：银行存款

例：缴纳增值税2万元
借：应交税费——未交增值税 20,000
    贷：银行存款 20,000
```

## 八、利润结转

### 1. 结转收入类科目
```markdown
借：主营业务收入
    其他业务收入
    营业外收入
    投资收益
    贷：本年利润

例：结转本月收入25万元
借：主营业务收入 200,000
    其他业务收入 30,000
    营业外收入 20,000
    贷：本年利润 250,000
```

### 2. 结转费用类科目
```markdown
借：本年利润
    贷：主营业务成本
        其他业务成本
        税金及附加
        销售费用
        管理费用
        财务费用
        营业外支出

例：结转本月费用18万元
借：本年利润 180,000
    贷：主营业务成本 120,000
        税金及附加 5,000
        销售费用 20,000
        管理费用 30,000
        财务费用 5,000
```

### 3. 计提所得税
```markdown
借：所得税费用
    贷：应交税费——应交企业所得税

例：计提所得税（25万-18万）×25%=1.75万
借：所得税费用 17,500
    贷：应交税费——应交企业所得税 17,500

借：本年利润 17,500
    贷：所得税费用 17,500
```

### 4. 结转本年利润
```markdown
借：本年利润
    贷：利润分配——未分配利润（如为盈利）

或

借：利润分配——未分配利润
    贷：本年利润（如为亏损）

例：结转全年净利润
借：本年利润 52,500
    贷：利润分配——未分配利润 52,500
```

## 九、利润分配

### 1. 提取盈余公积
```markdown
借：利润分配——提取法定盈余公积
            ——提取任意盈余公积
    贷：盈余公积——法定盈余公积
                ——任意盈余公积

例：按净利润10%提取法定盈余公积
借：利润分配——提取法定盈余公积 5,250
    贷：盈余公积——法定盈余公积 5,250
```

### 2. 分配股利
```markdown
借：利润分配——应付现金股利
    贷：应付股利

例：宣告分配现金股利2万元
借：利润分配——应付现金股利 20,000
    贷：应付股利 20,000
```

### 3. 支付股利
```markdown
借：应付股利
    贷：银行存款

例：支付现金股利2万元
借：应付股利 20,000
    贷：银行存款 20,000
```

## 十、常见差错更正

### 1. 科目错误
```markdown
更正方法：红字更正法

例：应计入管理费用的1,000元误计入销售费用

原错账：
借：销售费用 1,000
    贷：银行存款 1,000

红字冲销：
借：销售费用 -1,000
    贷：银行存款 -1,000

蓝字更正：
借：管理费用 1,000
    贷：银行存款 1,000
```

### 2. 金额多记
```markdown
更正方法：红字冲销多记金额

例：银行存款应为10,000元，误记为100,000元

红字冲销多记部分：
借：银行存款 -90,000
    贷：应收账款 -90,000
```

### 3. 金额少记
```markdown
更正方法：补充登记法

例：银行存款应为10,000元，误记为1,000元

补充登记：
借：银行存款 9,000
    贷：应收账款 9,000
```
