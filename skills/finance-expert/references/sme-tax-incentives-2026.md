# 小微企业税收优惠政策（2026年更新）

## 一、小微企业所得税优惠

### 1. 小微企业认定标准

| 项目 | 标准 |
|------|------|
| 年度应纳税所得额 | ≤ 300万元 |
| 从业人数 | ≤ 300人 |
| 资产总额 | ≤ 5000万元 |

**注意**：三个条件需同时满足，任一条件超出即不符合小微企业认定。

### 2. 企业所得税优惠税率

#### 应纳税所得额 ≤ 100万元
- **实际税负**：2.5%（原25%税率的10%）
- **政策依据**：财税〔2019〕13号、财税〔2021〕12号
- **计算公式**：应纳税所得额 × 25% × 20% ÷ 2 = 应纳税所得额 × 2.5%

#### 100万元 < 应纳税所得额 ≤ 300万元
- **实际税负**：5%（原25%税率的20%）
- **政策依据**：财税〔2019〕13号
- **计算公式**：应纳税所得额 × 25% × 20% = 应纳税所得额 × 5%

### 3. 优惠示例

**案例：某小微企业年应纳税所得额200万元**

```
分段计算：
- ≤100万元部分：100万 × 2.5% = 2.5万元
- 100万-200万元部分：100万 × 5% = 5万元
- 合计应缴税额：7.5万元
- 税负率：7.5万 ÷ 200万 = 3.75%

对比：
- 无优惠税额：200万 × 25% = 50万元
- 节税：42.5万元
```

## 二、增值税小规模纳税人优惠

### 1. 小规模纳税人标准
- **年应税销售额**：≤ 500万元
- **征收率**：3%（可享受减免优惠）

### 2. 增值税优惠政策

#### 月销售额 ≤ 10万元
- **优惠**：免征增值税
- **条件**：按月纳税
- **政策依据**：财税〔2019〕13号

#### 月销售额 > 10万元
- **优惠**：征收率由3%减按1%征收
- **适用期间**：延续执行
- **计算公式**：销售额 ÷ (1+1%) × 1%

### 3. 示例计算

**案例：小规模纳税人月销售额15万元**

```
应纳增值税 = 150,000 ÷ 1.01 × 1% = 1,485.15元

对比：
- 按3%征收率：150,000 ÷ 1.03 × 3% = 4,368.93元
- 节税：2,883.78元
```

## 三、附加税费减免

### 1. "六税两费"减免

**减免税种**：
- 资源税
- 城市维护建设税
- 房产税
- 城镇土地使用税
- 印花税（不含证券交易印花税）
- 耕地占用税
- 教育费附加
- 地方教育附加

**优惠幅度**：最高50%减免

**适用对象**：
1. 小规模纳税人
2. 小型微利企业
3. 个体工商户

### 2. 计算示例

**案例：小规模纳税人缴纳增值税1万元**

```
城市维护建设税（市区7%）：
- 正常：10,000 × 7% = 700元
- 减免50%：700 × 50% = 350元
- 实缴：350元

教育费附加（3%）：
- 正常：10,000 × 3% = 300元
- 减免50%：150元
- 实缴：150元

地方教育附加（2%）：
- 正常：10,000 × 2% = 200元
- 减免50%：100元
- 实缴：100元

合计节税：(350 + 150 + 100) = 600元
```

## 四、研发费用加计扣除

### 1. 政策内容

| 企业类型 | 加计扣除比例 |
|----------|-------------|
| 制造业企业 | 100% |
| 科技型中小企业 | 100% |
| 其他企业 | 75% |

### 2. 适用范围
- 人员人工费用
- 直接投入费用
- 折旧费用
- 无形资产摊销
- 新产品设计费
- 其他相关费用

### 3. 计算示例

**案例：企业研发费用100万元**

```
加计扣除金额：
- 制造业：100万 × 100% = 100万元
- 税前扣除：100万 + 100万 = 200万元
- 节税：100万 × 25% = 25万元

非制造业：
- 加计扣除：100万 × 75% = 75万元
- 税前扣除：175万元
- 节税：75万 × 25% = 18.75万元
```

## 五、固定资产加速折旧

### 1. 政策内容

**适用范围**：
- 小微企业
- 所有行业企业

**优惠内容**：
- 价值 ≤ 500万元的设备、器具，可一次性税前扣除
- 价值 > 500万元的，可缩短折旧年限或加速折旧

### 2. 计算示例

**案例：购买设备200万元，原折旧年限10年**

```
直线折旧：
- 年折旧：200万 ÷ 10 = 20万元/年
- 年节税：20万 × 25% = 5万元/年

加速折旧（一次性扣除）：
- 当年扣除：200万元
- 当年节税：200万 × 25% = 50万元
- 资金时间价值：提前10年获得50万节税收益
```

## 六、政策申报要点

### 1. 备案要求
- **企业所得税优惠**：年度汇算清缴时自行申报，无需事前备案
- **增值税优惠**：季度申报时自动享受
- **研发费用加计扣除**：需留存研发项目资料备查

### 2. 留存资料
- 企业所得税优惠备案表
- 小微企业认定指标计算表
- 研发费用辅助账
- 固定资产购置凭证

### 3. 风险提示
- 准确划分应纳税所得额区间
- 按时享受优惠，过期不补
- 保留完整备查资料
- 关注政策延续期限

## 七、常见问题解答

### Q1：小微企业优惠可以和其他优惠叠加吗？
**A**：企业所得税优惠不能叠加享受。企业应根据自身情况选择最优惠的政策。

### Q2：年中超过小微企业标准怎么办？
**A**：当年可继续享受优惠，次年不符合条件则恢复正常税率。

### Q3：研发费用加计扣除需要专账管理吗？
**A**：是的，需要设置研发支出辅助账，准确归集研发费用。

### Q4：小规模纳税人可以开具专票吗？
**A**：可以。但开具专票部分不能享受免税优惠，需按征收率缴税。

## 八、实务建议

### 税务筹划要点
1. **收入确认时机**：合理规划收入确认，控制应纳税所得额区间
2. **成本费用归集**：确保所有合法成本费用充分扣除
3. **研发费用管理**：建立研发项目台账，规范费用归集
4. **固定资产安排**：合理规划采购时间，充分利用加速折旧

### 合规提醒
- 严禁虚假申报
- 保留完整财务资料
- 关注政策变化和延续期限
- 必要时咨询专业税务顾问
