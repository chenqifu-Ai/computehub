# 小微企业税收优惠政策

## 增值税优惠

### 小规模纳税人优惠
| 政策 | 内容 | 有效期 |
|------|------|--------|
| 免征政策 | 月销售额≤10万元免征 | 长期 |
| 减征政策 | 征收率3%减按1% | 至2027年底 |
| 季度申报 | 季度销售额≤30万元免征 | 长期 |

### 申报要点
- 销售额为不含税销售额
- 包含所有增值税应税销售额
- 差额征税以差额后销售额为准

## 企业所得税优惠

### 小型微利企业标准
| 指标 | 标准 |
|------|------|
| 从业人数 | ≤300人 |
| 资产总额 | ≤5000万元 |
| 应纳税所得额 | ≤300万元 |

### 优惠政策
| 应纳税所得额 | 税率 | 实际税负 |
|--------------|------|----------|
| ≤100万元 | 20%×25% | 5% |
| 100-300万元 | 20%×50% | 10% |

### 计算示例
年应纳税所得额80万元：
- 应纳税额 = 80万 × 5% = 4万元
- 相比25%税率节税：16万元

## 个人所得税优惠

### 个体工商户
| 政策 | 内容 |
|------|------|
| 减半征收 | 年应纳税所得额≤200万元减半 |
| 叠加享受 | 可叠加其他优惠政策 |

### 特殊扣除
- 子女教育：每月2000元/孩
- 继续教育：每月400元或每年3600元
- 大病医疗：每年最高80000元
- 住房贷款利息：每月1000元
- 住房租金：每月800-1500元
- 赡养老人：每月2000元

| 税费 | 优惠 |
|------|------|
| 资源税 | 减征50% |
| 城市维护建设税 | 减征50% |
| 房产税 | 减征50% |
| 城镇土地使用税 | 减征50% |

| 耕地占用税 | 减征50% |

| 地方教育附加 | 减征50% |

### 社会保险费
- 阶段性缓缴政策
- 稳岗返还补贴
- 就业见习补贴

## 申报注意事项

### 享受方式
| 优惠 | 享受方式 |
|------|----------|
| 增值税 | 申报时自动享受 |
| 企业所得税 | 季度预缴、年度汇算 |
| 个人所得税 | 月度累计计算 |
| 六税两费 | 自行申报享受 |

### 留存资料
- 企业规模说明
- 从业人员名单
- 资产负债表
- 其他相关证明

### 常见误区
1. 销售额未换算不含税金额
2. 从业人数计算错误
3. 资产总额时点把握不准
4. 优惠政策重复享受
