# 初创企业会计实务指南

## 为什么会计对初创企业重要

### 会计的作用
| 作用 | 说明 | 价值 |
|------|------|------|
| 记录经营 | 准确记录收入支出 | 知道钱花哪里了 |
| 税务合规 | 按时申报纳税 | 避免罚款风险 |
| 融资需要 | 提供财务报表 | 投资人要求 |
| 经营决策 | 数据支持决策 | 提升效率 |

### 不重视会计的后果
| 后果 | 说明 |
|------|------|
| 税务罚款 | 逾期申报、漏报 |
| 融资受阻 | 无法提供财务数据 |
| 现金流失 | 不知道钱花哪了 |
| 决策盲目 | 缺乏数据支撑 |

## 初创企业会计基础

### 会计基本假设
| 假设 | 说明 |
|------|------|
| 会计主体 | 公司独立核算，与个人分开 |
| 持续经营 | 假设公司会持续经营 |
| 会计分期 | 按月、季、年分期核算 |
| 货币计量 | 以货币为主要计量单位 |

### 会计要素
| 要素 | 说明 | 示例 |
|------|------|------|
| 资产 | 拥有的资源 | 现金、设备、存货 |
| 负债 | 承担的债务 | 借款、应付账款 |
| 所有者权益 | 资产减负债 | 实收资本、留存收益 |
| 收入 | 日常活动产生的利益 | 销售收入、服务收入 |
| 费用 | 日常活动产生的耗费 | 人工、房租、水电 |
| 利润 | 收入减费用 | 营业利润、净利润 |

### 记账方法：复式记账
```
每笔业务，有借必有贷，借贷必相等

示例：收到投资款10万
借：银行存款  100,000  （资产增加）
    贷：实收资本    100,000  （权益增加）

示例：支付工资1万
借：管理费用  10,000  （费用增加）
    贷：银行存款     10,000  （资产减少）
```

## 初创企业常见账务处理

### 1. 收到投资款
```markdown
## 收到股东投资

借：银行存款    XXX
    贷：实收资本——股东A    XX%
        实收资本——股东B    XX%

注意事项：
1. 保留银行回单
2. 出资证明书
3. 验资报告（如需）
4. 工商变更登记
```

### 2. 发生费用
| 费用类型 | 科目 | 说明 |
|----------|------|------|
| 办公房租 | 管理费用-房租 | 按月摊提 |
| 员工工资 | 管理费用-工资 | 含社保公积金 |
| 办公用品 | 管理费用-办公费 | 需发票 |
| 差旅费用 | 管理费用-差旅费 | 出差报销 |
| 招待费用 | 管理费用-业务招待费 | 有扣除限额 |
| 广告宣传 | 销售费用-广告费 | 需发票 |
| 服务器 | 管理费用-技术服务费 | 按月计提 |
| 软件购买 | 无形资产/管理费用 | 视金额和年限 |

### 3. 收入确认
```markdown
## 提供服务收到款项

借：银行存款    XXX
    贷：主营业务收入    XXX
        应交税费——应交增值税    XX

## 收到预付款
借：银行存款    XXX
    贷：预收账款    XXX

## 确认收入
借：预收账款    XXX
    贷：主营业务收入    XXX
        应交税费——应交增值税    XX
```

### 4. 固定资产
```markdown
## 购入电脑等设备
借：固定资产    XXX
    应交税费——应交增值税(进项)    XX
    贷：银行存款    XXX

## 每月折旧
借：管理费用-折旧费    XX
    贷：累计折旧    XX

固定资产标准：
- 使用年限超过1年
- 单价通常超过2000元
- 有实物形态
```

### 5. 无形资产
```markdown
## 购入软件著作权
借：无形资产    XXX
    贷：银行存款    XXX

## 每月摊销
借：管理费用-摊销费    XX
    贷：累计摊销    XX

无形资产包括：
- 软件
- 专利权
- 商标权
- 著作权
```

## 增值税处理

### 小规模纳税人 vs 一般纳税人
| 项目 | 小规模纳税人 | 一般纳税人 |
|------|--------------|------------|
| 标准 | 年应税销售额<500万 | 年应税销售额≥500万或自愿登记 |
| 税率 | 3%（减按1%） | 6%/9%/13% |
| 进项抵扣 | 不可抵扣 | 可抵扣 |
| 发票 | 可开普票或专票 | 可开普票和专票 |
| 申报 | 按季申报 | 按月申报 |

### SaaS企业增值税
| 服务类型 | 税率 | 说明 |
|----------|------|------|
| 信息技术服务 | 6% | 软件服务、数据处理 |
| 软件销售 | 13% | 独立软件产品 |
| 技术咨询 | 6% | 咨询服务 |

### VideoInsight增值税计算（小规模）
```
月销售额：10,000元
增值税率：1%（减按）
增值税额：10,000 ÷ 1.01 × 1% = 99.01元

季销售额：30,000元
季度免税额：<30万免税（2023年政策）
实际缴纳：0元
```

## 企业所得税

### 税率与优惠
| 企业类型 | 税率 | 说明 |
|----------|------|------|
| 一般企业 | 25% | 标准税率 |
| 小微企业 | 5%（实际） | 年应纳税所得额<300万 |
| 高新技术企业 | 15% | 需认证 |
| 软件企业 | 两免三减半 | 需认定 |

### 小微企业所得税计算
```
年应纳税所得额：50万元

计算公式：
应纳税所得额 × 25% × 20% = 实际税额

计算：
50万 × 25% × 20% = 2.5万元

实际税率：2.5万 ÷ 50万 = 5%
```

### 应纳税所得额计算
```
利润总额
+ 纳税调整增加额（招待费超标、罚款等）
- 纳税调整减少额（加计扣除等）
= 应纳税所得额

常见纳税调整：
1. 业务招待费：60%扣除，最高不超过收入0.5%
2. 广告费：不超过收入15%扣除（2027年前30%）
3. 研发费用：加计扣除100%
4. 公益捐赠：不超过利润总额12%
```

## 发票管理

### 发票类型
| 类型 | 说明 | 适用 |
|------|------|------|
| 增值税专用发票 | 可抵扣进项 | B2B |
| 增值税普通发票 | 不可抵扣 | B2C、小规模 |
| 电子发票 | 电子形式 | 全场景 |

### 发票开具
```markdown
## 开票要素（必填）
1. 购买方名称
2. 纳税人识别号（专票必填）
3. 地址电话、开户行账号（专票建议填）
4. 货物/服务名称
5. 金额、税率、税额
6. 开票日期
7. 销售方信息
8. 发票代码、号码
```

### 发票取得
| 注意事项 | 说明 |
|----------|------|
| 真实性 | 从正规渠道获取 |
| 完整性 | 抬头、金额、日期完整 |
| 合规性 | 业务真实发生 |
| 时效性 | 及时取得，避免跨年 |

## 税务申报

### 申报时间表
| 税种 | 申报周期 | 申报时间 |
|------|----------|----------|
| 增值税 | 月/季 | 次月15日前/季末次月15日前 |
| 企业所得税 | 季/年 | 季末次月15日前/次年5月31日前 |
| 个人所得税 | 月 | 次月15日前 |
| 印花税 | 月/季 | 次月15日前 |
| 社保 | 月 | 次月15日前 |

### 零申报
```markdown
适用情况：
- 无收入、无支出
- 已开业但暂无业务

注意：
- 零申报≠不申报
- 连续零申报可能被税务局关注
- 有支出不宜零申报
```

## 财务报表编制

### 月末结账流程
```
Day 1-28：日常账务处理
Day 28-30：核对账目、调整分录
Day 30-31：结转损益、生成报表
Day 1-15：税务申报
```

### 月末调整
| 调整事项 | 说明 |
|----------|------|
| 计提折旧 | 固定资产折旧 |
| 计提摊销 | 无形资产摊销 |
| 计提工资 | 应付未付工资 |
| 计提利息 | 借款利息 |
| 结转成本 | 收入与成本配比 |
| 损益结转 | 结转到本年利润 |

## 常见问题解答

### Q1: 公司刚成立没有业务，要报税吗？
```
答：要！
即使零申报也要按时申报，否则会被罚款。
增值税、企业所得税、个税都要按时申报。
```

### Q2: 股东借款给公司怎么处理？
```
分录：
借：银行存款    XXX
    贷：其他应付款——股东    XXX

注意：
1. 保留借款协议
2. 约定利息（如有）
3. 及时归还，避免被视同分红
```

### Q3: 公司替股东垫付费用怎么办？
```
分录：
借：其他应收款——股东    XXX
    贷：银行存款    XXX

注意：
1. 及时归还
2. 年末未归还被视同分红缴税
3. 建议签订借款协议
```

### Q4: 研发费用怎么处理？
```
费用化：
借：管理费用——研发费用    XXX
    贷：银行存款    XXX

资本化（符合条件）：
借：研发支出——资本化支出    XXX
    贷：银行存款    XXX

好处：
1. 费用化可加计扣除100%
2. 资本化形成无形资产摊销
```

|------|------|
