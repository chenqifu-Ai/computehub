# 成本控制：企业成本管理实务

## 一、成本控制概述

### 1. 成本控制定义
成本控制是指企业在生产经营过程中，对影响成本的各种因素进行规划、调节和监督，发现偏差及时纠正，将成本控制在目标范围内的一系列管理活动。

### 2. 成本控制的重要性
```markdown
对企业的影响：

提升盈利能力：
- 降低成本直接增加利润
- 成本每降低1%，利润可能增加10%以上
- 成本控制是最直接的利润来源

增强竞争力：
- 成本优势转化为价格优势
- 成本优势转化为利润空间
- 成本优势抵御市场风险

生存发展基础：
- 成本失控导致企业亏损
- 成本失控影响企业生存
- 成本控制是企业管理基础

可持续发展：
- 成本控制释放资源
- 资源可用于创新投资
- 支持企业长期发展
```

### 3. 成本控制原则
```markdown
全面性原则：
- 全员参与：从高层到基层
- 全过程：从设计到销售
- 全要素：人、财、物、时间

经济性原则：
- 成本控制本身有成本
- 控制成本不能超过节约收益
- 选择性价比高的控制方法

责权利结合原则：
- 明确成本责任
- 赋予成本权力
- 建立利益激励机制

动态控制原则：
- 成本控制是动态过程
- 及时发现问题
- 及时调整措施

例外管理原则：
- 关注重大差异
- 关注异常情况
- 重点管理关键成本
```

## 二、成本分类

### 1. 按成本性态分类
```markdown
固定成本：
- 不随产量变动的成本
- 如：厂房租金、管理人员工资、折旧
- 在一定范围内保持不变

变动成本：
- 随产量正比变动的成本
- 如：原材料、计件工资、包装费
- 与产量呈线性关系

混合成本：
- 既有固定部分，又有变动部分
- 如：水电费、维修费、通讯费
- 需要分解为固定和变动部分

半固定成本（阶梯成本）：
- 在一定范围内固定
- 超过范围跳跃式增加
- 如：检验员工资、仓库租金
```

### 2. 按成本可控性分类
```markdown
可控成本：
- 责任中心可以控制的成本
- 如：生产部门可控制的原材料消耗
- 可以通过管理活动调节

不可控成本：
- 责任中心无法控制的成本
- 如：生产部门无法控制的材料价格
- 需要上级或其他部门协调

注意：
- 可控与不可控是相对的
- 同一成本对某部门可控
- 对另一部门可能不可控
- 成本归口管理要明确
```

### 3. 按成本经济用途分类
```markdown
生产成本：
- 直接材料：产品实体材料
- 直接人工：直接生产人员工资
- 制造费用：车间管理费用

期间费用：
- 销售费用：销售环节费用
- 管理费用：企业管理费用
- 财务费用：资金成本

研发成本：
- 新产品研发费用
- 工艺改进费用
- 技术创新投入
```

### 4. 按决策相关性分类
```markdown
相关成本：
- 与决策相关的成本
- 决策时需要考虑的成本
- 如：边际成本、机会成本

无关成本：
- 与决策无关的成本
- 已经发生的沉没成本
- 如：历史成本、沉没成本

决策时注意：
- 区分相关成本和无关成本
- 决策只考虑相关成本
- 忽略无关成本
```

## 三、成本控制方法

### 1. 标准成本法
```markdown
定义：
预先制定标准成本，实际成本与标准成本比较，分析差异，采取措施。

标准成本制定：

直接材料标准成本：
- 标准用量：单位产品材料消耗量
- 标准价格：材料单价
- 标准成本 = 标准用量 × 标准价格

直接人工标准成本：
- 标准工时：单位产品工时消耗
- 标准工资率：小时工资
- 标准成本 = 标准工时 × 标准工资率

制造费用标准成本：
- 标准工时：单位产品工时
- 标准分配率：制造费用/总工时
- 标准成本 = 标准工时 × 标准分配率

差异分析：

材料成本差异：
- 材料价格差异 = (实际价格 - 标准价格) × 实际用量
- 材料数量差异 = (实际用量 - 标准用量) × 标准价格

人工成本差异：
- 工资率差异 = (实际工资率 - 标准工资率) × 实际工时
- 人工效率差异 = (实际工时 - 标准工时) × 标准工资率

制造费用差异：
- 开支差异 = 实际制造费用 - 预算制造费用
- 效率差异 = (实际工时 - 标准工时) × 标准分配率
```

### 2. 目标成本法
```markdown
定义：
以市场为导向，根据市场定价倒推成本目标。

目标成本计算：
目标成本 = 目标售价 - 目标利润

实施步骤：

第一步：确定目标售价
- 市场调研确定价格
- 分析竞争对手定价
- 考虑顾客接受度

第二步：确定目标利润
- 根据企业战略目标
- 参考行业利润率
- 考虑投资回报要求

第三步：计算目标成本
- 目标成本 = 目标售价 - 目标利润
- 分解到各部门各环节

第四步：设计成本实现方案
- 价值工程分析
- 成本功能分析
- 设计成本优化

第五步：持续改进
- 追踪成本执行
- 发现成本偏差
- 持续优化改进

目标成本法特点：
- 市场导向定价
- 成本设计前移
- 全员参与控制
- 持续改进优化
```

### 3. 作业成本法
```markdown
定义：
以作业为中心，通过作业动因分配成本，提高成本核算准确性。

核心概念：

作业：
- 企业为提供产品或服务所进行的活动
- 如：采购、生产、检验、发货

成本动因：
- 引起成本发生的原因
- 资源动因：作业消耗资源的原因
- 作业动因：产品消耗作业的原因

成本池：
- 相同性质成本的集合
- 按作业归集成本

实施步骤：

第一步：识别作业
- 分析业务流程
- 识别主要作业
- 确定作业中心

第二步：归集成本
- 将成本归集到成本池
- 确定资源消耗

第三步：确定成本动因
- 分析成本发生原因
- 选择合适的成本动因

第四步：计算作业成本
- 作业成本 = 作业成本池 / 成本动因总量
- 分配到产品或服务

第五步：分析优化
- 分析高成本作业
- 优化作业流程
- 消除非增值作业

优点：
- 成本核算更准确
- 揭示成本动因
- 支持成本决策
- 发现改进空间
```

### 4. 责任成本法
```markdown
定义：
将成本责任落实到各责任中心，进行成本考核。

责任中心类型：

成本中心：
- 只对成本负责
- 不对收入负责
- 如：生产车间、职能部门

利润中心：
- 对成本和收入负责
- 如：事业部、分公司

投资中心：
- 对成本、收入、投资负责
- 如：子公司、投资部门

责任成本核算：

责任成本归集：
- 确定责任归属
- 归集可控成本
- 剔除不可控成本

责任成本考核：
- 制定责任成本目标
- 追踪责任成本执行
- 计算责任成本差异
- 分析差异原因

责任成本控制：
- 建立责任制度
- 明确责任范围
- 实施责任考核
- 建立激励机制
```

### 5. 价值链分析法
```markdown
定义：
分析价值链各环节成本，寻找成本优化空间。

价值链环节：

上游环节：
- 采购：供应商管理、采购成本
- 研发：产品设计、工艺开发
- 物流：原材料运输、仓储

中游环节：
- 生产：制造加工、质量控制
- 仓储：库存管理、保管成本

下游环节：
- 销售：渠道管理、销售费用
- 配送：物流配送、运输成本
- 服务：售后服务、客户维护

分析步骤：

第一步：绘制价值链
- 识别价值链环节
- 分析各环节关系

第二步：归集成本
- 归集各环节成本
- 计算成本占比

第三步：识别关键成本
- 找出高成本环节
- 分析成本动因

第四步：优化成本
- 消除非增值活动
- 优化流程
- 降低关键成本
```

## 四、成本控制实施

### 1. 成本控制流程
```markdown
第一步：制定成本目标

成本目标类型：
- 目标成本：根据市场倒推
- 标准成本：根据技术制定
- 预算成本：根据计划编制

制定方法：
- 历史数据分析
- 同行业对比
- 技术定额计算
- 目标利润倒推

第二步：分解成本指标

分解原则：
- 按责任中心分解
- 按成本项目分解
- 按时间期间分解

分解要求：
- 指标具体明确
- 责任归属清晰
- 可计量可考核

第三步：执行成本控制

控制要点：
- 事前控制：预测、计划
- 事中控制：监督、纠偏
- 事后控制：分析、改进

控制手段：
- 制度控制
- 定额控制
- 预算控制
- 审批控制

第四步：成本核算

核算要求：
- 及时准确
- 分类清晰
- 便于分析

核算方法：
- 品种法
- 分批法
- 分步法
- 作业成本法

第五步：成本分析

分析方法：
- 比较分析：与目标、历史、同行对比
- 因素分析：分析各因素影响
- 结构分析：分析成本结构
- 趋势分析：分析成本变化趋势

第六步：成本考核

考核内容：
- 成本目标完成情况
- 成本控制措施执行情况
- 成本改进效果

考核方式：
- 定量考核：数据对比
- 定性考核：管理评价
- 综合考核：定量定性结合
```

### 2. 成本预测
```markdown
预测方法：

定性预测：
- 专家意见法
- 德尔菲法
- 主观判断法

定量预测：
- 高低点法
- 回归分析法
- 时间序列法
- 学习曲线法

预测步骤：
1. 收集历史数据
2. 分析成本变动规律
3. 选择预测方法
4. 建立预测模型
5. 进行预测计算
6. 分析预测误差
7. 调整预测结果
```

### 3. 成本决策
```markdown
常见成本决策：

自制与外购决策：
- 比较自制成本与外购价格
- 考虑产能利用
- 考虑质量风险
- 考虑战略影响

特殊订单决策：
- 分析订单收入
- 分析订单成本
- 判断是否接受订单

停产决策：
- 分析产品贡献毛益
- 分析固定成本分摊
- 判断是否停产

加工决策：
- 分析进一步加工增加的收入
- 分析进一步加工增加的成本
- 比较收入与成本增量
```

### 4. 成本预算
```markdown
预算编制方法：

增量预算法：
- 在历史基础上调整
- 简单易行
- 可能保留不合理支出

零基预算法：
- 从零开始编制
- 所有支出需证明合理性
- 更科学但工作量大

弹性预算法：
- 编制多套预算
- 适应不同业务量
- 便于差异分析

滚动预算法：
- 定期更新预算
- 保持预算连续性
- 及时反映变化

成本预算内容：

材料成本预算：
- 根据生产计划
- 计算材料消耗
- 预计材料价格

人工成本预算：
- 根据生产计划
- 计算工时需求
- 预计工资水平

制造费用预算：
- 分析费用项目
- 按成本性态分类
- 编制费用预算

期间费用预算：
- 销售费用预算
- 管理费用预算
- 财务费用预算
```

## 六、成本分析技术

### 1. 成本性态分析
```markdown
分析目的：
- 分离固定成本和变动成本
- 为成本预测和决策提供依据

分析方法：

高低点法：
- 找出最高点和最低点业务量
- 计算变动成本率
- 分离固定成本

回归分析法：
- 建立成本与业务量的回归方程
- 计算相关系数
- 预测成本

账户分析法：
- 分析各账户成本性态
- 分类归集固定和变动成本

工程法：
- 分析成本发生的技术原因
- 确定成本与业务量的关系
```

### 2. 本量利分析
```markdown
基本概念：

贡献毛益：
- 贡献毛益 = 销售收入 - 变动成本
- 单位贡献毛益 = 单价 - 单位变动成本
- 贡献毛益率 = 贡献毛益 / 销售收入

盈亏平衡点：
- 盈亏平衡销售量 = 固定成本 / 单位贡献毛益
- 盈亏平衡销售额 = 固定成本 / 贡献毛益率

安全边际：
- 安全边际 = 实际销售量 - 盈亏平衡销售量
- 安全边际率 = 安全边际 / 实际销售量

目标利润分析：
- 目标销售量 = (固定成本 + 目标利润) / 单位贡献毛益
- 目标销售额 = (固定成本 + 目标利润) / 贡献毛益率

敏感性分析：
- 分析各因素变动对利润的影响
- 识别敏感因素
- 进行风险防范
```

### 3. 成本差异分析
```markdown
差异类型：

有利差异：
- 实际成本 < 标准成本
- 成本节约

不利差异：
- 实际成本 > 标准成本
- 成本超支

分析步骤：

第一步：计算差异
- 总差异 = 实际成本 - 标准成本

第二步：分解差异
- 价格差异 = (实际价格 - 标准价格) × 实际数量
- 数量差异 = (实际数量 - 标准数量) × 标准价格

第三步：分析原因
- 价格差异原因：市场价格变化、采购策略
- 数量差异原因：生产效率、材料利用率

第四步：采取措施
- 有利差异：总结经验，推广
- 不利差异：分析原因，改进
```
