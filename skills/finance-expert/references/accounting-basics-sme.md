# 小微企业会计实务要点

## 会计基础知识

### 会计等式
**基本等式**：资产 = 负债 + 所有者权益

### 会计要素
| 要素 | 定义 | 示例 |
|------|------|------|
| 资产 | 企业拥有或控制的资源 | 现金、存货、设备 |
| 负债 | 企业承担的债务 | 应付账款、借款 |
| 所有者权益 | 资产减负债后的余额 | 实收资本、留存收益 |
| 收入 | 日常活动产生的利益 | 销售收入 |
| 费用 | 日常活动产生的消耗 | 工资、租金 |
| 利润 | 收入减费用 | 净利润 |

### 借贷记账法
- **有借必有贷，借贷必相等**
- 资产、费用：借增贷减
- 负债、权益、收入：贷增借减

| 科目 | 说明 |

## 常见账务处理

### 销售业务
```
借：银行存款/应收账款
    贷：主营业务收入
        应交税费-应交增值税（销项税）
```

### 采购业务
```
借：原材料/库存商品
    应交税费-应交增值税（进项税）
    贷：银行存款/应付账款
```

### 工资发放
```
借：管理费用/销售费用
    贷：应付职工薪酬
借：应付职工薪酬
    贷：银行存款
```

## 发票管理

### 发票类型
| 类型 | 适用场景 |
|------|----------|
| 增值税专用发票 | 一般纳税人 |

### 开票注意
- 真实业务、真实开票
- 发票信息准确
- 及时认证抵扣
- 妥善保管
