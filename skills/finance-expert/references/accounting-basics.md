# 会计实务：会计核算基础

## 一、会计基本概念

### 1. 会计定义
会计是以货币为主要计量单位，运用专门的方法，对企业、机关、事业单位的经济活动进行连续、系统、全面、综合的核算和监督，并提供经济信息的一种管理活动。

### 2. 会计职能
```markdown
核算职能：
- 确认：确定经济业务是否应确认
- 计量：确定经济业务的金额
- 记录：将经济业务记录在账簿中
- 报告：编制财务报表

监督职能：
- 事前监督：经济业务发生前
- 事中监督：经济业务发生时
- 事后监督：经济业务发生后

关系：
- 核算是监督的基础
- 监督是核算的保障
- 相互依存、相互促进
```

### 3. 会计假设
```markdown
会计主体假设：
- 会计核算以企业为主体
- 区分企业与企业所有者
- 区分企业与其他企业

持续经营假设：
- 假设企业将持续经营
- 不会破产清算
- 资产按历史成本计量

会计分期假设：
- 将持续经营期间划分为若干期间
- 定期结算账目、编制报表
- 分为年度、半年度、季度、月度

货币计量假设：
- 以货币为主要计量单位
- 假设币值相对稳定
- 可以补充非货币信息
```

### 4. 会计信息质量要求
```markdown
可靠性：
- 以实际发生的交易为依据
- 如实反映经济活动
- 内容真实、数字准确

相关性：
- 与决策相关
- 具有预测价值
- 具有反馈价值

可理解性：
- 清晰明了
- 便于理解和使用

可比性：
- 纵向可比：不同时期可比
- 横向可比：不同企业可比
- 采用一致的会计政策

实质重于形式：
- 按经济实质确认
- 不局限于法律形式

重要性：
- 区分重要与不重要
- 重要项目单独反映
- 不重要项目合并反映

谨慎性：
- 不高估资产和收益
- 不低估负债和费用
- 充分估计风险

及时性：
- 及时收集会计信息
- 及时处理会计信息
- 及时报告会计信息
```

## 二、会计要素

### 1. 资产
```markdown
定义：
- 企业过去的交易或事项形成
- 企业拥有或控制
- 预期带来经济利益

确认条件：
- 与该资源有关的经济利益很可能流入企业
- 该资源的成本或价值能够可靠计量

分类：

流动资产：
- 货币资金：库存现金、银行存款、其他货币资金
- 交易性金融资产：短期投资
- 应收款项：应收账款、其他应收款
- 存货：原材料、在产品、产成品

非流动资产：
- 长期股权投资：对其他企业的股权投资
- 固定资产：房屋、机器设备、运输工具
- 无形资产：专利权、商标权、土地使用权
- 长期待摊费用：摊销期超过一年的费用
```

### 2. 负债
```markdown
定义：
- 企业过去的交易或事项形成
- 预期会导致经济利益流出企业
- 现时义务

确认条件：
- 与该义务有关的经济利益很可能流出企业
- 未来流出的经济利益能够可靠计量

分类：

流动负债：
- 短期借款：一年内到期的借款
- 应付账款：购买商品或接受劳务应付未付款项
- 预收账款：预收的款项
- 应付职工薪酬：应付未付的工资、福利费等
- 应交税费：应缴纳的各项税费
- 其他应付款：其他应付暂收款项

非流动负债：
- 长期借款：一年以上到期的借款
- 应付债券：发行的债券
- 长期应付款：融资租入固定资产应付款
```

### 3. 所有者权益
```markdown
定义：
- 企业资产扣除负债后由所有者享有的剩余权益
- 所有者对企业的投资及其增值

来源：
- 所有者投入的资本
- 直接计入所有者权益的利得和损失
- 留存收益

构成：

实收资本（股本）：
- 投资者投入的注册资本
- 股份有限公司称为股本

资本公积：
- 资本溢价：投资者投入超过注册资本部分
- 其他资本公积：直接计入所有者权益的利得

盈余公积：
- 法定盈余公积：按净利润10%提取
- 任意盈余公积：根据股东会决议提取

未分配利润：
- 历年累计未分配利润
- 留存于企业的利润
```

### 4. 收入
```markdown
定义：
- 日常活动中形成
- 会导致所有者权益增加
- 与所有者投入资本无关

确认条件：
- 合同存在
- 履约义务识别
- 交易价格确定
- 交易价格分摊
- 履行履约义务时确认收入

分类：

主营业务收入：
- 销售商品收入
- 提供劳务收入
- 让渡资产使用权收入

其他业务收入：
- 材料销售收入
- 包装物出租收入
- 固定资产出租收入
```

### 5. 费用
```markdown
定义：
- 日常活动中发生
- 会导致所有者权益减少
- 与向所有者分配利润无关

确认条件：
- 与费用相关的经济利益很可能流出企业
- 未来流出的经济利益能够可靠计量

分类：

营业成本：
- 主营业务成本：销售商品、提供劳务的成本
- 其他业务成本：其他业务的成本

期间费用：
- 销售费用：销售商品、提供服务的费用
- 管理费用：组织和管理生产经营的费用
- 财务费用：筹集资金的费用
```

### 6. 利润
```markdown
定义：
- 一定会计期间的经营成果
- 收入减去费用后的净额
- 直接计入当期利润的利得和损失

计算公式：

营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用 - 资产减值损失 + 公允价值变动收益 + 投资收益 + 其他收益

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用

利润分配：
- 弥补以前年度亏损
- 提取法定盈余公积
- 提取任意盈余公积
- 向投资者分配利润
```

## 三、会计等式

### 1. 基本会计等式
```markdown
资产 = 负债 + 所有者权益

含义：
- 资产是企业拥有或控制的资源
- 负债是债权人权益
- 所有者权益是所有者权益
- 资产来源于负债和所有者权益

应用：
- 编制资产负债表的基础
- 复式记账法的理论基础
- 试算平衡的依据
```

### 2. 扩展会计等式
```markdown
资产 = 负债 + 所有者权益 + (收入 - 费用)

或

资产 + 费用 = 负债 + 所有者权益 + 收入

含义：
- 收入增加导致所有者权益增加
- 费用增加导致所有者权益减少
- 利润增加所有者权益

应用：
- 收入费用要素纳入会计等式
- 动态反映企业财务状况
```

### 3. 经济业务对会计等式影响
```markdown
四种类型：

资产与权益同时增加：
- 资产增加，负债增加
- 资产增加，所有者权益增加
例：接受投资者投入资本

资产与权益同时减少：
- 资产减少，负债减少
- 资产减少，所有者权益减少
例：用银行存款归还短期借款

资产内部增减变动：
- 一项资产增加，另一项资产减少
例：用银行存款购买固定资产

权益内部增减变动：
- 一项负债增加，另一项负债减少
- 一项所有者权益增加，另一项所有者权益减少
- 负债增加，所有者权益减少
- 所有者权益增加，负债减少
例：盈余公积转增资本
```

## 四、会计科目与账户

### 1. 会计科目
```markdown
定义：
- 对会计要素进行分类的项目
- 设置账户的依据

分类：

按经济内容分类：
- 资产类科目：库存现金、银行存款、应收账款等
- 负债类科目：短期借款、应付账款等
- 所有者权益类科目：实收资本、资本公积等
- 成本类科目：生产成本、制造费用等
- 损益类科目：主营业务收入、销售费用等

按提供信息详细程度分类：
- 总分类科目：一级科目
- 明细分类科目：二级科目、三级科目

设置原则：
- 全面反映经济业务
- 满足管理需要
- 符合会计准则
- 相对稳定
```

### 2. 账户
```markdown
定义：
- 根据会计科目设置
- 具有一定格式和结构
- 用于分类记录经济业务

基本结构：
- 借方：账户的左边
- 贷方：账户的右边
- 余额：期初余额、期末余额

账户格式：
┌─────────────────────────────────────┐
│ 日期 │ 摘要 │ 借方 │ 贷方 │ 余额 │
├─────────────────────────────────────┤
│      │      │      │      │      │
└─────────────────────────────────────┘

账户分类：
- 资产类账户：借增贷减，余额在借方
- 负债类账户：贷增借减，余额在贷方
- 所有者权益类账户：贷增借减，余额在贷方
- 成本类账户：借增贷减，余额在借方
- 损益类账户：期末结转后无余额
```

## 五、借贷记账法

### 1. 基本原理
```markdown
定义：
- 以"借"和"贷"为记账符号
- 以"有借必有贷，借贷必相等"为记账规则
- 反映各会计要素增减变动

记账符号：
- 借：账户左边
- 贷：账户右边

记账规则：
- 有借必有贷：每笔业务同时登记借贷两方
- 借贷必相等：借贷金额相等
```

### 2. 账户结构
```markdown
资产类账户：
借方登记增加，贷方登记减少，余额在借方
期末余额 = 期初余额 + 借方发生额 - 贷方发生额

负债类账户：
借方登记减少，贷方登记增加，余额在贷方
期末余额 = 期初余额 + 贷方发生额 - 借方发生额

所有者权益类账户：
借方登记减少，贷方登记增加，余额在贷方
期末余额 = 期初余额 + 贷方发生额 - 借方发生额

成本类账户：
借方登记增加，贷方登记减少，余额在借方

收入类账户：
借方登记减少（结转），贷方登记增加，期末无余额

费用类账户：
借方登记增加，贷方登记减少（结转），期末无余额
```

### 3. 会计分录
```markdown
定义：
- 标明经济业务应借应贷的账户名称及金额
- 是登记账簿的依据

要素：
- 账户名称（会计科目）
- 记账方向（借方或贷方）
- 记账金额

类型：

简单分录：
- 一借一贷
例：借：银行存款 10000
        贷：实收资本 10000

复合分录：
- 一借多贷
例：借：原材料 10000
        应交税费-应交增值税(进项税额) 1300
        贷：银行存款 11300

- 多借一贷
例：借：管理费用 8000
        贷：银行存款 5000
            库存现金 3000

- 多借多贷（一般不使用）
```

### 4. 试算平衡
```markdown
发生额平衡：
- 所有账户本期借方发生额合计 = 所有账户本期贷方发生额合计

余额平衡：
- 所有账户期末借方余额合计 = 所有账户期末贷方余额合计

试算平衡表：
┌───────────────────────────────────────────────────┐
│  账户名称  │ 期初余额 │ 本期发生额 │ 期末余额 │
│            │ 借  贷  │ 借    贷  │ 借   贷 │
├───────────────────────────────────────────────────┤
│            │      │        │       │      │
├───────────────────────────────────────────────────┤
│  合  计    │  相等 │  相等   │  相等 │
└───────────────────────────────────────────────────┘

注意：
- 试算平衡不一定说明记账正确
- 漏记、重记、方向颠倒等错误可能平衡
- 需要结合其他方法检查
```

## 六、会计凭证

### 1. 原始凭证
```markdown
定义：
- 经济业务发生或完成时取得或填制
- 记录经济业务内容和完成情况
- 具有法律效力

分类：

按来源分：
- 外来原始凭证：购货发票、银行收款通知等
- 自制原始凭证：领料单、工资结算单等

按填制方法分：
- 一次凭证：一次填制完成
- 累计凭证：多次累计填制
- 汇总凭证：汇总多个凭证

基本内容：
- 凭证名称
- 填制日期
- 填制单位名称
- 经济业务内容
- 数量、单价、金额
- 经办人员签章

填制要求：
- 内容真实
- 项目齐全
- 计算准确
- 填制及时
- 书写规范

审核内容：
- 真实性：业务是否真实发生
- 合法性：是否符合法规制度
- 合理性：是否符合计划预算
- 完整性：项目是否齐全
- 正确性：金额计算是否正确
```

### 2. 记账凭证
```markdown
定义：
- 根据审核无误的原始凭证填制
- 确定会计分录
- 登记账簿的直接依据

分类：

按内容分：
- 收款凭证：现金和银行存款收入
- 付款凭证：现金和银行存款支出
- 转账凭证：不涉及现金和银行存款

按填制方法分：
- 复式凭证：完整借贷关系
- 单式凭证：只填借方或贷方

基本内容：
- 凭证名称
- 填制日期
- 凭证编号
- 摘要
- 会计科目
- 金额
- 所附原始凭证张数
- 填制、审核、记账人员签章

填制要求：
- 依据审核无误的原始凭证
- 内容完整
- 编号连续
- 书写规范

审核内容：
- 是否附有原始凭证
- 内容是否一致
- 科目使用是否正确
- 金额是否相等
- 项目是否齐全
```

## 七、会计账簿

### 1. 账簿种类
```markdown
按用途分：
- 序时账簿：日记账，按时间顺序登记
- 分类账簿：总分类账、明细分类账
- 备查账簿：补充登记账簿

按外表形式分：
- 订本式账簿：账页固定，防止散失
- 活页式账簿：账页可增减
- 卡片式账簿：卡片形式

按账页格式分：
- 三栏式：借方、贷方、余额
- 多栏式：多栏明细分类账
- 数量金额式：数量、单价、金额
```

### 2. 账簿登记规则
```markdown
基本要求：
- 以审核无误的会计凭证为依据
- 登记内容完整、准确
- 使用蓝黑墨水或碳素墨水书写
- 不得随意涂改

错账更正方法：
- 划线更正法：结账前发现文字或数字错误
- 红字更正法：记账后发现科目或金额错误
- 补充登记法：记账后发现金额少记

对账：
- 账证核对：账簿与凭证核对
- 账账核对：账簿之间核对
- 账实核对：账簿与实物核对

结账：
- 月结：月末结出本期发生额和期末余额
- 年结：年末结出本年累计发生额和年末余额
```

### 3. 总分类账与明细分类账
```markdown
总分类账：
- 根据总分类科目设置
- 提供总括核算资料
- 采用三栏式账页

明细分类账：
- 根据明细科目设置
- 提供详细核算资料
- 可采用三栏式、多栏式、数量金额式

平行登记：
- 同期登记：同一会计期间
- 同向登记：借贷方向相同
- 等额登记：总账金额等于明细账金额之和
- 同据登记：依据相同
```

## 八、财务报表

### 1. 资产负债表
```markdown
定义：
- 反映企业特定日期财务状况
- 静态报表

结构：
资产 = 负债 + 所有者权益

编制方法：
- 根据总账科目余额填列
- 根据明细账科目余额填列
- 根据总账和明细账余额分析填列

主要内容：

资产：
- 流动资产：货币资金、应收账款、存货等
- 非流动资产：长期股权投资、固定资产、无形资产等

负债：
- 流动负债：短期借款、应付账款等
- 非流动负债：长期借款、应付债券等

所有者权益：
- 实收资本、资本公积、盈余公积、未分配利润
```

### 2. 利润表
```markdown
定义：
- 反映企业一定会计期间经营成果
- 动态报表

编制方法：
- 根据损益类科目发生额填列

主要内容：

- 销售费用、管理费用、财务费用

- 投资收益
- 营业利润
- 营业外收入、营业外支出

- 所得税费用
- 净利润
```

```markdown

筹资活动现金流量：
```
