# 会计实务：企业账务处理全流程

## 一、会计核算基础

### 1. 会计核算原则
```markdown
会计核算基本假设：

会计主体假设：
- 会计核算的对象是企业
- 区分企业与企业所有者
- 区分企业与其他企业

持续经营假设：
- 企业将持续经营下去
- 不会清算或大幅缩减
- 资产按历史成本计价

会计分期假设：
- 将企业经营活动分期
- 分为年度、季度、月度
- 定期编制财务报表

货币计量假设：
- 以货币为主要计量单位
- 假设币值稳定
- 外币业务需折算
```

### 2. 会计信息质量要求
```markdown
可靠性：
- 内容真实
- 数据准确
- 资料完整

相关性：
- 与决策相关
- 提供预测价值
- 提供反馈价值

可理解性：
- 清晰明了
- 便于理解
- 格式规范

可比性：
- 不同时期可比
- 不同企业可比
- 会计政策一致

实质重于形式：
- 按经济实质判断
- 不局限于法律形式
- 如：融资租入资产

重要性：
- 重要事项详细披露
- 次要事项简化处理
- 重要性判断需合理

谨慎性：
- 不高估资产收益
- 不低估负债费用
- 合理计提减值

及时性：
- 及时确认
- 及时计量
- 及时报告
```

### 3. 会计要素
```markdown
资产：
- 企业过去的交易形成
- 由企业拥有或控制
- 预期带来经济利益

负债：
- 企业过去的交易形成
- 现时义务
- 预期导致经济利益流出

所有者权益：
- 资产减负债后的余额
- 所有者投入
- 留存收益

收入：
- 日常活动形成
- 导致所有者权益增加
- 与所有者投入无关

费用：
- 日常活动形成
- 导致所有者权益减少
- 与利润分配无关

利润：
- 一定会计期间的经营成果
- 收入减费用后的净额
- 直接计入当期利润的利得和损失
```

## 二、账务处理流程

### 1. 原始凭证
```markdown
原始凭证分类：

按来源分：
- 自制原始凭证：企业内部制作
- 外来原始凭证：外部单位取得

按填制方法分：
- 一次凭证：一次填制完成
- 累计凭证：多次累计填制
- 汇总凭证：汇总填制

原始凭证要素：
- 凭证名称
- 填制日期
- 填制单位名称
- 接受单位名称
- 经济业务内容
- 金额
- 填制单位签章
- 经办人员签章

原始凭证审核：
- 真实性审核
- 合法性审核
- 合规性审核
- 完整性审核
- 正确性审核
```

### 2. 记账凭证
```markdown
记账凭证分类：

收款凭证：
- 现金收款凭证
- 银行存款收款凭证

付款凭证：
- 现金付款凭证
- 银行存款付款凭证

转账凭证：
- 不涉及现金和银行存款
- 内部转账

记账凭证要素：
- 凭证名称
- 填制日期
- 凭证编号
- 经济业务摘要
- 会计科目
- 金额
- 所附原始凭证张数
- 填制人员签章

记账凭证填制要求：
- 内容完整
- 摘要简明
- 科目正确
- 金额准确
- 编号连续
- 附件齐全
```

### 3. 会计账簿
```markdown
账簿分类：

按用途分：
- 序时账簿：日记账
- 分类账簿：总账、明细账
- 备查账簿：辅助账

按外表形式分：
- 订本式账簿
- 活页式账簿
- 卡片式账簿

按账页格式分：
- 三栏式账簿
- 多栏式账簿
- 数量金额式账簿

账簿登记规则：
- 准确完整
- 符号规范
- 连续登记
- 结出余额
- 过次承前
- 定期对账
```

### 4. 账务处理程序
```markdown
记账凭证账务处理程序：
- 根据原始凭证编制记账凭证
- 根据记账凭证登记日记账
- 根据记账凭证登记明细账
- 根据记账凭证登记总账
- 编制财务报表

汇总记账凭证账务处理程序：
- 根据原始凭证编制记账凭证
- 根据记账凭证编制汇总记账凭证
- 根据汇总记账凭证登记总账
- 编制财务报表

科目汇总表账务处理程序：
- 根据原始凭证编制记账凭证
- 根据记账凭证编制科目汇总表
- 根据科目汇总表登记总账
- 编制财务报表
```

## 三、日常业务核算

### 1. 现金核算
```markdown
现金管理：

现金使用范围：
- 职工工资、津贴
- 个人劳务报酬
- 根据国家规定颁发给个人的奖金
- 各种劳保、福利费用
- 向个人收购农副产品和其他物资的价款
- 出差人员必须随身携带的差旅费
- 结算起点以下的零星支出

现金限额：
- 库存现金限额由开户银行核定
- 一般不超过3-5天日常零星开支
- 超过限额及时送存银行

现金核算：
- 设置"库存现金"科目
- 现金日记账逐日逐笔登记
- 每日终了进行现金盘点
- 编制现金日报表
```

### 2. 银行存款核算
```markdown
银行存款账户：

基本存款账户：
- 日常转账结算和现金收付
- 工资、奖金等现金支取
- 只能开立一个

一般存款账户：
- 借款转存
- 借款归还
- 不得支取现金

临时存款账户：
- 临时经营活动
- 有效期最长2年

专用存款账户：
- 专项资金管理
- 专款专用

银行存款核算：
- 设置"银行存款"科目
- 银行存款日记账逐日逐笔登记
- 月末与银行对账单核对
- 编制银行存款余额调节表
```

### 3. 应收账款核算
```markdown
应收账款确认：
- 销售商品、提供劳务应收的款项
- 包括货款、增值税、代垫费用
- 按实际发生额入账

应收账款核算：
- 设置"应收账款"科目
- 按客户设置明细账
- 定期与客户对账

坏账处理：

坏账确认条件：
- 债务人破产或死亡
- 债务人逾期3年以上
- 经批准核销

坏账核算方法：
- 直接转销法：实际发生时核销
- 备抵法：按期计提坏账准备

坏账准备计提：
- 余额百分比法
- 账龄分析法
- 个别认定法
```

### 4. 存货核算
```markdown
存货分类：
- 原材料
- 在产品
- 库存商品
- 周转材料

存货取得计价：
- 外购：买价+运费+其他费用
- 自制：直接材料+直接人工+制造费用
- 委托加工：材料成本+加工费+运费

存货发出计价：

先进先出法：
- 先购入的先发出
- 期末存货接近市价
- 物价上涨时利润偏高

月末一次加权平均法：
- 加权平均单价 = 总成本/总数量
- 简化核算
- 不利于日常控制

移动加权平均法：
- 每次进货后计算平均单价
- 及时反映存货成本
- 工作量大

个别计价法：
- 按个别存货的实际成本计价
- 适用于贵重物品
- 核算准确

存货盘点：
- 定期盘点
- 永续盘存
- 盘盈盘亏处理
```

### 5. 固定资产核算
```markdown
固定资产确认：
- 为生产商品、提供劳务、出租或经营管理持有
- 使用寿命超过一个会计年度
- 有形资产

固定资产分类：
- 房屋建筑物
- 机器设备
- 运输设备
- 电子设备
- 其他设备

固定资产取得：
- 外购：购买价款+相关税费+运输费+安装费
- 自建：建造过程中发生的全部支出
- 融资租入：租赁开始日公允价值与最低租赁付款额较低者

固定资产折旧：

折旧方法：
- 年限平均法：原值×(1-残值率)/使用年限
- 工作量法：原值×(1-残值率)/预计总工作量×实际工作量
- 双倍余额递减法：期初净值×2/使用年限
- 年数总和法：原值×(1-残值率)×剩余年限/年数总和

折旧范围：
- 所有固定资产（除已提足折旧、单独计价入账的土地）
- 当月增加当月不提折旧
- 当月减少当月照提折旧

固定资产处置：
- 固定资产转入清理
- 发生清理费用
- 残值收入
- 结转清理损益
```

### 6. 应付账款核算
```markdown
应付账款确认：
- 购买商品、接受劳务应付的款项
- 包括货款、增值税
- 按实际发生额入账

应付账款核算：
- 设置"应付账款"科目
- 按供应商设置明细账
- 定期与供应商对账

应付账款管理：
- 按合同约定付款
- 合理利用信用期
- 控制应付款规模
- 避免逾期付款
```

### 7. 应付职工薪酬核算
```markdown
职工薪酬内容：
- 职工工资、奖金、津贴
- 职工福利费
- 医疗保险费、养老保险费、失业保险费、工伤保险费、生育保险费
- 住房公积金
- 工会经费、职工教育经费
- 非货币性福利
- 辞退福利

职工薪酬核算：
- 设置"应付职工薪酬"科目
- 按薪酬项目设置明细账
- 计提：借：成本/费用，贷：应付职工薪酬
- 发放：借：应付职工薪酬，贷：银行存款/现金
```

### 8. 应交税费核算
```markdown
增值税核算：

进项税额：
- 购进货物、接受劳务支付的增值税
- 取得增值税专用发票
- 准予抵扣

销项税额：
- 销售货物、提供劳务收取的增值税
- 开具增值税发票
- 计入应交税费

应交增值税 = 销项税额 - 进项税额

企业所得税核算：
- 按季预缴，年终汇算清缴
- 计提：借：所得税费用，贷：应交税费
- 缴纳：借：应交税费，贷：银行存款
```

## 四、期末账务处理

### 1. 成本结转
```markdown
制造费用分配：
- 按生产工时分配
- 按机器工时分配
- 按直接人工分配
- 按直接材料分配

生产成本结转：
- 完工产品成本结转
- 在产品成本保留
- 借：库存商品，贷：生产成本

销售成本结转：
- 月末计算销售成本
- 借：主营业务成本，贷：库存商品
- 采用先进先出法、加权平均法等
```

### 2. 损益结转
```markdown
收入结转：
- 借：主营业务收入
- 借：其他业务收入
- 借：营业外收入
- 贷：本年利润

费用结转：
- 借：本年利润
- 贷：主营业务成本
- 贷：其他业务成本
- 贷：税金及附加
- 贷：销售费用
- 贷：管理费用
- 贷：财务费用
- 贷：营业外支出
```

### 3. 利润核算
```markdown
利润计算：

营业利润 = 营业收入 - 营业成本 - 税金及附加
         - 销售费用 - 管理费用 - 财务费用
         + 投资收益 + 公允价值变动收益

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用

利润分配：
- 提取法定盈余公积（10%）
- 提取任意盈余公积
- 向投资者分配利润
```

### 4. 对账与结账
```markdown
对账：

账证核对：
- 账簿记录与记账凭证核对
- 确保账证相符

账账核对：
- 总账与日记账核对
- 总账与明细账核对
- 确保账账相符

账实核对：
- 现金日记账与库存现金核对
- 银行存款日记账与银行对账单核对
- 存货明细账与存货盘点表核对
- 固定资产明细账与固定资产盘点表核对
- 确保账实相符

结账：

月结：
- 计算本月发生额
- 计算月末余额
- 划通栏单红线

年结：
- 计算本年发生额
- 计算年末余额
- 划通栏双红线
- 余额转入下年
```

## 五、财务报表编制

### 1. 资产负债表
```markdown
资产负债表结构：
- 资产 = 负债 + 所有者权益
- 报告式或账户式
- 年初余额、期末余额

资产项目：
- 流动资产：货币资金、应收账款、存货等
- 非流动资产：固定资产、无形资产等

负债项目：
- 流动负债：短期借款、应付账款等
- 非流动负债：长期借款、应付债券等

所有者权益项目：
- 实收资本
- 资本公积
- 盈余公积
- 未分配利润

编制方法：
- 根据总账科目余额填列
- 根据明细账科目余额填列
- 根据总账和明细账余额分析填列
```

### 2. 利润表
```markdown
利润表结构：
- 多步式利润表
- 本期金额、上期金额

计算步骤：
1. 营业收入 - 营业成本 = 毛利
2. 毛利 - 税金及附加 - 销售费用 - 管理费用 - 财务费用 = 营业利润
3. 营业利润 + 营业外收入 - 营业外支出 = 利润总额
4. 利润总额 - 所得税费用 = 净利润

编制方法：
- 根据损益类科目发生额填列
- 营业收入 = 主营业务收入 + 其他业务收入
- 营业成本 = 主营业务成本 + 其他业务成本
```

### 3. 现金流量表
```markdown
现金流量分类：

经营活动现金流量：
- 销售商品、提供劳务收到的现金
- 购买商品、接受劳务支付的现金
- 支付给职工的现金
- 支付的各项税费

投资活动现金流量：
- 收回投资收到的现金
- 取得投资收益收到的现金
- 购建固定资产支付的现金
- 投资支付的现金

筹资活动现金流量：
- 吸收投资收到的现金
- 取得借款收到的现金
- 偿还债务支付的现金
- 分配股利支付的现金

编制方法：
- 直接法：直接列示现金收入和支出
- 间接法：从净利润调整为经营现金流量
```

### 4. 所有者权益变动表
```markdown
所有者权益变动内容：
- 实收资本变动
- 资本公积变动
- 盈余公积变动
- 未分配利润变动

变动项目：
- 综合收益总额
- 所有者投入和减少资本
- 利润分配
- 所有者权益内部结转

编制方法：
- 根据所有者权益科目明细填列
- 反映所有者权益增减变动
```
