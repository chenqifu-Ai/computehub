# 小微企业税收优惠政策

## 企业所得税优惠

### 小型微利企业
**认定标准**（同时满足）：
- 年度应纳税所得额 ≤ 300万元
- 从业人数 ≤ 300人
- 资产总额 ≤ 5000万元

**优惠税率**：
| 应纳税所得额 | 税率 | 计算 |
|---------------|------|------|
| ≤ 100万元 | 2.5% | 实际税负 |
| 100-300万元 | 5% | 实际税负 |

### 计算示例
```
应纳税所得额：150万元
分段计算：
- 100万 × 2.5% = 2.5万
- 50万 × 5% = 2.5万
合计应纳税：5万元
实际税负：5/150 = 3.33%
```

## 增值税优惠

### 小规模纳税人
| 项目 | 政策 |
|------|------|
| 起征点 | 月销售额 ≤ 10万元免征 |
| 征收率 | 3%（可减按1%） |
| 适用范围 | 年销售额 ≤ 500万元 |

### 进项抵扣
| 类型 | 说明 |
|------|------|
| 可抵扣 | 取得专票、海关缴款书 |
| 不可抵扣 | 用于简易计税、免税项目 |

## 优惠政策申报流程

### 准备材料
1. 企业基本情况表
2. 财务报表
3. 从业人数证明
4. 资产总额证明

### 申报步骤
1. 判断是否符合条件
2. 准备相关证明材料
3. 填写纳税申报表
4. 自行判断享受优惠
5. 保存备查资料

## 注意事项

### 常见风险
| 风险点 | 说明 |
|--------|------|
| 指标超限 | 年末收入/人数超标 |
| 申报错误 | 优惠计算错误 |
| 资料缺失 | 备查资料不完整 |
| 政策变化 | 政策到期或调整 |

### 管理建议
- 定期测算应纳税所得额
- 关注人数和资产变化
- 及时享受优惠政策
- 保存相关证明材料
