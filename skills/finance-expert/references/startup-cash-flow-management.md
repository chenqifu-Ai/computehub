# 初创企业现金流管理指南

## 现金流的重要性

### 现金流 vs 利润
| 维度 | 利润 | 现金流 |
|------|------|--------|
| 定义 | 收入-成本 | 现金流入-流出 |
| 时间点 | 权责发生制 | 收付实现制 |
| 重要性 | 长期生存 | 短期生存 |
| 风险 | 账面利润 | 现金断裂 |

### 企业死亡原因
| 原因 | 占比 |
|------|------|
| 现金流断裂 | 80% |
| 市场需求不足 | 40% |
| 团队问题 | 30% |
| 竞争失败 | 20% |

**结论**：现金流是企业生存的血液，比利润更重要。

## 现金流分类

### 三大现金流
| 类型 | 内容 | 健康状态 |
|------|------|----------|
| 经营现金流 | 日常经营收支 | 正数为优 |
| 投资现金流 | 投资活动收支 | 成长期可为负 |
| 筹资现金流 | 融资活动收支 | 视情况而定 |

### 现金流组合分析
| 经营 | 投资 | 筹资 | 企业状态 | 建议 |
|------|------|------|----------|------|
| + | - | + | 快速成长 | 保持增长 |
| + | - | - | 成熟盈利 | 投资扩张 |
| + | + | - | 收缩期 | 寻找增长点 |
| - | + | + | 困难期 | 改善经营 |
| - | - | + | 困难期 | 谨慎融资 |
| - | + | - | 危机期 | 紧急应对 |

### 现金流监控指标
| 指标 | 公式 | 健康值 |
|------|------|--------|
| 现金余额 | 期末现金 | >3个月支出 |

| 现金燃烧率 | 月支出-月收入 | 负值越小越好 |

## 总结

### 现金流管理要点
1. **预测先行**：提前预测现金流
2. **动态监控**：每周监控现金流
3. **预警机制**：设置预警信号
4. **开源节流**：增加流入、减少流出
5. **未雨绸缪**：提前融资规划
