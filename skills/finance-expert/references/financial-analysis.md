# 财务报表分析

## 三大财务报表

### 资产负债表
**反映企业某一特定日期的财务状况**

| 项目 | 内容 |
|------|------|
| 资产 | 流动资产 + 非流动资产 |
| 负债 | 流动负债 + 非流动负债 |
| 所有者权益 | 实收资本 + 留存收益 |

**平衡公式**：资产 = 负债 + 所有者权益

### 利润表
**反映企业一定期间的经营成果**

| 项目 | 计算 |
|------|------|
| 营业收入 | 主营 + 其他业务 |
| 营业成本 | 直接成本 |
| 营业利润 | 收入 - 成本 - 费用 - 税金 |
| 利润总额 | 营业利润 + 营业外收支 |
| 净利润 | 利润总额 - 所得税 |

### 现金流量表
**反映企业现金及现金等价物流入流出**

| 类型 | 内容 |
|------|------|
| 经营活动 | 销售商品、购买商品、支付工资 |
| 投资活动 | 购建资产、处置资产 |
| 筹资活动 | 吸收投资、分配股利、借款还款 |

## 财务指标分析

### 盈利能力
| 指标 | 公式 | 标准 |
|------|------|------|
| 毛利率 | (收入-成本)/收入 | >30% |
| 净利率 | 净利润/收入 | >10% |
| ROE | 净利润/净资产 | >15% |
| ROA | 净利润/总资产 | >8% |

### 偿债能力
| 指标 | 公式 | 标准 |
|------|------|------|
| 流动比率 | 流动资产/流动负债 | >2 |
| 速动比率 | (流动资产-存货)/流动负债 | >1 |
| 资产负债率 | 负债/资产 | <70% |

### 运营能力
| 指标 | 公式 | 含义 |
|------|------|------|
| 存货周转率 | 营业成本/平均存货 | 周转速度 |
| 应收周转率 | 营业收入/平均应收 | 回款速度 |
| 总资产周转率 | 营业收入/平均资产 | 资产效率 |

### 发展能力
| 指标 | 公式 | 含义 |
|------|------|------|
| 营收增长率 | (本期-上期)/上期 | 成长性 |
| 利润增长率 | (本期-上期)/上期 | 盈利增长 |
