# 会计实务操作

## 记账基础

### 会计科目分类
| 类别 | 科目示例 | 借贷方向 |
|------|---------|---------|
| 资产 | 库存现金、银行存款、应收账款 | 借增贷减 |
| 负债 | 短期借款、应付账款、预收账款 | 贷增借减 |
| 所有者权益 | 实收资本、盈余公积、未分配利润 | 贷增借减 |
| 收入 | 主营业务收入、其他业务收入 | 贷增借减 |
| 费用 | 管理费用、销售费用、财务费用 | 借增贷减 |

### 常用会计分录

**收到投资款**
```
借：银行存款
  贷：实收资本
```

**支付费用**
```
借：管理费用/销售费用
  贷：银行存款
```

**确认收入**
```
借：银行存款/应收账款
  贷：主营业务收入
      应交税费-应交增值税(销项税)
```

**结转成本**
```
借：主营业务成本
  贷：库存商品
```

**计提工资**
```
借：管理费用/销售费用/生产成本
  贷：应付职工薪酬
```

## 财务报表编制

### 资产负债表结构
```
资产 = 负债 + 所有者权益

流动资产
├── 货币资金
├── 应收账款
├── 预付账款
└── 存货

非流动资产
├── 固定资产
├── 无形资产
└── 长期待摊费用

流动负债
├── 短期借款
├── 应付账款
├── 预收账款
└── 应付职工薪酬

非流动负债
└── 长期借款

所有者权益
├── 实收资本
├── 盈余公积
└── 未分配利润
```

### 利润表结构
```
营业收入
  减：营业成本
      税金及附加
      销售费用
      管理费用
      财务费用
  = 营业利润
  加：营业外收入
  减：营业外支出
  = 利润总额
  减：所得税费用
  = 净利润
```
