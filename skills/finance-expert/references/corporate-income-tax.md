# 税收政策：企业所得税实务

## 一、企业所得税概述

### 1. 企业所得税定义
企业所得税是国家对企业和其他取得收入的组织的生产经营所得和其他所得征收的一种税。

### 2. 纳税义务人
```markdown
居民企业：
- 在中国境内成立
- 在中国境外成立但实际管理机构在中国境内
- 就其来源于中国境内、境外的所得缴纳企业所得税

非居民企业：
- 在中国境内设立机构、场所
- 就其所设机构、场所取得的所得缴纳企业所得税
- 未设立机构、场所但来源于中国境内的所得缴纳企业所得税
```

### 3. 税率
```markdown
基本税率：25%

优惠税率：
- 小型微利企业：20%（实际税负5%或2.5%）
- 高新技术企业：15%
- 技术先进型服务企业：15%
- 重点软件企业：10%
- 重点集成电路设计企业：10%
- 西部地区鼓励类产业企业：15%

预提所得税率：20%（实际征收10%）
```

## 二、应纳税所得额计算

### 1. 收入总额
```markdown
收入范围：
- 销售货物收入
- 提供劳务收入
- 转让财产收入
- 股息、红利等权益性投资收益
- 利息收入
- 租金收入
- 特许权使用费收入
- 接受捐赠收入
- 其他收入

收入确认原则：
- 权责发生制原则
- 实质重于形式原则
- 收入与成本配比原则

收入确认时间：
- 销售货物：按照合同约定的收款日期
- 提供劳务：按照完工进度或完成的工作量
- 转让财产：按照合同约定的收款日期
```

### 2. 不征税收入
```markdown
财政拨款：
- 各级人民政府对纳入预算管理的事业单位、社会团体等组织拨付的财政资金

行政事业性收费、政府性基金：
- 依法收取并纳入财政管理的行政事业性收费、政府性基金

其他不征税收入：
- 国务院规定的其他不征税收入

注意：
- 不征税收入支出形成的费用不得扣除
- 不征税收入支出形成的资产折旧不得扣除
```

### 3. 免税收入
```markdown
国债利息收入：
- 企业购买国债取得的利息收入免税
- 转让国债收益不免免税

符合条件的居民企业之间股息、红利：
- 直接投资于其他居民企业取得的投资收益
- 不包括连续持有居民企业公开发行并上市流通的股票不足12个月取得的投资收益

在中国境内设立机构、场所的非居民企业从居民企业取得与该机构、场所有实际联系的股息、红利：
- 按权益性投资收益免税

符合条件的非营利组织收入：
- 接受捐赠收入
- 政府补助收入
- 会费收入
- 不征税收入孳生的银行存款利息收入
```

### 4. 税前扣除项目
```markdown
基本扣除原则：
- 真实性原则：实际发生
- 相关性原则：与取得收入有关
- 合理性原则：符合经营常规
- 合法性原则：符合法律法规规定

成本：
- 销售成本
- 劳务成本
- 固定资产折旧
- 无形资产摊销
- 长期待摊费用摊销

费用：
- 销售费用
- 管理费用
- 财务费用

税金：
- 消费税
- 城市维护建设税
- 教育费附加
- 资源税
- 土地增值税
- 房产税
- 车船税
- 印花税
- 注意：增值税不得扣除

损失：
- 资产盘亏、报废、毁损损失
- 坏账损失
- 资产减值损失
- 自然灾害等不可抗力造成的损失
```

### 5. 扣除标准
```markdown
职工福利费：
- 工资薪金总额14%以内准予扣除
- 超过部分不得扣除

工会经费：
- 工资薪金总额2%以内准予扣除

职工教育经费：
- 工资薪金总额8%以内准予扣除
- 超过部分准予结转以后年度扣除

业务招待费：
- 发生额的60%扣除
- 但最高不得超过当年销售（营业）收入的5‰
- 两者取较低值扣除

广告费和业务宣传费：
- 销售（营业）收入15%以内准予扣除
- 超过部分准予结转以后年度扣除
- 烟草企业广告费不得扣除

公益性捐赠：
- 年度利润总额12%以内准予扣除
- 超过部分准予结转以后三年内扣除
- 必须通过公益性社会组织或县级以上人民政府

利息支出：
- 向金融企业借款：据实扣除
- 向非金融企业借款：不超过金融企业同期同类贷款利率

手续费及佣金：
- 财产保险企业：15%
- 人身保险企业：10%
- 其他企业：合同金额的5%
```

### 6. 不得扣除项目
```markdown
不得扣除项目：
- 向投资者支付的股息、红利
- 企业所得税税款
- 税收滞纳金
- 罚金、罚款和被没收财物的损失
- 超过规定标准的捐赠支出
- 赞助支出
- 未经核定的准备金支出
- 与取得收入无关的其他支出
- 企业之间支付的管理费
- 企业内营业机构之间支付的租金和特许权使用费
- 非银行企业内营业机构之间支付的利息
```

## 三、应纳税额计算

### 1. 计算公式
```markdown
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 以前年度亏损

应纳税额 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额
```

### 2. 亏损弥补
```markdown
亏损弥补规定：
- 企业纳税年度发生的亏损，准予向以后年度结转
- 结转年限最长不得超过五年
- 弥补顺序：先弥补先发生的亏损

注意事项：
- 亏损弥补必须连续
- 五年内没有弥补完的，不得再弥补
- 境外亏损不得用境内盈利弥补
- 境内亏损可用境外盈利弥补
```

### 3. 境外所得抵免
```markdown
抵免范围：
- 居民企业来源于中国境外的应税所得
- 非居民企业在中国境内设立机构、场所，取得发生在中国境外但与该机构、场所有实际联系的应税所得

抵免方法：
- 分国不分项抵免
- 综合抵免法（可选）

抵免限额计算：
抵免限额 = 中国境内、境外所得依照企业所得税法规定计算的应纳税总额 × 来源于某国（地区）的应纳税所得额 ÷ 中国境内、境外应纳税所得总额

注意事项：
- 超过抵免限额的部分可在以后五个年度内抵免
- 不足抵免限额的部分据实抵免
```

## 四、企业所得税优惠政策

### 1. 小型微利企业优惠
```markdown
认定条件：
- 从事国家非限制和禁止行业
- 年度应纳税所得额不超过300万元
- 从业人数不超过300人
- 资产总额不超过5000万元

优惠政策（2023年）：
- 年应纳税所得额不超过100万元的部分：
  - 减按25%计入应纳税所得额
  - 按20%税率缴纳
  - 实际税负5%

- 年应纳税所得额超过100万元但不超过300万元的部分：
  - 减按25%计入应纳税所得额
  - 按20%税率缴纳
  - 实际税负5%

计算示例：
某小型微利企业年应纳税所得额280万元
- 100万元部分：100 × 25% × 20% = 5万元
- 180万元部分：180 × 25% × 20% = 9万元
- 应纳税额：5 + 9 = 14万元
- 实际税负：14 ÷ 280 = 5%
```

### 2. 高新技术企业优惠
```markdown
认定条件：
- 注册成立一年以上
- 拥有核心自主知识产权
- 属于国家重点支持的高新技术领域
- 科技人员占比不低于10%
- 研发费用占比符合要求
- 高新技术产品收入占比不低于60%
- 企业创新能力评价达标

优惠政策：
- 减按15%税率征收企业所得税
- 延续亏损结转年限至10年
- 技术转让所得优惠

研发费用加计扣除：
- 研发费用实际发生额的100%加计扣除
- 无形资产成本的200%摊销
```

### 3. 研发费用加计扣除
```markdown
适用范围：
- 科技型中小企业
- 高新技术企业
- 所有开展研发活动的企业

加计扣除比例：
- 研发费用100%加计扣除（2023年）
- 制造企业研发费用100%加计扣除
- 科技型中小企业研发费用100%加计扣除

可加计扣除的研发费用：
- 人员人工费用
- 直接投入费用
- 折旧费用与长期待摊费用
- 无形资产摊销
- 新产品设计费等
- 其他相关费用

申报方式：
- 预缴享受
- 年度汇算清缴
- 留存备查资料
```

### 4. 固定资产加速折旧
```markdown
适用范围：
- 由于技术进步产品更新换代较快的固定资产
- 常年处于强震动、高腐蚀状态的固定资产

加速折旧方法：
- 双倍余额递减法
- 年数总和法
- 缩短折旧年限（不得低于规定年限的60%）

特殊规定：
- 小型微利企业研发仪器设备加速折旧
- 信息技术、生物技术等特定行业加速折旧
- 企业可自行选择是否采用加速折旧
```

### 5. 创业投资企业优惠
```markdown
创业投资企业：
- 采取股权投资方式直接投资于初创科技型企业满2年
- 可按投资额的70%抵扣应纳税所得额
- 当年不足抵扣的可在以后年度结转抵扣

有限合伙制创业投资企业：
- 法人合伙人投资于初创科技型企业满2年
- 可按投资额的70%抵扣分得的应纳税所得额
```

### 6. 区域性优惠
```markdown
西部地区鼓励类产业企业：
- 减按15%税率征收企业所得税
- 需要以《西部地区鼓励类产业目录》规定项目为主营业务
- 主营业务收入占企业收入总额70%以上

海南自由贸易港：
- 鼓励类产业企业减按15%征收企业所得税
- 符合条件的企业免税（2025年前）

前海深港现代服务业合作区：
- 符合条件的企业减按15%征收企业所得税

横琴粤澳深度合作区：
- 符合条件的企业减按15%征收企业所得税
```

## 五、企业所得税申报

### 1. 纳税期限
```markdown
纳税年度：
- 公历1月1日至12月31日
- 实际经营期不足12个月的，以实际经营期为一个纳税年度
- 清算期间作为一个纳税年度

预缴：
- 按月或按季预缴
- 自月份或季度终了之日起15日内预缴

汇算清缴：
- 年度终了之日起5个月内汇算清缴
- 多退少补
```

### 2. 申报方式
```markdown
电子申报：
- 通过电子税务局申报
- 使用企业所得税年度纳税申报表
- 附报相关财务报表和资料

纸质申报：
- 到办税服务厅申报
- 提交纸质申报表和相关资料
```

### 3. 申报资料
```markdown
必报资料：
- 企业所得税年度纳税申报表（A类）
- 企业所得税月（季）度预缴纳税申报表（A类）
- 财务报表
- 企业所得税优惠事项备案表

留存备查资料：
- 资产损失相关资料
- 优惠事项相关资料
- 研发费用加计扣除相关资料
- 其他需要留存备查的资料
```

## 六、常见税务风险

### 1. 收入确认风险
```markdown
常见问题：
- 收入确认不及时
- 隐瞒收入
- 收入确认金额不准确
- 视同销售未确认收入

风险提示：
- 加强收入管理
- 准确确认收入
- 规范视同销售处理
- 及时结转收入
```

### 2. 成本费用扣除风险
```markdown
常见问题：
- 虚列成本费用
- 扣除凭证不合规
- 超标准扣除
- 扣除项目与经营无关

风险提示：
- 真实发生原则
- 取得合规凭证
- 按标准扣除
- 相关性原则
```

### 3. 优惠事项风险
```markdown
常见问题：
- 不符合优惠条件享受优惠
- 优惠事项资料不完整
- 优惠金额计算错误

风险提示：
- 严格把握优惠条件
- 完善优惠事项资料
- 准确计算优惠金额
```

### 4. 关联交易风险
```markdown
常见问题：
- 关联交易定价不公允
- 关联交易披露不完整
- 利润转移

风险提示：
- 独立交易原则
- 完整披露关联交易
- 准备转让定价文档
```
