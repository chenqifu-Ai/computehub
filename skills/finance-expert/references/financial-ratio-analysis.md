# 财务分析：财务比率分析体系

## 一、财务比率分析概述

### 1. 财务比率分析定义
财务比率分析是通过计算财务报表中各项目之间的比率，评估企业财务状况、经营成果和现金流量的分析方法。

### 2. 分析意义
```markdown
横向比较：与同行业企业对比，判断企业在行业中的位置
纵向比较：与企业历史数据对比，判断发展趋势
目标比较：与预定目标对比，判断目标完成情况
```

### 3. 分析框架
```
                    财务比率分析
                         │
    ┌────────┬─────────┼─────────┬────────┐
    │        │         │         │        │
 偿债能力 盈利能力  运营能力  发展能力  现金流量
```

## 二、偿债能力分析

### 1. 短期偿债能力

#### 流动比率
```markdown
公式：流动比率 = 流动资产 ÷ 流动负债

标准值：2:1（经验值）
含义：每1元流动负债有2元流动资产作保障

判断标准：
- > 2：偿债能力较强，但可能资金运用效率低
- 1-2：偿债能力适中
- < 1：偿债能力不足，存在风险

注意：
- 行业差异大，需与同行业比较
- 过高可能表示资金使用效率低
- 需结合流动资产质量分析
```

#### 速动比率
```markdown
公式：速动比率 = (流动资产 - 存货) ÷ 流动负债
       或 = 速动资产 ÷ 流动负债

标准值：1:1（经验值）
速动资产 = 流动资产 - 存货 - 预付账款

判断标准：
- > 1：短期偿债能力强
- 0.5-1：偿债能力一般
- < 0.5：偿债能力弱

注意：
- 扣除了变现能力较弱的存货
- 更准确反映即时偿债能力
```

#### 现金比率
```markdown
公式：现金比率 = (货币资金 + 交易性金融资产) ÷ 流动负债

标准值：0.2-0.3（经验值）
含义：直接用现金偿还流动负债的能力

判断标准：
- > 0.3：现金充裕，但可能资金利用效率低
- 0.2-0.3：合理水平
- < 0.2：现金紧张
```

#### 现金流量比率
```markdown
公式：现金流量比率 = 经营活动现金流量净额 ÷ 流动负债

含义：经营活动产生的现金对流动负债的保障程度

判断标准：
- > 1：经营现金流充足，偿债能力强
- 0.5-1：偿债能力一般
- < 0.5：偿债能力弱
```

### 2. 长期偿债能力

#### 资产负债率
```markdown
公式：资产负债率 = 负债总额 ÷ 资产总额 × 100%

标准值：40%-70%（行业差异大）
含义：总资产中由负债筹集的比例

判断标准：
- < 40%：财务风险低，但可能杠杆不足
- 40%-70%：合理范围
- > 70%：财务风险高

注意：
- 行业差异大，重资产行业通常较高
- 需结合盈利能力分析
- 过低可能表示财务杠杆利用不足
```

#### 产权比率
```markdown
公式：产权比率 = 负债总额 ÷ 所有者权益 × 100%

含义：债权人提供的资本与股东提供的资本的相对关系

判断标准：
- < 100%：财务风险较低
- 100%-200%：财务风险适中
- > 200%：财务风险较高
```

#### 权益乘数
```markdown
公式：权益乘数 = 资产总额 ÷ 所有者权益

含义：资产是所有者权益的多少倍

与资产负债率的关系：
权益乘数 = 1 ÷ (1 - 资产负债率)

判断标准：
- 权益乘数越大，财务杠杆越高
- 需结合盈利能力判断杠杆是否合理
```

#### 利息保障倍数
```markdown
公式：利息保障倍数 = 息税前利润 ÷ 利息费用
       = (利润总额 + 利息费用) ÷ 利息费用

标准值：> 3（经验值）
含义：企业支付利息的能力

判断标准：
- > 5：支付利息能力强
- 3-5：支付利息能力一般
- 1-3：支付利息能力较弱
- < 1：无法支付利息，存在风险
```

## 三、盈利能力分析

### 1. 销售盈利能力

#### 销售毛利率
```markdown
公式：销售毛利率 = (营业收入 - 营业成本) ÷ 营业收入 × 100%
       = 毛利润 ÷ 营业收入 × 100%

含义：每1元销售收入扣除成本后的毛利

影响因素：
- 产品定价能力
- 成本控制能力
- 产品结构
- 行业竞争程度

分析要点：
- 与同行业比较
- 与企业历史比较
- 分析变动原因
```

#### 销售净利率
```markdown
公式：销售净利率 = 净利润 ÷ 营业收入 × 100%

含义：每1元销售收入最终形成多少净利润

影响因素：
- 毛利率水平
- 期间费用控制
- 其他收入支出
- 所得税影响
```

### 2. 资产盈利能力

#### 资产净利率（ROA）
```markdown
公式：资产净利率 = 净利润 ÷ 平均资产总额 × 100%

含义：企业运用全部资产获取利润的能力

杜邦分解：
ROA = 销售净利率 × 资产周转率

判断标准：
- 越高越好
- 需与同行业比较
- 结合资产结构分析
```

#### 总资产报酬率
```markdown
公式：总资产报酬率 = 息税前利润 ÷ 平均资产总额 × 100%

含义：企业运用全部资产获取息税前利润的能力

优点：排除了资本结构的影响
```

### 3. 资本盈利能力

#### 净资产收益率（ROE）
```markdown
公式：净资产收益率 = 净利润 ÷ 平均所有者权益 × 100%

含义：股东投入资本的回报率
这是股东最关心的指标

杜邦分解：
ROE = 销售净利率 × 资产周转率 × 权益乘数

影响因素：
- 销售盈利能力
- 资产运营效率
- 财务杠杆

判断标准：
- > 15%：优秀
- 10%-15%：良好
- 5%-10%：一般
- < 5%：较差

注意：
- 需关注是高杠杆还是高盈利
- 与行业平均水平比较
- 分析可持续性
```

#### 投入资本回报率（ROIC）
```markdown
公式：ROIC = 息税前利润 × (1-税率) ÷ 投入资本 × 100%

投入资本 = 有息负债 + 所有者权益

含义：企业投入的全部资本（股权+债权）的回报率

优点：排除了资本结构的影响，反映企业真实的经营回报能力
```

## 四、运营能力分析

### 1. 资产周转效率

#### 总资产周转率
```markdown
公式：总资产周转率 = 营业收入 ÷ 平均资产总额

含义：企业运用全部资产创造收入的能力

判断标准：
- 越高越好
- 需与行业比较
- 分析变动原因
```

#### 流动资产周转率
```markdown
公式：流动资产周转率 = 营业收入 ÷ 平均流动资产

含义：流动资产周转速度

周转天数 = 365 ÷ 流动资产周转率
```

#### 固定资产周转率
```markdown
公式：固定资产周转率 = 营业收入 ÷ 平均固定资产

含义：固定资产利用效率

注意：
- 重资产行业通常较低
- 分析固定资产利用程度
- 关注产能与销售匹配
```

### 2. 应收账款管理

#### 应收账款周转率
```markdown
公式：应收账款周转率 = 萓业收入 ÷ 平均应收账款

应收账款周转天数 = 365 ÷ 应收账款周转率

含义：应收账款周转速度，反映收款效率

判断标准：
- 周转率越高越好
- 周转天数越短越好
- 一般30-60天为合理

分析要点：
- 与信用政策对比
- 分析应收账款账龄
- 关注坏账风险
```

#### 应收账款占收入比
```markdown
公式：应收账款占收入比 = 应收账款 ÷ 营业收入 × 100%

含义：每1元收入中未收回的款项

判断标准：
- 越低越好
- 与行业比较
- 分析变化趋势
```

### 3. 存货管理

#### 存货周转率
```markdown
公式：存货周转率 = 营业成本 ÷ 平均存货

存货周转天数 = 365 ÷ 存货周转率

含义：存货周转速度，反映存货管理效率

判断标准：
- 周转率越高越好
- 周转天数越短越好
- 行业差异大

分析要点：
- 与行业比较
- 分析存货结构
- 关注滞销风险
```

### 4. 应付账款管理

#### 应付账款周转率
```markdown
公式：应付账款周转率 = 营业成本 ÷ 平均应付账款

应付账款周转天数 = 365 ÷ 应付账款周转率

含义：应付账款付款速度

判断标准：
- 周转天数适当延长可占用供应商资金
- 过长可能影响供应商关系
- 分析付款政策
```

### 5. 现金循环周期
```markdown
公式：现金循环周期 = 应收账款周转天数 + 存货周转天数 - 应付账款周转天数

含义：从投入现金到收回现金的时间

判断标准：
- 越短越好，资金效率越高
- 负值表示占用供应商资金能力强

优化方向：
- 缩短应收账款周转天数
- 缩短存货周转天数
- 延长应付账款周转天数
```

## 五、发展能力分析

### 1. 收入增长能力

#### 营业收入增长率
```markdown
公式：营业收入增长率 = (本期营业收入 - 上期营业收入) ÷ 上期营业收入 × 100%

含义：企业收入增长速度

判断标准：
- > 20%：高速增长
- 10%-20%：较快增长
- 0%-10%：缓慢增长
- < 0：负增长

分析要点：
- 分析增长来源（销量增长/价格增长）
- 分析增长可持续性
- 与行业增长比较
```

#### 三年营业收入复合增长率
```markdown
公式：CAGR = (本期营业收入 ÷ 三年前营业收入)^(1/3) - 1

含义：三年收入平均增长速度

优点：消除短期波动影响
```

### 2. 利润增长能力

#### 净利润增长率
```markdown
公式：净利润增长率 = (本期净利润 - 上期净利润) ÷ 上期净利润 × 100%

含义：企业利润增长速度

分析要点：
- 分析增长来源
- 与收入增长匹配度
- 盈利质量分析
```

### 3. 资产增长能力

#### 总资产增长率
```markdown
公式：总资产增长率 = (本期总资产 - 上期总资产) ÷ 上期总资产 × 100%

含义：企业资产扩张速度

分析要点：
- 分析资产增长来源
- 与收入增长匹配
- 增长质量分析
```

### 4. 资本积累能力

#### 资本积累率
```markdown
公式：资本积累率 = (期末所有者权益 - 期初所有者权益) ÷ 期初所有者权益 × 100%

含义：所有者权益增长速度

判断标准：
- > 0：资本积累为正
- < 0：资本减少，需关注原因
```

## 六、现金流量分析

### 1. 现金流量结构

#### 现金流量类型
```markdown
经营活动现金流：企业日常经营产生的现金流量
投资活动现金流：企业投资活动产生的现金流量
筹资活动现金流：企业筹资活动产生的现金流量
```

#### 现金流量组合分析
| 经营现金流 | 投资现金流 | 筹资现金流 | 企业状态 |
|-----------|-----------|-----------|---------|
| + | - | + | 成长期，需要外部融资 |
| + | - | - | 成熟期，还债和分红 |
| + | + | - | 衰退期，变卖资产 |
| - | + | + | 困难期，借钱维持 |
| - | - | + | 困难期，借钱投资 |
| - | + | - | 困难期，难以持续 |

### 2. 现金流量比率

#### 经营现金流量对流动负债比
```markdown
公式：经营现金流量对流动负债比 = 经营活动现金流量净额 ÷ 流动负债 × 100%

含义：经营活动现金流量对流动负债的保障程度

判断标准：
- > 50%：较强
- 20%-50%：一般
- < 20%：较弱
```

#### 经营现金流量对债务比
```markdown
公式：经营现金流量对债务比 = 经营活动现金流量净额 ÷ 负债总额 × 100%

含义：经营活动现金流量对总负债的保障程度
```

#### 经营现金流量对利息比
```markdown
公式：经营现金流量对利息比 = 经营活动现金流量净额 ÷ 利息费用

含义：经营活动现金流量对利息支付的保障程度

判断标准：
- > 3：保障程度高
- 1-3：保障程度一般
- < 1：保障程度低
```

### 3. 盈利质量分析

#### 经营现金流与净利润比
```markdown
公式：经营现金流与净利润比 = 经营活动现金流量净额 ÷ 净利润

含义：净利润的现金含量

判断标准：
- > 1：盈利质量高，现金充足
- 接近 1：盈利质量正常
- < 0.5：盈利质量较差，可能存在应收账款问题
- < 0：盈利质量问题严重，需重点关注
```

#### 销售现金比率
```markdown
公式：销售现金比率 = 经营活动现金流量净额 ÷ 营业收入 × 100%

含义：每1元销售收入带来的经营现金流入

判断标准：
- 越高越好
- 反映销售收入的质量
```

## 七、杜邦分析体系

### 1. 杜邦分析原理
```markdown
核心思想：将ROE分解为三个维度

ROE = 销售净利率 × 资产周转率 × 权益乘数
     = (净利润/收入) × (收入/资产) × (资产/权益)

三个维度：
- 销售净利率：反映盈利能力
- 资产周转率：反映运营能力
- 权益乘数：反映财务杠杆
```

### 2. 杜邦分析图
```
                    ROE
                     │
        ┌────────────┼────────────┐
        │            │            │
    销售净利率   资产周转率    权益乘数
        │            │            │
    净利润/收入   收入/资产    资产/权益
        │            │            │
        │        ┌───┴───┐        │
        │        │       │        │
      净利润   收入   资产   资产   权益
        │        │       │        │
    ┌───┴───┐    │       │    ┌───┴───┐
  收入-成本 流量资产 非流动资产 负债+权益
```

### 3. 杜邦分析应用
```markdown
ROE下降分析：
- 销售净利率下降？→ 成本上升或价格下降
- 资产周转率下降？→ 资产闲置或销售下降
- 权益乘数下降？→ 去杠杆或亏损

ROE提升分析：
- 销售净利率提升？→ 成本控制或提价
- 资产周转率提升？→ 运营效率提升
- 权益乘数提升？→ 加杠杆（需关注风险）
```

## 八、财务比率综合分析

### 1. Z计分模型
```markdown
Z = 1.2X1 + 1.4X2 + 3.3X3 + 0.6X4 + 0.999X5

X1 = 营运资金/总资产
X2 = 留存收益/总资产
X3 = 息税前利润/总资产
X4 = 股东权益/负债总额
X5 = 销售额/总资产

判断标准：
- Z > 2.99：财务状况良好
- 1.81 < Z < 2.99：存在财务风险
- Z < 1.81：可能破产
```

### 2. 沃尔评分法
```markdown
步骤：
1. 选择财务指标
2. 确定标准值
3. 计算实际值
4. 计算相对比率
5. 加权计算总分

示例：
指标          标准值  实际值  相对比率  权重   得分
流动比率       2.0     1.8     0.9      0.25   22.5
资产负债率(%)  50      45      0.9      0.25   22.5
净资产收益率(%) 15     18      1.2      0.25   30.0
总资产周转率   1.0     1.2     1.2      0.25   30.0
合计                                        105.0
```

## 九、财务分析注意事项

### 1. 数据质量
```markdown
- 关注财务报表真实性
- 分析会计政策影响
- 关注表外信息
- 结合非财务信息
```

### 2. 比较基准
```markdown
- 与历史数据比较
- 与同行业比较
- 与预算目标比较
- 与行业标杆比较
```

### 3. 指标局限
```markdown
- 时点指标vs时期指标
- 账面价值vs市场价值
- 历史成本vs公允价值
- 单一指标的片面性
```

### 4. 行业特点
```markdown
- 不同行业财务特征不同
- 行业周期影响指标
- 商业模式影响指标
- 需结合行业分析
```
