# 成本核算方法

## 成本核算概述

### 成本核算定义
成本核算是对生产经营过程中发生的各种费用进行归集、分配和计算的过程。

### 成本分类
| 类型 | 说明 | 案例 |
|------|------|------|
| 直接材料 | 直接用于产品生产 | 原材料、零部件 |
| 直接人工 | 直接参与生产 | 生产工人工资 |
| 制造费用 | 间接生产成本 | 车间折旧、水电 |

## 成本核算方法

### 品种法
**适用**：大批量、单步骤生产

| 特点 | 说明 |
|------|------|
| 计算对象 | 产品品种 |
| 计算周期 | 月末 |
| 适用企业 | 电力、采掘、供水 |

**计算步骤**：
1. 归集直接材料、直接人工
2. 分配制造费用
3. 计算产品成本

### 分批法
**适用**：单件、小批生产

| 特点 | 说明 |
|------|------|
| 计算对象 | 产品批次 |
| 计算周期 | 批次完工 |
| 适用企业 | 机械、船舶、建筑 |

**计算步骤**：
1. 按批次归集成本
2. 完工批次计算成本
3. 未完工批次累计成本

### 分步法
**适用**：大量、多步骤生产

| 特点 | 说明 |
|------|------|
| 计算对象 | 生产步骤 |
| 计算周期 | 月末 |
| 适用企业 | 纺织、冶金、化工 |

**计算步骤**：
1. 按步骤归集成本
2. 计算各步骤成本
3. 结转下步骤成本

## 制造费用分配

### 分配标准
| 标准 | 公式 | 适用 |
|------|------|------|
| 生产工时 | 制造费用/总工时 | 机械化程度相近 |
| 直接人工 | 制造费用/直接人工 | 劳动密集 |
| 机器工时 | 制造费用/机器工时 | 机械化程度高 |
| 直接材料 | 制造费用/直接材料 | 材料相关 |

### 分配计算
```
分配率 = 制造费用总额 ÷ 分配标准总数
某产品分配额 = 该产品标准数 × 分配率
```

## 完工产品与在产品

### 成本分配
```
月初在产品成本 + 本月生产费用 = 完工产品成本 + 月末在产品成本
```

### 分配方法
| 方法 | 说明 | 适用 |
|------|------|------|
| 不计算在产品 | 在产品成本为0 | 在产品很少 |
| 在产品按年初数 | 年初数固定 | 在产品波动小 |
| 在产品按材料成本 | 只计材料 | 材料占比大 |
| 约当产量法 | 折合完工产品 | 在产品较多 |
| 定额成本法 | 按定额计算 | 定额完善 |

### 约当产量法
```
约当产量 = 在产品数量 × 完工程度
单位成本 = (月初在产品成本 + 本月费用) ÷ (完工产品 + 约当产量)
完工产品成本 = 完工产品数量 × 单位成本
在产品成本 = 在产品约当产量 × 单位成本
```

## 成本核算流程

### 核算流程
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   要素费用归集 → 成本项目归集 → 成本分配 → 成本计算 │
│                                                     │
│   要素费用：                                        │
│   ├── 外购材料                                    │
│   ├── 外购燃料动力                                │
│   ├── 工资                                        │
│   ├── 折旧                                        │
│   └── 其他                                        │
│                                                     │
│   成本项目：                                        │
│   ├── 直接材料                                    │
│   ├── 直接人工                                    │
│   └── 制造费用                                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 账务处理
| 步骤 | 分录 |
|------|------|
| 归集费用 | 借：生产成本/制造费用 |
| 分配费用 | 借：生产成本 贷：制造费用 |
| 完工入库 | 借：库存商品 贷：生产成本 |

## 成本分析

### 成本项目分析
| 分析内容 | 方法 |
|----------|------|
| 材料成本 | 价差×量差分析 |
| 人工成本 | 效率×工资率分析 |
| 制造费用 | 预算差异分析 |

### 成本差异分析
```
材料成本差异 = 实际成本 - 标准成本
材料价格差异 = (实际价格 - 标准价格) × 实际数量
材料数量差异 = (实际数量 - 标准数量) × 标准价格
```

## 成本控制

### 控制要点
| 环节 | 控制要点 |
|------|----------|
| 采购 | 采购价格、采购量 |
| 生产 | 消耗定额、生产效率 |
| 管理 | 费用预算、审批 |

### 成本改善
| 措施 | 效果 |
|------|------|
| 标准成本 | 明确目标 |
| 价值工程 | 降低成本 |
| 精益生产 | 消除浪费 |
| 目标成本 | 成本前置 |
