# 成本控制：企业成本管理方法

## 一、成本控制概述

### 1. 成本控制定义
成本控制是指企业按照预定目标，对成本形成过程中的一切耗费进行计算、调节和监督，及时发现偏差，纠正偏差，使成本限制在目标范围内，以实现成本目标的管理活动。

### 2. 成本控制意义
```markdown
降低成本：减少不必要的支出，提高资源利用效率
提高利润：成本降低直接增加利润空间
增强竞争力：成本优势是核心竞争力之一
支持定价：成本是定价决策的重要依据
促进管理：成本控制推动管理精细化
```

### 3. 成本控制原则
| 原则 | 说明 |
|------|------|
| 全面性原则 | 全员、全面、全过程控制 |
| 经济性原则 | 控制成本不应超过控制收益 |
| 责权利结合原则 | 责任、权力、利益相结合 |
| 例外管理原则 | 关注例外情况和重大差异 |
| 全程控制原则 | 事前、事中、事后全过程控制 |

## 二、成本分类与控制

### 1. 按成本习性分类

#### 固定成本控制
```markdown
定义：在一定范围内不随业务量变化的成本

控制要点：
- 合理确定固定成本规模
- 提高固定成本的利用效率
- 分摊固定成本到更多产品

常见固定成本：
- 折旧费用
- 管理人员工资
- 租金
- 保险费

控制方法：
- 预算控制：设定固定成本预算上限
- 效率提升：提高产能利用率
- 外包决策：考虑是否外包更经济
```

#### 变动成本控制
```markdown
定义：随业务量正比例变化的成本

控制要点：
- 降低单位变动成本
- 优化供应商降低采购成本
- 提高效率降低单位消耗

常见变动成本：
- 直接材料
- 直接人工
- 销售佣金

控制方法：
- 标准成本：设定单位消耗标准
- 采购控制：降低采购单价
- 效率提升：减少单位消耗
```

#### 混合成本控制
```markdown
定义：既有固定成分又有变动成分的成本

分解方法：
- 高低点法
- 散布图法
- 回归分析法

控制要点：
- 区分固定部分和变动部分
- 分别采取控制措施
- 分析成本动因
```

## 三、成本控制方法

### 1. 标准成本法

#### 标准成本制定
```markdown
直接材料标准成本：
= 材料标准用量 × 材料标准价格

直接人工标准成本：
= 标准工时 × 标准工资率

制造费用标准成本：
= 标准工时 × 标准制造费用率

制定依据：
- 历史数据分析
- 行业标杆对比
- 技术测定
- 专家判断
```

#### 成本差异分析
```markdown
材料成本差异：
材料价格差异 = (实际价格 - 标准价格) × 实际数量
材料数量差异 = (实际数量 - 标准数量) × 标准价格

人工成本差异：
工资率差异 = (实际工资率 - 标准工资率) × 实际工时
效率差异 = (实际工时 - 标准工时) × 标准工资率

制造费用差异：
耗费差异 = 实际费用 - 预算费用
效率差异 = (实际工时 - 标准工时) × 标准费用率
```

#### 差异处理
```markdown
有利差异（节约）：
- 分析原因，总结经验
- 防止以牺牲质量为代价

不利差异（超支）：
- 分析原因，明确责任
- 制定改进措施
- 追踪改进效果

差异调查标准：
- 差异金额超过一定比例
- 差异持续发生
- 关键项目出现差异
```

### 2. 预算控制法

#### 预算编制
```markdown
增量预算：
- 在上年基础上调整
- 简单易行
- 可能包含不合理因素

零基预算：
- 从零开始编制
- 每项支出需说明必要性
- 工作量大，但更科学

滚动预算：
- 定期调整预算
- 保持预算连续性
- 适应环境变化
```

#### 预算执行控制
```markdown
预算审批：
- 预算内支出按流程审批
- 预算外支出需特别审批
- 超预算支出需追加预算

预算监控：
- 定期分析预算执行情况
- 及时发现偏差
- 分析偏差原因

预算调整：
- 重大环境变化时可调整
- 调整需经审批
- 记录调整原因
```

### 3. 目标成本法

#### 目标成本确定
```markdown
目标成本 = 目标售价 - 目标利润

计算步骤：
1. 确定目标售价（市场定价）
2. 确定目标利润（企业要求）
3. 计算目标成本（售价-利润）
4. 分解目标成本到各环节

特点：
- 以市场为导向
- 从售价推导成本
- 强调成本规划
```

#### 目标成本分解
```markdown
按产品分解：
- 将目标成本分配到各产品
- 各产品成本不能超标

按部件分解：
- 将产品成本分解到各部件
- 部件成本不能超标

按责任分解：
- 将成本分解到各责任中心
- 明确成本责任
```

#### 价值工程分析
```markdown
价值 = 功能 / 成本

提高价值的方法：
1. 功能提高，成本降低（最佳）
2. 功能提高，成本不变
3. 功能大幅提高，成本略增
4. 功能不变，成本降低
5. 功能略降，成本大幅降低

应用步骤：
1. 明确产品功能
2. 分析功能重要性
3. 计算功能成本
4. 识别改进机会
5. 实施改进方案
```

### 4. 作业成本法

#### 基本原理
```markdown
传统成本法：
- 按单一分配率分摊间接费用
- 可能扭曲产品成本
- 不利于成本控制

作业成本法：
- 按作业动因分配成本
- 更准确的产品成本
- 揭示成本动因
```

#### 实施步骤
```markdown
第一步：识别作业
- 分析企业的各项作业
- 确定作业中心
- 归集作业成本

第二步：确定成本动因
- 分析各作业的成本动因
- 选择合适的成本动因
- 计算作业成本分配率

第三步：分配作业成本
- 按成本动因分配作业成本
- 计算产品成本
- 分析成本构成
```

#### 成本控制应用
```markdown
识别增值作业和非增值作业：
- 增值作业：客户愿意付费的作业
- 非增值作业：不创造价值的作业

控制措施：
- 保留增值作业，提高效率
- 减少非增值作业，降低成本
- 优化作业流程
- 消除无效作业
```

### 1. 研发环节成本控制

#### 研发成本管理
```markdown
设计成本控制：
- 设计决定了成本的70-80%
- 在设计阶段控制成本最有效
- 采用目标成本法指导设计

研发费用控制：
- 制定研发预算
- 控制研发周期
- 评估研发效果
```

#### 设计优化
```markdown
标准化设计：
- 采用标准件、通用件
- 减少零件种类
- 降低采购和库存成本

价值工程：
- 分析产品功能
- 剔除不必要功能
- 优化功能成本比

可制造性设计：
- 考虑生产便利性
- 减少生产难度

```

### 2. 采购环节成本控制

#### 采购价格控制
```markdown

- 多家比价
- 定期评估供应商

采购方式：
- 招标采购
- 集中采购

- 电子采购平台

价格控制方法：
- 市场价格跟踪
- 成本分析定价
- 竞价采购
```

#### 采购数量控制
```markdown
经济订货量（EOQ）：
Q = √(2DS/H)
D：年需求量
S：每次订货成本
H：单位存储成本

安全库存：
- 考虑需求波动
- 考虑供应不确定性
- 平衡库存成本和缺货风险

采购批量策略：
- 小批量多频次
- 大批量少频次
- 根据实际情况选择
```

#### 采购质量控制
```markdown
供应商审核：
- 质量体系审核
- 生产能力审核
- 技术能力审核

来料检验：
- 抽样检验
- 全检或免检
- 质量问题处理

质量成本：
- 预防成本
- 鉴定成本
- 内部失败成本
- 外部失败成本
```

## 五、成本分析与改善

### 1. 成本分析方法

#### 比较分析法
```markdown
与预算比较：
- 分析预算执行情况
- 找出差异原因
- 制定改进措施

与历史比较：
- 与上年同期比较
- 与历史最好水平比较
- 分析变化趋势

与行业比较：
- 与行业平均水平比较
- 与行业标杆比较
- 找出差距
```

#### 结构分析法
```markdown
成本结构分析：
- 分析各成本项目占比
- 找出主要成本项目
- 确定成本控制重点

成本变化分析：
- 分析成本结构变化
- 找出变化原因
- 预测未来趋势
```

#### 因素分析法
```markdown
连锁替代法：
- 分析各因素对成本的影响
- 确定各因素影响程度
- 找出主要影响因素

应用：
材料成本 = 产量 × 单耗 × 单价
- 分析产量变化影响
- 分析单耗变化影响
- 分析单价变化影响
```

### 2. 成本改善策略

#### 战略性成本降低
```markdown
价值链优化：
- 分析企业价值链
- 找出非增值环节
- 优化或外包

供应链整合：
- 优化供应商结构
- 深化供应商合作
- 降低供应链成本

商业模式创新：
- 改变盈利模式
- 降低成本结构
- 提升价值创造
```

#### 战术性成本控制
```markdown
流程优化：
- 简化流程
- 消除瓶颈
- 提高效率

精益管理：
- 减少浪费
- 持续改进
- 全员参与

技术改造：
- 设备升级
- 自动化改造
- 信息化建设
```
