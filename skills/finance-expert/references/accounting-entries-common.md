# 常用会计分录实务指南

## 概述

会计分录是会计核算的基础，是将经济业务转化为会计语言的过程。本文整理企业日常经营中常用的会计分录，帮助财务人员快速查找和应用。

---

## 一、会计分录基础

### 1. 会计分录格式

**基本格式**：
```
借：科目名称    金额
贷：科目名称    金额
```

**原则**：有借必有贷，借贷必相等

### 2. 科目分类

| 类型 | 借方 | 贷方 |
|-----|------|------|
| 资产类 | 增加 | 减少 |
| 负债类 | 减少 | 增加 |
| 所有者权益类 | 减少 | 增加 |
| 成本类 | 增加 | 结转 |
| 损益类（收入） | 减少 | 增加 |
| 损益类（费用） | 增加 | 减少 |

## 二、资金业务分录

### 1. 库存现金

**提取现金**：
```
借：库存现金
贷：银行存款
```

**现金存入银行**：
```
借：银行存款
贷：库存现金
```

**现金短缺（待查明原因）**：
```
借：待处理财产损溢——待处理流动资产损溢
贷：库存现金
```

**现金溢余（待查明原因）**：
```
借：库存现金
贷：待处理财产损溢——待处理流动资产损溢
```

### 2. 银行存款

**收到投资款**：
```
借：银行存款
贷：实收资本
    资本公积——资本溢价（如有溢价）
```

**收到借款**：
```
借：银行存款
贷：短期借款/长期借款
```

**支付借款利息**：
```
借：财务费用——利息支出
    应付利息（已计提部分）
贷：银行存款
```

### 3. 其他货币资金

**存出投资款**：
```
借：其他货币资金——存出投资款
贷：银行存款
```

**银行汇票存款**：
```
借：其他货币资金——银行汇票存款
贷：银行存款
```

## 三、采购业务分录

### 1. 原材料采购

**一般纳税人，收到发票**：
```
借：原材料/在途物资
    应交税费——应交增值税（进项税额）
贷：银行存款/应付账款/应付票据
```

**小规模纳税人**：
```
借：原材料/在途物资
贷：银行存款/应付账款/应付票据
```

**材料入库（从在途物资转入）**：
```
借：原材料
贷：在途物资
```

### 2. 采购费用

**运杂费计入采购成本**：
```
借：原材料/在途物资
    应交税费——应交增值税（进项税额）
贷：银行存款/应付账款
```

### 3. 采购退货

**已付款退货**：
```
借：银行存款
贷：原材料
    应交税费——应交增值税（进项税额转出）
```

**未付款退货**：
```
借：应付账款
贷：原材料
    应交税费——应交增值税（进项税额转出）
```

### 4. 应付账款

**购货发生应付账款**：
```
借：原材料/库存商品
    应交税费——应交增值税（进项税额）
贷：应付账款
```

**偿还应付账款**：
```
借：应付账款
贷：银行存款
```

**现金折扣（付款方）**：
```
借：应付账款（原金额）
贷：银行存款（实付金额）
    财务费用（折扣金额）
```

## 四、生产业务分录

### 1. 材料领用

**领用原材料**：
```
借：生产成本——直接材料（产品生产）
    制造费用（车间一般耗用）
    管理费用（管理部门耗用）
    销售费用（销售部门耗用）
贷：原材料
```

### 2. 人工成本

**计提工资**：
```
借：生产成本——直接人工（生产工人）
    制造费用（车间管理人员）
    管理费用（管理人员）
    销售费用（销售人员）
贷：应付职工薪酬——工资
```

**计提社保公积金**：
```
借：生产成本——直接人工
    制造费用
    管理费用
    销售费用
贷：应付职工薪酬——社会保险费
    应付职工薪酬——住房公积金
```

**发放工资**：
```
借：应付职工薪酬——工资
贷：银行存款
    应交税费——应交个人所得税
    其他应付款——社会保险费（个人部分）
    其他应付款——住房公积金（个人部分）
```

### 3. 制造费用

**归集制造费用**：
```
借：制造费用
贷：累计折旧（车间固定资产折旧）
    原材料（车间材料消耗）
    应付职工薪酬（车间人员工资）
    银行存款（其他费用）
```

**结转制造费用**：
```
借：生产成本——制造费用
贷：制造费用
```

### 4. 产品完工入库

**结转完工产品成本**：
```
借：库存商品
贷：生产成本——直接材料
    生产成本——直接人工
    生产成本——制造费用
```

## 五、销售业务分录

### 1. 确认收入

**销售商品确认收入**：
```
借：银行存款/应收账款/应收票据
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

**销售商品（小规模纳税人）**：
```
借：银行存款/应收账款
贷：主营业务收入
    应交税费——应交增值税
```

### 2. 结转成本

**结转销售成本**：
```
借：主营业务成本
贷：库存商品
```

### 3. 销售退回

**销售退回（冲减收入）**：
```
借：主营业务收入
    应交税费——应交增值税（销项税额）
贷：银行存款/应收账款
```

**销售退回（冲减成本）**：
```
借：库存商品
贷：主营业务成本
```

### 4. 销售折让

**发生销售折让**：
```
借：主营业务收入（折让金额）
    应交税费——应交增值税（销项税额）
贷：银行存款/应收账款
```

### 5. 销售费用

**发生销售费用**：
```
借：销售费用
贷：银行存款/库存现金/应付账款
```

**常见销售费用**：
```
借：销售费用——广告费
    销售费用——运输费
    销售费用——展览费
    销售费用——销售人员工资
贷：银行存款/应付职工薪酬等
```

### 6. 应收账款

**确认应收账款**：
```
借：应收账款
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

**收回应收账款**：
```
借：银行存款
贷：应收账款
```

**现金折扣（收款方）**：
```
借：银行存款（实收金额）
    财务费用（折扣金额）
贷：应收账款（原金额）
```

**坏账准备**：
```
计提：
借：信用减值损失
贷：坏账准备

核销：
借：坏账准备
贷：应收账款

收回已核销：
借：应收账款
贷：坏账准备
借：银行存款
贷：应收账款
```

## 六、期间费用分录

### 1. 管理费用

**常见管理费用分录**：
```
借：管理费用——办公费
    管理费用——差旅费
    管理费用——业务招待费
    管理费用——管理人员工资
    管理费用——折旧费
贷：银行存款/应付职工薪酬/累计折旧等
```

### 2. 财务费用

**利息支出**：
```
借：财务费用——利息支出
贷：银行存款/应付利息
```

**利息收入**：
```
借：银行存款
贷：财务费用——利息收入
```

**手续费**：
```
借：财务费用——手续费
贷：银行存款
```

**汇兑损益**：
```
汇兑损失：
借：财务费用——汇兑损益
贷：银行存款/应收账款等

汇兑收益：
借：银行存款/应收账款等
贷：财务费用——汇兑损益
```

## 七、固定资产分录

### 1. 固定资产取得

**外购固定资产**：
```
借：固定资产（购买价款+相关税费+运输费等）
    应交税费——应交增值税（进项税额）
贷：银行存款/应付账款
```

**自行建造固定资产**：
```
归集成本：
借：在建工程
贷：原材料/应付职工薪酬/银行存款等

完工转入：
借：固定资产
贷：在建工程
```

### 2. 固定资产折旧

**计提折旧**：
```
借：制造费用（生产用）
    管理费用（管理用）
    销售费用（销售用）
贷：累计折旧
```

### 3. 固定资产处置

**转入清理**：
```
借：固定资产清理
    累计折旧
    固定资产减值准备
贷：固定资产
```

**发生清理费用**：
```
借：固定资产清理
贷：银行存款
```

**收到处置价款**：
```
借：银行存款
贷：固定资产清理
    应交税费——应交增值税（销项税额）
```

**结转处置损益**：
```
净收益：
借：固定资产清理
贷：资产处置损益

净损失：
借：资产处置损益
贷：固定资产清理
```

## 八、无形资产分录

### 1. 无形资产取得

**外购无形资产**：
```
借：无形资产
贷：银行存款
```

### 2. 无形资产摊销

**计提摊销**：
```
借：管理费用（管理用）
    制造费用（生产用）
贷：累计摊销
```

### 3. 无形资产处置

**出售无形资产**：
```
借：银行存款
    累计摊销
    无形资产减值准备
贷：无形资产
    应交税费——应交增值税（销项税额）
    资产处置损益（差额，或借方）
```

## 九、投资业务分录

### 1. 交易性金融资产

**取得**：
```
借：交易性金融资产——成本
    投资收益（交易费用）
    应收股利/应收利息（已宣告未发放）
贷：银行存款
```

**持有期间公允价值变动**：
```
公允价值上升：
借：交易性金融资产——公允价值变动
贷：公允价值变动损益

公允价值下降：
借：公允价值变动损益
贷：交易性金融资产——公允价值变动
```

**处置**：
```
借：银行存款
贷：交易性金融资产——成本
    交易性金融资产——公允价值变动（或借方）
    投资收益（差额，或借方）
```

### 2. 长期股权投资

**取得（同一控制）**：
```
借：长期股权投资（被投资方净资产份额）
贷：银行存款/股本等（支付对价）
    资本公积——股本溢价（差额，或借方）
```

**取得（非同一控制）**：
```
借：长期股权投资（支付对价公允价值）
    商誉（如有）
贷：银行存款/股本等
    营业外收入（差额，如有）
```

**成本法核算**：
```
宣告分派股利：
借：应收股利
贷：投资收益

收到股利：
借：银行存款
贷：应收股利
```

**权益法核算**：
```
被投资方实现净利润：
借：长期股权投资——损益调整
贷：投资收益

被投资方宣告分派股利：
借：应收股利
贷：长期股权投资——损益调整
```

## 十、负债业务分录

### 1. 短期借款

**借入**：
```
借：银行存款
贷：短期借款
```

**计提利息**：
```
借：财务费用
贷：应付利息
```

**偿还**：
```
借：短期借款
    应付利息
    财务费用（最后一期利息）
贷：银行存款
```

### 2. 应付职工薪酬

**计提工资**：
```
借：生产成本/制造费用/管理费用/销售费用
贷：应付职工薪酬——工资
```

**发放工资**：
```
借：应付职工薪酬——工资
贷：银行存款
    应交税费——应交个人所得税
    其他应付款——社保（个人）
```

### 3. 应交税费

**增值税销项税额**：
```
借：银行存款/应收账款
贷：主营业务收入
    应交税费——应交增值税（销项税额）
```

**增值税进项税额**：
```
借：原材料/库存商品等
    应交税费——应交增值税（进项税额）
贷：银行存款/应付账款
```

**缴纳增值税**：
```
借：应交税费——应交增值税（已交税金）
贷：银行存款
```

**计提企业所得税**：
```
借：所得税费用
贷：应交税费——应交企业所得税
```

**缴纳企业所得税**：
```
借：应交税费——应交企业所得税
贷：银行存款
```

## 十一、所有者权益分录

### 1. 实收资本

**收到投资者投入资本**：
```
借：银行存款/固定资产/无形资产等
贷：实收资本
    资本公积——资本溢价（如有溢价）
```

**资本公积转增资本**：
```
借：资本公积
贷：实收资本
```

### 2. 留存收益

**盈余公积转增资本**：
```
借：盈余公积
贷：实收资本
```

**宣告分派现金股利**：
```
借：利润分配——应付现金股利
贷：应付股利
```

**支付现金股利**：
```
借：应付股利
贷：银行存款
```

## 十二、期末结转分录

### 1. 收入结转

**结转收入**：
```
借：主营业务收入
    其他业务收入
    营业外收入
    投资收益
    公允价值变动损益（收益）
贷：本年利润
```

### 2. 费用结转

**结转费用**：
```
借：本年利润
贷：主营业务成本
    其他业务成本
    税金及附加
    销售费用
    管理费用
    财务费用
    资产减值损失
    营业外支出
```

### 3. 所得税

**计提所得税**：
```
借：所得税费用
贷：应交税费——应交企业所得税
```

**结转所得税**：
```
借：本年利润
贷：所得税费用
```

### 4. 利润结转

**年末结转利润**：
```
盈利：
借：本年利润
贷：利润分配——未分配利润

亏损：
借：利润分配——未分配利润
贷：本年利润
```

**提取盈余公积**：
```
借：利润分配——提取法定盈余公积
    利润分配——提取任意盈余公积
贷：盈余公积——法定盈余公积
    盈余公积——任意盈余公积
```

## 十三、常见错误及纠正

### 1. 借贷方向错误

**错误**：
```
借：银行存款
    主营业务收入
贷：应收账款
```

**正确**：
```
借：银行存款
贷：应收账款
```

### 2. 科目使用错误

**错误**：
```
借：管理费用（应为销售费用）
贷：银行存款
```

### 3. 金额错误

**错误**：
```
借：库存商品 10000
贷：银行存款 100000
```

**正确**：借贷金额必须相等

### 4. 缺少摘要

**建议**：每笔记账凭证都应有清晰摘要

---

## 总结

会计分录是会计核算的基础，掌握常用分录是财务人员的必备技能。关键要点：

1. **借贷平衡**：有借必有贷，借贷必相等
2. **科目准确**：正确使用会计科目
3. **摘要清晰**：经济业务表述清楚
4. **附件完整**：原始凭证齐全

**学习方法**：多做实务练习，理解经济业务实质，熟练掌握常用分录。
