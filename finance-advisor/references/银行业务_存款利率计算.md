# 银行业务：存款利率计算

## 学习时间
2026-03-19 08:49

## 核心知识点

### 1. 单利与复利

**单利计算公式：**
```
利息 = 本金 × 年利率 × 存期（年）
本息和 = 本金 + 利息
```

**复利计算公式：**
```
本息和 = 本金 × (1 + 年利率)^存期
```

### 2. 活期存款利率
- 当前央行基准：0.20%（2024年调整后）
- 银行可上浮：最高基准利率的1.5倍
- 计息方式：按季结息，本金不变

### 3. 定期存款利率参考（以某大型银行为例）
| 存期 | 利率范围 |
|------|----------|
| 3个月 | 1.15%-1.40% |
| 6个月 | 1.35%-1.65% |
| 1年 | 1.45%-1.80% |
| 2年 | 1.65%-2.10% |
| 3年 | 1.95%-2.60% |
| 5年 | 2.00%-2.80% |

### 4. 大额存单特点
- 起存金额：20万元起
- 利率优势：比普通定期高10-50个基点
- 流动性：可转让、可质押
- 风险等级：低风险（存款保险覆盖50万元）

### 5. 实际案例分析

**案例：10万元存3年定期**

方案A - 普通定期（利率2.30%）：
```
到期利息 = 100,000 × 2.30% × 3 = 6,900元
本息和 = 106,900元
```

方案B - 大额存单（利率2.60%）：
```
到期利息 = 100,000 × 2.60% × 3 = 7,800元
本息和 = 107,800元
差额 = +900元
```

## 客户建议要点

1. **资金规划**：根据流动性需求选择存期
2. **利率比较**：不同银行利率有差异，货比三家
3. **分段存储**：避免全部资金锁死，可分期存入
4. **关注调息**：央行政策变化会影响存款利率
5. **存款保险**：单家银行50万元以内保本保息
