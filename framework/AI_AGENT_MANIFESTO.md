## 📋 使用流程

### 1. 给任务
```
"分析今日股票持仓"
```

### 2. AI 思考
```
分析：需要获取持仓、价格、计算盈亏
计划：
  1. 获取持仓列表
  2. 获取实时价格
  3. 计算盈亏
  4. 生成报告
```

### 3. AI 写代码
```python
# step1: 获取持仓
positions = get_positions()

# step2: 获取价格
prices = get_prices(positions)

# step3: 计算盈亏
profit = calculate_profit(positions, prices)

# step4: 生成报告
generate_report(profit)
```

### 4. 执行
```bash
python3 task.py
```

### 5. 反馈
```
结果：总盈亏 -¥8,011
状态：成功
```

### 6. 完成
```
✅ 任务完成
```
