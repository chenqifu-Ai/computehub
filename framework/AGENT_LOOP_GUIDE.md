## 🔄 循环流程

```
Task (任务)
  ↓
Plan (规划)
  ↓
Code (写代码)
  ↓
Execute (执行)
  ↓
Feedback (反馈)
  ↓
[循环...]
```

---

## 📋 使用说明

### 1. 定义任务
```python
task = "分析今日股票持仓盈亏"
```

### 2. 创建循环
```python
loop = AgentLoop(task)
```

### 3. 执行
```python
results = loop.run()
```
