# AI 智能体框架

**Python 就是 AI 的手脚**

## 🚀 快速开始

```python
from framework.ai_agent import AIAgent

# 创建智能体
agent = AIAgent(name="投资助手")

# 执行任务
task = "分析今日股票持仓盈亏"
result = agent.run(task)

# 输出结果
print(result)
```
