## 第二条 服务概述

2.1 本合同项下，乙方为甲方提供 **AI 智能体安装部署服务**，包括但不限于：
- AI 智能体系统的安装、配置与调试
- 模型服务接入（包括 qwen3.6-35b 等大语言模型 API 配置）
- 智能体技能模块加载与定制
- 系统联调测试与验收

2.2 服务地点：甲方指定服务器或云端环境（具体以附件《部署环境清单》为准）

2.3 服务期限：自合同生效之日起至系统验收合格之日止

## 第三条 服务范围与内容

### 3.1 智能体系统架构

乙方为甲方部署以下 AI 智能体系统：

| 模块 | 说明 |
|------|------|
| **智能体核心引擎** | OpenClaw 智能体框架，支持多轮对话、任务执行、工具调用 |
| **模型路由系统** | 多模型智能路由（DeepSeek、Qwen 等），支持负载均衡与故障转移 |
| **技能模块** | 图片识别、网页搜索、天气查询、视频分析等标准化技能 |
| **专家智能体** | CEO 战略顾问、财务专家、法律顾问、营销专家等 7 大专家模块 |
| **记忆系统** | 每日记忆、主题记忆、长期记忆三级架构 |
| **心跳监控系统** | 定期系统巡检、异常预警、状态报告 |

### 3.2 模型服务接入

- **Primary 模型：** ollama-cloud-2/deepseek-v4-flash（主模型）
- **辅助模型：** qwen3.6-35b（图片识别、多模态任务）
- **NewAPI 模型：** qwen3.6-35b-common（极速响应 ~0.7s）
- **本地模型：** Ollama 本地部署 8+ 模型（视硬件条件）

### 3.3 部署环境要求

甲方需提供以下基础环境：

| 项目 | 最低要求 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Ubuntu 20.04+ / CentOS 7+ | Ubuntu 22.04 LTS |
| CPU | 4 核 | 8 核以上 |
| 内存 | 8 GB | 16 GB 以上 |
| 磁盘 | 50 GB 可用空间 | 100 GB 以上 |
| 网络 | 公网带宽 ≥ 10 Mbps | 带宽 ≥ 50 Mbps |
| Python | 3.10+ | 3.12 LTS |

## 第七条 验收标准

### 7.1 验收条件

系统满足以下全部条件视为验收合格：

- [ ] 系统可正常启动运行，无严重错误
- [ ] 智能体对话响应时间 ≤ 5 秒（简单问答）
- [ ] 图片识别功能可用（支持 JPG/PNG 格式）
- [ ] 技能模块调用正常（搜索、天气、系统监控等）
- [ ] 专家智能体可正常执行（7 大专家模块）
- [ ] 心跳监控系统运行正常
- [ ] 系统日志无致命错误

### 7.2 验收流程

1. 乙方完成部署后通知甲方验收
2. 甲方在 5 个工作日内组织验收测试
3. 验收通过后双方签署《验收报告》
4. 如验收不通过，乙方应在 10 个工作日内整改
