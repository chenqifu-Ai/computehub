OpenClaw TUI界面含义
==================================================

┌─ 连接状态
│  ├─ connected - 连接正常
│  ├─ connecting - 正在连接
│  ├─ disconnected - 断开连接
│  ├─ reconnecting - 重新连接中
│  ├─ auth_failed - 认证失败
│
┌─ 运行状态
│  ├─ running - 运行中
│  ├─ starting - 启动中
│  ├─ stopped - 已停止
│  ├─ error - 错误状态
│  ├─ idle - 空闲状态
│
┌─ 时间显示
│  ├─ • 18s - 秒级运行时间
│  ├─ • 2m 30s - 分钟级时间
│  ├─ • 1h 5m - 小时级时间
│  ├─ uptime - 总运行时间
│
┌─ 动画指示
│  ├─ ⠹ - 旋转动画(处理中)
│  ├─ ✓ - 完成标记
│  ├─ ✗ - 错误标记
│  ├─ ⏳ - 等待标记
│  ├─ ⚠️ - 警告标记
│
┌─ 资源状态
│  ├─ memory: 45% - 内存使用率
│  ├─ cpu: 12% - CPU使用率
│  ├─ network: 2.3M/s - 网络流量
│  ├─ sessions: 3 - 活跃会话数
│  ├─ queue: 2 - 任务队列
│
┌─ 操作状态
│  ├─ processing - 处理中
│  ├─ waiting - 等待输入
│  ├─ completed - 已完成
│  ├─ failed - 操作失败
│  ├─ cancelled - 已取消
│
