## 🖼️ HTML图片替换代码

### 替换示例
```html
<!-- 替换前 -->
<div class="image-placeholder">
    📊 全球算力分布图
    <br><small>(可视化展示全球AI算力需求热点)</small>
</div>

<!-- 替换后 -->
<img src="images/global_compute_map.png" alt="全球AI算力分布热力图" style="width: 100%; height: 250px; object-fit: cover; border-radius: 15px;">
```

### 完整图片替换指南

1. **创建images文件夹**
   ```bash
   mkdir images
   ```

2. **放入图片文件**
   - 将所有图片放入images文件夹
   - 确保文件名符合命名规范

3. **替换所有占位符**
   - 搜索所有`image-placeholder`div
   - 替换为实际的img标签
   - 添加适当的alt文本


### 图片优化建议
1. 使用WebP格式获得更好压缩
2. 添加懒加载属性`loading="lazy"`
3. 为重要图片添加width和height属性
4. 使用CDN加速图片加载
