# 🎮 量子竞技场 - 四人游戏服务器原型

## 📋 项目概述
这是一个基于Go和WebSocket的四人实时对战游戏服务器原型，支持2v2或自由对战模式。

## 🚀 快速开始

### 环境要求
- Go 1.18+
- Node.js (可选，用于前端)

### 安装运行
```bash
# 1. 进入项目目录
cd /root/.openclaw/workspace/ai_agent/code/game_server/prototype

# 2. 安装依赖
go mod init game-server
go mod tidy

# 3. 启动服务器
go run main.go

# 4. 打开浏览器访问
# http://localhost:8080
```

### Docker运行
```bash
# 构建镜像
docker build -t quantum-arena .

# 运行容器
docker run -p 8080:8080 quantum-arena
```

## 📡 API接口

### WebSocket消息格式
```json
{
  "type": "join|leave|action|chat",
  "player_id": "string",
  "player_name": "string", 
  "player_class": "string",
  "room_id": "string",
  "action": "string",
  "message": "string"
}
```

### HTTP API
- `GET /api/status` - 服务器状态
- `GET /` - 游戏客户端界面

## 🐛 问题排查

### 常见问题
1. **WebSocket连接失败**
   - 检查服务器是否正常运行
   - 确认端口8080未被占用

2. **玩家无法加入房间**
   - 检查房间管理逻辑
   - 验证玩家数据格式

3. **消息不同步**
   - 检查状态同步机制
   - 验证网络通信稳定性
