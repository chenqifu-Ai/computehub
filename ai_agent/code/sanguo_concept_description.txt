# 《烽火三国：乱世棋局》概念效果图描述

三国题材3D策略战棋游戏概念图

## 视觉元素
1. **3D六边形战场**: 立体六边形网格地形，包含平原、丘陵、河流等地形
2. **三大势力部队**: 
   - 魏国(蓝色): 重装步兵阵列，曹操旗帜
   - 蜀国(绿色): 轻骑兵冲锋，刘备旗帜  
   - 吴国(红色): 弓箭手方阵，孙权旗帜
3. **技能特效**:
   - 诸葛亮火攻: 火焰粒子效果
   - 关羽青龙斩: 青色刀气轨迹
   - 赵云龙胆枪: 银色旋风特效
4. **历史氛围**: 古风战场，烽火台，战旗飘扬，古代兵器

## 艺术风格
- **风格**: 写实偏卡通渲染
- **色彩**: 三国阵营色系(魏蓝、蜀绿、吴红)
- **光影**: 动态光照，战场烟雾效果
- **视角**: 45度俯视角，展现全局战场
