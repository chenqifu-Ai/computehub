## 第一条 合作目标

1.1 各方一致同意，本协议的合作目标是：**在项目公司依法设立后，由乙方向项目公司推荐甲方作为全过程咨询服务供应商，并协调推动项目公司与甲方签订正式的全过程咨询服务合同**（以下简称"目标合同"）。

1.2 "全过程咨询服务"的范围包括但不限于：
（1）项目前期策划与可行性研究咨询；
（2）项目投融资结构设计与融资顾问服务；
（3）项目建设期全过程管理咨询；
（4）项目运营期管理咨询；
（5）产业招商及资源对接服务；
（6）其他与项目相关的专项咨询服务。

1.3 目标合同的具体内容、服务期限、服务费用等，由甲方与项目公司在目标合同签订时另行协商确定，本协议不构成对目标合同具体条款的承诺。

## 第四条 丙方的权利与义务（保证金监管）

4.1 丙方作为独立的保证金监管方，负责履约保证金的收取、保管和按本协议约定退还。

4.4 丙方有权收取保证金监管服务费，具体金额及支付方式由甲丙双方另行约定。

## 第五条 保证金的处置

### 5.1 保证金的退还条件

出现下列情形之一时，甲方有权要求丙方在 **___ 个工作日内** 将保证金全额无息退还至甲方指定账户：

（1）**本协议签署后满 ___ 个月**，项目公司仍未依法设立，且非因甲方原因造成的；

（2）**项目公司设立后满 ___ 个月**，甲方与项目公司仍未签订目标合同，且非因甲方原因造成的；

（3）本项目因政策调整、招标取消或其他不可归责于各方的原因而终止或无法继续推进的；

（4）乙方违反本协议第三条约定的义务，导致甲方无法获得公平的签约机会；

（5）经各方书面同意终止本协议的。

### 5.2 保证金的转化

甲方与项目公司成功签订目标合同后 **___ 个工作日内**，丙方应将保证金转入甲方指定账户，转为甲方支付目标合同项下的履约保证金或服务预付款（具体以目标合同约定为准）。

### 5.3 保证金的扣除

有下列情形之一的，乙方有权要求丙方将保证金作为违约金不予退还（甲丙双方配合执行）：

（1）甲方明确书面放弃与项目公司签订目标合同；
（2）因甲方资质、能力等自身原因导致目标合同无法签订；
（3）甲方提供虚假资质材料，经核实确认的。

### 5.4 保证金操作流程

保证金的退还、转化或扣除，均须由甲方和乙方共同向丙方出具书面通知（双方签字盖章），丙方在收到通知后 **___ 个工作日内** 执行相应操作。如甲乙双方就保证金处置产生分歧，丙方暂停执行，待各方协商一致或司法机关出具生效法律文书后执行。

## 📝 合同使用说明

**协议性质定位**：
本合同的法律性质是"**标前合作框架协议**"（或称"合作意向协议"），核心逻辑是：

1. **A公司（甲方）**支付200万保证金给**B公司（丙方）**→ 显示诚意和能力
2. **中标方（乙方）**承诺推动项目公司与A公司签咨询服务合同
3. **如果没签成** → 200万全额退还

**需填写的空白项（建议）**：

| 项目 | 建议填写 |
|------|---------|
| 甲方 | 你的公司全称 |
| 丙方 | 指定收款的公司B全称 |
| 保证金支付期限 | 协议签署后 5-7 个工作日 |
| 乙方推荐期限 | 项目公司设立后 15 日内 |
| 退还期限（未设项目公司） | 签署后 6 个月 |
| 退还期限（签不了合同） | 项目公司设立后 3 个月 |
| 保密期限 | 3 年 |
| 管辖法院 | 建议填甲方所在地法院（或丙方所在地） |
| 丙方监管费 | 建议收很少或象征性（0.5%-1%） |

**风险提示**：
- 200万是**履约保证金**不是"服务费"也不要写成"定金"（避免适用定金罚则）
- 退还条件一定要写清楚：**非因甲方原因**未签成→全退
- 丙方（B公司）必须是独立可靠的第三方，不能是乙方关联方
- 如果乙方最终没有履约诚意，仅靠这份协议追索可能力量有限

**后续衔接**：
- 成功签约后，200万转为目标合同的履约保证金或预付款
- 这份协议签署后，建议甲方同步准备正式的全过程咨询服务方案
