# 中医药建模比赛数据收集框架

## 数据结构

### 1. 比赛基本信息
- 比赛名称
- 主办单位
- 年份
- 级别（国家级/省级/校级）
- 参赛队伍数量

### 2. 获奖项目信息
- 获奖等级（一等奖/二等奖/三等奖）
- 获奖团队
- 所属院校
- 指导老师
- 项目名称

### 3. 技术实现方案
- 使用算法
- 数据处理方法
- 模型架构
- 创新点
- 技术难点

### 4. 实际应用案例
- 应用领域
- 实际效果
- 推广应用情况
- 经济效益
- 社会效益

### 5. AI/机器学习应用
- AI技术类型
- 机器学习算法
- 深度学习模型
- 自然语言处理
- 计算机视觉

## 数据来源
1. 各中医药大学官网
2. 数学建模竞赛官网
3. 学术论文数据库
4. 教育部门网站
5. 相关新闻报道

## 输出格式
- CSV格式数据表
- JSON格式结构化数据
- Markdown报告
- Excel表格
- PDF文档
