# 数字营销漏斗模型

## 核心概念

数字营销漏斗（Digital Marketing Funnel）描述用户从认知到转化的完整旅程。

## 漏斗四层

```
┌─────────────────────┐
│     认知 (AWARENESS)     │  ← 最上层：曝光、触达
├─────────────────────┤
│     兴趣 (INTEREST)      │  ← 了解、浏览
├─────────────────────┤
│     意向 (INTENT)        │  ← 加入购物车、询价
├─────────────────────┤
│     转化 (CONVERSION)     │  ← 下单、付费
└─────────────────────┘
```

## 各层常用渠道

| 层级 | 常用渠道 | KPI |
|------|---------|-----|
| 认知 | 信息流广告、品牌广告、社媒曝光 | 曝光量、CPM |
| 兴趣 | 内容营销、SEO、短视频 | 点击率、停留时长 |
| 意向 | 再营销广告、邮件营销、直播 | 加购率、咨询量 |
| 转化 | 促销活动、限时优惠、落地页优化 | 转化率、ROI |

## 关键指标

### AOV（平均订单价值）
```
AOV = 总收入 ÷ 订单数
```

### CAC（用户获取成本）
```
CAC = 营销总投入 ÷ 新增用户数
```

### LTV（用户终身价值）
```
LTV = 平均购买频次 × 客单价 × 用户生命周期
```

**健康比例**：LTV ≥ 3 × CAC

## 漏斗优化思路

1. **逐层分析流失率**
2. **识别最大流失环节**
3. **针对性优化**：
   - 认知层低 → 加大投放、优化素材
   - 兴趣层低 → 优化落地页、内容质量
   - 转化层低 → 简化流程、增加激励

## 实战案例

某电商发现：访问→加购转化率正常，但加购→下单流失60%。

**排查**：购物车页面加载慢、运费显示晚、支付方式少

**优化**：预加载支付页、前置运费说明、增加微信支付

**结果**：加购→下单转化率从40%提升至65%
