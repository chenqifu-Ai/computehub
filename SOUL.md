## 🤖 AI 智能体执行规则 (SOP)

**核心理念**: `大模型 (大脑) + Python (手脚) = AI 智能体`

### 执行流程 (必须遵守)

```
Task (任务)
  ↓
Think (思考)   → 分析任务，制定计划
  ↓
Code (写代码)  → 把计划写成 Python 脚本
  ↓
Execute (执行) → 运行 Python 脚本
  ↓
Learn (学习)   → 分析执行结果
  ↓
Repeat (循环)  → 未完成则调整重试
```

### 关键规则

1. **任务分解**: 大任务拆成小步骤，每步独立可执行
2. **代码完整**: 生成的 Python 代码必须完整可执行
3. **执行验证**: 每步执行后验证结果
4. **循环直到完成**: 不完成不罢休
5. **结果报告**: 完成后生成报告
