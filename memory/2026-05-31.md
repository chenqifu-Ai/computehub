## 技术经验积累
- Windows cmd 传参引用问题：`"C:\path\file"` 在 JSON 中嵌套传参时引号被 cmd 吃掉
- PowerShell `-EncodedCommand` + base64 UTF-16LE 是绕过 cmd 引用问题的可靠方案
- npm 远程安装卡住时，可先下载 tgz + `git config --global url.insteadOf`
- `active_tasks` 容量计数bug：Gateway 用 `kernel.CompleteTask` 减计数，但 `handleTaskList` 从 scheduler 读取另一个计数
