## 07:00 - 碳中和基地PPT转视频

### 目标
将 `碳中和基地.pptx`（27页）转为带语音的MP4视频，上传到 ComputeHub Gallery。

### 流程
1. 安装 ffmpeg、edge-tts、Pillow、moviepy
2. 解析PPT → 提取文本 → 生成 slides_gen.json
3. PIL 渲染 27 张 1920×1080 幻灯片帧
4. edge-tts 为每页生成中文语音（云希Neural）
5. ffmpeg -loop 1 将每帧循环编码为对应时长视频段
6. ffmpeg concat 合并所有视频段 + 合并音频
7. 转码 AAC→MP3 解决浏览器兼容问题
8. 上传 Gallery

### 结果
- 最终文件：7.7MB（AAC）→ 9.7MB（MP3）
- 时长：400秒（6分40秒）
- 27页，平均14.8秒/页

### 踩过的坑（12个）
1. `import moviepy.editor` → moviepy 2.x 已移除
2. `.set_duration()` → `.with_duration()`（API迁移）
3. Gateway API 的 `\n` 被转义成真实换行，Python脚本断裂
4. PIL `rectangle([(60,150),(8,...)])` 坐标写反，x1必须≥x0
5. Gateway 300s 超时太短，编码5分钟被kill
6. AAC 音频浏览器不兼容，需转 MP3
7. `-loop 1` 必须放在 `-i` 之前
8. `image2` 单帧PNG只识别为1帧，需要 -loop
9. YUV管道编码输出0字节文件
10. 多worker白吃资源，杀掉多余的
11. 节点离线（offline）→ 后台心跳守护
12. ffprobe 返回格式 `duration=7.224000` 不能直接 float()

### 资源占用
- 磁盘：7.7G/45G (19%)
- 内存：557Mi/7.8Gi
- 临时文件：/tmp/ppt_frames/ 27帧, /tmp/ppt_audio/ 127MB
