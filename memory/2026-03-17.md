## 使用方法

### 检查状态
```bash
python3 /data/data/com.termux/files/usr/lib/node_modules/openclaw/skills/web-automation/scripts/web_tool.py status
```

### 搜索
```bash
python3 web_tool.py search "关键词" 10
```

### 打开网页
```bash
python3 web_tool.py open "https://example.com"
```

### 截图
```bash
python3 web_tool.py screenshot "https://example.com" /tmp/output.png
```

## 前置要求

