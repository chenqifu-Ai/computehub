## Windows 远程登录方法

### 1. 远程桌面 (RDP)
- 端口：3389 (TCP)
- 需要：Windows Pro/Enterprise 版本
- 启用：设置 → 系统 → 远程桌面 → 启用
- 安全：建议启用 NLA

### 2. WinRM / PowerShell Remoting
- 端口：5985 (HTTP), 5986 (HTTPS)
- 认证：Kerberos 或 NTLM
- 加密：始终加密
- 需要：管理员权限

### 3. SMB / PsExec
- 端口：445 (SMB)
- 需要：管理员账户 + 密码/哈希
- 方法：通过 ADMIN$ 共享上传执行
- 风险：会被杀毒软件检测

### 4. SSH (Windows 10+)
- 端口：22
- 需要：安装 OpenSSH Server

### 5. 其他方法
- DCOM 远程执行
- WMI 远程管理
- 计划任务
- 服务创建

### 目标状态
- RDP (3389)：关闭
- WinRM：关闭
- SSH：关闭
- SMB：开放但被防火墙阻止
- Ollama (11434)：开放 ✓
