# 设备连接管理

## 连接经验 (2026-04-04)

### 问题场景
小米平板OpenClaw TUI客户端连接问题

### 问题分析
1. **Token信息分散**: 凭证信息存储不集中
2. **版本落后**: 客户端版本与服务器不匹配
3. **网络隔离**: 192.168.1.x ↔ 192.168.2.x网络隔离
4. **Android权限**: Termux环境需要特殊网络权限处理

### 解决方案
16分钟快速诊断，提供正确token，恢复连接

## 核心经验总结

- 避免凭证信息分散存储
- 建立统一凭证管理系统
- 定期更新和验证Token有效性

- 华为设备标识: HWI-AL00 - Huawei/generic_a15/generic_a15:9/PPR1.180610.011/root202005281627:user/test-keys
- 即使IP变化，通过设备指纹也能准确识别

### 3. 跨网段访问
- 处理不同子网设备连接
- 配置正确的网络路由
- 处理NAT和防火墙规则

### 4. Android适配
- Termux环境特殊权限处理
- 网络配置优化
- 服务启动脚本适配
