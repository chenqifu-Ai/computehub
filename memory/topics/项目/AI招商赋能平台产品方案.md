# AI 招商赋能平台 — 产品方案 & 开发需求说明书

基于原有的"AI智能招商演示主机（硬件盒子）"升级为**AI招商赋能平台**（SaaS + 硬件 + AI），面向品牌方、连锁企业、会销公司提供一站式智能招商解决方案。

## 四、核心功能模块

### 4.1 AI 引擎模块（核心）
| 功能 | 描述 | 技术要点 |
|------|------|----------|
| 离线语音识别 | 无网环境下的语音指令识别 | 本地模型 + 关键词唤醒 |
| 在线语音识别 | 联网环境下自然语言理解 | 云端大模型 API |
| 招商全流程自动讲解 | AI 按预设流程自动讲解全案 | 语音合成(TTS) + 页面联动 |
| 智能问答 | 回答政策、费用、利润、扶持、回本等问题 | RAG + 知识库 |
| AI 话术生成 | 自动生成/优化招商话术 | LLM + 行业模板 |
| 多轮对话 | 支持追问、澄清、深入问答 | 对话管理(Dialog Management) |

### 4.2 内容管理模块
| 功能 | 描述 |
|------|------|
| 招商全案编辑器 | 拖拽式全案编辑，支持图文/视频/数据图表 |
| 素材库管理 | 品牌、产品、模式、投资、盈利、扶持、案例、视频统一管理 |
| 版本管理 | 多版本控制，历史可追溯 |
| 远程统一更新 | 总部一键发布，全国渠道同步 |
| 内容加密 | 防拷贝、防篡改、DRM 保护 |

### 4.3 语音控制大屏
| 功能 | 指令示例 |
|------|----------|
| 打开全案 | "打开招商全案" |
| 播放视频 | "播放品牌宣传片" |
| 翻页 | "翻到加盟政策" |
| 展示数据 | "展示利润分析" |
| 放大/缩小 | "放大产品图" |
| 返回 | "返回首页" |

### 4.4 数据分析模块（V2 核心增值功能）
| 功能 | 描述 |
|------|------|
| 高频问题统计 | 记录加盟商最常问的问题 |
| 停留页面分析 | 统计每页停留时间，识别关注点 |
| 转化漏斗 | 从展示→咨询→意向→签约 全链路追踪 |
| 话术效果评估 | 分析不同话术版本的转化效果 |
| 行业热力图 | 跨品牌/行业数据汇总分析 |

### 4.5 渠道与权限管理
| 功能 | 描述 |
|------|------|
| 多级权限 | 总部 → 区域代理 → 门店 分级管理 |
| 渠道分发 | 每个渠道独立内容/品牌配置 |
| 激活码管理 | 硬件激活码批量生成、绑定 |
| 使用统计 | 各渠道/设备使用情况报表 |

## 五、技术栈建议

| 层级 | 技术选型 |
|------|----------|
| 前端（管理后台） | React/Vue3 + TypeScript + Ant Design |
| 前端（大屏端） | 定制 Electron/Web 应用 |
| 移动端 | 微信小程序 + H5 |
| 后端 | Go（核心服务）/ Python（AI 服务） |
| 数据库 | PostgreSQL + Redis |
| AI 服务 | 语音识别：Whisper（本地）/ 讯飞（云端） |
| | 大模型：通义千问 / DeepSeek |
| | TTS：Azure TTS / 火山引擎 |
| 硬件端 | Android 系统（工控主板）/ Linux |
| 部署 | Docker + K8s（云服务）/ 本地私有化部署 |
| 存储 | 阿里云 OSS / 腾讯云 COS（内容分发） |
