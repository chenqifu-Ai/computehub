## 📝 详细记录

### E-001 | 改IP只改缓存没改主配置
- **日期**: 2026-04-30 19:18
- **严重度**: 🟡重要
- **场景**: 老大要求把 vllm-proxy 的 IP 从 192.168.2.54 改为 192.168.1.17
- **根因**: 
  1. 没有先读取主配置确认结构
  2. 只改了 `/root/.openclaw/agents/main/agent/models.json`（本地缓存）
  3. 没改 `/root/.openclaw/openclaw.json`（OpenClaw 主配置）
  4. 跳过 MCP-STD-001 标准流程
- **影响**: 
  - 主配置一直是旧IP，服务不可用
  - 老大发现后才补救，浪费约 10 分钟
  - 暴露标准文档存在但不会触发的系统性问题
- **修复**: 补改 openclaw.json 主配置，同步 models.json 和 auth-profiles
- **教训**: 
  - 改配置前必须读主配置确认文件关系
  - 任何配置修改必须走 MCP-STD-001 4步流程
  - 修改后必须检查所有相关文件
- **标准化**: → **STD-SOP-001**（标准匹配检查机制）
- **重复次数**: 1
- **状态**: ✅ 已标准化
