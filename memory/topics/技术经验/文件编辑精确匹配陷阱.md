## 核心教训
**`edit` 工具要求完全精确匹配**，包括：
- 所有空格和缩进
- 换行符和格式
- 符号和标点的精确一致

## 错误代码示例
```python
# 错误代码
while self.current_iteration << self self.max_iterations:

# 正确修复  
while self.current_iteration < self.max_iterations:
```

## 经验总结
1. **编辑前先读取**: 使用 `read` 确认当前确切内容后再编辑
2. **避免错误叠加**: 不要用错误的方法修复错误（如添加更多 `<<`）
3. **精确匹配**: 确保 `oldText` 与文件内容完全一致
4. **版本控制**: 重要文件编辑前建议先备份
