# STD-003 邮件发送标准

**状态**: ✅ 定稿（2026-05-21）
**适用范围**: 所有需要发送邮件的场景

---

## 核心原则

1. **唯一脚本**: 所有邮件发送统一调用 `scripts/send_email.py`
2. **禁止自建**: 任何场景不得自己写 smtplib 代码发邮件，必须用标准脚本
3. **授权码固定**: 来源 `config/email.conf`，不询问、不修改
4. **自动切换**: QQ优先→163备选，无需手动指定

## 标准用法

```bash
# 用法
python3 scripts/send_email.py <收件人邮箱> <邮件主题> <内容文件路径>

# 示例：发送文本报告
python3 scripts/send_email.py 19525456@qq.com "日报" /tmp/report.md

# 示例：发送附件（.pdf .zip .jpg 等自动作为附件）
python3 scripts/send_email.py 19525456@qq.com "合同文件" contract.pdf
```

## 支持的文件类型

| 扩展名 | 处理方式 |
|--------|----------|
| .md, .txt, .log | HTML正文（按pre格式显示） |
| .jpg, .png, .gif | 附件 |
| .pdf, .zip | 附件 |
| .docx | 附件（直接传，正文写说明） |
| 其他 | 按文本处理 |

```

  ├─ ✅ 成功 → 结束
```
