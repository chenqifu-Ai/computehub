### 踩坑记录
1. **PATH 不生效**: 安装程序加了 PrependPath=1 但 PATH 没更新 → 验证必须用完整路径
2. **setx 语法坑**: `setx PATH "..." /M` 中 `/M` 放位置不对会报 "Invalid syntax"
3. **引号三层嵌套**: Python 字符串 + JSON + Windows cmd 三层引号导致 Shell 语法错误
4. **msiexec 可能挂死**: MSI 安装任务可能长时间无响应
5. **LTSC 2019 不兼容 Node 20+**: MSI exit 1620，EXE 也不兼容 → ZIP 二进制版是王道
6. **FFmpeg 无安装器**: 下载单 exe 放 Program Files 就够，补 PATH 搞定
