### API端点
- GET  /api/market/stocks - 股票列表
- GET  /api/market/quote - 行情数据
- GET  /api/market/hot - 热门股票
- GET  /api/market/summary - 市场概况
- POST /api/auth/login - 登录
- GET  /api/account/info - 账户信息
- POST /api/trading/buy - 买入
- POST /api/trading/sell - 卖出
