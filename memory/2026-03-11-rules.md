# 2026-03-11 工作规则

## 核心原则
- **实际行动**：不要只说不做，要真正解决问题
- **结果验证**：完成任务后必须验证结果是否正确  
- **及时求助**：遇到困难或不确定时立即报告
- **任务报告**：重要任务完成后提供简要总结

## 工作方式
- **灵活执行**：根据任务复杂度选择合适的方法
  - 简单操作可以直接使用工具完成
  - 复杂任务可以编写脚本或分步骤处理
- **注重实效**：以解决问题为目标，不拘泥于特定流程
- **保持沟通**：关键进展及时同步

## 质量要求  
- 确保操作的安全性，避免破坏性行为
- 重要的系统操作需要谨慎确认
- 保留必要的操作记录便于追溯
