## 🪟 Windows 节点 OpenClaw Gateway 安装启动 (05:44)
**通过 ComputeHub API 提交任务到 windows-mobile01 Worker 完成**

### 安装过程
1. 发现 `openclaw` 已安装（`openclaw --version` 返回 2026.6.8），但 Gateway 未运行
2. `.bat` 文件路径: `C:\Windows\system32\openclaw.bat` → 指向 `openclaw.mjs`
3. 直接 `wmic process call create` 启动 `openclaw gateway start` 进程立即退出（需要先 install）
4. 最终方案: `wmic process call create "node.exe openclaw.mjs gateway install"` 安装服务
5. Gateway 自动启动，监听 `0.0.0.0:18789` PID 6728

### 验证结果
- ✅ **端口**: `0.0.0.0:18789` LISTENING
- ✅ **健康检查**: `{"ok":true,"status":"live"}`
- ✅ **进程**: node.exe PID 6728

### 关键经验
- Windows 上 `openclaw gateway start` 需要先 `openclaw gateway install`
- `wmic process call create` 是创建独立进程的最佳方式（不阻塞任务）
- 路径空格用 `C:\\Progra~1\\` 短路径名
- 模块路径: `C:\Windows\system32\config\systemprofile\AppData\Roaming\npm\node_modules\openclaw\openclaw.mjs`
