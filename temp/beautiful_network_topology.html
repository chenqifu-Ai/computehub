🛠️ 修复方案

在华为手机上执行以下命令：

# 1. 查看当前配置
cat ~/.openclaw/config/gateway.conf

# 2. 修改bind地址为0.0.0.0
sed -i 's/127\.0\.0\.1/0.0.0.0/g' ~/.openclaw/config/gateway.conf
sed -i 's/localhost/0.0.0.0/g' ~/.openclaw/config/gateway.conf

# 3. 重启OpenClaw服务
pkill -f openclaw
cd ~/.openclaw && nohup openclaw gateway start > gateway.log 2>&1 &

# 4. 验证修复
curl http://192.168.2.156:18789/status
