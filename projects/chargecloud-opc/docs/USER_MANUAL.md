# 充电云 - 用户操作手册

## 第一章 系统概述

### 1.1 产品简介
充电云是专业的充电桩运营管理SaaS平台，为充电站运营商提供一站式管理解决方案，包括充电站管理、充电桩管理、订单管理、用户管理、财务结算等功能。

### 1.2 功能模块
- 数据看板：实时展示运营数据
- 充电站管理：站点信息维护
- 充电桩管理：设备状态监控
- 订单管理：充电订单处理
- 用户管理：用户信息维护
- 财务管理：收入、提现、发票
- 系统设置：账户、权限配置

## 第二章 快速开始

### 2.1 注册登录
1. 访问充电云官网
2. 点击"免费试用"
3. 输入手机号，获取验证码
4. 设置密码，完成注册
5. 使用手机号密码登录

### 2.2 首次配置
#### 添加充电站
1. 进入"充电站管理"
2. 点击"添加站点"
3. 填写站点信息：
   - 站点名称
   - 所在城市
   - 详细地址
   - 经纬度（可地图选点）
   - 停车费设置
4. 保存

#### 添加充电桩
1. 进入"充电桩管理"
2. 点击"添加充电桩"
3. 选择所属站点
4. 填写充电桩信息：
   - 充电桩编号
   - 充电桩名称
   - 类型（快充/慢充）
   - 功率
   - 电价
   - 服务费
5. 保存

#### 对接充电桩硬件
1. 进入"系统设置 > 硬件对接"
2. 选择充电桩品牌
3. 输入充电桩SN码
4. 测试连接
5. 完成对接

## 第三章 功能详解

### 3.1 数据看板
进入首页可看到：
- 今日收入
- 今日订单数
- 今日充电量
- 设备在线率
- 最近订单列表
- 数据趋势图

### 3.2 充电站管理

#### 3.2.1 站点列表
展示所有充电站信息，支持：
- 搜索筛选
- 状态切换
- 编辑删除

#### 3.2.2 新增站点
必填信息：
- 站点名称（必填）
- 所在城市（必填）
- 详细地址（必填）
- 经纬度（选填，可地图选点）
- 停车费（选填）

#### 3.2.3 站点详情
点击站点可查看：
- 基本信息
- 充电桩列表
- 订单统计
- 收入分析

### 3.3 充电桩管理

#### 3.3.1 充电桩列表
展示所有充电桩信息：
- 充电桩编号
- 所属站点
- 类型（快充/慢充）
- 功率
- 状态（空闲/充电中/离线/故障）
- 电价

#### 3.3.2 充电桩状态
- 🟢 空闲：可正常使用
- 🟡 充电中：正在充电
- 🔴 离线：设备断网
- ⚫ 故障：设备异常

#### 3.3.3 远程控制
支持远程操作：
- 启动充电
- 停止充电
- 重启设备
- 设置价格

### 3.4 订单管理

#### 3.4.1 订单列表
展示所有充电订单：
- 订单号
- 用户信息
- 充电桩
- 充电量
- 金额
- 状态
- 创建时间

#### 3.4.2 订单详情
点击订单可查看：
- 订单基本信息
- 充电过程数据
- 支付信息
- 操作日志

#### 3.4.3 订单处理
支持操作：
- 查看详情
- 退款
- 导出

### 3.5 用户管理

#### 3.5.1 用户列表
展示所有注册用户：
- 手机号（脱敏）
- 昵称
- 余额
- 等级
- 注册时间

#### 3.5.2 用户详情
点击用户可查看：
- 基本信息
- 充电记录
- 消费统计

### 3.6 财务管理

#### 3.6.1 收入统计
- 今日收入
- 本月收入
- 收入明细
- 收入趋势

#### 3.6.2 提现管理
1. 绑定银行卡

## 第六章 常见问题

### 6.1 充电桩无法连接
1. 检查网络连接
2. 重启充电桩
3. 检查SN码是否正确
4. 联系技术支持

### 6.2 订单金额异常
1. 进入订单详情
2. 点击"修正金额"
3. 输入正确金额
4. 保存

### 6.3 用户无法充电
1. 确认充电桩状态为"空闲"
2. 确认用户余额充足
3. 检查小程序版本

### 6.4 数据同步延迟
正常情况下数据实时同步，如遇延迟：
1. 刷新页面
2. 清除缓存
3. 联系技术支持

## 第七章 安全规范

### 7.1 账户安全
- 设置强密码
- 定期更换密码
- 不透露密码给他人
- 发现异常及时修改密码

### 7.2 操作安全
- 重要操作二次确认
- 定期查看操作日志
- 异常操作及时处理

### 7.3 数据安全
- 定期导出备份
- 敏感数据加密存储
- 不在公共场所登录
