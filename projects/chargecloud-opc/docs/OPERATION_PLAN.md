### 2.2 激活策略

#### 新用户引导流程
```
注册 → 完善信息 → 创建站点 → 添加充电桩 → 试用功能 → 转化付费
```

#### 关键转化点
- 注册后24小时内：引导创建第一个站点
- 创建站点后：引导添加充电桩
- 添加桩后：展示核心功能价值
- 试用7天后：推送付费优惠

### 2.3 留存策略

#### 用户分层运营
| 层级 | 定义 | 运营策略 |
|------|------|---------|
| 高活跃用户 | 日活>10次 | VIP服务、专属优惠 |
| 中活跃用户 | 日活3-10次 | 定期关怀、功能推荐 |
| 低活跃用户 | 日活<3次 | 召回活动、问题调研 |
| 流失用户 | 30天未登录 | 电话回访、优惠召回 |

#### 用户生命周期运营
- **新手期（0-7天）**：引导使用、答疑解惑
- **成长期（7-30天）**：功能推荐、使用技巧
- **成熟期（30天+）**：深度运营、增值服务
- **流失期**：召回活动、优惠促销
