# 充电云SaaS平台 - 目标客户画像

## 一、客户分层

### 1. A类客户：小型运营商
- **规模**：1-5个充电站，10-50个充电桩
- **痛点**：缺乏管理系统，靠人工统计
- **预算**：3000-10000元/月
- **决策周期**：1-2周
- **数量**：市场占比40%

### 2. B类客户：中型运营商
- **规模**：5-20个充电站，50-200个充电桩
- **痛点**：管理效率低，数据分析弱
- **预算**：10000-30000元/月
- **决策周期**：2-4周
- **数量**：市场占比35%

### 3. C类客户：物业/停车场
- **规模**：单个站点，5-20个充电桩
- **痛点**：缺乏运营能力，收益不清晰
- **预算**：500-2000元/月
- **决策周期**：1周
- **数量**：市场占比25%

## 五、客户决策流程

### 小型运营商决策流程
```
认识需求 → 搜索方案 → 对比评估 → 试用体验 → 购买决策
    ↓           ↓           ↓           ↓           ↓
  痛点明确    找3-5家    比价格功能   免费试用    价格+服务
```

**关键触点**：
1. 搜索时：SEO/竞价排名
2. 对比时：官网案例、价格透明
3. 试用时：1对1客服指导
4. 决策时：优惠活动、服务承诺

### 物业/停车场决策流程
```
上级要求 → 找供应商 → 快速决策 → 采购
    ↓           ↓           ↓        ↓
  需要管理    要省心      要快     走流程
```

**关键触点**：
1. 需求：明确充电桩管理需求
2. 找供应商：行业推荐、采购平台
3. 决策：价格低、服务好、无风险

## 六、客户生命周期价值（LTV）

### A类客户（小型运营商）
- **月费**：500-1000元
- **留存时间**：2-3年
- **LTV**：12,000-36,000元
- **获客成本（CAC）**：500-1000元
- **LTV/CAC**：12-36倍

### B类客户（中型运营商）
- **月费**：2000-5000元
- **留存时间**：3-5年
- **LTV**：72,000-300,000元
- **获客成本（CAC）**：2000-5000元
- **LTV/CAC**：14-60倍

### C类客户（物业/停车场）
- **月费**：300-500元
- **留存时间**：1-2年
- **LTV**：3,600-12,000元
- **获客成本（CAC）**：300-500元
- **LTV/CAC**：7-24倍
