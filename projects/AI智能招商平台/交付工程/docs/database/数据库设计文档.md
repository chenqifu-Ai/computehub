## 1. 数据库概述

| 项目 | 内容 |
|------|------|
| 数据库类型 | MySQL 8.0 |
| 字符集 | utf8mb4 |
| 排序规则 | utf8mb4_unicode_ci |
| 存储引擎 | InnoDB |

### 1.1 ER关系图（核心）

```
用户表 ────< 角色表 >──── 权限表
   │
   └──< 设备表
          │
          └──< 内容版本表 >──< 内容发布记录
                │
                └──< 品牌/产品/模式/案例表
                │
```

## 2. 核心表结构

### 2.1 用户表 (sys_user)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password_hash | VARCHAR(255) | bcrypt哈希密码 |
| real_name | VARCHAR(50) | 真实姓名 |
| phone | VARCHAR(20) | 手机号 |
| email | VARCHAR(100) | 邮箱 |
| role_id | BIGINT UNSIGNED | 角色ID（外键） |
| status | TINYINT | 0=禁用 1=正常 |
| last_login_at | DATETIME | 最后登录时间 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- UNIQUE KEY uk_username (username)
- INDEX idx_phone (phone)
- INDEX idx_status (status)

### 2.2 设备表 (sys_device)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| device_id | VARCHAR(64) | 设备唯一标识 |
| device_name | VARCHAR(100) | 设备名称 |
| device_type | VARCHAR(20) | 类型：box/工控机/其他 |
| region | VARCHAR(50) | 所属区域 |
| city | VARCHAR(50) | 所在城市 |
| online_status | TINYINT | 0=离线 1=在线 |
| last_heartbeat_at | DATETIME | 最后心跳时间 |
| current_version | VARCHAR(20) | 当前版本 |
| target_version | VARCHAR(20) | 目标版本 |
| ip_address | VARCHAR(45) | 最后IP |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- UNIQUE KEY uk_device_id (device_id)
- INDEX idx_region (region)
- INDEX idx_online (online_status)
- INDEX idx_heartbeat (last_heartbeat_at)

### 2.3 品牌表 (biz_brand)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| name | VARCHAR(100) | 品牌名称 |
| slogan | VARCHAR(200) | 品牌口号 |
| logo_url | VARCHAR(500) | Logo地址 |
| intro | TEXT | 品牌介绍 |
| honor_list | JSON | 荣誉资质列表 |
| sort_order | INT | 排序 |
| status | TINYINT | 0=下架 1=上架 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- INDEX idx_status (status)

### 2.4 产品表 (biz_product)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| brand_id | BIGINT UNSIGNED | 品牌ID |
| name | VARCHAR(100) | 产品名称 |
| description | TEXT | 产品描述 |
| images | JSON | 产品图片列表 |
| specs | JSON | 产品规格参数 |
| price_range | VARCHAR(100) | 价格区间 |
| highlight | TEXT | 核心卖点 |
| sort_order | INT | 排序 |
| status | TINYINT | 0=下架 1=上架 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- INDEX idx_brand (brand_id)

### 2.5 问答库表 (biz_qa)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| category | VARCHAR(50) | 分类：政策/费用/利润/扶持/回本 |
| question | VARCHAR(500) | 问题 |
| answer | TEXT | 标准答案 |
| keywords | JSON | 关键词（用于模糊匹配） |
| audio_url | VARCHAR(500) | 语音答案地址 |
| use_count | INT | 使用次数 |
| status | TINYINT | 0=禁用 1=启用 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- INDEX idx_category (category)
- INDEX idx_status (status)

### 2.6 内容版本表 (biz_content_version)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| version | VARCHAR(20) | 版本号 |
| content_type | VARCHAR(50) | 内容类型 |
| changelog | TEXT | 更新说明 |
| status | TINYINT | 0=草稿 1=已发布 2=已回滚 |
| created_by | BIGINT UNSIGNED | 创建人 |
| published_at | DATETIME | 发布时间 |
| created_at | DATETIME | 创建时间 |

索引：
- PRIMARY KEY (id)
- UNIQUE KEY uk_version (version)
- INDEX idx_status (status)

### 2.7 设备版本绑定表 (sys_device_version)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| device_id | VARCHAR(64) | 设备ID |
| content_version | VARCHAR(20) | 内容版本 |
| app_version | VARCHAR(20) | App版本 |
| update_status | TINYINT | 0=未更新 1=更新中 2=已更新 |
| updated_at | DATETIME | 更新时间 |

索引：
- PRIMARY KEY (id)
- UNIQUE KEY uk_device_ver (device_id, content_version)

### 2.8 统计记录表 (stat_query_log)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| device_id | VARCHAR(64) | 设备ID |
| query_text | VARCHAR(500) | 查询文本 |
| query_result | TEXT | 查询结果摘要 |
| duration_ms | INT | 响应耗时(ms) |
| slide_id | BIGINT | 当前页面ID |
| created_at | DATETIME | 创建时间 |

索引：
- PRIMARY KEY (id)
- INDEX idx_device (device_id)
- INDEX idx_created (created_at)

### 2.9 页面访问统计 (stat_page_view)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT UNSIGNED | 主键 |
| device_id | VARCHAR(64) | 设备ID |
| slide_id | BIGINT | 页面ID |
| view_start_at | DATETIME | 开始时间 |
| view_end_at | DATETIME | 结束时间 |
| duration_sec | INT | 停留时长(秒) |

索引：
- PRIMARY KEY (id)
- INDEX idx_device (device_id)
- INDEX idx_slide (slide_id)

## 3. 表关系说明

```
sys_user (1) ──< (N) sys_user_device
sys_role (1) ──< (N) sys_user
sys_device (1) ──< (N) sys_device_version
sys_device (1) ──< (N) stat_query_log
sys_device (1) ──< (N) stat_page_view
biz_brand (1) ──< (N) biz_product
biz_content_version (1) ──< (N) sys_device_version
biz_qa (1) ──< (N) stat_query_log (通过query_text关联)
```

---

## 4. 数据加密存储方案

- 用户密码：bcrypt哈希
- 招商全案数据：AES-256-CBC加密存储
- API密钥：HMAC-SHA256签名
- 敏感字段（手机号/邮箱）：AES-128加密存储
