## 第一部分：终端用户使用手册

### 1.1 开机与连接

#### 1.1.1 硬件准备
1. 确认AI智能招商主机已通电
2. 通过WiFi或HDMI连接显示设备（电视/投影仪/会议屏）
3. 等待系统启动完成（约30秒）

#### 1.1.2 连接步骤
1. 主机通电后自动开机
2. 显示设备自动检测并切换信号源
3. 系统显示招商平台首页

### 1.2 语音控制操作

#### 1.2.1 基础语音指令

| 语音指令 | 功能 |
|----------|------|
| "打开招商全案" | 开始招商全案演示 |
| "播放品牌宣传片" | 播放品牌宣传视频 |
| "翻到加盟政策" | 跳转到加盟政策页面 |
| "展示利润分析" | 展示利润分析页面 |
| "放大产品图" | 放大当前产品图片 |
| "返回首页" | 返回招商全案首页 |
| "暂停" | 暂停自动讲解 |
| "继续" | 继续被暂停的讲解 |

#### 1.2.2 自动讲解模式
1. 说"打开招商全案"或"开始讲解"
2. 系统自动从头到尾演示全部内容
3. 可随时通过语音中断
4. 说"继续"可恢复讲解

#### 1.2.3 语音问答
直接说出问题即可，例如：
- "加盟费多少钱？"
- "有区域保护吗？"
- "多久可以回本？"
- "总部提供什么扶持？"

### 1.3 常见问题处理

#### 1.3.1 无法识别语音
- 确认麦克风未被遮挡
- 确认环境噪音不过大（<65dB）
- 确认普通话发音清晰

#### 1.3.2 无法连接显示设备
- 检查HDMI线是否插好
- 检查WiFi投屏网络是否正常
- 重启主机

## 第二部分：管理后台操作手册

### 2.1 登录系统

1. 打开浏览器，访问管理后台地址
2. 输入用户名和密码
3. 点击"登录"

### 2.2 发布新内容

1. 登录管理后台
2. 进入"内容管理" → "发布管理"
3. 点击"新建发布"
4. 选择发布类型（全案更新/单页更新）
5. 上传/编辑内容
6. 选择目标设备/区域
7. 点击"发布"

### 2.3 设备管理

1. 进入"设备管理" → "设备列表"
2. 可查看设备在线状态、版本信息
3. 点击设备可查看详情
4. 支持批量操作（强制更新、重启等）
