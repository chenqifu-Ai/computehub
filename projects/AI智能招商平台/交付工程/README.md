## 系统模块

1. **AI智能体模块** - 语音识别、自动讲解、语音问答、内容加密
2. **大屏交互控制模块** - 语音翻页、播放、暂停、放大、切换
3. **招商全案管理模块** - 品牌、产品、模式、投资、盈利、扶持、案例
4. **数据统计模块** - 高频问题、停留分析、转化率
5. **管理后台模块** - 内容发布、版本管理、设备管理、权限管理

## 技术栈

- **前端**: Vue3 + TypeScript + Element Plus
- **后端**: Python 3.10+ / Go 1.21+
- **AI引擎**: 离线语音识别 + 在线大语言模型
- **数据库**: MySQL 8.0 + Redis 7.0
- **部署**: Docker + Kubernetes
- **硬件适配**: Android TV Box / 迷你工控主机
