# 问题记录 - efinance 安装失败

**时间**: 2026-03-23 11:36
**问题**: Termux 环境下无法安装 efinance

---

## ❌ 错误信息

```
ERROR: Failed to build 'pandas' when installing build dependencies for pandas
```

## 🔍 原因分析

1. **Termux 环境限制** - 缺少 C 编译器和开发库
2. **pandas 需要编译** - efinance 依赖 pandas，pandas 需要编译 C 扩展
3. **架构问题** - ARM64 架构编译复杂

---

## ✅ 解决方案

### 方案 1：使用模拟券商（当前方案）
- ✅ 功能完整
- ✅ 无需额外依赖
- ✅ 可以测试交易逻辑
- ⚠️ 不是真实交易

### 方案 2：在 Ubuntu 虚拟机部署
- 192.168.1.3 (红茶) 可以安装
- Ubuntu 环境完整，支持编译
- 需要配置网络访问

### 方案 3：使用其他免编译库
- 寻找替代的券商接口库
- 直接用 HTTP API 调用

### 方案 4：预编译 wheel
- 查找 Termux 专用的预编译包
- 成功率较低
