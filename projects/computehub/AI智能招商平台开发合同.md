## 第一条 项目概述

### 1.1 项目名称

**AI智能招商平台**（暂定名，最终以双方确认为准）

### 1.2 项目定位

打造一套集AI智能体、语音交互、大屏演示、全案管理于一体的智能招商系统，实现"会说话的智能招商盒子"——张嘴就能讲全案。

### 1.3 项目目标

1. 实现招商全流程语音控制，无需手、电脑、U盘操作
2. 实现AI全自动讲解，新人也能专业完成招商演示
3. 实现招商资料统一管理、远程更新、加密防泄密
4. 适配多种硬件形态和显示设备，覆盖多场景应用

## 第二条 开发内容与范围

### 2.1 系统架构

本项目包含以下五大模块：

| 序号 | 模块名称 | 说明 |
|------|----------|------|
| 1 | AI智能体模块（招商大脑） | 语音识别、自动讲解、语音问答、内容加密 |
| 2 | 大屏交互控制模块 | 语音翻页、播放、暂停、放大、切换 |
| 3 | 招商全案管理模块 | 品牌、产品、模式、投资、盈利、扶持、案例、视频统一管理 |
| 4 | 数据统计模块 | 加盟商高频问题统计、页面停留分析、转化率分析 |
| 5 | 管理后台模块 | 内容发布、版本管理、设备管理、用户权限 |

### 2.2 AI智能体模块详细功能

#### 2.2.1 语音识别

| 功能项 | 技术要求 |
|--------|----------|
| 在线语音识别 | 支持普通话实时识别，准确率≥95%，响应延迟≤500ms |
| 离线语音识别 | 支持基础指令离线识别（翻页、播放、暂停、返回首页等） |
| 语音唤醒 | 支持自定义唤醒词，识别率≥98% |
| 抗噪能力 | 在65dB以下环境噪音中正常识别 |

#### 2.2.2 自动讲解

| 功能项 | 技术要求 |
|--------|----------|
| 全流程自动讲解 | 唤醒后可自动从头到尾演示招商全案（品牌、产品、模式、投资、盈利、扶持、案例、视频） |
| 讲解内容管理 | 支持自定义讲解词，AI按设定内容自动讲解 |
| 讲解中断/恢复 | 支持语音随时中断，指定话题重新开始讲解 |

#### 2.2.3 语音问答

| 功能项 | 技术要求 |
|--------|----------|
| 政策问答 | 支持加盟政策、区域政策等问答 |
| 费用问答 | 支持加盟费、保证金、设备费等问答 |
| 利润问答 | 支持投资回报周期、利润分析等问答 |
| 扶持问答 | 支持总部扶持政策问答 |
| 回本分析 | 支持回本周期计算与展示 |
| 自定义问答库 | 支持甲方自由配置问答对 |

#### 2.2.4 内容安全

| 功能项 | 技术要求 |
|--------|----------|
| 内容加密 | 招商全案内容采用AES-256或同等强度加密存储 |
| 防拷贝 | 内容不可被复制、导出、截屏录制 |
| 防篡改 | 内容完整性校验，篡改自动告警 |
| 防解析 | 反编译保护，禁止直接提取加密资源 |

### 2.3 大屏交互控制模块详细功能

| 功能项 | 功能描述 |
|--------|----------|
| 语音翻页 | "翻到下一页""翻到上一页""翻到加盟政策" |
| 语音播放 | "播放品牌宣传片""播放产品视频" |
| 语音暂停 | "暂停""继续" |
| 语音放大 | "放大产品图""缩小" |
| 语音切换 | "切换到盈利分析""切换到案例展示" |
| 语音首页 | "返回首页" |
| 多轮对话 | 支持上下文关联的多轮语音交互 |
| 错误容错 | 语音识别错误时友好提示并重试 |

### 2.4 招商全案管理模块详细功能

| 功能项 | 功能描述 |

| 模式管理 | 加盟模式、合作模式、代理模式展示 |
