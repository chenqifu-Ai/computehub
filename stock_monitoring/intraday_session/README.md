# 📈 盘中股票监控系统

## 系统概述

盘中股票监控系统是OpenClaw股票监控体系的重要组成部分，负责在交易时段(10:00-15:00)定期执行市场监控任务。

## 功能特性

- **实时监控**: 每30分钟监控主要市场指数
- **多指标分析**: 价格变动、成交量、市场趋势
- **智能警报**: 基于阈值自动触发警报
- **报告生成**: 自动生成详细监控报告
- **历史记录**: 保存所有监控结果供分析

## 监控目标

| 指数代码 | 名称 | 说明 |
|---------|------|------|
| SH000001 | 上证指数 | 上海证券交易所综合股价指数 |
| SZ399001 | 深证成指 | 深圳证券交易所成份股价指数 |
| CYBZ | 创业板指 | 创业板指数 |
| KZZ | 可转债指数 | 可转换债券指数 |

## 警报阈值

- **价格变动**: ≥3.0% (涨跌幅)
- **成交量比率**: ≥2.0 (相对平均成交量)
- **波动率指数**: ≥25.0

## 文件结构

```
stock_monitoring/

    ├── intraday_report_*.md         # 监控报告

    └── README.md                    # 说明文档
```

## 调度配置

监控任务通过OpenClaw的cron事件系统调度：

- **频率**: 每30分钟 (10:00-14:00交易时段)
- **配置文件**: `/config/intraday_stock_schedule.json`
- **执行脚本**: `/scripts/intraday_stock_monitor.sh`

## 输出示例

每次执行生成：
1. **JSON详细数据**: 包含所有监控指标的原始数据
2. **Markdown报告**: 格式化的监控报告，便于阅读
3. **内存记录**: 在每日记忆文件中记录执行情况

## 集成架构

```
OpenClaw Cron Event → 执行脚本 → 监控脚本 → 生成报告 → 更新内存
                      ↓           ↓           ↓
                  调度配置     市场数据     历史记录
```
