## 🚀 控制方案

### 第一阶段：OpenClaw初始化 (立即执行)
```bash
# 1. 初始化配置
openclaw init --accept-defaults

# 2. 启动Gateway服务  
openclaw gateway start

# 3. 检查服务状态
openclaw gateway status

# 4. 获取连接信息
openclaw gateway url
```

### 第二阶段：设备功能控制

#### 📞 通信控制
```bash
# 发送短信
termux-sms-send -n 号码 "消息内容"

# 拨打电话
termux-telephony-call 号码

# 读取短信
termux-sms-list
```

#### 📍 位置服务
```bash
# 获取当前位置
termux-location

# 持续位置监控
termux-location -p 5000  # 每5秒更新
```

#### 📷 媒体控制
```bash
# 拍照
termux-camera-photo -c 0 photo.jpg

# 录像  
termux-microphone-record -l 10 audio.mp3
```

#### 🔔 通知管理
```bash
# 发送通知
termux-notification -t "标题" -c "内容"

# 振动控制
termux-vibrate -d 1000  # 振动1秒
```

#### 🌐 网络控制
```bash
# WiFi管理
termux-wifi-connectioninfo

# 移动数据
termux-telephony-cellinfo
```

### 第三阶段：高级控制

#### 📱 应用管理
```bash
# 列出已安装应用
pm list packages

# 启动应用
am start -n 包名/活动名

# 停止应用  
am force-stop 包名
```

#### 🔒 设备管理
```bash
# 屏幕控制
input keyevent 26  # 电源键
input keyevent 3   # Home键
input keyevent 4   # 返回键

# 音量控制
input keyevent 24  # 音量+
input keyevent 25  # 音量-
```

#### 📊 系统监控
```bash
# CPU使用率
top -n 1

# 电池状态
dumpsys battery

# 内存信息
dumpsys meminfo
```
