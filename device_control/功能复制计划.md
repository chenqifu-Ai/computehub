# 🚀 OpenClaw功能复制计划

## 🎯 复制目标
将本机完整的OpenClaw功能复制到华为手机 (192.168.1.9)

## 📋 复制内容

### 1. 核心配置文件
- `openclaw.json` - 主配置文件
- `exec-approvals.json` - 执行批准配置
- `update-check.json` - 更新检查配置

### 2. 工作空间数据
- `workspace/` - 完整工作空间
- `memory/` - 记忆文件
- `agents/` - 代理配置
- `devices/` - 设备配置

### 3. 扩展和技能
- `extensions/` - 所有扩展
- 内置技能和自定义技能

### 4. 运行状态
- Cron任务配置
- 子代理状态
- 画布状态

## 🔧 复制策略

### 方法1: 完整部署包传输
```bash
# 传输完整部署包
scp -P 8022 ~/.openclaw/openclaw_full_deploy.tar.gz u0_a46@192.168.1.9:~/

# 在目标设备解压
tar -xzf openclaw_full_deploy.tar.gz -C ~/.openclaw/
```

### 方法2: 增量同步
```bash
# 使用rsync同步关键目录
rsync -avz -e 'ssh -p 8022' ~/.openclaw/workspace/ u0_a46@192.168.1.9:~/.openclaw/workspace/
rsync -avz -e 'ssh -p 8022' ~/.openclaw/extensions/ u0_a46@192.168.1.9:~/.openclaw/extensions/
```

### 方法3: 选择性复制
```bash
# 只复制最重要的配置和技能
scp -P 8022 ~/.openclaw/openclaw.json u0_a46@192.168.1.9:~/.openclaw/
scp -P 8022 ~/.openclaw/workspace/ u0_a46@192.168.1.9:~/.openclaw/workspace/
```
