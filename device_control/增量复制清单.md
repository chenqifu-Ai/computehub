# 📋 增量复制清单

## 🎯 复制策略
只复制关键配置和技能，避免系统冲突

## 📁 需要复制的目录

### 核心配置
- `workspace/` - 完整工作空间 ✅
- `openclaw.json` - 主配置文件
- `exec-approvals.json` - 执行批准配置

### 技能和扩展  
- `extensions/` - 所有扩展技能
- 自定义技能目录

### 数据和记忆
- `memory/` - 历史记忆文件
- `agents/` - 代理配置

## 🚫 不需要复制的目录

### 系统相关
- `canvas/` - 画布状态（设备特定）
- `devices/` - 设备配置（会冲突）
- `identity/` - 身份配置（设备特定）
- `logs/` - 日志文件（重新生成）
- `cron/` - 定时任务（重新配置）

### 运行时数据
- `subagents/` - 子代理状态
- 临时文件和缓存

## 🔧 复制步骤

### 第一步：准备工作空间
```bash
# 备份现有workspace
mv ~/.openclaw/workspace ~/.openclaw/workspace.backup

# 创建新workspace
cp -r ~/temp_deploy/workspace ~/.openclaw/
```

### 第二步：复制扩展
```bash
# 复制扩展目录（如果存在）
if [ -d ~/temp_deploy/extensions ]; then
    cp -r ~/temp_deploy/extensions ~/.openclaw/
fi
```

### 第三步：选择性复制配置
```bash
# 复制主配置（谨慎操作）
cp ~/temp_deploy/openclaw.json ~/.openclaw/openclaw.json.new

# 合并配置而不是覆盖
# 需要手动检查重要设置
```

### 第四步：清理临时文件
```bash
rm -rf ~/temp_deploy
rm ~/openclaw_full_deploy.tar.gz
```
