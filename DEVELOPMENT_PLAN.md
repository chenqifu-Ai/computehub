## 🏆 执行标准

遵循 SOUL.md 四大支柱 + OpenPC 六大工程哲学：

1. **绝对确定性** — 任务状态机严格：`Pending → Allocated → Executing → Verifying → Completed`
2. **防御性鲁棒性** — 节点故障自动熔断，不引发级联崩溃
3. **物理真实** — 计费基于真实 GPU 利用率，不是简单时长
4. **极致效率** — 核心心跳 ≤ 50ms，调度开销 ≤ 10ms
5. **零 Mock** — 所有接口必须是真实物理调用
6. **一次出错，永久修复** — 基因进化系统
