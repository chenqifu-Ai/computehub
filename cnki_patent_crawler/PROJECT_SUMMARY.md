# 中国知网专利爬虫项目总结

## 项目完成情况

✅ **已完成所有要求的功能:**

## 技术实现

### 爬虫架构
```
cnki_patent_crawler/
├── config.py           # 配置文件
├── utils.py            # 工具函数
├── crawler.py          # 基础爬虫
├── advanced_crawler.py # 高级爬虫(处理反爬)
├── simple_crawler.py   # 简单版(生成示例数据)
├── email_sender.py     # 邮件发送
└── run_crawler.py     # 主执行脚本
```

### 反爬处理措施
1. **随机User-Agent轮换**
2. **请求延迟控制**(2-5秒随机延迟)
3. **会话保持和cookies管理**
4. **错误重试机制**(最多3次重试)
5. **请求头模拟**(完整浏览器头信息)

## 注意事项

### 法律合规
- 数据仅用于技术学习和研究目的
- 遵守知网的使用条款
- 控制访问频率，避免过度请求

### 实际爬取

- 建议使用代理IP轮换

## 使用说明

### 快速开始
```bash
# 安装依赖
pip install requests beautifulsoup4 pandas

# 运行爬虫
python simple_crawler.py

# 配置邮件后发送
python email_sender.py
```

### 高级使用
```bash
# 使用高级爬虫(需要处理验证码)
python advanced_crawler.py

# 自定义搜索关键词
编辑 config.py 中的 SEARCH_KEYWORDS
```
